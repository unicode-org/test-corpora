Yeroo akkasii immoo dhuguma copha jiidhina bishaan dhangalaʼaa keessaa akka hubatte yaaduu qabda yoo taʼe, danfaa isaa irraa kan kaʼe akka walitti qabaman qofa akka hin taane akkamitti beekta; ykn akkamitti copha wal fakkaatu kan gubbaadhaan ciccituu boolla bishaan keessaa dhangalaʼee fi fiixee mataa qurxummii galaanaa sanaa keessa faallaa lixu sana akka hin taane beekta? Sababni isaas, yeroo tasgabbiidhaan galaana walakkaa guyyaa tasgabbaa’aa ta’een, hump isaa olka’aa akka dromedary’s gammoojjii keessatti aduudhaan gogfamee jiruu illee; yeroo sanattillee, qurxummiin galaanaa kun yeroo hunda bishaan xiqqoo mataa isaa irra kan baatu siʼa taʼu, aduu bobaʼaa jalatti yeroo tokko tokko dhagaa roobaan guutame keessaa qaawwa tokko ni argita.

Akkasumas, adamsaan sun maalummaa sirrii dhangalaʼaa qurxummii galaanaa tuquuf fedhii guddaa qabaachuunsaa matumaa of eeggannoo miti. Isa keessa ilaalee, fuula isaas keessa kaa’uun isaaf hin fayyadu. Qaruuraa kee wajjin gara burqaa kanaa deemtee guuttee, fiduu hin dandeessu. Sababni isaas, yeroo xiqqoo danfaa ciccitaa jeetichaa wajjin wal qunnamuu illee, yeroo baayʼee kan taʼu, gogaan kee akka hoʼaa taʼetti ni qaroo taʼa, wanti akkasitti isa tuqe sanaa irraa kan kaʼe. Akkasumas tokko nan beeka, kan amma iyyuu dhiyeenyaan kan wal qunnamu, meeshaa saayinsii tokko tokko ijaan ilaaluudhaan, yookaan kan biraatiin, gogaan qoochoo fi harka isaa irraa ba’e jechuu hin danda’u. Kanaaf, namoota qurxummii galaanaa qaban biratti, dhangala’aan sun summii akka ta’etti ilaalama; jalaa miliquuf yaalu. Waan biraa; Jeetichi ija keessanitti haqaan yoo dhangala’e akka isin jaamsu dhaga’eera, baay’ee hin shakku. Wanti ogummaan qorataan yeroo sana gochuu danda’u, akka natti fakkaatutti, dhangala’aa du’aa kana dhiisuudha.

Ammas, mirkaneessuu fi hundeessuun yoo dadhabnellee, yaada (hypothesis) gochuu dandeenya. Yaadni koo kana: dhangala’aan awwaara malee homaa akka hin taane. Akkasumas sababoota birootiin alatti, gara xumura kanaatti kan na kakaasu, yaadannoowwan kabajaa fi ol’aantummaa uumamaan guddaa Sperm Whale tuquun; Sagaleewwan irratti, ykn qarqara galaanaa biratti gonkumaa akka hin argamne dhugaa hin mormamne waan ta’eef, uumama waliigalaa, gadi fagoo akka hin taane nan isa ilaala; qurxummiin galaanaa kaan hundi yeroo tokko tokko akkasuma. Inni nama ulfaataa fi gadi fagoodha. Akkasumas mataa uumamtoota gadi fagoo ulfaataa ta’an hunda irraa, kan akka Pilaatoo, Piiroo, Diyaabiloos, Juupitaar, Daantee fi kkf, yeroo hundumaa danfi walakkaa mul’atu murtaa’e tokko akka ol ba’u nan amana, osoo gocha yaada gadi fagoo yaaduu keessatti. Barreeffama xiqqoo waa’ee Bara Baraa ilaalchisee osoon qindeessuu, daawwitii tokko fuuldura koo kaa’uuf fedhiin beekuu qaba ture; fi ere long saw achitti calaqqise, fedhiin beeku tokko qilleensa mataa koo irra jiru keessatti raammoo fi undulation hirmaachise. Jiidhinni rifeensa koo hin jijjiiramne, osoo yaada gadi fagoo keessa lixuu, shaayii ho’aa ja’a booda mana gubbaa koo isa haphii shingled keessatti dhugee booda, kan Hagayya sa’aatii walakkaa; kun tilmaama armaan olii kanaaf falmii dabalataa fakkaata.

Akkasumas, galaana tiroopikaalaa tasgabbaaʼaa taʼe keessa dooniin isaa kabajaan yeroo deemu ilaaluun, bineensa jabaa, awwaara qabu sanaaf of tuulummaa keenya akkam kabajaan nu kaasa; mataan isaa bal’aa fi lallaafaan golgaa danfaatiin kan fannifame, kan xiinxala isaa isa hin dabarreen kan dhalate, danfis sun—akkuma yeroo tokko tokko argitan—akka waan Samiin mataan isaa yaada isaa irratti chaappaa isaa kaa’eetti, bokkaa bokkaatiin ulfina argate. Sababni isaas, d'ye see, bokkaa qilleensa qulqulluu hin daawwatan; isaan danfaa qofa ibsu. Kanaaf, awwaara furdaa shakkii dukkanaa’aa sammuu koo keessa jiru hunda keessa darbee, hubannoon waaqummaa yeroo yerootti dhukaasa, awwaara koo balaqqeessa samiitiin qabsiisu. Kanaafis Waaqayyoon nan galateeffadha; hunduu shakkii waan qabaniif; baay’een ni haalu; garuu shakkiin ykn haaluun, isaan waliin muraasni, hubannoo (intuition) qabu. Shakkii wantoota lafa irraa hundumaa, fi hubannoo wantoota samii tokko tokko; walnyaatinsi kun mu’minaas ta’ee kaafira hin taasisu, garuu nama lamaan isaanii iji wal qixaan ilaalu taasisa.
