Imi deg uneẓruf am wa n tkerrust n tkerrust, ideg assaɣen-nsen yemgaraden yemgaraden deg yigenni ur zmiren ara dima ad ttwaferqen s wayen yettwalin fell-asen deg tɣerɣert ; yerna mi ara ttwaḍeggren yiɣef n tkeṛṛust n yiɣil-a yal kra n tedqiqin seg tɣawsiwin n tɣawsiwin, ad yili d lmut n tɣawsiwin kan, ma yella, war ma yella-d yiwen n uɛessas n yal tikkelt, aɣerrabu-nni yettwasfeḍen ad yettwaḍegger s kra n tɣawsiwin ur nesɛi lḥeqq n yixeddamen n tkeṛṛust yerna ad yeɣli akk s tɣawla ɣer yilel. Ihi, lḥeqq n Axab di temsalt- agi ur yelli ara d ayen ur nelli ara d lqayen; taɣawsa kan i ten-yessewhamen iban-d d akken, Starbuck, qrib d argaz-nni kan i t-yesxeṣren ad t-iqabel s kra n tɣawsa s tɣawla tamecṭuḥt kan i d-iqerben ɣer ɣɣer-is – yiwen seg wid-nni ula d netta, s lḥeqq-is ɣef tmuɣli-nni, iban-d amzun yella yettcukku kra ; s tlelli yettak-d tudert-is meṛṛa deg ifassen n umdan am wa ur nesɛi ara laman s tɣawsiwin nniḍen.

Tura, tikkelt tamezwarut ideg Axab yeqqim s lqaɛa; ere yella dinna mraw n tedqiqin; yiwen seg yiɣersiwen-nni n yilel n tmeẓẓuɣt taberkant ay yettfeggiḍen s lqerb s lqerb ɣef yiqerra n tmeẓẓuɣt n yiɣersiwen n yilel ay yesɛan imdanen deg tmiwa-a n tlemmast; yiwen seg yigḍaḍ-agi yusa-d yettmuqul yerna yettɛeggiḍ ɣef uqerruy-is deg yiwen n lbabur n tɣawsiwin s tɣawla ur nesɛi lḥeqq. Syin, teffeɣ-d srid ɣer yigenni s alef n yiḍarren; syin yezger-d s wadda, yerna yedda yettmuqul-d tikkelt nniḍen ɣef uqerruy-is.

Maca s tmuɣli-ines tettmuqul deg tlemmast n tmurt i d-yeffɣen s lbeɛd yerna s lbeɛd, Axab iban-d amzun ur iḥemmel ara agḍiḍ-agi n tẓegwa; ula d netta, s tidet, ula d yiwen nniḍen ur t-id-yesmektay ara aṭas, imi mačči d tagnit ur nesɛi lmeɛna; ala tura qrib tiṭ ur nesɛi ara aṭas n tmuɣli amzun tettwali kra n ṣṣenf n unamek n tḥerci deg qrib yal tamuɣli.

“Aqcic-ik, aqcic-ik, Mass!” deg yiwet n tikkelt, i d-yessuɣ umaru n yilel asisili, imi yella yettwasbedd deg uqerru n mizen-mast, ibedd srid deffir Ahab, ɣas akken yella kra ddaw n uswir-is, yerna s yiwen n lbabur n lhawa lqayen i ten-yebḍan.

Maca yakan afriwen n sable yella zdat wallen n umɣar-nni; lfakya ɣezzifen i d-yeqqnen ɣer uqerruy-is : s lḥir, aɣersiw-nni aberkan yeffeɣ-d s warraz-is.

Yiwen n lbaz yeffeɣ-d kraḍ n tikkal ɣef uqerru n Tarquin, yekkes-as aqerruy-is akken ad t-id-yesbedd, dɣa s waya, Tanaquil, tameṭṭut-is, tenna-d belli Tarquin ad yili d agellid n Ruma. Maca ala s ubeddel n tqacuct i d-yettwaḥseb lxir-nni. Ur t-id-yerri ara lqecc n Ahab; lbaz n tẓegwa yettfeggiḍ-d yid-s; bɣiɣ zdat n tmeẓẓuɣt : yerna taggara-a yeɣli-d; ma d seg wemkan n tmettant-nni, tettban-d s tmuɣli n tmeẓẓuɣt yiwet n tmeẓẓuɣt taberkant n tedqiqt, teɣli-d seg lqedd-nni ameqran ɣer yilel.
