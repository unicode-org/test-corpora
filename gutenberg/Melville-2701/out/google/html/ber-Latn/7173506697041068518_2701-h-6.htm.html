ADLISS II. ( Oktavu ), IXEF WIS SIN. ( Iselman Iberkanen ).—Ttakeɣ-d ismawen n yiselman imeqranen i yiselman-agi akk, imi s umata d nitni ay d igerrzen akk. ADLISS II. ( Oktavu ), IXEF wis III. ( Narwhale ), dɣa, Nostril whale .—D amedya niḍen n yiwen n uɣersiw n yilel umi semman s tɣawla, ttwaliɣ d akken s yisem-nnes ay d-yellan s tɣawla imi ay t-ttwalin deg tazwara d akken d aɣersiw ay yesɛan anzaren n tqacuct. Taxelqit-a tesɛa azal n mraw d sḍis n yiḍarren deg teɣzi, ma d aqerruy-nnes yesɛa xemsa n yiḍarren deg tlemmast, ɣas akken kra ttqadaren mraw, yerna ttawin ula d xemsa n yiḍarren. S tɣawla, aqerruy-a d aɣersiw kan yettwasɣezfen, yettnerni-d seg tɣerɣert deg yiwen n lḥirfa i d-yesɣezfen cwiṭ seg tɣerɣert. Maca yettwaf-d kan deg tama n tmeṭṭut n tmeṭṭut, yesɛan azal n diri, yettakf i bab-is kra n tɣawsa yettemcabin ɣer yidis n wergaz azelmaḍ ixuṣṣen. Anwa iswi s tidet i d-yettarra lḥenni-agi n lgaz neɣ lmus-agi, ad yili d ayen iweɛren ad d-nini. Ur iban ara ad yettwaseqdec am lqecc n yiselman n tafrut d yiselman n tmeẓẓuɣt; ɣas akken kra n yimariren qqaren-iyi-d d akken Narwhale yesseqdac-it i lmend n tmeẓẓuɣt deg uɛewwel ɣef lqaɛ n yilel i lmend n tuččit. Charley Coffin yenna-d d akken yettwasseqdec i tmeẓdit n ugris; acku Narwhale, mi d-yeffeɣ ɣer tqacuct n Yilel Aqbuṛ, yerna yufa-t yettwasfeḍ s wegris, yessufeɣ-d aqerruy-nnes ɣer sdat, dɣa s waya ay d-yeqceɛ. Maca ur tezmireḍ ara ad d-tessekneḍ belli ula d yiwen seg yiḥezziben-agi d tidet. Rray-inu s timmad-inu d akken, akken yebɣu yili lḥal, aqerruy-a n yiwen n yidis yezmer ad t-yesseqdec s tidet Narwhale – akken yebɣu yili waya – s tidet ad as-iɛeǧba aṭas i yiwen n uḍris deg yiḍrisen n tɣuri. Iɣersiwen n Narwhale ay sliɣ qqaren-as iɣersiwen n Tusked, iɣersiwen n Horned, d iɣersiwen n Unicorn. Netta d amedya n tmuɣli n Unicornisme ara d-naf qrib deg yal tageldit n ugama yettwaddren. Seg kra n yemseḍruyen iqburen ay yellan ttwaḥebsen, snejmaɛeɣ-d d akken aɣersiw-a n tmeẓẓuɣt n yilel-nni kan, deg wussan iqburen, yettwaḥseb d netta ay d aɣersiw ameqran mgal ssem, yerna s waya, aheggi-nnes yettawi-d ssuma ur nesɛi lqima. Yettwasseqdec daɣen ɣer yiwen n lmelḥ yettwasqedcen i tmeṭṭut ay la yettfaḍen, akken ay ttwaxedmen yiqjan n yizerfan n wergaz ɣer yiqjan n yilef (hartshorn). Deg tazwara yella s timmad-is yettwaḥseb d ayen n tmuɣli tameqrant. Black Letter tenna-yi-d belli Sir Martin Frobisher mi d-yuɣal seg tikli-nni, asmi ay s-d-tessufeɣ Tagellidt Bess afus-nnes n tẓegwa s lǧehd seg yiwen n usqif n lqaɛa n Greenwich, asmi ay d-yella uɣerrabu-nnes ay yesɛan tabɣest yettɛedday ɣef wasif n Thames ; “mi d-yuɣal Sir Martin seg tikli-nni”, ay d-tenna Black Letter, “s yiḍarren-nnes ay d-yeqqnen, yessken-as i tgelda-nnes yiwen n lḥenni ɣezzifen n Narwhale, ay yellan yettwaɛelleq ɣef leqṣer n Windsor ɣef yiwet n tallit ɣezzifen.” Yiwen n umeskar irlandi yenna-d d akken Earl n Leicester, s yiḍarren-nnes yeqqnen, yesskanay-d ula d netta i tgelda-nnes yiwen n yiqjan niḍen, ay yerzan aɣersiw n wakal n tgemmi (nature) n yiwen n uɣersiw.

Narwhale yesɛa tamuɣli n tmeẓẓuɣt (picture) aṭas, am tmeẓẓuɣt n tmeẓẓuɣt, imi ay yesɛa ccbaḥa n lqaɛa d tamellalt am uyefki, yerna llant deg-s tɣawsiwin n tmeẓẓuɣt d tmeẓẓuɣt n tmeẓẓuɣt. Zit-is tgerrez aṭas, iban yerna tgerrez; maca drus deg-s, yerna qlil ma yettwaḥḍer. Yettwaf aṭas deg yilel n tcirḍart.
