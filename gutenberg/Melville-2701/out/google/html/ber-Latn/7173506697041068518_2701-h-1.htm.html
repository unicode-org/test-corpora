IXEF 1. Tilufa.

Ssiwlet-iyi-d Ismaɛil. Kra n yiseggasen aya – ur ḥesbeɣ ara acḥal n lweqt s tidet – imi sɛiɣ drus n yedrimen neɣ ulac-iten deg tḥemla-inu, yerna ulac kra n wayen ara yi-iɛeǧben s wazal-is deg tlisa, ɣileɣ ad sfeḍḍeɣ cwiṭ s lbabur ad waliɣ aḥric n umaḍal yesɛan aman. D abrid sɛiɣ n ussufeɣ n tfekka d uslugen n tcirḍart. Yal mi ara d-afeɣ iman-iw ttnerniɣ s lḥir ɣef yimi; yal mi ara yili d Novembre iḥemmlen, iḥemmlen lḥemla deg ṛṛuḥ-iw; yal mi ara d-afeɣ iman-iw ur bɣiɣ ara ad ḥebseɣ zdat n tḥuna n tmeqbert, yerna ad d-ssufɣeɣ aḍar n yal tameḍḍelt i d-ttmagareɣ; yerna ladɣa yal tikkelt aydeg hypos-inu ad iyi-d-yefk afus n tɣawsa, dɣa yessefk-iyi ad sɛuɣ amenzay (principe) adduklan (moral) ijehden akken ad iyi-yeḥbes ad t-id-sfeḍḥeɣ s lqedd deg webrid, yerna ad sneqseɣ s tmeẓri (méthode) tiqbucin n medden – sakkin, ttwaliɣ d lawan ad awḍeɣ ɣer yilel akken kan ara zemreɣ. Wagi d abeddel-iw n tmeẓdit d tkurt. S tfekka n tfelsafit Cato yettḍeggir iman-is ɣef tmeẓẓuɣt-is; Nekk s tɣawla ttawiɣ ɣer uɣerrabu. Ulac ayen yessewhamen deg waya. Ma yella ssnen-t kan, qrib akk irgazen deg tlemmast-nsen, kra n tallit neɣ tayeḍ, ttḥemmilen aṭas qrib yiwen n lḥir ɣer lbabur yid-i.

Dinna tura temdint-nwen n tegzirt n Manhattoes, ttwazedɣent s yiɣerruba am tegzirin tihendiyin s yiɣerruba n tmerɣa (corail) – tanezzuyt tezzi-as-d s tmeẓẓuɣt-nnes. Yeffus d uzelmaḍ, iberdan ttawin-k ɣer waman. Taɣremt-is n tlemmast n temdint d aṭaṭar, anda ay tt-ssiriden yimuḍan-nni n tmeẓyant, yerna ttṣemmiḍen-t yiɣeblan, ay yellan kra n tsaɛtin uqbel ur ttwalin ara akal. Wali lɣaci n wid yettmuqulen aman dinna.

Ḥedded ɣef temdint n yiwen n wass n ssebt tameddit n targit. Ad t-id-tedduḍ seg Corlears Hook ɣer Coenties Slip, yerna seg din, s Whitehall, ɣer ugafa. D acu i twalaḍ ?— Ttwasbedden am yiɛessasen n tmesliwt yessusem di temdint meṛṛa, bedden lqern ɣef lqern n yergazen n lmut ttwaḥerzen deg tmuɣliwin n yilel. Kra ttkelfen ɣef tmeẓẓuɣin; kra qqimen ɣef yiqerra n tḥuna; kra ttmuqulen ɣef tɣerɣert n yiɣerruba n Ccinwa; kra s lqedd deg tmeqbert, amzun ttnadin ad d-afen aẓar n yilel mazal-it yelhan. Maca wigi akk d imdanen n tmurt; n wussan n ddurt ttwaḥebsen deg lḥebs d lplaster — ttwaqqnen ɣer tmeẓdiyin, ttwasemman ɣer yiḍebsiyen, ttwaḥebsen ɣer yiḍebsiyen. Amek ihi akka ? Ma llan yigran izewwaɣen? D acu xeddmen dagi?

Maca ẓer! dagi ttasen-d ugar n lɣaci, tteddun srid ɣer waman, yerna amzun ttwaqqnen ɣer tɣerɣert. Abeṛṛani! Ulac d acu ara ten-iqeblen anagar lḥedd n tmurt; loitering ddaw n tlemmast n tlemmast n tḥuna n dinna ur t-id-yettaf ara. Ala. Ilaq ad qriben akken zemren ɣer waman mebla ma ɣlin deg-sen. Yerna dinna bedden — s yikilumitren seg-sen — s lmitra. Inlanders akk, ttasen-d seg yiberdan ed yiberdan, seg yiberdan ed yiberdan n tmurt – seg ugafa, seg wegmuḍ (ccerq), seg wenẓul ed seg wegmuḍ (lɣerb). Maca dagi dduklen akk. Ini-yi-d, ma yella lxir n tmagnit n tmeẓẓuɣin n tbuciḍant n yiɣerruba-nni akk i ten-ijebden ɣer din ?

Tikkelt nniḍen. Ini-d telliḍ di tmurt; deg kra n tmurt ɣlayen n yiɣeẓran. Ad t-id-tawiḍ qrib akk abrid i tḥemmleḍ, yerna mraw ɣer yiwen ad k-id-yawi ɣer ddaw deg yiwen n wedrar, ad k-yeǧǧ dinna ɣer tama n yiwen n lbabur deg wasif. Yella deg-s sḥur. Eǧǧ win ur nesɛi ara lbal n yemdanen ad yeɣli deg tmuɣliwin-is lqayen akk – bedd argaz-nni ɣef yiḍarren-is, ssers iḍarren-is a-yeddu, ad k-yawi s lxilaf ɣer waman, ma yella llan waman di temnaḍt-nni meṛṛa. Ma yella kra n wass ad teffudeḍ deg uneẓruf ameqran n Marikan, ɛreḍ ajerred-a, ma yella s tɣawla ad t-id-tessiweḍ tkerrust-ik s yiwen n uselmad n tmeṭṭut. Ih, akken yeẓra yal yiwen, ameslay d waman ttwaɛezlen i lebda.

Maca atan unaẓur. Yebɣa ad ak-d-yesnulfu aḥric n tmurt n tmeṭṭut i d-yettawin targit, i d-yettawin lḥir, i d-yessusmen, i d-yettawin lḥir deg wakk tala n Saco. D acu i d aferdis agejdan i yesseqdac? Dinna bedden yisekla-ines, yal yiwen s lqecc-is ixussen, amzun yella daxel-is d aḥerfi d lmeṣnuɛ ; yerna dagi iṭṭes lqaɛa-s, dinna iṭṭes lmal-is; yerna s ufella seg uxxam n dinna iteddu yiwen n dexxan n yiḍes. Deg lqaɛ n tẓegwa ibeɛden, aḍu yettḥaz abrid n tmeẓẓuɣt, yettaweḍ ɣer tɣerɣert n yidurar yettwaɛezlen s tmeẓẓuɣt-nsen n tlemmast n wedrar. Maca ɣas akken tugna-nni tezga-d akka tban-d, yerna ɣas akken aseklu-agi n tẓegwa yettḥezzib s tmeẓẓuɣt-is am yiferran ɣef uqerru n umeksa-agi, maca kullec d lbaṭel, ma yella ur tettmuqul ara wallen n umeksa-nni ɣef wasif n sḥur yellan zdat-s. Ruḥ ad terzeḍ ɣer Prairies deg Yunyu, mi ara tedduḍ ɣef tmerwin n yikilumitren gar Tiger-lilies armi d iḍarren-ik — d acu i d lḥirfa-nni kan i yebɣa ? — Aman — ulac dinna ula d yiwet n tqejjirt n waman ! Lemmer d Niagara kan d katarakt n lḥemla, ad tedduḍ alef n yikilumitren-ik akken ad tt-twaliḍ? Ayɣer amedyaz-nni ameɣbun n Tennessee, mi d-yewwi deg yiwet n tikkelt snat n tfekkiwin n lfeṭṭa, ixemmem ma ad as-d-yaɣ aɛekkaz, ay t-iḥwajen s lḥif, neɣ ad yessefti idrimen-nnes deg yiwen n usikel n yiḍarren ɣer Rockaway Beach? Ayɣer qrib yal aqcic s tezmert iǧehden yesɛan yiwen n ṛṛuḥ s tezmert iǧehden deg-s, deg kra n tallit neɣ tayeḍ d aḥeqrani akken ad iruḥ ɣer yilel? Ayɣer deg tikli-inek tamezwarut d imsiklen, kečč s timmad-ik teḥṣiḍ s tmeẓẓuɣt n tmeẓẓuɣt am ta, asmi i k-d-nnan tikkelt tamezwarut belli kečč d uɣerrabu-inek tura ur tettwalin ara akal? Ayɣer ay ḥesben Ifarsiyen iqburen d akken ilel d amqeddes? Ayɣer ay as-fkan Yigrikkiyen yiwen n ṛebbi yemgerraden, d gma-s n Jove? S tidett, ayagi meṛṛa mačči mebla anamek. Yerna mazal-it lqayen anamek n tmacahut-nni n Narcisse, imi ur yezmir ara ad yeṭṭef tugna-nni n lḥif, n tmeẓẓuɣt i d-iwala deg tmeẓẓuɣt, iɣli-d deg-s, iɣreq. Maca tugna-nni kan, nekni s timmad-nneɣ nettwali-tt deg wakk isafen d yilel. D tugna n lfanṭum n tudert ur nettwaṭṭef ara; yerna d aya ay d lsas n waya akk.

Tura, mi ara d-iniɣ d akken sɛiɣ lɛada n ussiweḍ ɣer yilel yal tikkelt aydeg ara bduɣ ad ttḥulfuɣ s tmeẓẓuɣt ɣef wallen, yerna ad bduɣ ad ttḥulfuɣ s lḥir ɣef yiɣes-inu, ur bɣiɣ ara ad d-iniɣ ad d-yili lḥeqq-inu ad dduɣ ɣer yilel kra n wass d amsebrid. Imi akken ad truḥeḍ d amsebrid ilaq ad tesɛuḍ tasarut, yerna tasarut d aɛeqqa kan ma yella ur tesɛiḍ ara kra deg-s. Syin akkin, imsiklen ttwaḍḍfen s waṭṭan n yilel — ttnernin deg ccwal — ur ttnadin ara yiḍan — ur ttḥemmilen ara aṭas, am wakken d ayen amatu ; — ala, ur ttruḥeɣ ara d imsiklen ; ula d nekk, ɣas akken nekk d kra n lmelḥ, ur ttruḥeɣ ara ɣer yilel d Akumudur, neɣ d Kabṭan, neɣ d Aseggaḍ. Nekk ttaǧǧaɣ azal d lxilaf n yibiḍaren am wigi i wid i ten-iḥemmlen. Seg tama-w, ttkukruɣ akk lxedma n lḥeṛma, lḥeṛma d lḥif n yal ṣṣenf. D ayen akk zemreɣ ad t-xedmeɣ akken ad ḥadreɣ iman-iw, war ma ḥadreɣ iɣerruba, ibarkiyen, ibrigen, iɣerruba n tkeṛṛusin, d wayen nniḍen. Yerna ɣef wayen yerzan ad dduɣ d aseggaḍ, ɣas akken ttqadareɣ belli yella ccbaḥa d tameqrant deg waya, imi aseggaḍ d ṣṣenf n ufesyan deg uɣerrabu, maca, s kra n tɣawsa, ur ḥemmleɣ ara ad sqeɛdeɣ iɣersiwen ; lbaṭaṭa n tmes n wayen ara bɣuɣ. D seg tmeẓẓuɣin n tmeẓẓuɣin n Yimaṣriyen iqburen ɣef tmeẓẓuɣin n tmeẓẓuɣin d yizem n wasif yettwasxedmen, ay twaliḍ timummiyin n yimdanen-nni deg yixxamen-nsen imeqranen n tmeẓẓuɣin n tmeẓẓuɣin n tmeẓẓuɣin.

Ala, mi ara dduɣ ɣer yilel, ttṛuḥeɣ am yiwen n umeṣ̣ruf afraray, zdat n tmeẓẓuɣt, ttḥezzibeɣ ɣer tmeẓẓuɣt, s afella dinna ɣer uqerru n tmeẓẓuɣt tagellidt. D tidet, ttḥemmilen-iyi ad iyi-d-sbedden ɣef kra, yerna ttgen-iyi ad nḍeggeɣ seg spar ɣer wayeḍ, am tmeẓẓuɣt deg tẓegwa n Mayyu. Yerna deg tazwara, ṣṣenf-agi n tɣawsa ur t-iɛeǧba ara mliḥ. Tettḥaz lḥeqq n wemdan, ladɣa ma yella d tawacult taqburt ay d-yesbedden iman-nnes deg wakal-nni, d tin n Van Rensselaers, neɣ n Randolphs, neɣ n Hardicanutes. Yerna ugar n waya, ma yella qbel kan ad tessekneḍ afus-ik deg tmeqbert n lqaṭar, tellaḍ tettḥekkim fell-as am uselmad n uɣerbaz n tmurt, tgiḍ arrac iɣezfanen ad bedden fell-ak s lxuf. Tazmert-nni d tin iḥerzen, ttwekkideɣ-k, seg uselmad n uɣerbaz ɣer umesni, yerna teḥwaǧ-d aḍebsi ijehden n Seneca d Yistoyiyen akken ad tzemreḍ ad t-tḥemmleḍ yerna ad t-tesɛeddiḍ. Maca ula d aya yettfakka deg wakud.

Acu ara d-iniɣ, ma yella kra n yiḥricen iqburen n ukapitan n yilel ssutren-iyi-d ad d-awiɣ tafekka ad sfeḍeɣ tizegzewt ? D acu i d lqima n lbaṭel-nni, yettwaɛzel, bɣiɣ ad d-iniɣ, deg lmizan n Lɛahd Amaynut ? Tettwaliḍ d akken lmelk ameqqran Jibril yettxemmim fell-i kra n wayen ur nelli ara d ameqqran, acku ttqadareɣ s tɣawla yerna s leqder i yiḥeffaḍen-nni iqburen deg tegnit-nni s timmad-is? Anwa ur nelli d akli? Ini-d aya. Ihi, ihi, akken ma llan yiqeddacen iqburen n yilel ad iyi-d-sbedden lameṛ ɣef waya – akken ma llan ad iyi-d-sfeḍḍen yerna ad iyi-sfeḍḍen, sɛiɣ lfeṛḥ imi ẓriɣ belli kullec d ayen yelhan ; belli yal yiwen nniḍen d yiwen n ubrid neɣ wayeḍ yettwaqeddec s waṭas n yiwen n ubrid- ama deg tmuɣli n tfekka neɣ n tmeṭṭut, dɣa; yerna akka i d-yettili uqeclaw amaḍlan s tɣawla, yerna ifassen akk yessefk ad sfeḍḍen iḍarren n wayeḍ, yerna ad ttqadaren.

I tikkelt niḍen, ttedduɣ dima ɣer yilel d amesni, imi ay ttgen lḥeqq ad iyi-xellṣen ɣef wuguren-inu, deg wakud aydeg ur ttxellisen i yimsiklen ula d yiwen n yidinaṛen ay ɣef slan. Am wakken daɣen, ilaq imsiklen s timmad-nsen ad xellṣen. Yerna yella akk lxilaf deg umaḍal gar uxelleṣ d uxelleṣ. Tigawt n uxelleṣ ahat d ayen ur nḥemmel ara akk ay d-ssawḍen sin-nni n yimeksawen n tẓegwa. Maca ma yella yettwaxelleṣ ,—d acu ara t-id-yesqedcen ? Tigawt n temdint ay s-d-yettaf wemdan idrimen d tin ay yessewhamen s tidet, ma nwala d akken nettamen s lqerb d akken idrimen d aẓar n wakk waḍḍanen n ddunit, yerna s kra n ssebba ur yezmir wemdan ay yesɛan idrimen ad yekcem ɣer yigenni. Ah! amek s lferḥ i neţţaǧǧa iman- nneɣ ɣer tmeɣra!

Taggara, ttedduɣ dima ɣer yilel am umesni, ɣef ljal n tmeɣra n tfekka d lḥamu zeddigen n tɣerɣert n lqaɛa n zdat. Imi am wakken yella deg umaḍal-a, aḍu n uqerruy yettwassen s waṭas ugar n waḍu n tlemmast (dɣa, ma yella ur tḍurreḍ ara lqanun n Pythagore), dɣa deg tmurt-a, amur ameqran seg Commodore yellan deg tquarter-deck yettaf-d atmosphère-nnes s ufus wis sin seg yimesnallunen yellan deg forecastle. Yettxemmim d netta ay t-id-yessuffuɣen d amezwaru; maca mačči akka. S tɣawla am wakken ay ttgent tdukliwin (communautés) imḍebbren-nsent deg waṭas n tɣawsiwin niḍen, deg tallit-nni aydeg imḍebbren ur ttcukkuten ara mliḥ. Maca ayɣer ay d-yella waya, deffir ma snuɣeɣ aṭas n tikkal ilel am yiwen n umesni n tmerkantit, ad t-id-awiɣ tura deg uqerruy-iw akken ad dduɣ deg yiwen n usikel n tmeẓdit n yilef; aya d amsulta ur nettwaẓer n Fates, yesɛan lḥeqq fell-i dima, yerna yettḍurru-yi s tuffra, yerna yettḍurru-yi s kra n tɣawsa ur nesɛi lḥisab – yezmer ad d-yerr s lxir ugar n wayeḍ. Yerna, war ccek, tikli-inu deg tikli-a n tmeẓdiyin (baleines), tella-d d aḥric seg wahil ameqran n Providence ay d-yettwaheyyan zik. Tekcem-d d ṣṣenf n tmeẓẓuɣt (interlude) tawezlant (interlude) d tmeẓẓuɣt (solo) gar yisura s wazal-nsen ugar. Ttwaliɣ belli aḥric-agi n usenfar n lqanun ilaq-as ad yeddu kra am wa:

“ Tafrant Tameqrant n Tselwit n Yiwunak Yeddukklen. “IMENƔI YIDAMEN DI AFƔANISTAN.”

Xas akken ur zmireɣ ara ad d-iniɣ ayɣer ay d-ssersen yimḍebbren-nni n tmezgunt, Fates, ɣef weḥric-a n tmeẓẓuɣt n yiwen n usikel n tmeẓẓuɣt, asmi ay ttwassersen wiyaḍ ɣef yiḥricen imeqranen deg tmeɣriwin n tmeɣriwin n tmeɣriwin n tmeɣriwin n tmeɣriwin, d yiḥricen igejdanen yerna isehlen deg tmeɣriwin n tmeɣriwin n tmeɣriwin n tmeɣriwin, d yiḥricen n tmeɣriwin n tmeɣriwin n tmeɣriwin n tmeɣriwin – ɣas akken ur zmireɣ ara ad d-iniɣ acimi aya ur yelli ara s tidet; maca, tura mi d-mmektiɣ akk tilufa, ttwaliɣ zemreɣ ad waliɣ cwiṭ deg tefsutin d tmuɣliwin ay d-ttwaqeddmen fell-i s tḥerci s tɣawsiwin yemgerraden, ay iyi-yessawḍen ad bduɣ ad xedmeɣ aḥric ay xedmeɣ, sdat n tɣawsa-nni ay iyi-d-yesbedden deg tmuɣli-nni ay d-yeqqaren d akken d afran ay d-yellan seg lebɣi-inu d tɣawsiwin ur nesɛi lḥeqq.

Aɣella gar tmuɣliwin-a d tikti-nni ay d-yesnekren aṭas n yilef ameqran s timmad-nnes. Aɣersiw-agi n tmeẓẓuɣt d lbaḍna am wa, yessekcem-iyi-d akk lḥir. Syin lebḥer n tẓegwa d wid ibeɛden anda i d-yesfeḍ tigzirt-is s lqedd; lxiq ur nesɛi ara isem, ur nesɛi isem n yilef; wigi, s wakk lmuɛjizat i d-iḥeḍren n alef n tmuɣliwin d tɣawsiwin n Patagonia, ɛawnent-iyi ad sfeḍḥeɣ ɣer lebɣi-w. S yergazen nniḍen, ahat, ur llint ara tɣawsiwin am tigi d lḥeqq; maca nekk, ttwaɛezleɣ s lḥir n lebda ɣef tɣawsiwin ibeɛden. Ḥemmleɣ ad ssiwḍeɣ s lbabur deg yilel yettwaḥerrmen, d wakal deg yiberdan iberraniyen. Ur ttḥeqriɣ ara ayen yelhan, ttɣawleɣ ad ḥulfuɣ s lxuf, yerna mazal zemreɣ ad iliɣ d anmetti yid-s – ad iyi-ǧǧen – imi d ayen yelhan kan ad iliɣ deg tmuɣliwin n tmeddukal ed wakk imeḥbas n wemkan aydeg yettɛedday yiwen.

S ssebba n tɣawsiwin-a, ihi, tikli n tmeẓdiyin (baleines) tella-d s lxir; tiwwura timeqqranin n lḥemla n umaḍal n lmuɛjizat ttwafeṣṣlent, yerna deg tmuɣliwin n tmeẓẓuɣt i yi-d-yesfeḍḍen ɣer yiswi-inu, snat d snat dinna ttfeggiḍent deg ṛṛuḥ-iw n daxel, tikliwin ur nesɛi taggara n tmeẓẓuɣt, yerna, gar tuget-nsent akk, yiwen n lfanṭum ameqqran s tmeẓẓuɣt, am yiwen n wedrar n wedfel deg yigenni.





IXEF WIS 2. Taqcict-nni n tkerrust.

Ssexsiɣ yiwet n tceṭṭiḍt neɣ snat deg tkerrust-iw taqburt n tkerrust, sserseɣ-tt ddaw ufus-iw, bdiɣ ɣer Cape Horn d Pacific. Ssexsiɣ tamdint yelhan n Manhatto taqdimt, uwḍeɣ s lḥeqq ɣer New Bedford. D iḍ n ssebt deg Dujember. Aṭas i s-d-sfehmeɣ mi sliɣ belli lbaṭaṭa tamecṭuḥt n Nantucket teffeɣ-d yakan, yerna ulac abrid ara d-yefken akken ad awḍeɣ ɣer wemkan-nni, alamma d ass n lḥedd i d-iteddun.

Akken amur ameqran seg yimazwaren ilmeẓyen i lmend n lḥif ed lḥebs n tmeẓdiyin (baleines) ttḥebbsen deg New Bedford-a kan, seg din akken ad bdun tikli-nsen, yezmer lḥal ad d-yili dakken nekk, ɣef yiwen, ur sɛiɣ ara tikti ɣef waya.
