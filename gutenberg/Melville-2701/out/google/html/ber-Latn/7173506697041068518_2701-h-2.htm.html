IXEF WIS 4. Aqcic n tmeqbert.

Mi d-ffɣeɣ sbeḥ-nniḍen ɣef tafat n wass, ufiɣ afus n Queequeg yettwaḍegger fell-i s tmuɣli n tayri d lḥir akk. Qrib ad t-id-tiniḍ d nekk ay d tameṭṭut-nnes. Tafelwit n tmeqbert tella n tmeqbert, teččur d iqudas d triangles n tmeqbert tamecṭuḥt n tmeqbert n tmeqbert ; yerna afus-agi-ines yettwaṭṭef akk s yiwen n lbabur n Kṛiṭa ur nesɛi tagara n yiwen n wudem, ulac sin yiḥricen deg-s llan s yiwen n lḥirfa s tɣawla – ɣef ljal n tmuɣli-ines i yeṭṭef afus-is deg yilel s tɣawla deg yiṭij d lḥirfa, lqecc n tceṭṭiḍt-is ttwaḍeggren s lqanun deg talliyin yemgaraden – ayagi qqaren-d yiwen n lḥirfa amzun d afus n, quilt n tmeqbert. D tidet, amur qqimen fell-as am wakken i t-id-yesbedd ufus mi d-ffɣeɣ tikkelt tamezwarut, s lqella n tmussni ssawḍeɣ ad t-id-iniɣ seg tmeẓẓuɣt, s waya sxeṣren lḥirfat-nsen; yerna ala s tmuɣli n lqis d tɣawsa ay zemreɣ ad d-iniɣ d akken Queequeg la iyi-d-yettḥemmil.

Iḥulfan-iw llan d iɛewwiqen. Eǧǧ-iyi ad ɛerḍeɣ ad ten-sfehmeɣ. Asmi lliɣ d ameẓyan, cfiɣ mliḥ ɣef yiwet n liḥala yecban kra n tɣawsiwin i yi-d-yeḍran; ama d tilawt neɣ d targit, ur zmireɣ ara ad sbeddeɣ iman-iw s lekmal-is. Liḥala tella akka. Ssneɣ kra n tkapert neɣ nniḍen – ttwaliɣ d akken teɛreḍ ad t-id-tessufeɣ s tmeẓẓuɣt, akken walaɣ cwiṭ n tmeẓẓuɣt tga kra n wussan uqbel; akked yemma-inu n tmeṭṭut-iw i yellan, s kra n tɣawsa neɣ s kra nniḍen, tettḥezzib-iyi akk tikkal, neɣ tettazen-iyi ɣer usu war imensi, — yemma tessufeɣ-iyi-d s yiḍarren-iw seg tmeqbert, tessewjed-iyi-d ɣer usu, ɣas akken d snat kan n tmeddit n wass n 21 yunyu, ass iɣezfanen akk deg useggas deg tlemmast n tmurt-nneɣ. Ḥulfaɣ s lxuf. Maca ulac lemɛawna fell-as, ihi s tɣawla ṛuḥeɣ ɣer texxamt-iw tamecṭuḥt yellan deg lqaɛa tis kraḍt, sfeḍḍeɣ iman-iw s tɣawla akken ad nɣeɣ lweqt, s yiwen n usnuffes rẓayen uwḍeɣ gar tfelwiyin.

Nekk qqimeɣ dinna s lḥir ttḥessibeɣ belli ilaq ad ɛeddint mraw d setta n tsaɛtin s lekmal-nsent uqbel ad ssarameɣ lḥeqq n tnekra. Setta n tsaɛtin deg usu! ameẓyan n tɣerɣert-iw yeqreḥ mi ara t-id-xemmeɣ. Yerna tella d tafat ula d nettat; iṭij yettfeǧǧiǧ deg tɣerɣert, d tmeẓẓuɣt tameqrant n tkeṛṛusin deg yiberdan, d ṣṣut n tmeslayin n yigayyen deg wexxam akk. Ḥulfaɣ s lxir ugar n lxir – taggara kkseɣ-d, lseɣ, s tɣawla ḥeḍreɣ deg yiḍarren-iw s tcekkaṛt, nudan-d ɣef yemma-inu n tmeṭṭut, s kra n tɣawsa sderɣleɣ iman-iw ɣer yiḍarren-is, ttḥellileɣ-tt am lxir s umata akken ad iyi-d-tefk sliḍ yelhan ɣef lbaṭel-iw ; kra s tidett anagar ad iyi-ḥekmen ad skerḍeɣ abed am taɣect ur nesɛi lḥeqq. Maca d nettat ay d tameṭṭut ay igerrzen akk yerna tesɛa lḥeqq gar yemma-s n tmeṭṭut, yerna ɣer deffir ilaq-iyi ad dduɣ ɣer texxamt-inu. Acḥal n tsaɛtin lliɣ ttɣiliɣ dinna s tɣawla, ttḥulfuɣ s lxir meqqren ugar n wayen lliɣ ttḥulfuɣ seg wasmi ay lliɣ, ula seg lbaṭel ameqran ay d-yeḍran sdeffir-s. Taggara ilaq ɣliɣ deg yiwen n yiḍ n tmeɣra n tmeɣra; yerna s tɣawla ttfeggiḍeɣ seg-s – azgen s tmeẓẓuɣt n targit – ldiɣ allen-iw, yerna texxamt-nni uqbel i d-yesfaydi yiṭij, tura tettwagdel deg ṭṭlam n berra. Deg yiwen n wakud ḥulfaɣ s yiwen n usḥissef yettazzalen deg wakk akaram-iw; ulac d acu ara d-yettwawalin, ulac d acu ara d-yeslan; maca afus n tmeṭṭut iban-d amzun yettwassers deg win-iw. Ifusen-iw ttwaḍeggren ɣef tmeẓẓuɣt, yerna lbenna neɣ lfanṭum ur nesɛi isem, ur nesɛi lmeɛna, yessusem, iɣef yella ufus-nni, iban-d amzun yeqqim qrib ɣer tama n wusu-w. Imi ayen ibanen d leqrun ttwaferqen ɣef leɛmeṛ, lliɣ ttɣiliɣ dinna, sgerrzeɣ s tugdiwin tiqburin akk, ur zmireɣ ara ad sneqseɣ afus-iw; maca ttxemmimeɣ kra n tikkelt belli ma yella zemreɣ ad t-sfeḍḍeɣ kan yiwen n lmitra, ad t-yeqḍeɛ lḥirfa-nni n lḥir. Ur ẓriɣ ara amek i yi-d-teffeɣ tmusni-agi taggara-agi; maca mi d-ffɣeɣ sbeḥ, s tɣawla ttmektayeɣ-d akk, yerna ussan d dduṛt d wayyuren deffir waya sxeṣreɣ iman-iw deg ɛewwiqen n tkerkas akken ad d-sfehmeɣ lbaḍna-nni. Ala, armi d ssaɛa-agi, ttḥeqriɣ iman-iw s waya.

Tura, kkset-d tugdi-nni n tkerkas, yerna tḥulfa-inu mi ḥulfaɣ s ufus n tmeẓẓuɣt yellan deg-i mxallafent aṭas, deg tmeẓẓuɣt-nsent, d tid ay ḥulfaɣ mi d-ffɣeɣ seg tfekka-inu yerna ẓriɣ afus n Queequeg ay d-yellan d aɛebbuḍ yettwaḍḍef fell-i. Maca ɣer taggara, akk tidyanin n yiḍ yezrin ttkemmilent-d s tɣawla, yiwet yiwet, deg tilawt n tɣawsiwin, syin lliɣ ttɣimiɣ kan ddreɣ ɣer wuguren-nni n tmedyazt. Imi ɣas akken ɛerḍeɣ ad sneqseɣ afus-is – ad sfeḍeɣ aqerruy-is n usli – maca, yeṭṭes akken yella, mazal-it yeḥḍer-iyi s lǧehd, amzun ulac lmut ara aɣ-yebḍun sin. Nekk ssawḍeɣ tura ad t-sfehmeɣ — “Queequeg!”— maca tiririt-is kan d aḥebbus. Nekk syin sfeḍḍeɣ, aqerruy-iw ḥulfaɣ amzun yella deg tmeẓẓuɣt n yizem; yerna deg yiwet n tikkelt ḥulfaɣ s tmeẓẓuɣt n tmeẓẓuɣt. Yessers ɣer yidis lqaɛa n tmeẓẓuɣt, dinna i d-yezga tomahawk-nni yeṭṭes ɣer tama n tmeṭṭut-nni n tmeẓẓuɣt, amzun d aɛebbuḍ s wudem n tmeẓẓuɣt. A pickle yelhan, s tidet, ttxemmimeɣ; abed dagi deg yiwen uxxam n tmura tiberraniyin deg wass ameqqran, s yiwen n umdan n wemdan d yiwen n tomahawk! “Quequeg!—s yisem n lxir, a Quequeg, sfehmeḍ-d!” Deg taggara, s lmendad n waṭas n tmuɣliwin, d tmuɣliwin s ṣṣut ɛlayen yerna ur nettḥebbis ara ɣef lbaṭel n tḥemla-ines n umdakkel-is n wergaz s ṣṣenf-nni n zzwaǧ, ssawḍeɣ ad d-ssufɣeɣ yiwen n usgunfu ; yerna tura kan, yerra-d afus-is ɣer deffir, iḥebbes iman-is akk am yiwen n uqjun n Newfoundland kan seg waman, yerna yeqqim deg usu, yeḥbes am tmeẓẓuɣt n tmeẓẓuɣt, yettmuqul-iyi-d, yerna yettḥezzib allen-is amzun ur d-yemmekti ara akk amek ay d-usiɣ din, ɣas akken kra n tɣawsa n tmeẓẓuɣt tettḥulfu-yi-d s tɣawla. Di lawan-nni, lliɣ ttɣimiɣ ttmuquleɣ-t s wallen-iw s tɣawla, ur sɛiɣ ara ccek d ameqran tura, yerna ttqerribeɣ ad ttwaliɣ s tɣawla amdan-nni ay d-yellan d aɛewwiq am wa. Mi, taggara, iban-d d akken lbal-is yeqbel ad iḥaz ṣṣifa n umddakkel-is n yiḍes, yerna yuɣal, amzun d ayen i d-yesfaydin seg lḥeqq ; yeffeɣ-d ɣef lqaɛa, yerna s kra n tmitar d tɣawsiwin yefka-yi-d ad fehmeɣ belli, ma yella yeɛǧeb-iyi, ad yels uqbel, syin ad iyi-yeǧǧ ad lseɣ deffir-s, ad yeǧǧ axxam-nni akk i yiman-iw. Txemmem nekk, Queequeg, deg tegnit, aya d uverture s tɣerma aṭas ; maca, tidett, iɣersiwen-agi sɛan lḥirfa n tmeẓẓuɣt, ini-d ayen tebɣiḍ ; d ayen yessewhamen amek llan s lsas d iḥeqqiyen. Nekk ttxelliseɣ tajmilt-a (compliment) s umata i Queequeg, imi ay iyi-d-yefka aṭas n tɣerma ed tmuɣli, deg wakud aydeg nekk lliɣ ttwaɛaqbeɣ s lbaṭel ameqran; tmuqel-it seg usu, yerna tmuqel akk tikliwin-is n tmeqbert ; i lmend n wakud-nni lḥir-iw yettnerni deg tmeṭṭut-iw. Dacu kan, argaz am Queequeg ur t-tettwaliḍ ara yal ass, netta d yiberdan-is llan s wazal-nsen ur nelli ara d lqayen deg wayen yerzan.

Yebda yettḍeggiɛ s tqacuct s tmeqbert-is n lbeɛd, d tin ɣezzifen aṭas, s tlemmast, syin – mazal-it s lqella n tceṭṭiḍt-is – iḥudd iḍarren-is. Acu yellan ddaw igenwan i t-yexdem, ur zmireɣ ara ad d-iniɣ, maca amussu-ines i d-iteddun d akken ad iḥebbes iman-is – lbuṭat deg ufus, d tqacuct ɣef – ddaw usu ; mi, seg tmeqwa n ccwal d tmeqwa n ccwal, ssawḍeɣ ad d-sfehmeɣ belli yella ixeddem s lǧehd yettbuṭṭu iman-is; ɣas akken s kra n lqanun n lḥeqq ur sliɣ ara fell-as, yal argaz yettwaḥettem fell-as ad yili d aḥerfi mi ara yessers iḍarren-is. Maca Queequeg, twaliḍ, yella d amdan yellan deg tallit n tmuɣli n tɣawsiwin – la d aɣersiw n tmeẓẓuɣt, la d aɣersiw n tmeẓẓuɣt. Yella kan d aɣerman akken ad d-yessken lbaṭel-is s tɣawsiwin tiberraniyin akk. Ur yekfi ara leqraya-s. Yella d anelmad n lbak. Lemmer ur yelli ara d aɣerman s tɣawla tamecṭuḥt, ahat ur yelli ara yettḥaz iman-is s tbuciḍant akk ; maca syen, lemmer ur yelli ara mazal-it d aɛewwiq, ur yettargu ara ad yekcem ddaw usu akken ad ten-yessers. Taggara, yeffeɣ-d s tceṭṭiḍt-is i d-yesfeḍḍen aṭas yerna tettwaḍegger ɣef wallen-is, yebda yettqerrib yerna yettqerrib deg texxamt, amzun, ur t-iɛedda ara aṭas ɣef yiḍarren, tiqbucin-is n tfunast i d-yesfeḍḍen, i d-yesfeḍḍen – ahat ula d nutenti ur tt-id-sbedden ara s lqanun – ttqerribent-as yerna ttqerribent-as-d s tmeẓẓuɣt tamezwarut.

Mi iwala, tura, ur llint ara tkurtin ɣer tɣerɣert, yerna abrid-nni d aẓiḍan aṭas, axxam yellan qbala-s yumer-d tamuɣli s tɣawla ɣer texxamt, yerna yettwali s waṭas lḥirfa ur nesɛi lḥeqq i d-yesnulfa Queequeg, yettɛedday s kra nniḍen anagar aqerruy-is d yiḍarren-is ; Ttḥellileɣ-t akken zemreɣ, ad yessiweḍ kra n tɣawsiwin s tɣawla, ladɣa ad yekcem deg yipantalunen-is akken ilaq. Yeḍfer, syin yebda yettsirid iman-is. Di lweqt-nni di tmeddit yal amasiêi ad yessired udem-is; maca Queequeg, s lfeṛḥ-iw, yeqbel iman-is s tḥeqranit n tẓallit-is deg yidmaren-is, deg yifassen-is, deg yifassen-is. Yessers-d syin lqecc-is, yerna yeddem-d yiwen n ssebɣa iqesḥen ɣef ṭṭabla n tlemmast n tmeqbert n usired, yessers-it deg waman, yebda yettfeggiḍ udem-is. Ttwaliɣ anda i yeṭṭef lqecc-is, mi d-yeffeɣ lo and behold, yekkes-d lḥerma-nni seg tlemmast n usu, yessufeɣ-d lḥenni-nni n useklu ɣezzifen, yefser-d aqerruy-is, iḥebbes-it cwiṭ ɣef tbuciḍant-is, yerna yezger-d ɣer tɣawsa n lmitra ɣef lḥiḍ, yebda-d lḥenni s lǧehd, neɣ s tɣawla, s lḥenni-ines. Txemmem nekk, Queequeg, aya d aseqdec n tmeqbert n Rogers igerrzen s tmeqbert. Syin steqsaɣ iman-iw ɣef tɣawsa-agi mi ẓriɣ d acu n wuzzal igerrzen ay d-yettwaxedmen uqerru n tmeẓẓuɣt, d wamek ay ttwaḥerzen dima yiɣezran-nnes usriden ɣezzifen s tɣawla.

Ayen nniḍen n tmeqbert-is ur tɛeṭṭel ara yewweḍ, yerna yeffeɣ-d s lferḥ seg texxamt, yettwaḍḍef s tmeqbert-is tameqrant n tmeqbert n tmeqbert, yerna yettḍeggiɛ lḥerpun-is am tmeqbert n umaru.





IXEF WIS 5. Iḍelli.

Nekk ḍefren-iyi-d s tɣawla, yerna mi ḥeḍreɣ ɣer texxamt n lbar, ssawḍeɣ ad sqedceɣ bab n wexxam-nni ay d-yesɣermen s lfeṛḥ d ameqran. Ur ḥemmleɣ ara lxir fell-as, ɣas akken yella yettmeslay-d yid-i mačči cwiṭ deg temsalt n umddakkel-iw n yiḍes.

Maca, taḍsa yelhan d ayen yelhan ijehden, yerna d ayen yelhan qlilet aṭas; Akken yugar lḥir. Ihi, ma yella kra n wemdan, deg wemdan-is, yezmer ad yesɛu tiɣawsiwin i lmend n tmeɣra yelhan i yal yiwen, ur yettili ara d aɛewwiq, maca ad yeǧǧ iman-is s lferḥ ad yesɛeddi yerna ad yettwasɛeddi s webrid-nni. Yerna argaz-nni yesɛan kra n tɣawsa s lxir fell-as, ḥeqq belli yella deg wergaz-nni ugar n wayen i tḥesbeḍ ahat.

Tazeqqa n lbar teččur tura d iɛeggalen n tɣiwant i d-yeɣlin deg yiḍ yezrin, yerna ur ten-ẓriɣ ara ar ass-a mliḥ. Qrib akk d iɣersiwen n yilel; imeqqranen imeqqranen, d imdukal wiss tnayen, d imeqqranen wis tlata, d ineǧǧaren n lebḥar, d ixeddamen n lḥenni n lebḥar, d iḥeddaden n lebḥer, d iḥeddaden n tmeẓẓuɣin d iɛessasen n iɣerruba. tkebbanit n tmeẓyant d tmeẓyant, s tmeẓẓuɣin n tmeẓẓuɣin; yiwen n uswir ur nesɛi ara iqecwal, d aɛeqli, lsan akk tijakiḍin n yilef i tmeqwa n ṣṣbeḥ.

Tzemreḍ ad d-tiniḍ s tɣawla acḥal aya seg wasmi ay yella yal yiwen deg-sen deg yilel. Taɣect n yilemẓi-agi yesɛan tadawsa tella am tferra n yiṭij deg tmeẓẓuɣt, yerna ad d-tban amzun tesɛa rriḥa qrib am tmeẓẓuɣt n tmeẓẓuɣt; ur yezmir ara ad yili kraḍ wussan deg tmurt seg tikli-ines n tmurt n Lhend. Argaz-nni yellan zdat-s yettban-d kra n tfelwiyin n tfekka; tzemreḍ ad d-tiniḍ lḥess n usɣar n satin yella deg-s. Deg tfekka n yiwen wis kraḍ mazal-it yettkemmil yiwen n tawn n tmura tiberraniyin, maca yettwasfeḍ cwiṭ s tmeẓẓuɣt; ur yelli ccek belli yeqqim ddurt s lekmal-is deg yilel. Maca anwa i izemren ad d-yessken lḥenk am Queequeg? llant, ttwaḥebsent s tɣawsiwin yemgerraden, ttbanent-d am tɣerɣert n wegmuḍ n Andes, akken ad d-sseknent deg yiwen n uswir, ttbeddilent tnemsal, tamnaḍt ɣef temnaḍt.

“Grub, ho!” tura i d-yessuɣ bab n uxxam, yeldi-d tawwurt, deg-s nekcem ad nečč lqahwa.

Qqaren-d belli irgazen i d-yeẓran ddunit, s waya, ttɣimin s lxir aṭas deg tmuɣli, ttḥulfun s lxir mliḥ deg tdukli. Mačči dima, d acu kan: Ledyard, amsebrid ameqran n Langliz Tamaynut, d Mungo Park, win n Skutlanda; deg yemdanen merra, sɛan lḥeqq ameẓyan akk deg uxxam n tmeɣra. Maca ahat d aɛeddi kan n Sibiriya s tkerrust ay d-jebden yiqjan am wakken ay t-yexdem Ledyard, neɣ d aɛeddi ɣezzifen n yiman-nnes s yiwen n uɛebbuḍ d aɛebbuḍ, deg wul n yineggura n Tefriqt, d tin ay yellan d ssuma n tmezgunin n Mungo ameɣbun – ṣṣenf-a n usikel, qqareɣ-d, yezmer lḥal ur yelli ara d abrid anmetti ay d-yellan s wazal-nnes akk akken ad naweḍ ɣer yiwen n polish aɣerfan. Maca, deg uḥric ameqran, ṣṣenf-nni n tɣawsa ad yili deg yal amkan.

Tiktiwin-agi kan dagi ttwaxedment s tegnit-nni aydeg deffir ma neqqim akk ɣef ṭṭabla, yerna lliɣ ttheyyiɣ-d iman-iw ad sleɣ i kra n tmucuha yelhan ɣef tmeẓdit n yilef; s lfeṛḥ-iw ur nelli d ameẓyan, qrib yal argaz yeṭṭef tassusmi lqayen. Yerna mačči kan d aya, maca ttwalin-ten s lḥeq. Ih, dagi tella yiwet n terbaɛt n yiqjan n yilel, aṭas seg-sen war ma sḥissifen cwiṭ, rnan rnan ɣef yiɣersiwen imeqranen deg yilel ameqran – d iberraniyen s lekmal-nsen – yerna ttnaɣen fell-asen d imettanen war ma sqedcen tiṭ-nsen; yerna ɣas akken, dagi qqimen ɣer yiwet n ṭṭabla n lqahwa tanmettit – akk s yiwet n tiɣri, akk s tɣawsiwin n tmeṭṭut – ttmuqulen gar-asen s lḥir amzun ur llin ara sdat n tmuɣli n kra n tmeqbert n lmal gar yidurar izegzawen. Tamuɣli n tmusni; ursuten-agi yettḥeqriqen, iɣerruba-agi n yimenɣi yettḥeqriqen!

Maca ma d Queequeg – ayɣer, Queequeg yeqqim dinna gar-asen – deg uqerru n ṭṭabla, ula d netta, d ayen i d-yeḍran s lbaḍna ; am usemmiḍ am ugris. Akken ad tt-id-iniɣ ur zmireɣ ara ad d-iniɣ aṭas ɣef tmeṭṭut-is. Amḥaddi-ines ameqqran ur yezmir ara ad d-yesbedd s lḥir imi ay d-yewwi lḥerma-nnes ɣer tmeddit yid-s, yerna yesseqdec-itt din war ma yella-d tmeɣra; yewweḍ s-s ɣef ṭṭabla, ɣer lḥers i d-iteddun n waṭas n yiqerra, yerna yettḥebbis iɣersiwen n lmal ɣer tama-s. Maca aya s tidett yella-d s tmeẓẓuɣt (cool) aṭas sɣur-s, yerna yal yiwen yeẓra belli deg tmuɣli n tuget n medden, ad tgeḍ kra s tmeẓẓuɣt d ayen ara t-txedmeḍ s tmeẓẓuɣt.

Ur d-nettmeslay ara dagi ɣef wakk tiɣawsiwin n Queequeg; amek i yettɛeṭṭil lqahwa d tmeqwa n lḥemla, yerna yefka azal-is ur yettwabḍan ara ɣef tmeqwa n lmal, yettwaxedmen s lqella. Imeslay-d Rebbi i yiselman, yerna seg usemmiḍ d tmeẓẓuɣt n yilel i d-yesḥissifen, yusa-d lbaṭaṭa-nni s tmeẓẓuɣt ɣer yiṭij iḥemmlen yerna d ayen igerrzen, d wakk lfeṛḥ n lḥaǧa d lqaɛa, yerna ‘tessufeɣ-d Yuna ɣef wakal iqerḥen’; mi d-yusa wawal n Sidi tikkelt tis snat;u Yuna, yettwaqbel yerna yettwawet — imeẓẓuɣen-is, am sin n yiqjan n yilel, mazal-iten ttmuqulen s lɣaci n yilel — Yerna d acu-t waya, ay imdukal n lbabur Ad t-id-berrḥen Tidet i wudem n Lekdeb!

"Tagi, ay imdukal n lbabur, d tamsirt-nni nniḍen, u aɛaqeb n umsebrid-nni n Rebbi yeddren i t-iḥeqren. Aɛaqeb n win i gḥemmel ddunit-agi s lḥeqq n Linjil! Aɛaqeb n win yettnadin ad issers zzit ɣef waman mi ara ten-id-yesfeḍ Rebbi d aḍu! win, di ddunit-agi, ur yettḥeqri ara lḥeqq i win ur nelli ara d tidet, ɣas akken ad yili d lekdeb d asfel, Ih, aɛeqli i win, akken i t-id-yefka Upiluṭ ameqqran Bulus, deg wakud ideg yettbeccir i wiyaḍ, d netta s timmad-is i d-yesbedden lḥeqq !

Yeɣli-d yerna yeɣli-d seg yiman-is kra n lweqt; syin yerfed udem-is ɣur-sen tikkelt nniḍen, yessken-d lferḥ lqayen deg wallen-is, mi d-yessuɣu s lferḥ n yigenni, — « Maca ay imdukal n lbabur! deg ufus n tlemmast n yal lḥif, yella lferḥ s lḥeqq ; s lbeɛd, s lbeɛd s afella, d lferḥ n daxel- win mgal irebbiten d yiqeddacen n lqaɛa-agi, yal tikkelt ibedd s timmad-is ur nesɛi lḥeqq Lferḥ d win i t-iɛawnen s yifassen-is ijehden, mi ara d-yeɣli uɣerrabu n umaḍal-agi ixeddaɛen ddaw-as Lferḥ d win i s-d-yefkan ur yettak ara ddnub, netta d tidet. ddaw n tmeqwa n Senator d Lqaḍiyen.Lferḥ n tqacuct d netta, win ur yettqadaren ula d yiwen n lqanun neɣ d bab, maca d Sidi Rebbi-ines, yerna d netta kan i d aḥemmel n tmurt ɣer yigenni Lferḥ d netta, win ur zmiren ara ad t-id-sfeḍḥen akk lmuḥal n tmeẓẓuɣin n yilel n lɣaci n tmeẓẓuɣin. ad t-id-yessers, yezmer ad d-yini s usnuffes-is aneggaru — a Baba !

Ur d-yenni ara s waṭas, maca s tɣawla i d-yesfeḍ lbaṛaka, iḍegger udem-is s ifassen-is, akka i yeqqim yeqqim ɣef yiḍarren-is, armi ffɣen akk medden, yeqqim weḥd-s deg umkan-nni.
