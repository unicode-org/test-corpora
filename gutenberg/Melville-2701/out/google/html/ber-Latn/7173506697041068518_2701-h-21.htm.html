Mi yella mazal-it ibeɛdeḍ cwiṭ ɣef tmeqbert, Ahab s lḥir yeḥbes; armi d taggara, Perth, yekkes-d wuzzal-is seg tmes, yebda yettḥezzib-it ɣef tmeẓẓuɣt-nni – taɣect taberkant tessufuɣ-d tifekkiwin-nni s yifeggiḍen n tmeẓẓuɣt n tmeẓẓuɣt, kra seg-sent ttfeggiḍent qrib ɣer Ahab.

"Ma d wigi i d igerdan n Yemma-k Carey, Perth? ttfeggiḍen dima deg tfekka-k; d igḍaḍ n lxir, ula d nutni, maca mačči i yal yiwen;—muqel dagi, ttqerriben; maca kečč—teddred gar-asen mebla ma tesɛiḍ lḥemla n tmes."

“Axaṭer sqeɛdeɣ akk, a Kapṭan Ahab”, i s-d-yerra Perth, yessufeɣ kra n lweqt ɣef tmeẓẓuɣt-is; “Nekk ɛeddaɣ s tmes, ur tezmireḍ ara ad tesreɣeḍ s lfeɛl.”

"Aha, aha; ulac ugar. Taɣect-ik s lḥir, s lḥir, tettban-iyi-d s lḥif. Deg ulac lǧennet s timmad-iw, ur sɛiɣ ara ṣṣber ɣef lḥif akk yellan deg wiyaḍ ur nelli ara d aḥeqrani. Ilaq ad t-teḥbesḍ, a lḥedd; ini-d, ayɣer ur tettḥeqriḍ ara? Amek ara tiliḍ mebla ma yella-d d aḥeqrani? belli ur tezmireḍ ara ad t-texḍuḍ ?— D acu i tgiḍ dinna?”

"Sselḥuɣ aqerruy aqbur n lbaṭaṭa, Mass; llant deg-s tmeqwa d tmeqwa."

“U ur tezmireḍ ara ad t-terreḍ akk d aḥlawan, a lḥedd, deffir n useqdec iqesḥen am wakken yella?”

“Ɣileɣ akka, Mass.”

"U ɣaleɣ tzemreḍ ad tesselḥuḍ qrib akk tiɣbula d yiɣeblan; ur tḥeqqeḍ ara amek i t-yeqseḥ wuzzal, a aḥeddad?"

"Ih, Mass, ɣileɣ zemreɣ; akk tiɣbula d tɣemmar anagar yiwet."

“Muqel-d dagi, ihi,” i d-yenna Ahab, s lḥir yettkemmil, yerna yettkel s sin ifassen-is ɣef yiḍarren n Perth; “muqel-d dagi— dagi —tzemrem ad tesfeḍ lḥirfa am tagi, a lḥedd,” iḥebbes yiwen ufus ɣef tmeẓẓuɣt-is yesɛan iɣes; "lemmer tzemreḍ, a lḥedd, s lferḥ ad sserseɣ aqerruy-iw ɣef tmeẓẓuɣt-ik, ad ḥulfuɣ s tmeẓẓuɣt-ik iẓewren akk gar wallen-iw. Tiririt! Ur tezmireḍ ara ad tesselḥuḍ aẓeḍḍa-agi?"

"Ah! D winna, Mass! Nniɣ-d mačči d akk tiɣbula d tiɣbula maca yiwet?"

"Ah, a lḥedd, d netta kan; ay argaz, ur yettwasfeḍ ara; imi ɣas ma yella twaliḍ-t kan dagi deg uksum-iw, maca yewweḍ armi d iɣes n tqacuct-iw – d ayen kan i d-yeɣlin! t-id-yesgerwen lqecc n lqecc, amzun yeččur d lḥenni n wureɣ. "Nekk, ula d nekk, bɣiɣ ad d-ttwaxedmen yiwen n lḥerma, win ur zmiren ara ad t-ferqen alef n yiɛewwiqen, Perth; kra ara yeṭṭfen deg yiwen n uɣersiw n yilel am yiɣes n tfekka-s. Dinna i d-yella lḥaǧa-nni", i d-yessers taɛkemt-nni ɣef tmeẓẓuɣt. “Muqel keččini, a lḥedd, wigi d iɣeblan n tmeẓẓuɣt yettwajemɛen n yiḍarren n wuzzal n yiysan n tcirḍart.”

"Iḍarren n yizem, a mass? Ayɣer, a Kapṭan Ahab, tesɛiḍ dagi, ihi, ayen igerrzen akk yerna d aɛewwiq akk ay nexdem nekni s yiḥeddaden."

"Ssneɣ-t, ay amɣar; tiɣbula-agi ad msefhament am lgaz seg yiɣsan n yimenɣayen. S tɣawla! Snulfu-yi-d lḥerma. Snulfu-yi-d di tazwara, mraw d sin n yiɣil i tɣerɣert-is; syen s waḍu, s tɣawla, s marṭul n tnac-agi am lxiḍ d yiferdisen n lḥiḍ n usɣerreb."

Mi ttwaxedmen taggara tnac n tmeẓyanin-nni, Axab yeɛreḍ-itent, yiwet yiwet, s tmeẓẓuɣt-nsent, s ufus-is, ɣef yiwen n lḥebs n wuzzal ɣezzifen yerna ẓẓayen. “D lɛib!” tugi aneggaru. “Xeddem ayagi tikkelt nniḍen, a Perth.”

Aya yexdem, Perth qrib ad yebdu ad yessiweḍ tnac-nni ɣer yiwen, mi yeqqim Ahab afus-is, yenna-d ad yessiweḍ wuzzal-is. Akken, imir-nni, s tmeẓẓuɣin n lqanun, s tmeẓẓuɣin n tmeẓẓuɣin, yettḥezzib ɣef tmeẓẓuɣt, Perth yettɛedday-as-d tiɣbula yettfeǧǧiǧen, yiwet deffir tayeḍ, d tmeẓẓuɣin-nni yettwaḥebsen s lǧehd tettḍeggiɛ-d lḥemla-ines srid n tmeẓẓuɣt, Parsee yezger-d s tsusmi, yerna yeɣli-d ɣef uqerruy-is ɣer tmes, amzun yettɛeggiḍ ɣef kra n tmeɣra neɣ ɣef kra n tmeẓẓuɣin. Maca, akken i d- imuqel Axab, iɛedda ɣer yidis.

“D acu i d lbaḍna-nni n lucifers i d-yettawin dinna?” » i d-yenna Stubb, yettmuqul seg tmeqbert. « Parsee-nni yessusem timest am fusee ; yerna yessusem-itt s timmad-is, am tmeqbert n lbuḍ n tmusket yeḥman ».

Di taggara lqecc, deg yiwen n lḥebs ikemlen, yewwi-d lḥemla-ines taneggarut; yerna mi t-tessers Perth, akken ad t-tesfeḍ, tessers-itt akk s tmeẓẓuɣt deg lqahwa n waman yellan qrib-is, lḥemla-nni n tmeẓẓuɣt tettfeggiḍ-d deg wudem n Ahab i d-yesɣermen.

“Ad iyi-t-id-tessekneḍ, a Perth?” tmuqel kra n lweqt s lḥif; “ma lliɣ kan ttṣeggimeɣ lmarkat-iw n wuzzal, ihi?”

"Ḍelbeɣ i Rebbi, mačči d aya, maca ttgaḍeɣ kra, a Kapṭan Ahab. Mačči d lḥerma-agi i Baleine Amellal?"

"I lbaṭel amellal! Maca tura i tmeẓẓuɣin; ilaq ad tent-txedmeḍ s timmad-ik, a wergaz. Atan tmeẓẓuɣin-iw – tid igerrzen akk deg wuzzal; atan, u rẓem tmeẓẓuɣin-nni d tizegzawin am tmeẓẓuɣin n yilel n Ugellid.”

Cwiṭ n lweqt, aḥeddad aqdim yettmuqul deg tmeẓẓuɣin-nni amzun ur tent-yesseqdac ara.

“Awi-ten, a wergaz, ur ten-ḥwaǧeɣ ara, axaṭer tura ur ttḥezzibeɣ ara, ur ttḥezzibeɣ ara, ur ttẓallaɣ ara alamma d — maca dagi — ad xedmeɣ!”

Yettwaxdem taggara-a s talɣa n tmeẓẓuɣt, yerna yesmel-it Perth ɣer tmeẓẓuɣt, ur yeɛjib ara ad d-yesmekti wuzzal-nni taggara n wuzzal ; yerna mi yella lḥedd-nni qrib ad yefk i tmeẓẓuɣin-nni lḥemla-nsen taneggarut, uqbel ad ten-yesfeḍ, iɛeggeḍ i Axab akken ad yerr qrib-is lqahwa n waman.

"Ala, ala - ulac aman i waya; bɣiɣ-t n tidet n lmut. Ahoy, dinna! Tashtego, Queequeg, Daggoo! D acu ara d-tinim, ay ibuḍanen! Ad iyi-d-tefkem idammen akken ara d-yeḍḍfen aɛekkaz-agi?" teṭṭef-it s lqedd. Yiwet n tqeḍɛit n yiḍarren iqesḥen terra-d, Ih. Kraḍ n tmeẓẓuɣin ttwaxedment deg uksum n yigenwan, yerna ttwaḥerzent sakkin tmeẓẓuɣin n White Whale.

“Nekk ur ttwaɣeḍseɣ ara s yisem n baba, sed s yisem n cceṛ!” » i d-yesɣer Axab s lḥir, akken i d-yečča wuzzal n lxir idammen n tmeɣra s lḥir.

Tura, mi d-yesnejmaɛ iḍarren n tmerniwt seg ddaw, yerna yefren yiwen n lḥemmam, s lqecc mazal-it yessefray-it, Ahab yessers taggara-nni ɣer tɣerɣert n wuzzal. Syin, teffeɣ-d yiwet n tmeẓẓuɣt n tmeẓẓuɣt tamaynut, yerna kra n tfekkiwin seg-s ttwawwint ɣer tmeẓẓuɣt n tmeẓẓuɣt, yerna ttwaḍeggrent ɣer tɣawsa tameqrant. Iḥettem aḍar-is fell-as, armi d asmi i d-yesḥissif lḥenni am lḥenni n lḥenni, syin s lḥir i d-yeɣli fell-as, yerna ur iwala ara lḥenni, Axab yenna-yas: “Ihi, tura i tmeẓẓuɣin.”

Deg yiwet n tlemmast, aẓru-nni ur yelli ara d aɛekkaz, yerna iferdisen-nni yettwaferqen yemxallafen ttwaɛezlen akk yerna ttwaɛezlen ɣef tɣerɣert n tmeẓẓuɣt n tmeẓẓuɣt; lqaɛa-nni ttwaḍeggren-d s lǧehd s aɛlayen deg tmeẓẓuɣt; seg tlemmast n ddaw, ttwaḍefren yiɣil-nni s uzgen n webrid ɣef teɣzi n tɣerɣert, yerna ttwaḍemnen s lǧehd akka, s tɣawsiwin n tɣawsiwin yettwasqedcen gar-asent. Aya yexdem, aɛekkaz, wuzzal, d lḥiḍ – am Kraḍ n Tmeẓẓuɣin – qqimen ur ttwabḍan ara, yerna Ahab s lḥir yeffeɣ-d s lbumbat-nni; ssut n uḍar-is n lgaz, d ssut n tmeẓẓuɣt n lḥik, sin-nni ttcewwiqen s tmeẓẓuɣt ɣef yal aḍebsi. Maca uqbel ad yekcem ɣer tkebbanit-is, tesla-d s ṣṣut ifesden, ur nelli ara d aɣelnaw, d azgen n tmeẓẓuɣt, maca d ayen i t-yesḥissifen aṭas. A, Bib! taḍsa-inek n lḥif, allen-inek ur nesɛi lxedma maca ur tettḥulfu ara; akk timmumin-ik tiberraniyin ur mxallafent ara war lmeɛna d tmeɣra taberkant n uɣerrabu n lḥif, yerna sḍehrent-t !
