IXEF 89. Iselman n Tmeẓẓuɣt d Yiselman n Tmeẓẓuɣt.

Allusion ɣef waif ed waif-poles deg yixef aneggaru maca yiwen, yessefk kra n leḥsabat ɣef yiṣuḍaf ed lqanun n tmerrit n yilef, aydeg waif yezmer ad yettwaḥseb d azamul ed tmeẓdit (badge) tameqrant.

S waṭas ay d-yettili mi ara tteddun aṭas n yiɣerruba deg tkebbanit, yiwen n uɣersiw n yilel yezmer ad t-yeṭṭef yiwen n uɣerrabu, sakkin ad yerwel, yerna ɣer taggara, ad t-ineɣ yerna ad t-yeṭṭef yiwen n uɣerrabu niḍen; yerna dagi ttwasemmant s wudem ur nelli d usrid aṭas n tɣawsiwin timecṭaḥ, akk ttekkant deg yiwet n tɣawsa-agi tameqrant. Ɣef umedya, —deffir n tḍeffer d tuḍḍfa n yiwen n uɣersiw n yilel s lqella n tmussni yerna s lxir, tafekka-nni tezmer ad teffeɣ seg uɣerrabu-nni s ssebba n yiwet n tmeẓẓuɣt n tmeẓẓuɣt; yerna yettfeggiḍ s lbeɛd ɣer tlemmast n yilel, ad t-id-yeṭṭef yiwen n uɛessas wis sin n yilel, d netta, s tɣawla, s tɣawla ara t-id-ijebden ɣer yidis, war ma yella-d lḥeṛs n tudert neɣ n lḥiḍ. S waya, s waṭas ay d-ttbanen yimenɣiyen ay d-yettawin ccwal ed ccwal akk gar yiselman, lemmer ur yelli kra n usaḍuf yettwarun neɣ ur yettwarun, amaḍlan, ur yettwaqbal ara, yerna ad yettwasseqdec deg wakk timsal.

Ahat lqanun unṣib kan n tmeẓdiyin (baleines) ay ɣef yettunefk ttesriḥ s lqanun, d win n Hulanda. D lqanun ay d-sbedden Yiwunak Yeddukklen deg 1695 sdeffir talalit n Lmasiḥ. Maca ɣas ma yella ulac aɣref niḍen ay yesɛan asaḍuf (lqanun) ay d-yettwarun ɣef tmeẓdiyin (baleines), maca iseggaḍen imarikaniyen llan d imseftiyen (législateurs) ed yibugaṭuten-nsen deg temsalt-a. Fkan-d anabaḍ (système) ay iɛeddan ɣef tɣawsiwin n tɣawsiwin n tɣawsiwin n Justinian ed Yilugan n Tdukli Tacinwat n Uḥbas n Tmeẓdiyin ed Tɣawsiwin n Weɣref niḍen. Ih; izerfan-agi zemren ad ttwaqedcen deg yiwen n farthing n Tgellidt Anne, neɣ deg tmeẓẓuɣt n tmeẓẓuɣt, yerna ad ttwaḍeggren deg uqerruy, s waya, d imeẓyanen.

I. A Fast-Fish yettili deg ukabar n tẓallit ɣur-s.

II. A Loose-Fish d urar n lḥeqq i yal yiwen izemren ad t-id-yeṭṭef s tɣawla.

Maca ayen ay yetturaren lbaṭel s lqanun-a (code) n tmussni d aɣezfan-nnes (brevité) ay yesɛan azal, yerna d aya ay yessefken ad d-yili yiwen n wedlis ameqran n tmuɣliwin (commentaires) akken ad t-id-sfehmen.

Tamezwarut: D acu i d Fast-Fish? Iselman yedder neɣ yemmut yettɛedday s tɣawla s tmussni, mi ara yettwaqqen ɣer yiwen n uɣerrabu neɣ ɣer yiwen n lbabur yettwaṭṭfen, s kra n ttawilat ay izemren ad t-ḥekmen win ay t-yeṭṭfen neɣ wid ay t-yeṭṭfen, — d aɣerrabu, d aɣerrabu, d aɣerrabu n tẓa n yisantimetren, d aɣerrabu n tiligraf, neɣ d aɣerrabu n tmeẓẓuɣt n tmeẓẓuɣt, d yiwen kan. Am wakken daɣen iselman ttɣawlen s tmussni (technique) mi ara d-awin waif, neɣ kra n tmeẓri nniḍen yettwasnen n tmeṭṭut; skud akabar ay t-yesbedden yesskanay-d s tɣawla tazmert-nsen deg yal tallit akken ad t-id-awi ɣer yidis, am wakken ay d-yesskanay lebɣi-nsen akken ad t-gen.

Wigi d isefka ussnanen; maca isefka n yilefḍan s timmad-nsen tikwal ttgen-d awalen iqesḥen ed tɣawsiwin iqesḥen ugar – d Coke-upon-Littleton n tmeẓẓuɣt. D tidet, gar yimeɣnasen (baleines) ay d-yesbedden iman-nsen s lḥeqq yerna sɛan cciɛa, ttwaxedmen dima yidrimen i lmend n temsal timezwura, anda ara yili d lbaṭel admayan (moral) ay d-yesnekren ccwal ma yella yiwen n ukabar ad d-yini d akken yesɛa yiwen n uɣersiw n yilel ay d-yeḍfer neɣ yenɣa yiwen n ukabar niḍen zik. Maca wiyaḍ ur llin ara s lḥeqq am wa.

Azal n xemsin n yiseggasen aya seg wasmi ay d-tella yiwet n temsalt (case) n tmeẓdiyin (baleines) ay yettwaḥebsen deg Langliz, anda ay d-sbedden yemcetkiyen-nni d akken deffir ma ḍefren s lqerb yiwen n uɣersiw n yilel deg yilel n Ugafa (nord); yerna asmi ay ssawḍen s tidet (imcetkiyen) ad sqedcen iselman-nni s tmeẓdiyin (harpons); llan taggara, s lxir n tudert-nsen, ttwaḥettmen ad ǧǧen mačči kan lḥiḍ-nsen, maca d aɣerrabu-nsen s timmad-is. Deg tgara, imḥaddiyen (ixeddamen n yiwen n uɣerrabu niḍen) usan-d s tmeẓdit-nni, wten-tt, nɣan-tt, ṭṭfen-tt, yerna ɣer taggara, sqedcen-tt sdat wallen n wid ay d-yessutren. Yerna asmi ay ttwaɛaqben yemḥaddiyen-nni, aqebṭan-nsen yessers iḍarren-nnes deg yiɣes n yemcetkiyen-nni, yerna iɛeqleḍ-asen-d s webrid n tɣawsiwin ay iga, ad yeḥrez tura lḥiḍ-nsen, iɣerruba-nsen ed lbabur-nsen, ay yeqqimen qqnen ɣer tmeẓdit-nni deg tallit n tuḍḍfa-nni. S waya, imcetkiyen-nni ttcetkin tura akken ad d-rren azal n tmeẓdit-nsen, n lḥiḍ, n tmeẓdiyin (harpons) ed lbabur-nsen.

Mass Erskine yella d abugaṭu n yemḥaddiyen-nni; Lord Ellenborough d netta ay d lqaḍi. Deg tegnit n tmenḍawt, Erskine ay yesɛan lḥirfa, ikemmel yesskanay-d amkan-nnes, s tmuɣli-nnes ɣef yiwet n tmenɣiwt ay d-yellan seg melmi kan. kun. lḥal, anda yiwen wergaz, deffir ma yeɛreḍ s lbaṭel ad yeḥbes lbaṭel n tmeṭṭut-is, taggara-yagi yeǧǧa-tt ɣef yilel n tudert ; maca deg yiseggasen, yettub seg uswir-nni, yesbedd-d tigawt akken ad tt-id-yerr. Erskine yella deg tama nniḍen; yerna syen yeɛna-tt s wawal-is, belli ɣas akken argaz-nni deg tazwara yessers-as lḥerma ɣef tmeṭṭut-nni, yerna yella yeẓẓum-itt yiwet n tikkelt, yerna ala s ssebba n lḥers ameqran n lbaṭel-is i d-yeɣlin, maca taggara-agi yeǧǧa-tt ; maca yeǧǧa-tt, akken ad tuɣal d aselman ixuṣṣen; yerna ɣef waya, asmi ay tt-id-yeɛreḍ yiwen n wergaz ay d-yeddan deffir-s, tameṭṭut-nni tuɣal sakkin d ayla n wergaz-nni ay d-yeddan deffir-s, ɣer yidis n wayen akk ay izemren ad tt-id-naf yeqqen deg-s.

Tura deg temsalt-a, Erskine yenna-d d akken imedyaten n tmeẓdit ed tmeṭṭut-nni llan ttbegginen-d gar-asen.

Tiɣriwin-agi, d tiɣriwin n tmuɣli nniḍen, mi slan-tent akken ilaq, lqaḍi-nni yelmed aṭas s tɣawsiwin yettwasbedden, yeqsed, d ayen icudden ɣer lbabur, yefka-t i wid yettcetkin, imi ǧǧan-t kan akken ad sselken tudert-nsen ; maca d akken deg wayen yerzan aɣersiw-nni n yilel, iɣerruba ed lḥiḍ ay ɣef yella wemjadel, llan d ayla n yemḥaddiyen; lbaṭaṭa-nni, imi ay tella d Loose-Fish deg tallit n tuṭṭfa taneggarut; akked yiharpun ed lḥiḍ imi mi ara ffɣen yiselman yid-sen, d netta (iselman) ay d-yewwin ayla deg yimagraden-nni; yerna seg waya, yal yiwen ara d-yeṭṭfen iselman-nni deffir-s, yesɛa azref fell-asen. Tura imḥaddiyen ṭṭfen-d sdeffir-s iselman-nni; ergo, imagraden-nni ay d-nebder llan d ayla-nsen.

Argaz amagnu yettmuqulen deg lqanun-agi n Lqaḍi yelmed aṭas, yezmer ad t-iḥettem. Maca ttwaḥerzen arma d aẓru amezwaru n temsalt-a, sin-nni n yimenzayen imeqranen ay d-yettwabedren deg yiṣ̣eḍrufen n tmeẓdiyin (baleines) n tkebbanit ay d-nebder yagi, yerna sseqdacen-ten yerna sfehmen-ten Lord Ellenborough deg temsalt-a ay d-nebder sdat-a; sin-a n yiṣuḍaf ay iḥuzan Fast-Fish ed Loose-Fish, qqareɣ-d, ad d-naf, ma nxemmem, d llsas n wakk lqanun n wemdan ; acku ɣas akken tesɛa lḥirfa n tmeṣkiwt ixuṣṣen, Taddart n Ccariɛa, am Taddart n Ifilistiyen, tesɛa kan snat n tɣawsiwin ara ibedden fell-asent.

Mačči d awal deg yimi n yal yiwen : Tazmert d azgen n ccaria : dɣa, akken yebɣu yili wamek i d-tusa tɣawsa-nni ɣer tmeṭṭut ? Maca s waṭas, lḥeqq d akk lqanun. D acu ay d iẓuran ed yiṛuḥaniyen n yiqeddacen irusiyen ed yiqeddacen iɣerfanen (républicains) anagar Fast-Fish, aydeg lḥeqq-nnes d akk lqanun? Acu i bab n tmurt n tmeṭṭut-nni n tmeṭṭut d aɛeqqa aneggaru n tmeṭṭut-nni maca d Fast-Fish? Acu yellan dinna ur nettwassen ara lḥebs n lmarmar n uɛdaw s tewwurt-nni i yiwen n waif; D acu-t waya siwa d Iselman Iɛewjen? D acu-t usnefli (discount) n tkerkas ay d-yettawi Murdukay, amesni, seg Woebegone ameɣbun, ameɣbun, ɣef lbanka akken ad teḥrez tawacult n Woebegone seg laẓ ; d acu i d asenqes-nni n tmeɣra maca d Fast-Fish? D acu ay d ayen ay d-yewwi Archibishop n Savesoul n 100.000 n yidinaṛen ay yettwaṭṭfen seg weɣrum d lbaṭaṭa n tmiḍiyin n yimelyunen n yixeddamen ay yesɛan aḍar yettwaqerḍen (akk ttwaɛeqlen s yigenni war ma yella-d kra seg lemɛawna n Savesoul) d acu ay d 100.000 n yidinaṛen-nni n tkurt n uḍar maca d Fast-Fish? D acu-tent temdinin d tɣiwanin n Duk n Dunder ay d-yellan s tmeṭṭut-nnes anagar Fast-Fish? Acu i umseḍru-nni n tmeẓẓuɣt, John Bull, d Irland ameɣbun, maca d Fast-Fish ? Acu i lancer-nni n umceggeɛ, gma-tneɣ Yunatan, d Tiksas maca d Fast-Fish ? U x wayagi meṛṛa, ma war iǧi d řḥeqq n ccaria meṛṛa?

Maca ma yella taɣuri n Fast-Fish tella s umata, taɣuri n Loose-Fish ay d-yeqqnen ɣer-s mazal-itt tettwasqedcem ugar. D ayen yettwasqedcen deg tmura tigraɣlanin d tmura n umaḍal.

D acu n Yunan i Uṭerki? D acu n Lhend ɣer Langliz? D acu ara tili taggara-a Miksik i Yiwunak Yeddukklen? Akk Iselman-Ilelliyen.

Yerna d acu-k, a ameɣri, maca d Iselman-Ilelli d Iselman-Iɛewwiqen, ula d kečč ?





IXEF 90. Iqerra neɣ Iḍarren.

“De balena vero sufficient, si rex habet caput, d regina caudam.” Braktun, l. 3, c. 3.

Talatinit seg yedlisen n Yilugan n Langliz, ay d-yettwaṭṭfen akk d tɣawsa-nni, d tin ay d-yesskanayen, seg wakk iɣerruba ay yettwaṭṭfen sɣur yal yiwen deg tlisa n tmurt-nni, agellid, am Honorary Grand Harpooneer, yessefk ad yesɛu aqerruy, yerna Tagellidt ad as-ttwaqeddmen s leqder s tmeẓẓuɣt-nnes. Tabḍit i yellan, deg yilef, tcudd aṭas ɣer uzgen n tfelwit; ulac ayen i d-yeqqimen d alemmas. Tura imi asaḍuf-a, s talɣa yettwabeddlen, yella ar ass-a deg Langliz ; yerna imi ay d-yettakf deg yiḥricen yemgerraden yiwet n tɣawsa ur nelli d tajḍiḍt (anomalie) ay iḥuzan asaḍuf amatu n Fast and Loose-Fish, yettwabder-d fell-as dagi deg yiwen n weḥric ay d-yellan s timmad-nnes, ɣef yiwen n wemenzay (principe) n tḥeqranit ay d-yettawin iberdan n tmacint n Langliz ad ilin ɣef ljal n yiwet n tkeṛṛust yemgerraden, ay yettwaḥerzen ladɣa i lmend n tnezduɣt n yigensasen (royaume). Deg umkan amezwaru, d ttbut n tmuɣli n tɣawsa-nni ay d-yesskanayen dakken asaḍuf-a ay d-nebder sdat-a mazal-it yettwaḍmen, ttkemmileɣ sserseɣ-d zdat-wen yiwet n liḥala ay d-yeḍran deg sin-a n yiseggasen ineggura.

Tban-d am wakken kra n yimariren n Dover, neɣ n Sandwich, neɣ kra seg yiɣerruba n Cinque, ssawḍen, deffir ma ḍefren-ten s lqerb, ad nɣen yerna ad sneqsen yiwen n uɣersiw n yilel (baleine) yelhan ay d-sfehmen deg tazwara s lbeɛd seg tlisa. Tura, Iɣerruba n Cinque llan s tɣawla neɣ s kra n tɣawsa ddaw n tmehla n yiwen n ṣṣenf n wemsulta neɣ n beadle, umi qqaren Lord Warden. Yeṭṭef lbiruwat srid seg tgelda, ttwaliɣ, akk tadyant n lmizan agellid ɣer wakal n Cinque Port ttɣimin s lxedma-s. Sɣur kra n yimura qqaren-as lbiruwat-agi sinecure. Maca mačči akka. Imi Lord Warden yettwaxdem s lxedma tikwal deg usseqdec n tmeẓdiyin-is (perquisites); llan d ayla-s s umata s lǧehd n fobbing-nni kan fell-asen.

Tura mi ssawḍen yimesnallunen-agi imeɣnasen i d-yesserɣen yiṭij, s yiḍarren-nsen ur nesɛi ara aɛebbuḍ, yerna s tceṭṭiḍin-nsen ttwaḍeggrent s lqedd ɣef yiḍarren-nsen n tmeẓẓuɣt, s lqella n tmussni ssawḍen ad sneqsen iselman-nsen n tfekka s lqedd d lqut, ttwaɛeḍḍlen i yiman-nsen 150 n yidinaren yelhan seg zzit d yiɣes s wazal-nsen ; yerna deg tmuɣli n tmeṭṭut ttṣewwiren lgaz n lqaɛa d tlawin-nsen, d lgaz yelhan d yimeddukal-nsen, ɣef tezmert n tmura-nsen ; sdat n tqacuct yiwen wergaz yelmed aṭas yerna d amasiḥi akk d lxir, s yiwet n tfelwit n Blackstone ddaw ufus-is; yerna yessers-it ɣef uqerru n lbaṭaṭa, yenna-yas: “Ifassen-iw, aselman-agi, a mass-iw, d Fast-Fish, ṭṭfeɣ-t d win n Lord Warden.” S waya, imarikaniyen igellilen deg tmeqbert-nsen n leqder – s tidett d Ingliziyen – ur ssinen ara d acu ara d-inin, ɣlin s lǧehd ttqerriben iqerra-nsen ɣer tama-nsen akk ; deg lawan-nni s lḥir yettmuqul seg lbaṭaṭa ɣer uɛraben. Maca aya ur yesselhu ara tamsalt, ne$ ur yesfeḍ ara ul n wergaz-nni yelmed s lqecc n Blackstone. Taggara, yiwen deg-sen, mi yeɛreḍ aṭas n lweqt ɣef tiktiwin-is, yuɣal d aɛessas n wawal,

“Ɛafak, Mass, anwa i d Lord Warden?”

“Duk.”

“Maca duq ur yesɛi ara kra n lxir deg uḍfar n yiselman-agi?”

“D ayla-s.”

“Nella deg wuguren imeqqranen, d lḥers, d kra n tmeṣkiwt, yerna ma yella ayagi akk ad iruḥ i lfayda n Duk, ur nettaf ara kra ɣef lḥif-nneɣ anagar tibuciḍanin-nneɣ?”

“D ayla-s.”

“Ma yella Duk-nni d ameɣbun aṭas armi ara yettwaḥettem ad yeg lḥal-agi n tkerkas akken ad d-yawi tudert?”

“D ayla-s.”

“Xemmemeɣ ad sfeḍḥeɣ yemma taqdimt yellan deg usu s yiwen n weḥric seg lḥeqq-iw n yilef-agi.”

“D ayla-s.”

“Ma ur yettqadar ara Duk s rbeɛ n lqidar neɣ s uzgen?”

“D ayla-s.”

S yiwen wawal, ttwaṭṭfen yilel-nni yerna ttwazenzen, yerna Grace-nnes Duke n Wellington ay d-yewwin idrimen-nni. S tmuɣli-nni, ma nmuqel-itt s kra n tmuɣliwin, taluft-a tezmer ad tettwaḥseb, s kra n tɣawsa tamecṭuḥt, d tin iweɛren aṭas, yiwen n lmuqeddem n temdint-nni s lḥeqq yura-d yiwet n tebrat i Grace-is, yettɛeggiḍ-it ad yeṭṭef taluft n yimariren-nni ur nesɛi lxir deg lbal-is s lekmal-is. I wacu ay d-yerra Lord Duke-inu deg lsas (snat-nni n tebratin ttwassuffɣent-d) d akken iga yakan aya, yerna yuwey-d idrimen-nni, yerna ad yili d aɛessas ɣef wergaz-nni yettwaɛezlen ma yella ɣer sdat, netta (argaz-nni yettwaɛezlen) ad yugi ad yettekki deg lecɣal n medden niḍen. Ma yella d amɣar-nni mazal-it d ameɣnas, ibedd di tɣemmar n kraḍ n tgeldiwin, ɣef ifassen-is meṛṛa yettḥettem lxir n yimessutur ?

Ad d-yettwaẓer s tɣawla dakken deg temsalt-a, azref ay d-yenna Duk ɣef tmeẓdit-nni d win ay d-yettwasbedden sɣur Uselway. Ilaq ad nesteqsi imir-nni ɣef wacu n umenzay ay d-yessefti Uselway deg tazwara s yizerfan-nni. Asaḍuf s timmad-is yettwasbedd yakan. Maca Plowdon yefka-aɣ-d ssebba-s. Yenna-d Plowdon, aɣersiw-nni n yilel ay yettwaṭṭfen s waya d ayla n Ugellid ed Tgellidt, “ɣef ljal n tmussni-nnes (excellence) ay d-yellan s waṭas.” Yerna sɣur imsefhamen (commentateurs) igerrzen akk, aya ur yelli ara d lḥeqq ay d-yellan deg temsal am tigi.

Maca ayɣer ilaq ad yesɛu Ugellid aqerruy, Tagellidt ad tesɛu taɣect? Ssebba n waya, ay ibugaṭuten!

Deg wedlis-nnes (traité) ɣef “Queen-Gold”, neɣ Queen-pinmoney, yiwen n umeskar aqbur n King’s Bench, yiwen seg-sen William Prynne, yenna-d akka: “Taɣect-nwen d tin n Tgellidt, akken ad tesɛuḍ lḥebs n Tgellidt s yiɣes n yilef.” Tura aya yura-d deg tallit ay deg iɣes n limber aberkan n Greenland neɣ n Right whale yella yettwasseqdac s waṭas deg yiɣes n tmeṭṭut. Maca iɣes-agi ur yelli ara deg tmeẓẓuɣt; yella deg uqerru, d tuccḍa n lḥif i ubugaṭu n tmusni am Prynne. Maca, ma yella Tagellidt d tameṭṭut n yilel, ad tt-id-qeddmen s tmeẓẓuɣt ? Anamek n tmedyazt yezmer ad d-yesbedd dagi.

Llan sin n yiselman igelliden ay sbedden yemyura n lqanun anglizi s ṣṣenf-a – d aɣersiw n yilel ed uɣersiw n tẓegwa (sturgeon); ama d ayla agellid ddaw kra n tlisa, ama d ayen ay d-yettaken s yisem aḥric wis mraw n tedrimt n tgelda n tgelda. Ur ẓriɣ ara belli yella umeskar nniḍen i d-yesmektan tamsalt-a; maca s tmuɣli-nni (inference) iban-iyi-d d akken ilaq ad yettwabḍu uɣersiw n tẓegwa (sturgeon) am wakken ay yettwabḍa uɣersiw n yilel, agellid ad yesɛu aqerruy-nni ay d-yellan s tɣawla yerna d aẓiḍan (élasticité) ay d-yellan i yiselman-nni, yerna, ma yella yettwaḥseb s tmeẓri, yezmer lḥal ad yili d win ay d-yellan s tmedyazt ɣef kra n tɣawsa ay d-yettwaḥesben d tin ay d-yellan s tmeẓẓuɣt.
