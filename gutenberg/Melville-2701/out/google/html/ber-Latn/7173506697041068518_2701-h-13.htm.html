Maca ur yelli ara akka deg taluft-a n wass-a akked yiɣersiwen n Pequod; ɣas akken, akken ad t-id-nini, yal argaz ur nesɛi ara lɛada n tmuɣliwin am tigi, ma yella imuqel ɣer tama-s iḍ-nni, qrib ad d-yini belli ilel-nni n tɣerɣert akk d yiwen n lbaṭaṭa tameqrant, yerna iɣersiwen-nni n tmes d iɣersiwen yellan deg-s.

Dacu kan, mi d-yessers Stubb lḥebs n lbabur deffir ma yekfa imensi-ines ; yerna asmi ay d-yusa, ɣef leḥsab-nnes, Queequeg ed yiwen n umaru n yilel n forecastle ɣef tkeṛṛust, ur d-yelli ara ccwal ameẓyan gar yiɣersiwen-nni n tmes; imi s tɣawla sbedden-d tignatin n ugezzum ɣef yidis, yerna sneqsen kraḍ n tfekkiwin, akken ad d-ssufɣen tifekkiwin ɣezzifen n tafat ɣef yilel n tmeẓẓuɣt, sin-agi n yimariren, s tmeẓẓuɣt-nsen ɣezzifen n tmeẓẓuɣt, ttkemmilen ttnaɣen ur nettḥebbis ara ɣef yiɣersiwen n yilel,* s tmeẓẓuɣt n tmeẓẓuɣt-nsen n wuzzal kan i d-yettbanen deg lqaɛa n yilel-nsen. Maca deg lbaṭel n tfekka n yemḍebbren-nsen ixuṣṣen yerna ttnaɣen, imsellḥen ur zmiren ara dima ad ssiwḍen ɣer tɣawsa-nsen ; yerna aya yessawḍ-d tiwlafin timaynutin n tmeɣra ur nesɛi lqima n weɛdaw. Ttqerriben s lbaṭel, mačči kan deg yiɣes n wayeḍ, maca am tmeẓẓuɣin n tmeẓẓuɣin, ttqerriben, ttqerriben ɣer wid-nsen ; alamma ttbanen-d yiɣes-nni ttwaselḥen tikkelt ɣef tikkelt s yiwen n yimi, ttwafeḍḍen s lqella n tmussni s lǧerḥ-nni i d-yeffɣen. Ula d aya ur yelli ara d ayen kan. Ur yelli ara d lḥeqq ad t-id-tessekcemḍ deg lǧeţţa d lǧennet n yimdanen-agi. Yiwen n ṣṣenf n tudert tagensit (générique) neɣ Pantheiste amzun tettḥulfu deg yiɣes-nsen d yiɣes-nsen s timmad-nsen, deffir ma teffeɣ-d wayen umi nezmer ad as-nini tudert n yal yiwen. Yenɣa-t yerna yettwasfeḍ ɣef tkeṛṛust ɣef ljal n tfekka-nnes, yiwen seg yiɣersiwen-a qrib ad yekkes afus n Queequeg ameɣbun, asmi ay yeɛreḍ ad yeqqen aqerruy-nnes yemmuten n tɣerɣert-nnes n tmenɣiwt.

*Aɣerrabu n tmeẓdiyin (whaling-spade) ay yettwasqedcen i lmend n ugezzum (cutting-in) yettwaxdem s wuzzal ay igerrzen akk; d ayen icudden ɣer tmeqrant n ufus n wergaz i d-yesfeḍḍen; yerna deg talɣa tamatut, yettemcabi ɣer tɣawsa n tẓegwa ay ɣef d-yettunefk yisem-nnes; ala iɣezran-is i d-yesfaydin s lekmal-nsen, yerna taggara-s n ufella tettnerni s waṭas ɣef tin n ddaw. Amrigen-a yettwaḥrez dima s tɣawla akken izemren; yerna mi ara yettwaseqdec yettwasfeḍ tikwal, am tmeẓẓuɣt. Deg tmeẓẓuɣt-is, yettwassekcem yiwen n lḥebs iqesḥen, seg ɛecrin ar tlatin n yiḍarren deg teɣzi, i lmend n tmeẓẓuɣt.

“Queequeg ur t-iḥemmel ara d acu n yillu i t-yeǧǧan ad yili d aɣersiw n yilel”, i d-yenna uɛewwiq-nni, yerfed afus-is s lḥif s afella d wadda; “Arebbi Fejee n zzwaǧ neɣ arebbi n Nantucket; maca de arebbi wat yexdem lbabur ilaq ad yili d yiwen n dam Ingin.”
