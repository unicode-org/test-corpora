IXEF 10. Ameddakel n Bosom.

Mi d-uɣaleɣ ɣer Spouter-Inn seg Tkapelt, ufiɣ Queequeg dinna weḥd-s s waṭas; netta mi yeffeɣ seg Tkapelt uqbel lbaṛaka kra n lweqt. Yella yeqqim ɣef yiwen n lbanka zdat n tmes, iḍarren-is ɣef tmes n tmes, yerna deg yiwen ufus yeṭṭef qrib ɣer wudem-is lmeṣnuɛ-is ameẓyan n yineggaren ; yettmuqul s lǧehd deg udem-is, yerna s tmeẓẓuɣt n lǧak yettḥezziben s lḥir ɣef tmeẓẓuɣt-is, di lawan-nni yettḥemmil iman-is s ubrid-is n tbuciḍant.

Maca imi tura yettwaqleɛ, yessers-d lmeṣnuɛ-nni; yerna ur yeɛjib ara, iṛuḥ ɣer ṭṭabla, yeddem-d yiwen wedlis ameqqran dinna, yerna yessers-it ɣef tɣerɣert-is yebda yettḥessis isebtar s lqanun s lqedd ; deg yal asebter wis xemsin – akken ttxemmimeɣ – yettḥebbis kra n lweqt, yettmuqul s lbaṭel deg tlemmast-is, yerna yettak-d awal i yiwen n usḥissef n tmeẓẓuɣt i d-yesɣezfen aṭas n lweqt. Ad yebdu syin tikkelt nniḍen di xemsin i d-iteddun; tban-d amzun yebda-d deg umḍan amezwaru yal tikkelt, amzun ur yezmir ara ad iḥess ugar n xemsin, yerna ala s umḍan ameqran n xemsin ay d-yettwafernen akk, ay d-yesḥissif lḥir-nnes ɣef waṭas n yisebtar.

S lferḥ ameqqran qqimeɣ ttwaliɣ-t. Ɣas akken yella d aɛewwiq, yerna yettwaḥbes s lḥir ɣef wudem-is – ma ulac ɣef leḥsab-iw – udem-is mazal-it yesɛa kra deg-s ur t-iɛeǧba ara s kra n tɣawsa. Ur tezmireḍ ara ad teffreḍ ṛṛuḥ. S akk tiwwura-ines ur nelli ara d tid n tmurt, ɣileɣ ẓriɣ iḍrisen n wul s tidett afraray; yerna deg wallen-is timeqqranin, lqayen, d tiberkanin am tmes yerna d tizegzawin, ttbanent-d tmitar n yiwen n rruê ara iɛewwlen alef n cciṭan. Yerna s lmendad n wayagi meṛṛa, yella kra n lḥeqq n tmeṭṭut n Upagan, ula d lbaṭel-is ur yezmir ara ad t-yesfeḍ akk. Yettban-d am wergaz ur nxeddem ara lḥir yerna ur yesɛi ara win ara t-iɛawnen. Ma yella ula d netta, aqerruy-is yettwaqleɛ, taɣect-is tettwakkes-d s tmeẓẓuɣt tilellit ugar, yerna tettban-d d tin i d-yesfaydin ugar n wayen ara t-id-yesfehmen ma ulac, aya ur t-ttḥeqriɣ ara ad t-id-sfehmeɣ ; maca s tidett d akken aqerruy-is yella d win igerrzen s tfranologie. Ahat ad yili d ayen yessewhamen, maca yesmekta-yi-d s uqerru n Jeneral Washington, akken ay d-yettwaẓer deg tbuciḍant-nnes (bustes) tiɣelnawin. Tesɛa yiwen n usqif ɣezzifen n tɣawsiwin s lqanun seg nnig n tɣawsiwin, d tid ay d-yettbanen daɣen aṭas, am snat n tɣawsiwin ɣezzifen aydeg llant tẓegwa s tɣawla ɣef tqacuct. Queequeg yella d George Wacinṭun ay d-yesnulfa s tmeṭṭut-nnes.

Mi lliɣ ttmuquleɣ-t akka s lqerb, ttgen-iyi-d azgen deg lawan-nni amzun ttmuquleɣ ɣer tmeẓẓuɣt seg lqaɛa, ur iḥemmel ara tilin-iw, ur iḥemmel ara iman-is ula d yiwet n tmuɣli ; maca iban-d d akken yeḥbes akk lḥess n yisebtar n wedlis-nni yessewhamen. Ma ttxemmimeɣ amek i nella neṭṭes s tmetti iḍ-nni yezrin, yerna ladɣa ma ttxemmimeɣ ɣef ufus n lḥir i d-ufiɣ yettwaḍegger fell-i mi d-ffɣeɣ sbeḥ, ttwaliɣ d akken lbaṭel-agi n tmeṭṭut-is d ayen igerrzen aṭas. Maca iɣersiwen d imdanen iberraniyen; tikwal ur teẓriḍ ara amek ara ten-teṭṭfeḍ akken iwata. Di tazwara ttqerriben; usdukkel-nsen n yiman s lḥir n tfehlit iban-d d lḥeqq n Socrate. Ssneɣ daɣen belli Queequeg ur yettmeslay ara akk, neɣ drus kan, d yimariren nniḍen yellan deg tḥanut-nni. Ur yexdim ara kra n tmuɣliwin akken yebɣu yili; iban-d ur yesɛi ara lebɣi ad yessemɣer lḥirfa n wid i t-yessnen. Aya akk yeɣli-d fell-i amzun d yiwen ijehden; maca, s tmuɣli tis snat, yella deg-s kra qrib ad yili d ayen igerrzen. Ha-t-an yiwen wergaz azal n ɛacrin n yimelyunen n yikilumitren seg wexxam, s webrid n Cape Horn, dɣa – d abrid kan i s-yezmer ad yaweḍ ɣer din – yettwaḍegger gar yemdanen amzun d aɛraben fell-as amzun yella deg uplanet n Jupiter ; yerna ɣas akken iban-d d akken yeḥbes akk; ḥerzen ɣef tɣellist (sécurité) tameqrant; yeqbel s tdukli-ines; dima d amsasi d yiman-is. S tidet, aya yella d aḥric n tfelsafit yelhan; ɣas akken ulac ccek ur yesli ara yella wayen yecban aya. Maca, ahat, akken ad nili d ifelsafen n tidet, nekni s yemdanen ur ilaq ara ad nesɛu lḥeqq ad nedder akka neɣ ad nḥareb akka. Ihi akken kan ara sliɣ belli argaz am wa neɣ am wa yettak-d iman-is i yiwen n ufelsuf, ttkemmileɣ belli, am tmeṭṭut-nni tameqqrant n tmeẓẓuɣt, ilaq-as “yeqceɛ aɛebbuḍ-is.”

Akken qqimeɣ dinna deg texxamt-nni tura d taḥeqranit; timest tettqerrib s lqedd, deg tallit-nni n tmeẓẓuɣt mi ara, deffir ma tesseḥmu tmeẓẓuɣt-is tamezwarut lḥamu, imiren tettfeǧǧiǧ kan akken ad tt-id-muqlen ; tifelwiyin n tmeddit d yifecka n tmeddit ttnejmaɛen ɣef tḥuna, yerna ttmuqulen-aɣ-d s snat n tɣawsiwin ssusmen, s yiwen ; tbuciḍant tettbubbu mebla deg tmeɣriwin n tmeɣra; Bdiɣ ttḥulfuɣ s tḥulfawin tiberraniyin. Ḥulfaɣ s lfeṭṭa deg-i. Ur tt-id-yezzi ara ul-iw i d-yesfeḍḍen d ufus-iw i d-yesfeḍḍen ɣer umaḍal n wuccen. Aḥerfi-agi yessefsayen yella yefdi-t. Dinna i yeqqim, lbaṭel-is s timmad-is yettmeslay-d ɣef yiwen n ugama ideg ur tt-id-yettaf ara lmunafiq n tɣerma d tkerkas n tmeẓẓuɣt. Wild yella; a s waṭas n tmuɣli n tmuɣliwin ad nwali; maca bdiɣ ttḥulfuɣ iman-iw ttwajebdeɣ s lbaḍna ɣer-s. Yerna tiɣawsiwin-nni kan ara yessufɣen amur ameqran seg wiyaḍ, d nutni ay d imagneten ay iyi-d-ijebden akka. Ad ɛerḍeɣ ameddakel n tbuciḍant, i d-nniɣ deg wul-iw, imi lxir amasiḥi yessken-d maca lḥeṛma n tmeẓẓuɣt. Jebdeɣ-d lbanka-inu ɣer tama-s, sxedmeɣ kra n tmuɣliwin d tmuɣliwin n tmeddukal, ttgenɣ akk ayen umi zemreɣ akken ad ssiwleɣ yid-s di lawan-nni. Di tazwara drus i d-yesmekta s tmuɣliwin-agi; maca tura, mi d-nniɣ ɣef tmeɣriwin-is n yiḍ yezrin, yeffeɣ-d ad iyi-d-yesteqsi ma yella ad nuɣal d imdukal n yiḍes. Nniɣ-as ih; anda ɣileɣ yettban-d d akken yefṛeḥ, ahat yettwasnemmer cwiṭ.

Nesbedd-d syin adlis-nni akk, yerna ɛerḍeɣ ad as-sfehmeɣ iswi n tɣuri-nni, d unamek n kra n tugniwin yellan deg-s. Akka ur ɛeṭṭleɣ ara sqedceɣ lfayda-s; yerna seg waya ay nedda ɣer tɣawsiwin yelhan ay nezmer ɣef tɣawsiwin yemgerraden n berra ara d-nwali deg temdint-a tamesbaɣurt. Cwi kan ssutureɣ-d ddexxan anmetti; yerna, yessufeɣ-d tasekkurt-is d tomahawk, yefka-yi-d s tɣawla yiwen n usfugel. Yerna syen neqqim nettbeddilen tifednin seg lbumbat-nni-ines n tmeqbert, yerna nettḥadar-itt yal ass ad tezger gar-aneɣ.

Ma yella mazal-it yella kra n ugris n tmuɣli ur nesɛi lḥeqq ɣur-i deg udmawen n Pagan, dexxan-agi igerrzen, n tmeṭṭut n tmeṭṭut i nesɛa, ur tɛeṭṭel ara ad t-yesfeḍ, yerna yeǧǧa-aɣ-d d iɛeggalen. Iban-d amzun yettawi-yi-d s tbaɣurt (nature) d lbaṭel am nekk ɣur-s; yerna mi yekfa ddexxan-nneɣ, iḥettem-d taɣect-is ɣef tin-iw, iḥettem-iyi-d ɣef tɣerɣert, yenna-yi-d belli seg wass-a d asawen ad nezweǧ ; anamek-is, deg tefyirt n tmurt-is, belli nella d imdukal n tfekka; Ad yemmet fell-i s lferḥ, ma yella ilaq. Deg yiwen n wemdan n tmurt, lḥemla-agi n tmeddukal s tɣawla ad d-tban d akken ur tesɛi ara lawan, d ayen ur nesɛi laman aṭas; maca deg tmeqbert-agi tafrarayt ur ttwaxdamen ara yilugan-nni iqburen.

Deffir imensi, d tmesliwt nniḍen n tmetti d tmes, nruḥ akk ɣer texxamt-nneɣ. Netta igga ayi d tarzeft n uzeif nnes ittwasfeḍen; yessufeɣ-d tafelwit-is tameqrant n tbaqut, yerna yettḥezzib ddaw tbaqut, yessufeɣ-d azal n tlatin n yidulaṛen n lfeṭṭa; syin yessers-iten ɣef ṭṭabla, yerna yebḍa-ten s tmeṣkiwt ɣef sin yiḥricen yemxalafen, iḥettem yiwen deg-sen ɣer tama-w, yenna-d d ayla-w. Nekk lliɣ ttedduɣ ad s-d-sfehmeɣ; maca yessusem-iyi s usfugel-nsen deg yijebbaden n tkerrusin-iw. Ǧǧiɣ-ten ad qqimen. Yedda syin ɣer tẓallit-is n tmeddit, yekkes-d lmeṣnuɛ-is, yekkes-d lfeṭṭa n lkarḍa. S kra n tmitar d tmitar, ɣileɣ iban-iyi-d d akken yettḥulfu s lḥir akken ad t-id-sdukkleɣ; maca imi ẓriɣ mliḥ d acu ara d-yeḍrun, sqedceɣ kra n lweqt ma yella, ma yella ad iyi-d-iɛreḍ, ad ḍefren neɣ ala.

Nekk lliɣ d amasiḥi yelhan; ilul yerna yettwasnerna deg tfekka n Tmezgida Tapṛesbitrit ur nesɛi lxilaf. Amek ihi ara ddukkleɣ d uɛebban-agi n lmeṣnuɛat n lbaṭel deg uɛbad n tfekka-ines n useklu? Maca d acu i d leɛbada? xemmem I. Tettwaliḍ tura, a Ismaɛil, belli Illu ameqqran n igenwan d lqaɛa – iɛewwiqen d wid akk yellan deg-sen – yezmer ad iḥess ɣef yiwen n lḥebs n useklu aberkan ur nesɛi ara azal? D awezɣi! Maca d acu i d leɛbada?—ad txedmeḍ lebɣi n Ṛebbi— d leɛbada. Yerna d acu i d lebɣi n Rebbi?—ad xedmeɣ i umdakkel-iw ayen bɣiɣ ad iyi-yexdem umdakkel-iw— d lebɣi n Rebbi. Tura, Queequeg d amdakkel-iw. Yerna d acu i ttmenniɣ ad iyi-d-yexdem Queequeg-agi ? Ayɣer, dduklen yid-i deg ṣṣenf-inu n leɛbad n Presbyterian s timmad-is. S waya, ilaq imiren ad sdukkleɣ yid-s deg win-is; ergo, ilaq ad uɣaleɣ d aɛebban n lmeṣnuɛat. Ihi sxeṣreɣ lḥenni; ɛawnen ad sbeɛden lmeṣnuɛ-nni ameẓyan ur nesɛi lḥeqq; t-id-yefka lbiskwit yeččuren d Queequeg; salamed zdat-s snat neɣ tlata n tikkal; yessuden anzaren-is; yerna aya yexdem, nekkes-d lqecc-nneɣ, nruḥ ɣer usu, di talwit d tnemmast-nneɣ d ddunit meṛṛa. Maca ur neddu ara ad neṭṭes mebla kra n tmeslayt tamecṭuḥt.

Amek i tella ur ẓriɣ ara; maca ulac amkan am usu i tmuɣliwin n tuffra gar yimeddukal. Argaz d tmeṭṭut, qqaren, dinna ldin lqaɛ n yiman-nsen i wayeḍ; yerna kra n yimezdaɣen iqburen ttkellixen s waṭas yerna ttmeslayen ɣef tallit taqburt armi d qrib ad d-yeffeɣ ṣṣbeḥ. Akka, ihi, deg wayyur n tmeɣra n ulawen-nneɣ, nekk d Queequeg llan ttnadin fell-aɣ — d sin n yirgazen igerrzen yerna ḥemmlen.





IXEF WIS 11. Taqcict n yiḍ.

Nella neqqim akka deg usu, nettmeslay yerna nettqerrib s tmeqbert, yerna Queequeg tura s lḥir yettḍeggir iḍarren-is iqehwan s tmeqbert ɣef wid-iw, syin yettjebbed-iten ɣer deffir ; akka i nella s lekmal-nneɣ d imettiyen yerna d ilelliyen yerna d isehlen; mi, taggara, s ssebba n tmucuha-nneɣ, ayen n tmeẓẓuɣt tamecṭuḥt i d-yeqqimen deg-neɣ yeffeɣ-d s lekmal-is, yerna neḥsa belli nebɣa ad d-nekker tikkelt nniḍen, ɣas akken lḥebs n wass mazal-it kra n tɣawsa ɣer sdat.

Ih, nuɣal d iɛewwiqen aṭas; armi d asmi yebda lḥal-nneɣ n tɣerɣert yettnerni, yerna cwiṭ cwiṭ nufa-d iman-nneɣ qqimen ; lqecc-nni yettwasfeḍ mliḥ fell-aɣ, yettkemmil ɣef tfelwit n uqerruy s ṛebɛa n yiḍarren-nneɣ ttwajerden qrib ɣer yiwen, yerna sin n yiḍarren-nneɣ ttwaḍeggren fell-asen, amzun d iḍarren-nneɣ i d-yesḥemmlen. Neḥsa s lxir aṭas yerna s lḥir, ugar imi d asemmiḍ aṭas berra n tewwura; s tidett seg lqecc n wusu ula d netta, imi ur telli ara tmes deg texxamt. Ugar n waya, qqareɣ-d, imi s tidet akken ad tesɛuḍ lḥemla n tfekka, ilaq kra n weḥric ameẓyan deg-k ad yili d asemmiḍ, imi ulac ṣṣifa deg umaḍal-agi ur nelli ara d ayen yellan s tmuɣli kan. Ulac ayen yellan deg yiman-is. Ma yella tḥemmleḍ iman-ik belli tḥemmleḍ akk, yerna tesɛiḍ aṭas n lweqt, ihi ur nezmir ara ad d-nini tḥemmleḍ s waṭas. Maca ma yella, am nekk d Queequeg deg usu, tmeẓẓuɣt n uqerruy-ik neɣ n tmeẓẓuɣt n uqerruy-ik tessemḍeḍ cwiṭ, ayɣer ihi, s tidet, deg tmusni tamatut ay tḥulfaḍ s lferḥ ameqran akk yerna s lḥir ur nesɛi lmeɛna. S ssebba-a, ur yessefk ara ad yettwasbedd uxxam n yiḍes s tmes, d yiwen seg wuguren n lxir n yimerkantiyen. Imi lqedd n ṣṣenf-agi n tmeẓẓuɣt d akken ur tesɛiḍ ara siwa lḥemla gar-ak d tmeẓẓuɣt-ik d usemmiḍ n lhawa n berra. Syin dinna tettḍeggiɛeḍ am tmeẓẓuɣt-nni kan iḥemmlen deg wul n lkristal n tmurt n Arktik.

Neqqim s tmuɣli-agi n tkerkas acḥal aya, mi d-nniɣ deg yiwet n tikkelt ad ldiɣ allen-iw; imi mi ara iliɣ gar tfelwiyin, ama deg wass neɣ deg yiḍ, ama deg yiḍes neɣ deg tfekka, sɛiɣ abrid n usḥulfu dima n wallen-iw, akken ad sqedceɣ ugar lḥir n tmeṭṭut deg usu. Imi ulac amdan izemren ad iḥulfu s tmagit-is s lḥeqq siwa ma yella ttwaqqnent wallen-is; amzun d tidett d ṭṭlam i d aferdis iwatan n tɣawsiwin-nneɣ, ɣas akken tafat tettqabal ugar aḥric-nneɣ n tẓegwa. Mi ldiɣ allen-iw imir-nni, yerna ffɣeɣ-d seg ṭṭlam-iw igerrzen yerna snulfaɣ-d iman-iw ɣer ṭṭlam n berra yettwaḥettmen yerna d aɛewwiq n tnac n tmeddit ur nesɛi tafat, ḥulfaɣ s lḥir ur nḥemmel ara. Ula d nekk ur ttqadareɣ ara akk lḥeqq n Queequeg ay d-yennan ahat d ayen yelhan ma neṭṭef tafat, imi ay nella nekkni s lqedd ameqran; yerna s zyada iḥulfa s lebɣi ameqqran akken ad yesɛu kra n tmeẓẓuɣin tisusmiwin seg Tomahawk-is. Ad d-nini, ɣas akken ḥulfaɣ s lḥir ijehden am wa ɣef tmes-is deg usu iḍ uqbel, maca ẓer amek ttnernin yiḥezziben-nneɣ iqesḥen mi ara d-tas tayri yiwet n tikkelt ad ten-teɛqel. Imir-a ur iyi-iɛeǧba ara wayen yelhan akk-n ad yili Queequeg yettḍeggir-iyi, ula deg usu, acku iban-d amzun yeččur d lferḥ n uxxam s lḥir am wa imir-nni. Ur ḥsiɣ ara s lḥir ur nesɛi lmeɛna ɣef tsertit n tḍmant n bab n wexxam. Nekk lliɣ ddreɣ kan s lxir n tuffra n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt d yiwen n umeddakel n tidet. S tcekkaṛin-nneɣ n tmeẓẓuɣt i d-nejbed ɣef yiḍarren-nneɣ, neɛḍel tura ɣef Tomahawk seg yiwen ɣer wayeḍ, armi d asmi s tɣawla i d-yennerna fell-aɣ yiwen n uɛessas n dexxan i d-yeqqnen s tmeẓẓuɣt, i d-yesfeḍḍen s tmeẓẓuɣt n tmeẓẓuɣt i d-yesfeḍḍen s tmeẓẓuɣt tamaynut.

Ma yella d akken aɛessas-agi n tmeẓẓuɣt i d-yesfeḍḍen aɛewwiq-nni ɣer tmuɣliwin ibeɛden aṭas, ur ẓriɣ ara, maca tura yessawel-d ɣef tegzirt-is n tmurt ; yerna, s lebɣi ad sleɣ i umezruy-is, ssutreɣ-as ad ikemmel ad t-id-yini. Yeḍfer-it s lfeṛḥ. Xas akken di tallit-nni ur fhimeɣ ara kra seg wawalen-is, maca tiwtilin i d-yeḍran sdeffir-s, asmi lliɣ ttḥulfuɣ ugar s tferdiwin-is i d-yeqqnen, ssawḍent-iyi tura ad d-ssumreɣ tamacahut-nni akk akken tezmer ad d-tessekneḍ deg tfekka kan i d-ttakeɣ.





IXEF WIS 12. Tasekla n tmeddurt.

Queequeg yella d aɣerman n Rokovoko, d tigzirt ay ibeɛden ɣef Utaram (lɣerb) ed Wenẓul. Ur yelli ara ddaw deg kra n tkarḍa; Imḍiqen n tidet ur llin ara.

Mi ara yili yiwen n uɛewwiq i d-yeffɣen s tmeẓẓuɣt tamaynut yettazzal s tmeẓẓuɣt ɣef tẓegwa-ines n tmurt deg yiwen n usqif n tẓegwa, ḍefren-t-id yiɣersiwen yettqerriben, amzun d aẓekka azegzaw; ula d asmi ay d-yella waya, deg ṛṛuḥ n Queequeg ay yesɛan lebɣi, tella tettḥulfu s lebɣi ijehden akken ad iẓer kra n wayen ay d-yellan ugar n tmasiḥit n yiwen n uɣersiw n yilel (baleine) n tmuɣli neɣ sin. Baba-s d Aqerru Aɛlayan, d Agellid; xali-s d Ameqqran n Ukehhan; yerna seg tama n yemma-s yettfaṛas xalitin yellan d tilawin n yimenɣayen ur nesɛi lḥeqq. Yella idamen ifazen deg yiẓuran-is — d tiɣawsiwin n ugellid; ɣas akken s lḥif, ttgaḍeɣ, s tmuɣli n tmeɣra n yemdanen ay d-yesfaydi deg temẓi-nnes ur yettwasselmed ara.
