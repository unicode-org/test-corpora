CAPITULO 2. Nopa alfombra-bolsa.

Nijtlalik se o ome camisa ipan no bolsa tlen uejkajkia, nijtlalik tlatsintla tlen noma, uan ​​nipejki nias ipan Cabo Horn uan Pacífico. Kej nopa miakej telpokamej tlen kinekij kiselisej nopa tlaijiyouilistli uan tlatsakuili tlen kichijtokej ika ballenas moketsa ipan ni New Bedford, uan ​​nopaya peuas inin viaje, uelis nojkia moijtos tlen na, pampa se, amo nijmatiyaya tleno nijchiuaskia. Pampa noyolo nimosentlalik ninejnemis ipan se barco tlen Nantucket, pampa onkayaya se tlamantli tlen kuali uan tlen tlauel chikauak tlen nochi tlen mosalojtoya ika nopa ueyi tlali tlen uejkajkia, tlen tlauel nechpaktiyaya. Uan kanke amo tlen Nantucket, nojkia, nopa achtoui pil barco tlen tlauel ixtlamatki, se kentsi temitoya ika tetl tlen kiuikayaya tlen seyok tlali —kej nopa kiijtoua nopa tlapoualistli— para kinmajkauis nopa ballenas, pampa kinekiyaya kimatis kemaj nechka eltoyaj uan ueliskia kiuikasej se arpón tlen nopa bowsprit?

Axkan pampa nijpixtoya se youali, se tonali uan seyok youali tlen nechtoquiliyaya ipan New Bedford, kema ayamo ueliyaya nias ipan nopa puerto tlen nijnekiskia, mochijki se tlamantli tlen nechtekipachouayaya kanke monekiyaya nitlakuas uan nikochis ipan nopa tonali. Ika se tlajko tlajko, ninejnemiyaya ipan calles, uan ​​nijpanoyaya nopa letrero tlen itoka “The Crossed Harpoons” —pero nesiyaya tlauel patiyo uan tlauel yolpakiyaya nopaya.
