*** จบโครงการกูเทนเบิร์ก อีบุ๊ก โมบี้ ดิก หรือ ปลาวาฬ ***
รุ่นที่อัปเดตจะเข้ามาแทนที่รุ่นที่แล้ว โดยรุ่นเก่าจะเปลี่ยนชื่อ
การสร้างผลงานจากฉบับพิมพ์ที่ไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา หมายความว่าไม่มีใครเป็นเจ้าของลิขสิทธิ์ของสหรัฐอเมริกาในผลงานเหล่านี้ ดังนั้นมูลนิธิ (และคุณ!) จึงสามารถคัดลอกและเผยแพร่ผลงานเหล่านี้ในสหรัฐอเมริกาโดยไม่ได้รับอนุญาตและไม่ต้องจ่ายค่าลิขสิทธิ์ กฎพิเศษที่ระบุไว้ในส่วนเงื่อนไขการใช้งานทั่วไปของใบอนุญาตนี้ใช้กับการคัดลอกและเผยแพร่ผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ เพื่อปกป้องแนวคิดและเครื่องหมายการค้าของ PROJECT GUTENBERG™ Project Gutenberg เป็นเครื่องหมายการค้าจดทะเบียน และห้ามใช้หากคุณเรียกเก็บเงินสำหรับ eBook ยกเว้นจะปฏิบัติตามเงื่อนไขของใบอนุญาตเครื่องหมายการค้า รวมถึงการจ่ายค่าลิขสิทธิ์สำหรับการใช้เครื่องหมายการค้าของ Project Gutenberg หากคุณไม่เรียกเก็บเงินใดๆ สำหรับสำเนา eBook นี้ การปฏิบัติตามใบอนุญาตเครื่องหมายการค้าก็ง่ายมาก คุณสามารถใช้ eBook นี้ได้เกือบทุกวัตถุประสงค์ เช่น การสร้างผลงานดัดแปลง รายงาน การแสดง และการวิจัย eBook ของ Project Gutenberg สามารถดัดแปลง พิมพ์ และแจกจ่ายได้ คุณสามารถทำอะไรก็ได้จริงในสหรัฐอเมริกาด้วย eBook ที่ไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา การแจกจ่ายซ้ำต้องเป็นไปตามใบอนุญาตเครื่องหมายการค้า โดยเฉพาะการแจกจ่ายซ้ำเชิงพาณิชย์
เริ่มต้น: ใบอนุญาตเต็มรูปแบบ

ใบอนุญาตโครงการ GUTENBERG ฉบับเต็ม

กรุณาอ่านสิ่งนี้ก่อนที่คุณจะเผยแพร่หรือใช้ผลงานนี้
เพื่อปกป้องภารกิจของ Project Gutenberg™ ในการส่งเสริมการแจกจ่ายผลงานอิเล็กทรอนิกส์ฟรี โดยการใช้หรือการแจกจ่ายผลงานนี้ (หรือผลงานอื่นใดที่เกี่ยวข้องในทางใดก็ตามกับวลี “Project Gutenberg”) คุณตกลงที่จะปฏิบัติตามข้อกำหนดทั้งหมดของใบอนุญาต Project Gutenberg™ ฉบับเต็มซึ่งมีอยู่ในไฟล์นี้หรือทางออนไลน์ที่ www.gutenberg.org/license
ส่วนที่ 1 เงื่อนไขการใช้งานทั่วไปและการแจกจ่ายผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™
1.A. การอ่านหรือใช้ส่วนใดส่วนหนึ่งของผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ นี้ ถือเป็นการแสดงว่าคุณได้อ่าน เข้าใจ ตกลง และยอมรับเงื่อนไขทั้งหมดของใบอนุญาตและข้อตกลงทรัพย์สินทางปัญญา (เครื่องหมายการค้า/ลิขสิทธิ์) ฉบับนี้แล้ว หากคุณไม่ตกลงที่จะปฏิบัติตามเงื่อนไขทั้งหมดของข้อตกลงนี้ คุณต้องหยุดใช้และส่งคืนหรือทำลายสำเนาผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ทั้งหมดที่คุณมี หากคุณจ่ายค่าธรรมเนียมในการรับสำเนาหรือการเข้าถึงผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ และคุณไม่ตกลงที่จะผูกพันตามเงื่อนไขของข้อตกลงนี้ คุณอาจขอคืนเงินจากบุคคลหรือหน่วยงานที่คุณจ่ายค่าธรรมเนียมตามที่ระบุไว้ในวรรค 1.E.8
1.B. “Project Gutenberg” เป็นเครื่องหมายการค้าจดทะเบียน สามารถใช้หรือเชื่อมโยงกับงานอิเล็กทรอนิกส์ได้เฉพาะบุคคลที่ตกลงผูกพันตามข้อตกลงนี้เท่านั้น มีบางอย่างที่คุณสามารถทำได้กับงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ส่วนใหญ่ แม้จะไม่ได้ปฏิบัติตามเงื่อนไขทั้งหมดของข้อตกลงนี้ก็ตาม โปรดดูย่อหน้า 1.C ด้านล่าง มีหลายอย่างที่คุณสามารถทำได้กับงานอิเล็กทรอนิกส์ของ Project Gutenberg™ หากคุณปฏิบัติตามเงื่อนไขของข้อตกลงนี้และช่วยรักษาการเข้าถึงงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ในอนาคตได้โดยเสรี โปรดดูย่อหน้า 1.E ด้านล่าง
1.C. มูลนิธิ Project Gutenberg Literary Archive Foundation (“มูลนิธิ” หรือ PGLAF) เป็นเจ้าของลิขสิทธิ์ในการรวบรวมผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ผลงานแต่ละชิ้นเกือบทั้งหมดในคอลเลกชันอยู่ในโดเมนสาธารณะในสหรัฐอเมริกา หากผลงานแต่ละชิ้นไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ในสหรัฐอเมริกาและคุณอยู่ในสหรัฐอเมริกา เราจะไม่ขอสิทธิ์ในการป้องกันไม่ให้คุณคัดลอก แจกจ่าย แสดง แสดง หรือสร้างผลงานดัดแปลงจากผลงานดังกล่าว ตราบใดที่ลบการอ้างอิงถึง Project Gutenberg ทั้งหมดออกไป แน่นอนว่าเราหวังว่าคุณจะสนับสนุนภารกิจของ Project Gutenberg™ ในการส่งเสริมการเข้าถึงผลงานอิเล็กทรอนิกส์โดยเสรีด้วยการแบ่งปันผลงาน Project Gutenberg™ อย่างเสรีตามข้อกำหนดของข้อตกลงนี้เพื่อรักษาชื่อ Project Gutenberg™ ที่เกี่ยวข้องกับผลงาน คุณสามารถปฏิบัติตามข้อกำหนดของข้อตกลงนี้ได้อย่างง่ายดายโดยเก็บผลงานนี้ในรูปแบบเดียวกันกับใบอนุญาต Project Gutenberg™ ฉบับเต็มที่แนบมาเมื่อคุณแบ่งปันผลงานดังกล่าวกับผู้อื่นโดยไม่เสียค่าใช้จ่าย
1.D. กฎหมายลิขสิทธิ์ของสถานที่ที่คุณอยู่ยังกำหนดสิ่งที่คุณสามารถทำกับงานนี้ได้ด้วย กฎหมายลิขสิทธิ์ในประเทศส่วนใหญ่มีการเปลี่ยนแปลงอยู่ตลอดเวลา หากคุณอยู่ภายนอกสหรัฐอเมริกา โปรดตรวจสอบกฎหมายของประเทศของคุณนอกเหนือจากข้อกำหนดของข้อตกลงนี้ก่อนดาวน์โหลด คัดลอก แสดง ดำเนินการ แจกจ่าย หรือสร้างผลงานดัดแปลงจากงานนี้หรือผลงานอื่นๆ ของ Project Gutenberg™ มูลนิธิไม่รับรองสถานะลิขสิทธิ์ของผลงานใดๆ ในประเทศอื่นนอกเหนือจากสหรัฐอเมริกา
1.E. เว้นแต่คุณจะลบการอ้างอิงทั้งหมดถึง Project Gutenberg:
1.E.1 ประโยคต่อไปนี้พร้อมลิงก์ที่ใช้งานได้หรือการเข้าถึงโดยตรงอื่นๆ ไปยังใบอนุญาต Project Gutenberg™ ฉบับเต็มจะต้องปรากฏอย่างเด่นชัดทุกครั้งที่มีการเข้าถึง แสดง ดำเนินการ ดู คัดลอก หรือแจกจ่ายสำเนาผลงาน Project Gutenberg™ (ผลงานใดๆ ที่มีวลี "Project Gutenberg" ปรากฏอยู่ หรือที่มีวลี "Project Gutenberg" เกี่ยวข้องด้วย)
eBook เล่มนี้เปิดให้ทุกคนใช้งานได้ฟรีทุกที่ในสหรัฐอเมริกาและส่วนอื่นๆ ของโลก โดยไม่มีค่าใช้จ่ายและไม่มีข้อจำกัดใดๆ ทั้งสิ้น คุณสามารถคัดลอก แจกจ่าย หรือใช้งานซ้ำได้ภายใต้เงื่อนไขของใบอนุญาต Project Gutenberg ที่รวมอยู่ใน eBook เล่มนี้ หรือทางออนไลน์ที่ www.gutenberg.org หากคุณไม่ได้อยู่ในสหรัฐอเมริกา คุณจะต้องตรวจสอบกฎหมายของประเทศที่คุณอยู่ก่อนใช้ eBook เล่มนี้
1.E.2 หากผลงานอิเล็กทรอนิกส์ Project Gutenberg™ แต่ละชิ้นได้มาจากข้อความที่ไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา (ไม่มีข้อความแจ้งว่าได้รับการโพสต์โดยได้รับอนุญาตจากผู้ถือลิขสิทธิ์) ผลงานดังกล่าวสามารถคัดลอกและเผยแพร่ให้กับบุคคลใดๆ ในสหรัฐอเมริกาได้โดยไม่ต้องเสียค่าธรรมเนียมหรือค่าบริการใดๆ หากคุณกำลังเผยแพร่ซ้ำหรือให้สิทธิ์เข้าถึงผลงานที่มีวลี “Project Gutenberg” เกี่ยวข้องหรือปรากฏอยู่ในผลงานดังกล่าว คุณต้องปฏิบัติตามข้อกำหนดในวรรค 1.E.1 ถึง 1.E.7 หรือขออนุญาตใช้ผลงานและเครื่องหมายการค้า Project Gutenberg™ ตามที่ระบุไว้ในวรรค 1.E.8 หรือ 1.E.9
1.E.3 หากโพสต์งานอิเล็กทรอนิกส์ Project Gutenberg™ แต่ละรายการโดยได้รับอนุญาตจากผู้ถือลิขสิทธิ์ การใช้งานและการแจกจ่ายของคุณจะต้องเป็นไปตามทั้งย่อหน้า 1.E.1 ถึง 1.E.7 และเงื่อนไขเพิ่มเติมใดๆ ที่ผู้ถือลิขสิทธิ์กำหนด เงื่อนไขเพิ่มเติมจะเชื่อมโยงกับใบอนุญาต Project Gutenberg™ สำหรับผลงานทั้งหมดที่โพสต์โดยได้รับอนุญาตจากผู้ถือลิขสิทธิ์ซึ่งระบุไว้ในตอนต้นของผลงานนี้
1.E.4 ห้ามยกเลิกการเชื่อมโยง แยก หรือลบข้อกำหนดสิทธิ์การใช้งาน Project Gutenberg™ ฉบับเต็มจากงานนี้ หรือไฟล์ใดๆ ที่ประกอบด้วยส่วนหนึ่งของงานนี้ หรืองานอื่นๆ ที่เกี่ยวข้องกับ Project Gutenberg™
1.E.5 ห้ามคัดลอก แสดง ดำเนินการ แจกจ่าย หรือแจกจ่ายซ้ำผลงานอิเล็กทรอนิกส์นี้ หรือส่วนใดส่วนหนึ่งของผลงานอิเล็กทรอนิกส์นี้ โดยไม่แสดงประโยคที่ระบุไว้ในวรรค 1.E.1 อย่างชัดเจน พร้อมลิงก์ที่ใช้งานได้หรือการเข้าถึงข้อกำหนดฉบับเต็มของใบอนุญาต Project Gutenberg™ ทันที
1.E.6 คุณสามารถแปลงและเผยแพร่ผลงานนี้ในรูปแบบไบนารี บีบอัด ทำเครื่องหมาย ไม่เป็นกรรมสิทธิ์ หรือเป็นกรรมสิทธิ์ รวมถึงรูปแบบการประมวลผลคำหรือไฮเปอร์เท็กซ์ อย่างไรก็ตาม หากคุณให้สิทธิ์เข้าถึงหรือเผยแพร่สำเนาผลงานของ Project Gutenberg™ ในรูปแบบอื่นที่ไม่ใช่ “Plain Vanilla ASCII” หรือรูปแบบอื่นที่ใช้ในเวอร์ชันอย่างเป็นทางการที่โพสต์บนเว็บไซต์ Project Gutenberg™ อย่างเป็นทางการ (www.gutenberg.org) คุณต้องจัดเตรียมสำเนา วิธีการส่งออกสำเนา หรือวิธีการรับสำเนาตามคำขอของผลงานในรูปแบบ “Plain Vanilla ASCII” ต้นฉบับหรือรูปแบบอื่น โดยไม่มีค่าใช้จ่ายหรือค่าใช้จ่ายเพิ่มเติมใดๆ ต่อผู้ใช้ รูปแบบทางเลือกใดๆ จะต้องมีใบอนุญาต Project Gutenberg™ ฉบับเต็มตามที่ระบุไว้ในวรรค 1.E.1
1.E.7. อย่าเรียกเก็บค่าธรรมเนียมสำหรับการเข้าถึง การดู การแสดง การดำเนินการ การคัดลอก หรือการแจกจ่ายผลงานใดๆ ของ Project Gutenberg™ เว้นแต่คุณจะปฏิบัติตามวรรค 1.E.8 หรือ 1.E.9
1.E.8 คุณอาจเรียกเก็บค่าธรรมเนียมที่สมเหตุสมผลสำหรับการคัดลอกหรือให้สิทธิ์เข้าถึงหรือแจกจ่ายผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ โดยมีเงื่อนไขว่า:
  • • คุณจ่ายค่าธรรมเนียมลิขสิทธิ์ 20% ของกำไรขั้นต้นที่คุณได้รับจากการใช้ผลงาน Project Gutenberg™ ซึ่งคำนวณโดยใช้วิธีที่คุณใช้ในการคำนวณภาษีที่เกี่ยวข้อง ค่าธรรมเนียมดังกล่าวจะต้องจ่ายให้กับเจ้าของเครื่องหมายการค้า Project Gutenberg™ แต่เจ้าของเครื่องหมายการค้าได้ตกลงที่จะบริจาคค่าลิขสิทธิ์ภายใต้ย่อหน้านี้ให้กับ Project Gutenberg Literary Archive Foundation การชำระค่าลิขสิทธิ์จะต้องชำระภายใน 60 วันหลังจากวันที่คุณจัดทำ (หรือถูกบังคับตามกฎหมายให้จัดทำ) แบบแสดงรายการภาษีประจำงวด การชำระค่าลิขสิทธิ์ควรระบุไว้อย่างชัดเจนและส่งไปยัง Project Gutenberg Literary Archive Foundation ตามที่อยู่ที่ระบุไว้ในมาตรา 4 “ข้อมูลเกี่ยวกับการบริจาคให้กับ Project Gutenberg Literary Archive Foundation”
  • • คุณจะคืนเงินเต็มจำนวนที่ชำระโดยผู้ใช้ที่แจ้งเป็นลายลักษณ์อักษร (หรือทางอีเมล) ภายใน 30 วันหลังจากได้รับแจ้งว่าผู้ใช้ไม่เห็นด้วยกับเงื่อนไขของใบอนุญาต Project Gutenberg™ ฉบับเต็ม คุณต้องขอให้ผู้ใช้ดังกล่าวส่งคืนหรือทำลายสำเนาผลงานทั้งหมดที่มีอยู่ในสื่อทางกายภาพ และหยุดใช้และเข้าถึงสำเนาผลงาน Project Gutenberg™ อื่นๆ ทั้งหมด
  • • คุณจัดเตรียมการคืนเงินเต็มจำนวนที่ชำระสำหรับงานหรือสำเนาทดแทนตามวรรค 1.F.3 หากพบข้อบกพร่องในงานอิเล็กทรอนิกส์และรายงานให้คุณทราบภายใน 90 วันหลังจากได้รับงานดังกล่าว
  • • คุณปฏิบัติตามเงื่อนไขอื่นๆ ทั้งหมดของข้อตกลงนี้สำหรับการแจกจ่ายผลงาน Project Gutenberg™ ฟรี
1.E.9 หากคุณต้องการเรียกเก็บค่าธรรมเนียมหรือแจกจ่ายงานอิเล็กทรอนิกส์ Project Gutenberg™ หรือกลุ่มงานในเงื่อนไขที่แตกต่างจากที่กำหนดไว้ในข้อตกลงนี้ คุณจะต้องได้รับอนุญาตเป็นลายลักษณ์อักษรจาก Project Gutenberg Literary Archive Foundation ซึ่งเป็นผู้จัดการเครื่องหมายการค้า Project Gutenberg™ โปรดติดต่อมูลนิธิตามที่ระบุไว้ในมาตรา 3 ด้านล่าง
1.ฉ.
1.F.1 อาสาสมัครและพนักงานของ Project Gutenberg ใช้ความพยายามอย่างมากในการระบุ ค้นคว้าลิขสิทธิ์ ถอดความและตรวจทานผลงานที่ไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ของสหรัฐอเมริกาในการสร้างคอลเลกชัน Project Gutenberg™ แม้จะมีความพยายามเหล่านี้ ผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ และสื่อที่อาจจัดเก็บไว้ก็อาจมี "ข้อบกพร่อง" เช่น แต่ไม่จำกัดเพียง ข้อมูลที่ไม่สมบูรณ์ ไม่ถูกต้อง หรือเสียหาย ข้อผิดพลาดในการถอดความ การละเมิดลิขสิทธิ์หรือทรัพย์สินทางปัญญาอื่น ๆ ดิสก์หรือสื่ออื่นที่ชำรุดหรือเสียหาย ไวรัสคอมพิวเตอร์ หรือรหัสคอมพิวเตอร์ที่ทำให้อุปกรณ์ของคุณเสียหายหรือไม่สามารถอ่านได้
1.F.2 การรับประกันแบบจำกัด การปฏิเสธความรับผิดต่อความเสียหาย - ยกเว้น "สิทธิในการเปลี่ยนหรือคืนเงิน" ตามที่อธิบายไว้ในวรรค 1.F.3 มูลนิธิ Project Gutenberg Literary Archive เจ้าของเครื่องหมายการค้า Project Gutenberg™ และบุคคลอื่นใดที่จำหน่ายผลงานอิเล็กทรอนิกส์ Project Gutenberg™ ตามข้อตกลงนี้ ปฏิเสธความรับผิดต่อคุณสำหรับความเสียหาย ค่าใช้จ่าย และค่าใช้จ่าย รวมถึงค่าธรรมเนียมทางกฎหมาย คุณตกลงว่าคุณไม่มีแนวทางแก้ไขใดๆ สำหรับความประมาท ความรับผิดที่เข้มงวด การละเมิดการรับประกัน หรือการละเมิดสัญญา ยกเว้นที่ระบุไว้ในวรรค 1.F.3 คุณตกลงว่ามูลนิธิ เจ้าของเครื่องหมายการค้า และผู้จัดจำหน่ายตามข้อตกลงฉบับนี้จะไม่รับผิดชอบต่อคุณสำหรับความเสียหายที่เกิดขึ้นจริง โดยตรง โดยอ้อม เป็นผลสืบเนื่อง เป็นการลงโทษ หรือโดยบังเอิญ แม้ว่าคุณจะแจ้งให้ทราบถึงความเป็นไปได้ของความเสียหายดังกล่าวก็ตาม
1.F.3 สิทธิ์จำกัดในการเปลี่ยนหรือคืนเงิน - หากคุณพบข้อบกพร่องในงานอิเล็กทรอนิกส์นี้ภายใน 90 วันหลังจากได้รับ คุณสามารถขอคืนเงิน (หากมี) ที่คุณจ่ายไปโดยส่งคำอธิบายเป็นลายลักษณ์อักษรถึงบุคคลที่คุณได้รับงานมา หากคุณได้รับงานในสื่อทางกายภาพ คุณต้องส่งสื่อพร้อมคำอธิบายเป็นลายลักษณ์อักษรคืน บุคคลหรือหน่วยงานที่มอบงานที่มีข้อบกพร่องให้กับคุณอาจเลือกที่จะให้สำเนาทดแทนแทนการคืนเงิน หากคุณได้รับงานทางอิเล็กทรอนิกส์ บุคคลหรือหน่วยงานที่มอบงานให้กับคุณอาจเลือกที่จะให้โอกาสครั้งที่สองแก่คุณในการรับงานทางอิเล็กทรอนิกส์แทนการคืนเงิน หากสำเนาที่สองมีข้อบกพร่องเช่นกัน คุณสามารถเรียกร้องคืนเงินเป็นลายลักษณ์อักษรโดยไม่ต้องมีโอกาสแก้ไขปัญหาเพิ่มเติม
1.F.4 ยกเว้นสิทธิ์จำกัดในการเปลี่ยนหรือคืนเงินตามที่ระบุไว้ในวรรค 1.F.3 งานนี้มอบให้คุณ "ตามที่เป็น" โดยไม่มีการรับประกันอื่นใด ไม่ว่าโดยชัดแจ้งหรือโดยนัย รวมถึงแต่ไม่จำกัดเฉพาะการรับประกันความสามารถในการขายหรือความเหมาะสมสำหรับจุดประสงค์ใดๆ
1.F.5. บางรัฐไม่อนุญาตให้มีการปฏิเสธการรับประกันโดยปริยายบางประการหรือการยกเว้นหรือจำกัดความเสียหายบางประเภท หากการปฏิเสธหรือการจำกัดใดๆ ที่ระบุไว้ในข้อตกลงนี้ละเมิดกฎหมายของรัฐที่บังคับใช้กับข้อตกลงนี้ ข้อตกลงนั้นจะต้องตีความเพื่อให้การปฏิเสธหรือการจำกัดสูงสุดได้รับอนุญาตตามกฎหมายของรัฐที่บังคับใช้ ความไม่ถูกต้องหรือไม่สามารถบังคับใช้ได้ของข้อกำหนดใดๆ ในข้อตกลงนี้จะไม่ทำให้ข้อกำหนดที่เหลือเป็นโมฆะ
1.F.6 การชดเชยความเสียหาย - คุณตกลงที่จะชดเชยความเสียหายและถือว่ามูลนิธิ เจ้าของเครื่องหมายการค้า ตัวแทนหรือพนักงานของมูลนิธิ บุคคลใดก็ตามที่จัดหาสำเนาผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ตามข้อตกลงนี้ และอาสาสมัครที่เกี่ยวข้องกับการผลิต การส่งเสริม และการจำหน่ายผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ พ้นจากความรับผิดชอบ ค่าใช้จ่าย และค่าใช้จ่ายใดๆ รวมถึงค่าธรรมเนียมทางกฎหมายที่เกิดขึ้นโดยตรงหรือโดยอ้อมจากสิ่งใดสิ่งหนึ่งต่อไปนี้ที่คุณทำหรือทำให้เกิดขึ้น: (a) การจำหน่ายผลงานนี้หรือผลงานใดๆ ของ Project Gutenberg™ (b) การเปลี่ยนแปลง การปรับเปลี่ยน หรือการเพิ่มเติมหรือการลบผลงานใดๆ ของ Project Gutenberg™ และ (c) ข้อบกพร่องใดๆ ที่คุณก่อขึ้น
ส่วนที่ 2 ข้อมูลเกี่ยวกับภารกิจของ Project Gutenberg™
โครงการ Gutenberg™ เป็นโครงการที่มีวัตถุประสงค์เพื่อแจกจ่ายงานอิเล็กทรอนิกส์ในรูปแบบต่างๆ ที่สามารถอ่านได้ด้วยคอมพิวเตอร์หลากหลายประเภท เช่น คอมพิวเตอร์ที่ล้าสมัย คอมพิวเตอร์รุ่นเก่า คอมพิวเตอร์วัยกลางคน และคอมพิวเตอร์รุ่นใหม่ โครงการนี้เกิดขึ้นได้จากความพยายามของอาสาสมัครหลายร้อยคนและการบริจาคจากผู้คนทุกระดับชั้น
อาสาสมัครและการสนับสนุนทางการเงินเพื่อให้ความช่วยเหลืออาสาสมัครตามที่ต้องการถือเป็นสิ่งสำคัญในการบรรลุเป้าหมายของ Project Gutenberg™ และเพื่อให้แน่ใจว่าคอลเลกชัน Project Gutenberg™ จะยังคงเปิดให้เข้าถึงได้ฟรีสำหรับคนรุ่นต่อๆ ไป ในปี 2001 มูลนิธิ Project Gutenberg Literary Archive Foundation ก่อตั้งขึ้นเพื่อมอบอนาคตที่มั่นคงและถาวรให้กับ Project Gutenberg™ และคนรุ่นต่อๆ ไป หากต้องการเรียนรู้เพิ่มเติมเกี่ยวกับมูลนิธิ Project Gutenberg Literary Archive Foundation และวิธีที่ความพยายามและการบริจาคของคุณจะช่วยเหลือได้ โปรดดูส่วนที่ 3 และ 4 และหน้าข้อมูลมูลนิธิที่ www.gutenberg.org
ส่วนที่ 3 ข้อมูลเกี่ยวกับมูลนิธิโครงการ Gutenberg Literary Archive
Project Gutenberg Literary Archive Foundation เป็นองค์กรการศึกษาที่ไม่แสวงหากำไร 501(c)(3) ซึ่งจัดตั้งขึ้นภายใต้กฎหมายของรัฐมิสซิสซิปปี้และได้รับสถานะยกเว้นภาษีจากกรมสรรพากร หมายเลข EIN หรือหมายเลขประจำตัวผู้เสียภาษีของรัฐบาลกลางของมูลนิธิคือ 64-6221541 เงินบริจาคให้กับ Project Gutenberg Literary Archive Foundation สามารถหักลดหย่อนภาษีได้ในขอบเขตสูงสุดที่กฎหมายของรัฐบาลกลางสหรัฐอเมริกาและกฎหมายของรัฐของคุณอนุญาต
สำนักงานธุรกิจของมูลนิธิตั้งอยู่ที่ 809 North 1500 West, Salt Lake City, UT 84116, (801) 596-1887 สามารถดูลิงก์ติดต่อทางอีเมลและข้อมูลติดต่อล่าสุดได้ที่เว็บไซต์ของมูลนิธิและหน้าทางการที่ www.gutenberg.org/contact
ส่วนที่ 4 ข้อมูลเกี่ยวกับการบริจาคให้กับมูลนิธิ Project Gutenberg Literary Archive
โครงการ Gutenberg™ พึ่งพาและไม่สามารถอยู่รอดได้หากขาดการสนับสนุนและการบริจาคจากสาธารณชนอย่างกว้างขวาง เพื่อดำเนินภารกิจในการเพิ่มจำนวนผลงานสาธารณสมบัติและผลงานที่ได้รับอนุญาต ซึ่งสามารถแจกจ่ายได้อย่างอิสระในรูปแบบที่เครื่องอ่านได้ซึ่งเข้าถึงได้โดยอุปกรณ์หลากหลายประเภท รวมถึงอุปกรณ์ที่ล้าสมัย การบริจาคจำนวนเล็กน้อยจำนวนมาก (ตั้งแต่ 1 ถึง 5,000 ดอลลาร์) มีความสำคัญอย่างยิ่งต่อการรักษาสถานะการยกเว้นภาษีกับกรมสรรพากร
มูลนิธิมีความมุ่งมั่นที่จะปฏิบัติตามกฎหมายที่ควบคุมองค์กรการกุศลและการบริจาคเพื่อการกุศลในทั้ง 50 รัฐของสหรัฐอเมริกา ข้อกำหนดในการปฏิบัติตามนั้นไม่ได้เหมือนกันและต้องใช้ความพยายามอย่างมาก เอกสารจำนวนมาก และค่าธรรมเนียมจำนวนมากเพื่อให้เป็นไปตามข้อกำหนดเหล่านี้ เราไม่ขอรับบริจาคในสถานที่ที่เราไม่ได้รับการยืนยันเป็นลายลักษณ์อักษรว่าปฏิบัติตาม หากต้องการส่งเงินบริจาคหรือตรวจสอบสถานะการปฏิบัติตามสำหรับรัฐใดรัฐหนึ่ง โปรดไปที่ www.gutenberg.org/donate
แม้ว่าเราไม่สามารถและจะไม่ขอรับการบริจาคจากรัฐที่เราไม่ได้ปฏิบัติตามข้อกำหนดในการขอรับบริจาค แต่เราไม่ทราบถึงข้อห้ามในการยอมรับการบริจาคที่ไม่ต้องการจากผู้บริจาคในรัฐดังกล่าวที่ติดต่อมาเสนอที่จะบริจาคกับเรา
เรายินดีรับเงินบริจาคจากต่างประเทศ แต่เราไม่สามารถออกแถลงการณ์ใดๆ เกี่ยวกับการจัดเก็บภาษีจากเงินบริจาคที่ได้รับจากนอกประเทศสหรัฐอเมริกาได้ กฎหมายของสหรัฐอเมริกาเพียงอย่างเดียวก็ทำให้พนักงานจำนวนน้อยของเราล้นมือแล้ว
