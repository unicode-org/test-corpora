CHAPITRE 55. N Tugniwin Tiqburin n Yilef.

Ad ak-d-sfehmeɣ ɣezzifen akken izemren yiwen mebla lkanvas, kra am tɣawsa n tidet n lbaṭaṭa akken i d-yettban s tidett i wallen n lbaṭaṭa mi ara yili deg tfekka-s n tmeẓẓuɣt n lbaṭaṭa yettwaqqen ɣer tama n lbaṭaṭa-nni akken ad t-id-sbedden s lḥeqq dinna. Yezmer ad yili d ayen yesɛan azal, ihi, zik ad d-nebder tifelwiyin-nni n tmuɣliwin (imaginaires) n tmuɣliwin-nnes (portraits imaginaires) ay d-yettawin ula d ass-a s lqerb ttqadaren liman n wemdan-nni n tmurt. D lawan ad nessemlil amaḍal deg temsalt-a, s ussekcem n tugniwin am tigi n yilef akk ur llint ara d tid yelhan.

Yezmer lḥal, aɣbalu amezwaru n wakk tiḥeqranin-nni n tfelwit ad t-naf gar tfelwiyin tiqburin akk n Yihinduyen, Imaṣriyen ed Yiyunaniyen. Imi seg tallit-nni n usnulfu maca ur nesɛi lḥeqq mi ara yili ɣef tfelwiyin n lmarmar n yiɣerbazen n tẓallit, ɣef tfelwiyin n tmeṣkiwt, ɣef tmeẓẓuɣin, ɣef tmeẓẓuɣin, ɣef tmeẓẓuɣin, ɣef tmeẓẓuɣin, ad yettwajerred udelfin s tmeẓẓuɣin n tmeẓẓuɣin n tmeẓẓuɣin am tin n Saladin, d uqerruy s lkaṣk am win n St. seg wasmi ay d-yella waya, yella kra n ṣṣenf n tlisa ay d-yellan, maci kan deg tugniwin ay d-yettwassnen aṭas n yilel, wanag deg waṭas n tqeddicin tussnanin ay d-yellan fell-as.

Tura, s wakk-n ay d-yella lḥal, taṣekka taqburt akk ay yellan, akken yebɣu yili lḥal, d tin n yilef, ad tt-naf deg yifri-pagoda yettwassnen n Elephanta, deg Lhend. Ibrahmiyen ttkemmilen ttwalin dakken deg tmeṣkiwt (sculptures) qrib ur ttfakkant n tmeqbert-nni ur nesɛi tagara, akk lḥirfat ed tɣawsiwin ay ɣef ara d-yili wemdan, yal ccɣel ay izemren ad d-yeḍru wemdan, ttwaḥesbent-d zik uqbel ma tella-d yiwet seg-sent s tidet. Ur yelli d lmuḥal ihi, ma yella s kra n ṣṣenf n lxedma-nneɣ n tmeɣra n tmeẓdiyin (baleines) ilaq ad tili dinna ttwaḥerzent. Iɣersiwen n Yihinduyen ay d-yettwabder, ḍerrun deg yiwen n weḥric ay d-yezgan deg lḥiḍ-nni, yerna yesskanay-d taɣawsa n Vishnu s talɣa n leviathan, ay yettwassnen s yisem n Matse Avatar. Maca ɣas akken taskulpt-a d azgen n wemdan d uzgen n yilef, akken kan ad d-tefk taɣect n uneggaru-a, maca aḥric-nni ameẓyan seg-s yeɣli-d akk. Tban-d ugar am tmeẓẓuɣt n tmeẓẓuɣt n anaconda, wala am yifassen n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt n tidet.

Maca ddu ɣer Galleries iqburen, ẓer tura s tmuɣli n yiwen n umeskar amasiḥi ameqqran n yiselman-agi ; acku ur yessaweḍ ara ad yili d ayen yelhan ugar n Yihinduyen n zik n tmes. D tugna n Guido n Perseus i d-isellken Andromeda seg uɣersiw n yilel neɣ seg uɣersiw n yilel. Anda i d-yewwi Guido lemtel n umdan n tmexluqt n tmexluqt am ta ? Ula d Hogarth, deg ussekles-nnes n yiwen n uswir deg « Perseus Descending »-nnes, ur d-yessuffeɣ ara yiwen n uswir ay igerrzen ugar. Tafekka tameqqrant n uɣersiw-nni n Hogarth tettmuqul deg tmurt, qrib ad d-tessufeɣ yiwen n lmitra n waman. Tesɛa ṣṣenf n howdah ɣef tɣerɣert-is, yerna aqemmuc-is n tmeẓẓuɣt yettwasfeḍen aydeg ttɣelliten yiɣerruba, yezmer ad yettwaṭṭef i Tewwurt n Yixeddamen ay d-yettawin seg Thames s waman ɣer Lbuṛj. Sakkin, llan yiɣersiwen n Prodromus n Scotch Sibbald aqbur, ed yiɣersiwen n Yuna, akken ay d-ttwaseknen deg tɣawsiwin ay d-yettwassuffɣen deg Yidlisen Iqburen ed yigezmen n tɣawsiwin tiqburin. D acu ara d-nini ɣef wigi ? Ma d ayen yerzan aɣersiw n tmeẓẓuɣt n yedlisen yettɛeddayen am tmeẓẓuɣt n tẓegwa ɣef tɣerɣert n yiwen n lbabur i d-yeɣlin – am wakken i d-yettwasbedd yerna yettwazedgen ɣef yiḍarren d yisebtar n tmeẓriwin n waṭas n yidlisen ama d iqburen ama d imaynuten – d ayen i d-yettawin lxir aṭas maca d ayen igerrzen kan, yettwaqbel, ttwaliɣ-t, seg yiḍrisen yecban wi. Ɣas ma yella qqaren-as deg umaḍal d adelfin, maca qqareɣ-as i yiselman-a n wedlis-a ay d-yesbedden adlis d aɛewwel ɣef yiwen n uɣersiw n yilel; acku akka ay d-yettwawehha asmi ay d-yeffeɣ wallal-a tikkelt tamezwarut. Tettwasekcem-d sɣur yiwen n umsuffeɣ aqbur aṭalyani anda-nniḍen azal n lqern wis 15, deg tallit n Tnekra n Uselmed ; yerna deg wussan-nni, yerna ula d tallit ay d-yellan s tɣawla, llan ttwalin medden d akken idelfan d ṣṣenf n Leviathan.

Deg tvignettes d tmeqwa nniḍen n kra n yedlisen iqburen ad temlileḍ tikwal s tmeẓẓuɣin n tmeẓẓuɣin aṭas deg tmeẓẓuɣin, anda akk ṣṣenf n tmeẓẓuɣin, jets d’eau, tiẓegwa n tmes d usemmiḍ, Saratoga d Baden-Baden, ttbubbun-d seg wallaɣ-is ur nxeddem ara. Deg usebter n tmezwarut n teẓrigt taẓurant n “Asnerni n Uselmed” ad tafeḍ kra n yiɣersiwen n tmussni.

Maca ma neǧǧa akk ɛewwiqen-a ur nelli d ixeddamen, ad d-nefk tamuɣli ɣef tugniwin-nni n leviathan ay d-yeqqaren d akken d tid ay d-yettwasbedden s tmussni, d tid ay d-sbedden wid yessnen. Deg wedlis aqbur n yiberdan n Harris llant kra n tfelwiyin n yiɣersiwen n yilel ay d-yekkan seg yiwen n wedlis ahulandi n yiberdan, 1671 sdeffir talalit n Lmasiḥ, s uzwel n “Asikel n yilel ɣer Spitzbergen deg uɣerrabu Jonas deg Yilel, Peter Peterson n Frisland, d aselway.” Deg yiwet seg tfelwiyin-nni, ttwaseknen-d yiɣersiwen n yilel, am tfekkiwin timeqranin n tẓegwa, ttwaḍeggren gar tegzirin n wegris, s yiɣersiwen imellalen ay la yettazzalen ɣef yiḍarren-nsen ay yeddren. Deg yiwet n tfelwit niḍen, lbaṭel-nni (blunder) ameqran yettwaxdem s ussemres n yilef s yiferdisen (fluques) ay d-yezgan ɣef tɣerɣert.

Sakkin, yella yiwen n quarto (quarto) ay d-yesmektayen s ljehd, ay d-yura yiwen n ukabṭan Colnett, d ukabṭan n Post deg yigen (armée) n yilel anglizi, s yisem-nnes “Asikel ɣef Cape Horn ɣer Yilel n Wenẓul, s yiswi n ussemɣer n Tmerrit n Yilel n Spermaceti.” Deg wedlis-a, yella yiwen n lqanun ay d-yeqqaren d akken d “Tugna n yiwen n uɣersiw n yilel n Physeter neɣ Spermaceti, ay d-yettwajerden s lqis seg yiwen ay yettwanɣan deg tlisa n Miksik, deg Ɣuct, 1793, yerna yettwasfeḍ ɣef tɣerɣert.” Ur ccukkeɣ ara belli aqebṭan-nni yesɛa tugna-agi n tidet i d-yeṭṭfen i lfayda n yimariren-is. Ma bderɣ-d yiwet kan n tɣawsa fell-as, ad d-iniɣ belli tesɛa tiṭ ay yettwasqedcen, ɣef leḥsab n tɣawsa ay d-yeddan yid-s, ɣef yiwen n uɣersiw n tmeẓẓuɣt (sperme) ay d-yefrurxen s lekmal-nnes, ad t-terr tiṭ n uɣersiw-nni d taɣect n tmeẓẓuɣt (bow-window) ay yesɛan azal n xemsa n yiḍarren deg teɣzi. Ah, a lqaḍi-inu n lǧehd, ayɣer ur aɣ-d-tefkiḍ ara Yuna yettmuqul seg wallen-nni !

Ula d tizrawin (compilations) n Umezruy Aɣelnaw ay d-yettwasbedden s tmussni (conscience) akk i lfayda n yilemẓiyen ed yilemẓiyen, ur llint ara d tilelliyin seg yiwet n tɣawsa n tuccḍa. Muqel axeddim- nni yettwassnen s waṭas “Agama n Goldsmith i d- iţwaḥerzen.” Deg teẓrigt n London n 1807 ay d-yeffɣen s tɣawla, llant tfelwiyin n yiwen n “uɣersiw n yilel” ay ɣef d-qqaren d yiwen n “uɣersiw n yilel” ed yiwen n “uɣersiw n yilel.” Ur bɣiɣ ara ad iliɣ d aɛeqli, maca aɣersiw-a ur nesɛi ara aɛeqli yettban-d aṭas am yiwen n uɣersiw yettwaqḍen; yerna, ɣef wayen yerzan narwhale, yiwet n tmuɣli kan fell-as tezmer ad tessewham yiwen, imi deg lqern-agi wis tẓa d mraw, aɣersiw am wa yezmer ad yettwaḥseb d tidet ɣef yal aɣref n yigerdan n uɣerbaz imeẓẓyanen.

Sakkin, tikkelt niḍen, deg 1825, Bernard Germain, Count de Lacépède, d amusnaw ameqran n tgemmi (nature), yessuffeɣ-d adlis ussnan n yiɣersiwen n yilel (baleine) ay d-yettwasbedden s tɣawla, yerna deg-s llant aṭas n tugniwin n tɣawsiwin yemgerraden n Leviathan. Wid-a akk ur llin kan d iɛewwiqen, wanag tugna n lbaṭaṭa n Mysticetus neɣ n Greenland (dɣa d lbaṭaṭa n Right), ula d Scoresby, d argaz ay yesɛan aṭas n tmussni am wakken ay yettaḍḍaf ṣṣenf-nni, yettbeccir-d ur yesɛi ara amciweṛ-nnes deg ugama.

Maca asras n tqacuct n tqacuct i wakk tamsalt-agi n tkerkas, yettwaḥrez i umusnaw Frederick Cuvier, gma-s n Baron yettwassnen. Deg 1836, yessuffeɣ-d adlis-nnes (Histoire naturelle des baleines), yerna deg-s yefka-d ayen umi isemma tugna n Yilef n Yilef. Uqbel ad d-tessekneḍ tugna-nni i yal yiwen n Nantucketer, ilaq ad t-id-tessewjedḍ i tmeɣra-inek n tgermant seg Nantucket. S yiwen wawal, Aɣersiw n Frederick Cuvier mačči d aɣersiw n tfekka, maca d aɣersiw n tfekka. D tidet, werjin yesɛa lfayda n tikli n tmeẓdit (irgazen am wi ur sɛin ara aṭas), maca anda ay d-yekka tugna-nni, anwa ay izemren ad d-yini? Ahat yufa-tt am win i t-id-yezwaren ussnan deg yiwen n unnar, Desmarest, yufa-d yiwen seg yigerdan-is n tmeṭṭut taḥeqqit ; dɣa, seg yiwen n usnulfu acinwat. Yerna d acu n ṣṣenf n yirgazen n tudert s lkalam n Yicinwaten-nni, aṭas n tkubbin d tmeqwa n queer ay aɣ-d-yesmektayen.

Ma d iɣersiwen n yimeskaren n tmeẓriwin yettwalin deg yiberdan ttwaḍeggren ɣef tḥuna n yimesnuzuyen n lpitrul, d acu ara d-nini fell-asen ? S umata d Ricarḍ wis 3. iɣersiwen, s yiɣil n dromedary, yerna d iɣersiwen aṭas; lqahwa ɣef kraḍ neɣ kuẓ n ttart n yimariren, dɣa d iɣerruba n yilel yeččuren d imarikaniyen : lqella n tmussni-nsen tettfeggiḍ deg yilel n yidammen d tmeẓẓuɣt tazegzawt.

Maca tuccḍiwin-a timeqranin deg ussekles n yilef ur ttwehhiment aṭas sdeffir ma tfuk temsalt. Meggez! Amur ameqran seg yisura ussnanen ttwakksen-d seg yiselman ay yettwaḍḍfen; yerna tid-a qrib ad ṣeḥḥent am wakken ara d-tessken tfelwit n yiwen n uɣerrabu iɣezfanen, s tɣerɣert iɣezfanen, ad d-tessken s lḥeqq aɣersiw-nni n lḥeṛma s timmad-is deg wakk lfeṛḥ-is ur yettwaḥeqren ara n tfekka d tfekka n tfekka. Xas akken ilfan bedden ɣef tɣerɣert-nsen, Leviathan yeddren ur yezmir ara ad d-yesfeḍ iman-is s lḥeqq ɣef tfelwit-is. Ilemẓi-nni yeddren, deg tmeqrant-is d lmeɛna-s, ur t-id-yettwali ara kan deg yilel deg waman ur nesɛi lmeɛna; yerna deg waman amur ameqran seg-s ur yettwali ara, am yiwen n uɣerrabu n tlisa n yimenɣi yettwasbedden; yerna seg uferdis-nni d ayen ur nezmir ara ad t-id-yesnekr wemdan s tfekka-s ɣer yigenni, akken ad yeḥrez akk tiɣbula-ines d tmeẓẓuɣin-is tijehden. Yerna, ur d-nettmeslay ara ɣef wemgerrad n tɣerɣert (contour) ay yettwaḥesben s waṭas gar yiwen n uɣersiw n yilel (baleine) ay d-yessussufen d yiwen n Leviathan n Platon ay d-yefrurxen s lekmal-nnes; maca, ula deg tegnit n yiwen seg yiɣersiwen-nni imeẓyanen ay d-yessussufen ay d-yettwasbedden ɣer tɣerɣert n yiwen n uɣerrabu, akka ay tella imir-nni ṣṣifa-nnes n tmura n beṛṛa, am tmeẓẓuɣt n tmeẓẓuɣt, n tmeẓẓuɣt, yerna temgerrad, yerna ur yezmir ara ad t-id-yeṭṭef cceṛ s timmad-nnes.

Maca yezmer ad d-yili uxemmem, belli seg tfekka n yilef-nni ay d-yeqqimen d aɛeryan, zemren ad d-ttwafernen yiḥezziben n tɣawsiwin s tɣawla ay iḥuzan talɣa-nnes n tidet. Xaṭi. Imi d yiwet seg tɣawsiwin i d-yettawin ccek ugar ɣef Leviathan-agi, d akken aɣersiw-is yettak-d drus n tmuɣli ɣef ṣṣifa-s tamatut. Xas akken aɣersiw n Jeremy Bentham, ay d-yeqqnen ɣer tmeẓdiyin (candelabres) deg temkarḍit n yiwen seg wid ay t-yesbedden, yesskanay-d s lḥeqq tikti n yiwen n wergaz aqbur n tmeẓdiyin (utilitariste) ay yesɛan aɣeddu n tmeẓdiyin, yerna yesɛa akk tiɣawsiwin niḍen n Jeremy ay d-yellan d tid ay d-yesbedden ; maca ulac kra n ṣṣenf-a ay izemren ad d-yettwassummel seg yiɣsan n yal leviathan ay d-yettwasqedcen. Deg tidet, akken ay d-yenna Uḥeddaḍ ameqran, aɣersiw kan n tmeẓdit yesɛa assaɣ ed uɣersiw-nni ay d-yesfaydin s lekmal-nnes yerna yettwaḍḍef am wakken ay yesɛa uɣersiw-nni ed tmeẓdit (chrysalis) ay t-yesfeḍḍen s tɣawla. Tazmert-a tettban-d s tɣawla deg uqerru, am wakken deg kra n weḥric n wedlis-a ara d-yettwasken s tɣawla. Yettwassken-d daɣen s tɣawla deg tfekka n yidis, iɣsan-nnes qrib ad rren ɣef yiɣsan n ufus n wemdan, s tɣawla kan, s tɣawla. Tafekka-a tesɛa 4 n yiḍarren n yiɣes n lqanun, d aḍad n tmeẓẓuɣt, n tlemmast, n tmeẓẓuɣt, d uḍad ameẓyan. Maca wigi akk ttwaḥerzen i lebda deg uḍebsi-nsen n uksum, am yiḍuḍan n wemdan deg uḍebsi n tmeṣkiwt. “Akken yebɣu yili tikwal lbaṭel ay aɣ-d-yefka lbaṭaṭa-nni,” ay d-yenna Stubb ay d-yesmektayen yiwen wass, “ur nezmir ara ad d-nini s tidet ad aɣ-d-iḥader mebla lmitten.”

S ssebbat-agi akk, ihi, akken tebɣiḍ ad t-tmuqleḍ, ilaq ad tgeḍ lḥeqq belli Leviathan ameqqran d amdan-nni yiwen deg umaḍal i yessefk ad yeqqim ur yettwasfeḍ ara armi d aneggaru. D tidet, yiwet n tfelwit tezmer ad teɣli ɣef tɣawsa-nni qrib aṭas n wayeḍ, maca ula d yiwet ur tezmir ad tt-teɣli s kra n tɣawsa s wazal-is aṭas. Ihi ulac abrid n tmurt akken ad naf s tɣawla amek ay yettwali lbaṭaṭa-nni s tidet. Yerna abrid kan aydeg ara d-tessufɣeḍ ula d yiwet n tekti yettwasfeḍen ɣef tɣerɣert-nnes (contour) n tudert, d akken ad tedduḍ s timmad-ik ɣer tmeɣra n yilef; maca s waya, ur tesɛiḍ ara lḥeṛs meẓẓiyen akken ad tiliḍ d lfurna n lebda yerna ad t-teɣliḍ sɣur-s. Ihi, iban-iyi-d d akken ilaq ur telliḍ ara d aɛewwiq aṭas deg tmuɣli-inek n tḥudda n Leviathan-agi.





IXEF 56. N tugniwin n yilef ur nesɛi ara aṭas n tuccḍa, d tugniwin n tidet n tmuɣliwin n yilef.

Deg wayen yerzan tugniwin n tmeẓyanin n yiɣersiwen, ttḥulfuɣ s ljehd dagi akken ad kecmeɣ ɣef tmucuha-nni n tmeẓyanin ugar n tmeẓyanin fell-asent ay d-yettwafernen deg kra n yedlisen, ama d iqburen, ama d imaynuten, ladɣa deg Plinius, Purchas, Hackluyt, Harris, Cuvier, atg. Maca ttɛeddayeɣ ɣef temsalt-nni.

Ssneɣ kan 4 n tfelwiyin yettwassuffɣen ɣef Uɣersiw n Sperm ameqran; n Kulnet, n Huggins, n Fridrik Kuvier, d n Bile. Deg yixef iɛeddan, nebder-d Colnett d Cuvier. Tin n Huggins telha aṭas n tin-nsen; maca, s lqedd ameqqran, tin n Beale d tin igerrzen akk. Akk tisura n Beale ɣef tmeẓdit-a lhan, anagar aḍris alemmas deg tugna n kraḍ n yiɣersiwen deg tmuɣliwin yemgerraden, d netta ay d-yesbedden aḥric-nnes wis sin. Tafelwit-nnes n tlemmast, iɣerruba ay yettḍurrun Iɣersiwen n Sperm, ɣas ma yella war ccek ttwaḥesbent akken ad sqeɛdent ccek aɣarim n kra n yergazen n tḥuna, maca d tin ay d-yellan s lḥeqq yerna tban-d am tudert deg tɣawsa-nnes tamatut. Kra seg yisura n Sperm Whale deg J. Ross Browne ttwaṣeggmen mliḥ deg tɣerɣert; maca ttwaqedcen s lḥif. D acu kan, mačči d lɣelṭa-s.

Deg Uɣersiw Aẓru, tugniwin n tfelwit igerrzen akk llant deg Scoresby; maca ttwajerden s lqidar meẓẓiyen aṭas akken ad d-ssiwḍen lḥirfa ibɣan. Yesɛa kan yiwet n tugna n tmuɣliwin n tmeẓdiyin (baleines), yerna aya d lexṣaṣ ay yessewhamen, imi d tugniwin am tigi kan, mi ara ttwaxedment mliḥ, ay tzemreḍ ad d-tessufɣeḍ kra am tikti n tidet ɣef tmeẓdiyin yeddren akken ay tt-ttwalin yiḥeddaden-nnes yeddren.

Maca, ma ttwaṭṭfent akk, s lbeɛd, tid ay igerrzen akk, ɣas akken deg kra n tɣawsiwin ur llint d tid ay d-yellan s lḥeqq, tiqdimin n yiɣersiwen n yilel ed tmuɣliwin n tmeẓdiyin (baleines) ay izemren ad tent-naf anda ma llan, d snat n tɣawsiwin timeqranin n tfelwiyin tifransiyin, ay d-yettwaxedmen mliḥ, yerna ttwakksent-d seg tfelwiyin n yiwen n Garnery. S tmerna, ttgensisen-d ineṭḍen ɣef Sperm d Right Whale. Deg tmeẓriwt tamezwarut, yettwassken-d yiwen n Uɣersiw n Sperm Whale s tmeqrant n tezmert-is, i d-yekkan kan ddaw lbabur seg lqaɛ n yilel, yerna yettawi-d s lqedd deg yigenni ɣef tɣerɣert-is aɣerrabu-nni ixuṣṣen n tfelwiyin n tmes. Taɣect n lbabur ur tettwakkes ara s tɣawla, yerna tettwajerred kan s lmizan ɣef tɣerɣert n uɣersiw-nni; yerna tebded deg tlemmast-nni, i yiwen n lbaḍna-nni n wakud ur nesɛi lḥisab, twaliḍ yiwen n umsebrid, azgen-is yettwaḍḍef s tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt, yerna deg tikli n tmeẓẓuɣt, amzun seg yiwen n usqif. Qbel ad d-yesneqs lbabur i lmend n uḍfar, ttwaṭṭfen yixef n tlemmast n tɣerɣert n tlemmast n tbuciḍant, yerna mi ara ɛeddin ɣef tɣerɣert n tẓegwa din, ad ttwaɛerḍen daɣen ɣer sdat ɣef teɣzi n lbabur-nni akk, ad sgunfun s tɣawla ɣef tɣawsa n tɣawsa neɣ ɣef tɣawsa n tɣawsa n yal amdan, akken ad yettɛeddi ɣef ufus-nnes deg uḍfar n tɣawsa ; yerna daɣen yettɛedday gar yergazen-nni, akken ttbeddilen deg yiɣerruba n tmeẓdiyin (gunwales) ay d-yezzin fell-asen, ɣer yiɣerruba neɣ iɣerruba ay yesɛan lḥemla deg tlemmast n tkeṛṛust-nni ay d-yellan s tɣawla, anda ay d-yettili yiwen n usɣar n useklu neɣ n usɣar n tmeẓdiyin s lqis n tmeẓdiyin n tmeẓdiyin, ay t-yettaǧǧan ad d-yeffeɣ. Seg tcekkaṛin yettɛeggiḍ deg yiwen n lfeṭṭa tameẓyant ɣef tmeẓẓuɣin, syin yettwaɛedda daxel n lbabur tikkelt nniḍen; yerna kra n mraw neɣ ɛacrin n yifaṭumen (yettusemma box-line) ttwaqnen ɣef tkeṛṛust deg yiɣil, tkemmel abrid-nnes ɣer gunwale mazal-itt cwiṭ ɣer deffir, sakkin tettwaqqen ɣer tɣawsa n tɣerɣert (short-warp) – d aɣerrabu ay yettwaqqnen srid ɣer tɣawsa n tɣawsa ; maca uqbel tuqqna-nni, short-warp yettɛedday s tmuɣliwin yemgaraden i d-yettawin lḥir s waṭas.

Akka, lḥiḍ n yilef (baleine) yettḍeggiɛ akk aɣerrabu-nni deg yiɣil-nnes (coils) ixuṣṣen, yerna yettḍeggiɛ-it yerna yettḍeggiɛ-it qrib deg yal tama. Akk imesnallunen ttekkant deg tmeẓẓuɣin-is n tmeẓẓuɣin; s waya, i wallen n tmeṭṭut n tmurt, ttbanen-d am yijeɛbuben ihendiyen, s yizerfan imeqranen akk ttfeggiḍen iḍarren-nsen s waddal. Ula d yiwen n mmi-s n tmeṭṭut tamettant, i tikkelt tamezwarut, ur yezmir ara ad yeqqim gar tmeẓẓuɣin-nni n lqeḍyan, yerna, mi ara yettɛeggiḍ akk ayen umi yezmer deg tmeẓẓuɣt, ad t-id-yefhem belli deg yal tallit ur nettwassen ara, yezmer ad yettwasfeḍ lḥerma, yerna akk tiɣawsiwin-agi n tmeẓẓuɣin n tmeẓẓuɣin ad ttwassersent deg urar am tmeẓẓuɣin n tmeẓẓuɣin ; ur yezmir ara ad yili akka mebla ma yella-d usḥissef i d-yettawin s timmad-is lḥirfa yellan deg yiɣes-is ad tettḥezzib deg-s am ljelba yettwaḥerrmen. Maca lɛada—d ayen yessewhamen! d acu ur tezmir ara ad t-tessiweḍ lɛada ?—Gayer sallies, ugar n lferḥ, lḥeq igerrzen, d repartees igerrzen, ur tesliḍ ara ɣef tmeẓẓuɣt-ik n tmeẓẓuɣt, akken ara tesleḍ ɣef tmeẓẓuɣt tamellalt n uzgen n lmitra n tkeṛṛust n yilef, mi ara tt-id-sbedden akka deg tmeẓẓuɣin n tmeẓẓuɣt n tmeẓẓuɣt ; yerna, am sḍis n yiburgen n Calais zdat n ugellid Edward, sḍis n yergazen-nni ay d-yesbedden lqeḍyan-nni ttjebden ɣer yiɣes n lmut, s yiwen n lḥelq deg yal aqerruy, akken ay tzemreḍ ad d-tiniḍ.

Ahat cwiṭ kan n uxemmem ara k-yeǧǧen tura ad d-tefkeḍ leḥsabat ɣef tmeskanin-nni n tmeẓdiyin (baleines) ay d-yellan s tɣawla – kra seg-sent ttwaḥesbent s tɣawla – n wergaz-a neɣ n wergaz-nni ay d-yeffɣen seg lbabur s tɣawla, yerna yeɣli-d. Axaṭer, mi ara d-yeffeɣ lḥiḍ, ad teqqimeḍ imiren deg lbabur, amzun teqqimeḍ deg tlemmast n tmeẓẓuɣin n tmeẓẓuɣin n tmeẓẓuɣin n tmeẓẓuɣin deg urar s lekmal-is, mi ara k-id-sfeḍḍen yal aẓru yettfeggiḍen, d tmeẓẓuɣt, d tmeẓẓuɣt. D ayen ixussen ugar; acku ur tezmireḍ ara ad teqqimeḍ ur tḥerzeḍ ara deg wul n tḥila-agi, acku aɣerrabu-nni yettḥezzib am tmeẓẓuɣt, yerna tḍeggreḍ-d yiwen ubrid d wayeḍ, mebla ma tesɛiḍ cwiṭ n tɣawsa ; yerna ala s kra n tmeẓẓuɣt i d-yesbedden iman-is d tmeẓẓuɣt n lebɣi d tigawt, i tzemreḍ ad terweleḍ ad k-rren d Mazeppa n, yerna ad terweleḍ s wanda iṭij yettwalin kullec s timmad-is ur yezmir ara ad k-yesfeḍ.

I tikkelt nniḍen: am wakken lḥir lqayen i d-yezgan kan s tmuɣli uqbel yerna yettwabder-d ɣef tmeẓẓuɣt, ahat d ayen ixuṣṣen ugar n tmeẓẓuɣt s timmad-is; axater, s tidet, lḥir-nni d ayen kan i d-yesfeḍḍen d tmeẓẓuɣt n tmeẓẓuɣt; yerna tesɛa-tt deg yiman-is, am wakken ay teṭṭef tmeẓdit-nni ay d-yettbanen d tin ur nesɛi lxir, aḍebsi n tmettant, d tkurt, d tbuciḍant ; ihi lḥirfa n tɣerɣert, akken tettmeslay s tsusmi ɣef yimesnallunen uqbel ad tt-id-ssiwḍen ɣer wurar n tidet – aya d ayen ay d-yettawin rrebrab n tidet ugar n yal aḥric niḍen n temsalt-a ay yesɛan lxir. Maca ayɣer ara d-nini ugar? Irgazen akk ttɣimin ttwaḍelqen deg yizirigen n yilef. Ttwaxelqen-d akk s yiḥemmalen deg yiɣil-nsen; maca ala mi ara ttwaṭṭfen deg tmuɣli n lmut i d-yettawin s tɣawla, s tɣawla, i d-ttwalin yimdanen lmut s tɣawsiwin n tudert i d-yessusem, i d-yettawin s tɣawla, i d-yettilin dima. Yerna ma yella d afelsuf, ɣas akken teqqimeḍ deg lbabur n yilef, ur tettḥulfuḍ ara deg wul-ik s yiwet n tɣawsa n tugdi ugar, wala ma yella teqqimeḍ zdat n tmes-ik n tmeddit s tmeẓẓuɣt, mačči s tmeẓẓuɣt, ɣer tama-k.
