Syin akkin, ad yekkes aṭas seg ccbaḥa n tmeɣra-nni lemmer ad yemlil St. George maca d yiwen n uɣersiw n tmurt yettɛeddayen, deg wemkan n uxeddim n yimenɣi ed uɣersiw ameqran n lqaɛa. Ur iban anda ara d-yesmekti belli yella kra n wass yessewḥac s tidet iselman-is, ala ma yella, s tidet, seg daxel. D acu kan, yezmer ad yettwaḥseb d ṣṣenf n umdan n yilel ur nelli d lebɣi; akken yebɣu yili lḥal, yeṭṭef-it lbaṭaṭa, ma yella ur t-id-yeṭṭef ara lbaṭaṭa. Nekk ttnadiɣ-t ɣef yiwen seg lɛerc-nneɣ.

Maca, sɣur yiduba (autorités) ay igerrzen akk ay d-yemxallafen, tamacahut-a tagrigit n Hercule ed tmeẓdit-nni ttwalin-tt d tin ay d-yekkan seg teqsiḍt taɛebrit n Yuna ed tmeẓdit-nni ay yellan mazal-itt d taqburt ugar; akked umḍiq-isâ; s tidet mxallafen aṭas. Ma yella ttnadiɣ ɣef demi-irebbi imir-nni, ayɣer mačči d nbi?

Mi d-yeqsed Brahma, neɣ Illu n Yilluten, i d-yenna Shaster, ad d-yerr amaḍal deffir yiwet seg tfekkiwin-is n tmeddurt, d netta ay d-yesnulfan Vishnoo, akken ad yesselhu ccɣel-nni; maca Vedas, neɣ idlisen n tmussni, ay ɣef ara d-yeffeɣ leqraya-nsen amzun d ayen ur nesɛi lmeɛna i Vishnoo uqbel ma yebda uxlaq, yerna ɣef waya ay yessefk ad yili deg-sen kra s talɣa n tmuɣliwin (indices) n tɣawsiwin n tɣawsiwin i yemseḍruyen (architectes) ilmeẓyen, Vedas-a llan ttwaḍḍfen deg tlemmast n waman; ihi Vishnoo yuɣal d aɣersiw deg yiwen n uɣersiw n yilel, yerna yessusem deg-s armi d lqaɛa, yesseḥbes iḍrisen iqedsen. IXEF 83. Yuna yettwaḥseb deg umezruy.

Yella-d wawal ɣef teqsiḍt n umezruy n Yuna d lbaṭaṭa deg yixef iɛeddan. Tura kra n Yinantucketen ur ttkelfen ara mliḥ ɣef teqsiḍt-a tamezruyant n Yuna ed tmeẓdit-nni. Maca syen llan kra n Yigrikkiyen ed Yiṛumyen ay yellan ttcukkuten, yerna, s tmuɣli-nsen ay d-yeffɣen seg yipaganiyen urtuduksiyen n tallit-nsen, ttcukkuten s tdukli deg teqsiḍt n Hercule ed tmeẓyant, d Arion ed tmeẓyant ; yerna ɣas akken, ccek-nsen deg wansayen-nni ur t-yeǧǧi ara ad yili wansayen-nni d yiwen n lḥeqq, ɣef waya akk.

Ssebba tameqrant ay t-yeǧǧan yiwen n wemɣar aqbur n Sag-Harbor ad yesteqsi tamacahut-nni n tɛebrit d tin-a:—Yella yesɛa yiwen seg Yidlisen-nni n zik n zik ay d-yesnulfan lḥir, yerna ttwacebbḥen s tfelwiyin n tmussni, ur nelli d tid n tussna; yiwen seg-sen yettgensis-d aɣersiw n Yuna s snat n tmeẓẓuɣin deg uqerruy-nnes – d taɣawsa ay d-yellan kan deg wayen yerzan yiwen n ṣṣenf n Leviathan (Aɣersiw n Yizerfan, ed tɣawsiwin n lqanun-nni), yerna ɣef waya ay sɛan yiselman awal-a: “Aḍris n yidinaren ad t-yesfeḍ”; Taqcict-is d tamecṭuḥt aṭas. Maca, ɣef waya, tiririt n tmuɣli n Bishop Jebb tella-d. Ur yessefk ara, i d-yesmekta Ubiṣup, ad nḥess Yuna amzun yettwaḍḍel deg uɛebbuḍ n yilef, maca amzun yettwaḍḍef s lqedd deg kra n tɣawsa n yimi-s. Yerna aya iban-d d ayen yesɛan lmeɛna deg Ubiṣup yelhan. Imi s tidet, aqemmum n Right Whale ad yesɛu snat n ttawilat n whist-tables, yerna ad yeqqim s lxir akk imyurar. Ahat ula d Yuna yezmer ad yeḥbes iman- is deg yiwen n usɣar ixuṣṣen; maca, ɣef tmuɣli tis snat, Aɣersiw n Yizerfan ur yesɛi ara tuɣmas.

Ssebba niḍen ay d-yessuter Sag-Harbor (yedda s yisem-nni) ɣef lebɣi-nnes n liman deg temsalt-a n nbi, d kra ur neẓri ara deg wayen yerzan tafekka-nnes ay yettwaḥebsen ed tɣawsiwin n tfekka n yilef. Maca ccwal-a ula d netta yeɣli-d ɣef lqaɛa, imi yiwen n umsefham (exegétisme) almani yettwali d akken Yuna yessefk ad yeṭṭef taɛessast deg tfekka n yiwen n uɣersiw n yilel yemmuten ay d-yettfeggiḍen – ula ma iserdasen ifṛensisen deg lḥemla n Rrus rran iɣersiwen-nsen yemmuten d iqiḍunen, yerna ttnadin deg-sen. Syin akkin, ttwaḥesben-d sɣur yemsefhamen niḍen n tmura n umaḍal, d akken asmi ay yettwaḍḍef Yuna ɣer yilel seg uɣerrabu n Yafa, srid, yerwel ɣer yiwen n uɣerrabu niḍen ay d-yezgan qrib-nnes, d aɣerrabu ay yesɛan yiwen n uɣersiw n yilel (baleine) i lmend n yiwen n uqerru n tmeṭṭut ; yerna, ad rnuɣ, ahat qqaren-as “Ilefḍan”, imi kra n tɣawsiwin n lḥirfa ttwaɣeḍsent ass-a “Aqcic n Yilel”, “Aɣersiw n Yilel”, “Aɣersiw n Yilel.” Ur llin ara ula d imnadiyen (exégétistes) ilmeden ur nebɣi ara, yerna qqaren-d belli aɣersiw-nni n yilel ay d-yebder wedlis n Yuna yella kan d ayen ara d-yesselḥun tudert —d aɛekkaz n waḍu ay d-yesfeḍḍen — ay ɣer yeɛum nbi-nni ay yellan deg tegnit n tmettant, yerna s waya ay d-yesselḥa seg lḥebs n waman. Sag-Harbor ameɣbun, ihi, iban-d d akken d ayen ixuṣṣen deg wakk timnaḍin. Maca mazal- as ssebba nniḍen i t- iǧǧan ur yesɛi ara liman. D aya, ma yella cfiɣ mliḥ: Yuna yečča-t lbaṭaṭa deg yilel n Mediterranean, sakkin, deffir kraḍ wussan, yessufeɣ-it-id anda-nniḍen deg kraḍ n wussan n tikli n Niniwa, d tamdint yellan ɣef Tigris, ugar n kraḍ n wussan n tikli ɣef tama n tlemmast n yilel n Mediterranean. Amek akka ?

Maca, eɛni ur yelli ara ubrid nniḍen i lbabur-nni akken ad yessiweḍ nnbi-nni ɣer tmurt-nni deg wemkan-nni n tlemmast n Niniwa? Ih. Ahat ad t-id-yawi s webrid n Cape of Good Hope. Maca ur d-nettmeslay ara ɣef uɛeddi s teɣzi n Yilel Agrakal akk, d uɛeddi niḍen ɣer sdat n Ugelmim n Faṛs d Yilel Azeggaɣ, tmuɣli am tagi ad d-tessiweḍ ɣer tɣerɣert n Tefriqt s lekmal-is deg kraḍ n wussan, ur d-nettmeslay ara ɣef waman n Tigris, qrib n wemkan n Nineveh, imi kra n tmuɣli-agi n Beha n tmurt n Nineveh, d tin ur nesɛi ara aṭas n lqaɛa. Ass-nni ay d-yesbedd Yuna ɣef Ugemmaḍ Utrim Yelhan deg yiwen n wass, ad yeqceɛ cciɛa n tuffɣa n tqacuct-nni tameqrant seg Bartholomew Diaz, win ay tt-id-yesfaydin s ccbaḥa-nnes, yerna s waya, amezruy atrar ad yerr d akeddab.

Maca akk lḥeqq-agi n tbuciḍant n Sag-Harbor aqbur yesskanay-d kan lfeṛḥ-is n tbuciḍant n lmeɛqul – d ayen mazal-it yettwaḥkem fell-as ugar, imi iwala belli drus kan n ulmud i yesɛa anagar ayen i d-yesfaydi seg yiṭij d yilel. Nniɣ-d d akken yesskanay-d kan lfeṛḥ-is aɛewwiq, aɛewwiq, d tnekra-ines n lbaṭel, n cceṛ mgal lmuqedmin yettwaqadren. Imi sɣur yiwen n lmuqeddem akaṭuli apurtugali, tikti-agi s timmad-is n tikli n Yuna ɣer Ninive s webrid n Cape of Good Espoir, tella-d sdat-s d ayen ara d-yesmektin s lbeɛd n lmuɛjiza tamatut. U akka i tella. Syin akkin, ar ass-a, Iṭerkiyen ay yesɛan aṭas n tmussni (éclairage) ttwalin s lqerb tamacahut tamezruyant n Yuna. Yerna azal n kraḍ n lqern aya, yiwen n umsebrid anglizi deg wedlis aqbur n Harris Voyages, yettmeslay-d ɣef yiwet n Tmesgida taṭerkit ay d-bnan i lmend n Yunus, yerna deg-s Tmesgida tella d lfeṭṭa n lmuɛjiza ay yettqerriben war ma yella kra n zzit.





IXEF 84. Asemres n tmeẓẓuɣt.

Akken ad tt-id-ssiwḍen s lfeɛl yerna s tɣawla, ttwadehnen yixef n tkeryas; yerna s lmeqsud-nni kan, kra n yimesnallunen (baleines) ttgen tamhelt (opération) am ta ɣef lbabur-nsen; ttṣeggimen ddaw. Ur yessefk ara daɣen ad ncukk belli imi abrid am wa ur yezmir ara ad d-yeglu s lxir, yezmer lḥal ur yesɛi ara lfayda yettwaḥeqren; s tmuɣli belli zzit d waman d iɛdawen; belli zzit d ayen yettḍeggiren, yerna d akken ayen yellan deg tmuɣli d akken ad yessiweḍ lbabur ad yettḍeggiɛ s tebɣest. Queequeg yumen s ljehd s tmeqwa n lbabur-nnes, yerna yiwet n tmeddit ur tɛeṭṭel ara deffir ma yeɣli-d lbabur almani Jungfrau, yeṭṭef ugar n lḥif n lɛada deg tmenɣiwt-nni; t-id-yesfeḍ ddaw n tlemmast-is, anda i d-yeɣli ɣef yidis, yerna yettḥezzib deg tmeẓẓuɣt-nni amzun yettnadi s lqerb ad d-yesḍemnen lɣella n yiɣes seg tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt. Iban-d amzun ixeddem s lḥeqq n kra n tmuɣli s timmad-is. Ula d netta ur yeqqim ara ur t-id-yesbedd ara tedyant-nni.

Sdat n tmeddit ttwakksen-d yiɣersiwen n yilel; maca akken kan i d-yeɣli lbabur-nni ɣur-sen, zzin-d, rewlen s tɣawla ; tikli ur nesɛi lqanun, am tbarra n Cleopatra seg Actium.

D acu kan, iɣerruba ḍefren-ten, yerna win n Stubb d netta ay d amezwaru. S lmejhud ameqran, Tashtego yessaweḍ, taggara-a, ad yeẓẓu yiwen n wuzzal; maca aɣersiw-nni yettwawten, war ma yessusem ula d yiwen, mazal-it ikemmel tikli-ines n tlemmast, s tfekka n tfekka i d-yernan. Tismin am tigi ur nettbeddil ara ɣef wuzzal yettwaẓẓan yessefk ad t-id-ssufɣent s tɣawla neɣ s tɣawla. Tuɣal d lḥeqq ad t-id-tessufɣeḍ i yilef yettfeggiḍen, neɣ ad t-tqebleḍ ad t-texṣeḍ. Maca akken ad yessiweḍ lbabur-nni ɣer tlemmast-is ur yezmir ara, yeɛum s tɣawla yerna s lɣiḍa. D acu i d-yeqqimen ihi?

Seg wakk allalen d tmussniwin yessewhamen, s tmeẓẓuɣin n ufus d tmeẓẓuɣin ur nesɛi lḥisab, ay ɣef yettwaḥettem s waṭas uɛessas n yilel aqbur, ula d yiwet ur tezmir ad tezger tmeẓẓuɣt-nni yelhan s tmeẓẓuɣt umi qqaren pitchpoling. Tafrut tameẓyant, neɣ tafruxt tameqqrant, deg wakk lxedma-s ur tettfaṛas ara am nettat. D ayen ur nesɛi lqima kan s yiwen n yilef yettazzalen inveterate; tidet-is tameqqrant d tfekka-s d azal igerrzen ay ɣer yettwaḍḍef s tɣawla lmus-nni ɣezzifen seg yiwen n lbabur ay d-yettḥezziben s lǧehd, ay d-yettḥezziben, ddaw n tɣawsiwin n tɣawsiwin n tɣawsiwin. Wuzzal d lḥenni llan deg-sen, lmus-nni akk yesɛa azal n mraw neɣ mraw d sin n yiḍarren deg teɣzi; lqecc-nni d ameẓyan aṭas ugar n win n lḥerma, yerna daɣen d ayen i d-yeffɣen s lfeṭṭa — d lpin. Yettwasseqdec s yiwen n lḥebs ameẓyan umi qqaren warp, s teɣzi tameqrant, s waya ay izemren ad t-id-yerr ɣer ufus deffir ma yettwaḍḍef.

Maca uqbel ad nruḥ ɣer sdat, yessefk ad d-nesmekti dagi, d akken ɣas akken lḥerpun tezmer ad tettwaḍḍef s yiwen n ubrid s tmeẓẓuɣt, maca qlil ay tt-yettgen ; yerna mi ara yettwaxdem, mazal-it ur yettkemmil ara s rrbeḥ, ɣef ljal n lqis ameqran ed teɣzi n tmeẓdit-nni ay d-yellan ddaw n tmeẓdit-nni, yerna aya yettuɣal, deg tidet, d lqella n tmussni. Am wakken d ayen amatu, ihi, yessefk ad t-id-tafeḍ di tazwara s tɣawla ɣer yiwen n uɣersiw n yilel, uqbel ad d-yeffeɣ kra n tɣawsa n tmeẓẓuɣt (pitchpoling) deg wurar.

Muqel tura ɣer Stubb; argaz ay d-yellan seg tmeẓẓuɣt-nnes (cool) d tmeẓẓuɣt-nnes (équilibre) ay d-yellan s lmeɛqul deg tegnit n tɣawsiwin n tɣawsiwin n tɣawsiwin, yella yettwassen s tɣawla akken ad yefru deg tɣawsiwin n tɣawsiwin n tɣawsiwin. Muqel-it; ibedd s lḥeqq deg tlemmast n lbabur yettfeggiḍen yettwaḍeggren; ttwaḍeggren s lqaɛa n lmal, aɣersiw-nni n tkerrust yella zdat-s rebɛin n yiḍarren. Yettḥadar lqecc-nni ɣezzifen s tfekka, yettmuqul snat neɣ tlata n tikkal ɣef teɣzi-ines akken ad iẓer ma yella srid akken iwata, Stubb yessemras s tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt deg yiwen ufus, akken ad yeḍmen taggara-ines tilellit deg ufus-is, ad yeǧǧ ayen nniḍen war ma yettwaḥbes. Syin yeṭṭef lmus-nni yeččur zdat n tlemmast n tmeẓẓuɣt-is, yessemlil-it ɣer tmeẓẓuɣt n yilel ; mi ara t-id-yeḍḍef s waya, ad iḥebbes s lqerb aqerruy n tbuciḍant yellan deg ufus-is, s waya ad yerfed lḥeqq alamma ibedd lbumbat-nni s lmizan ɣef tfekka-s, xemsa n yiḍarren deg yigenni. Yettxemmim-ik kra n tmeẓẓuɣt, yettqadar aɛekkaz ɣezzifen ɣef tɣenja-s. Aswir i d-iteddun s yiwen n uswir s tɣawla, ur nesɛi isem, deg yiwen n lqaɛa n tɣerɣert igerrzen, aẓru-nni icebḥen yettɛedday ɣef tɣerɣert n tɣerɣert, yerna yettqerrib deg umkan n tudert n yilef. Deg umkan n waman yettfeggiḍen, tura yettfeggiḍ idammen iberkanen.
