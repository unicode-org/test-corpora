Ula d, s tɣawla, ur yelli ara kra n lbaṭel deg usmenḍew-a.

IXEF 75. Aqerru n yilef n yidis ayeffus—Tamuɣli n tmuɣli yemgaraden.

Nezger lqaɛa, ad nwali tura tamuɣli ɣezzifen yelhan ɣef uqerru n Uqcic n Yilel Alemmas.

Am wakken ay d-yella deg ṣṣifa tamatut, aqerru n Sperm Whale n tmeqrant yezmer ad yettwasqerdec ed yiwen n ukamyun n yimenɣi n Rruman (ladɣa sdat, anda ay d-yesnulfa s tɣawla s waṭas); ihi, deg tmuɣli s wazal-is, aqerru n Right Whale yesɛa ccbaḥa ur nesɛi ara aṭas n tmussni am yiwen n uḍebsi ameqran s yiḍarren n galliot. Maca ma yella aɣersiw-agi d agellid, d amdakkel yettmuqulen s lḥir aṭas akken ad iɛeǧbeḍ taɛkemt.
