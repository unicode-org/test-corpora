Kraḍ n yifesyanen ed yergazen n yilel ay igerrzen ugar, yal yiwen s webrid-nnes, ur zmiren ad ten-id-afen s lqerb, yerna yal yiwen deg-sen d Amarikani ; yiwen n Nantuket, yiwen n Uẓekka, yiwen n wergaz n Kap. Tura, imi d Noël mi d-yeffeɣ uɣerrabu-nni seg beṛṛa n lḥebs-is, ɣef yiwen n umkan i nesɛa lḥal n tmurt n Polar, ɣas akken akk lweqt-nni nerwel seg-s ɣer tama n unẓul ; yerna s yal taswiɛt d yal dqiqa n tlemmast n tmurt i nessaweḍ, kra kra neǧǧa-d deffir-neɣ tagrest-nni ur nesɛi ara rreḥma, d wakk lḥala-ines ur nesɛi lḥeqq. D yiwen seg wid ur nesɛi ara aṭas n tɣawsiwin, maca mazal-it d aẓiḍan yerna d aẓiḍan aṭas n tmeddit n tmeɣra, mi s waḍu n lḥeqq, aɣerrabu-nni yettɛedday deg waman s ṣṣenf n tmeɣra n tmeɣra n tmeɣra, dɣa akken lliɣ ttṛuḥeɣ ɣer tɣerɣert s tɣawla n tmeɣra n tmeddit, akken kan lliɣ ttṛuḥeɣ ɣer tɣerɣert-iw. neki. Tilawt tezger lxuf; Ibedd ukabṭan Axab x rbeɛ n tqacuct nnes.

Ur iban ara kra n lbeḍna n waṭṭan n tfekka yettwasnen fell-as, ula d tujjya seg kra. Yettban-d am win yettwagezmen seg tmeqbert, mi ara tessexsi tmes s tmeqbert akk iferdisen war ma tečča-ten, neɣ tekkes-d yiwet n tfekka seg lǧehd-nsen aqbur i d-yesdukklen. Talɣa-ines akk ɣlayen, tban-d amzun d lbernz iǧehden, yerna ttwaxelqen-d deg yiwen n lqaleb ur yettbeddil ara, am Perseus n Cellini i d-yesnulfa. Tessufeɣ-d abrid-is n tuffɣa seg gar yiɣes-is aẓiḍan, yerna tkemmel srid ɣer yiwet n tama n wudem-is d tɣerɣert-is i d-yesɣerqen s tmeẓẓuɣt n tmeẓẓuɣt, alamma teɣli deg lqecc-is, twalaḍ yiwet n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt, d tamellalt n tmeẓẓuɣt. Ttcabin ɣer tmeẓẓuɣt-nni n tmeẓẓuɣt i d-yettilin tikwal deg tɣerɣert srid, n tɣerɣert n tteǧra tameqrant, mi ara tt-id-yesfeḍ lebraq n ufella s tmeẓẓuɣt, yerna mebla ma tesseɣli ula d yiwet n tfekka, tettqerrib yerna tessufuɣ-d lqecc-nni seg ufella ar ddaw, uqbel ad teffeɣ ɣer wakal, teǧǧa aseklu-nni mazal-it d azegzaw, maca mazal-it d azegzaw. Ma yella lbeɛd-nni ilul-d yid-s, neɣ d lḥirfa i d-yeǧǧa kra n lǧerḥ ur nesɛi lḥeqq, ulac win izemren ad d-yini s tɣawla. S kra n lqanun n tmuɣli, deg tikli-nni akk, drus neɣ ulac ayen ay d-yettwabedren fell-as, ladɣa seg yimeddukal-nnes. Maca yiwet n tikkelt, d ameqqran n Tashtego, yiwen n uhendi aqbur n Gay-Head gar yixeddamen n tkebbanit-nni, yenna-d s tmuɣli n tkerkas (superstition) d akken mačči armi ay d-yewweḍ rebɛin n yiseggasen ay d-yuɣal Ahab d win ay d-yettwasbedden s tɣawla-nni, sakkin, yusa-d fell-as waya, maci deg lɣiḍa n kra n ccwal n tmettant, wanag deg yiwen n ccwal n tɣawsiwin deg yilel. Maca, lḥeqq-agi n tmeẓẓuɣt iban-d d ayen ur nelli ara d ayen yelhan, s wayen ay d-yesmekta yiwen n Manxman aẓiḍan, d argaz aqbur n tmeqbert, ay werjin yeffeɣ seg Nantucket s lbabur, werjin yessers-d allen-nnes ɣef Ahab n tmeẓẓuɣt. D acu kan, ansay (tradition) aqbur n yilel, tiktiwin (crédulités) ur nesɛi lmeɛna, ssefran-d s weɣref (populaire) Manxman-a aqbur s tezmert (pouvoirs) n tmuɣli uqbel n tgemmi (prénature). Ihi ula d yiwen n umaru amellal ur t-iqebbel ara s lǧehd mi d-yenna belli ma yella kra n wass ad yettwasbedd Kapṭan Ahab s lḥir – d ayen ay izemren ad d-yeḍru s lqella, dɣa s tmuɣli-nnes – ihi, win ara ixeddmen ccɣel-nni aneggaru i wid yemmuten, ad as-d-yaf lḥeqq n tlalit seg tmeqbert arma d aqerruy-nnes.

S tezmert i yi-d-iḥuza akk lḥirfa n Ahab, d lmarka n lḥirfa i t-id-yesfeḍḍen, armi deg kra n teswiɛin timezwura ur ẓriɣ ara belli ula d cwiṭ seg lḥirfa-agi n lḥirfa ur d-telli ara s ssebba n uḍar-is amellal aberrani i ɣef ibedd s tɣawla. Yusa-d zik ɣur-i belli aḍar-agi n lgaz deg yilel yettwaxdem seg yiɣes n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt n tmeẓẓuɣt. “Aye, yettwasfeḍ ɣef Japun”, i d-yenna yiwen n wass Uhendi aqbur n Gay-Head; "maca am lbabur-is i d-yesfaydin seg lbabur-is, iceggeɛ-d lbabur nniḍen mebla ma yuɣal-d ɣer uxxam fell-as. Yesɛa aɛekkaz n lbabur-nsen."

Ttwawteɣ s tmuɣli n yiwen i yeṭṭef. Deg yal tama n tmeqbert n tquart n Pequod, yerna qrib aṭas ɣer tmeqbert n mizzen, yella yiwen n lḥebs n auger, yettwasfeḍ azal n uzgen n lmitra neɣ azal-is, deg tfelwit. Aḍar-is n yiɣes yeqqim deg tɣerɣert-nni; yiwen ufus yettwasnekren, yerna yettwaṭṭef s lqaɛa; Kapṭan Ahab ibedd s lqedd, yettmuqul srid nnig n tlemmast n uɣerrabu i d-yettawin yal tikkelt. Yella lbaḍna n lǧehd iǧehden, lebɣi iḥerzen, ur nettḥezzib ara, deg tmeɣra n tmuɣli-nni i d-yeqqimen yerna ur nesɛi tugdi, sdat. Ur d-yenni ula d yiwen wawal; ula d ifesyanen-is ur as-d-nnan ara kra; ɣas akken s wakk tignatin-nsen d tmuɣliwin-nsen timeẓyanin, sseknen-d s tɣawla lḥir ur nesɛi lḥir, ma ulac d ayen i t-iḥuzan, n tmuɣli n tmeṭṭut ddaw n yiwen n wallen n tmeṭṭut yettwaḥeqren.
