ໃບອະນຸຍາດ GUTENBERG ໂຄງການເຕັມ

ກະ​ລຸ​ນາ​ອ່ານ​ນີ້​ກ່ອນ​ທີ່​ທ່ານ​ຈະ​ແຈກ​ຢາຍ​ຫຼື​ນໍາ​ໃຊ້​ວຽກ​ງານ​ນີ້
ເພື່ອປົກປ້ອງພາລະກິດຂອງ Project Gutenberg™ ໃນການສົ່ງເສີມການແຈກຢາຍວຽກງານເອເລັກໂຕຼນິກໂດຍບໍ່ເສຍຄ່າ, ໂດຍການນໍາໃຊ້ ຫຼືການແຈກຢາຍວຽກງານນີ້ (ຫຼືວຽກງານອື່ນໆທີ່ກ່ຽວຂ້ອງກັບຄໍາວ່າ "Project Gutenberg"), ທ່ານຕົກລົງເຫັນດີທີ່ຈະປະຕິບັດຕາມເງື່ອນໄຂທັງຫມົດຂອງໃບອະນຸຍາດ Full Project Gutenberg™ ທີ່ມີຢູ່ໃນໄຟລ໌ນີ້ ຫຼືອອນໄລນ໌ຢູ່ www.gutenberg.org/license.
ພາກ​ທີ 1. ເງື່ອນ​ໄຂ​ທົ່ວ​ໄປ​ຂອງ​ການ​ນໍາ​ໃຊ້​ແລະ​ການ​ແຈກ​ຢາຍ​ໂຄງ​ການ​ການ​ຜະ​ລິດ​ເອ​ເລັກ​ໂຕຣ​ນິກ Gutenberg™​
1.ກ. ໂດຍການອ່ານ ຫຼືນຳໃຊ້ສ່ວນໃດສ່ວນໜຶ່ງຂອງວຽກອີເລັກໂທຣນິກຂອງໂຄງການ Gutenberg™ ນີ້, ທ່ານຊີ້ບອກວ່າທ່ານໄດ້ອ່ານ, ເຂົ້າໃຈ, ຍອມຮັບ ແລະຍອມຮັບເງື່ອນໄຂທັງໝົດຂອງຂໍ້ຕົກລົງໃບອະນຸຍາດ ແລະຊັບສິນທາງປັນຍາ (ເຄື່ອງຫມາຍການຄ້າ/ລິຂະສິດ). ຖ້າທ່ານບໍ່ເຫັນດີທີ່ຈະປະຕິບັດຕາມເງື່ອນໄຂທັງຫມົດຂອງຂໍ້ຕົກລົງນີ້, ທ່ານຕ້ອງຢຸດການນໍາໃຊ້ແລະສົ່ງຄືນຫຼືທໍາລາຍສໍາເນົາທັງຫມົດຂອງວຽກງານເອເລັກໂຕຣນິກ Project Gutenberg™ ຢູ່ໃນຄອບຄອງຂອງທ່ານ. ຖ້າທ່ານໄດ້ຈ່າຍຄ່າທໍານຽມສໍາລັບການໄດ້ຮັບສໍາເນົາຫຼືການເຂົ້າເຖິງວຽກງານເອເລັກໂຕຣນິກຂອງ Project Gutenberg™ ແລະທ່ານບໍ່ຍອມຮັບທີ່ຈະຜູກມັດໂດຍຂໍ້ກໍານົດຂອງສັນຍານີ້, ທ່ານອາດຈະໄດ້ຮັບເງິນຄືນຈາກບຸກຄົນຫຼືນິຕິບຸກຄົນທີ່ທ່ານໄດ້ຈ່າຍຄ່າທໍານຽມຕາມທີ່ໄດ້ກໍານົດໄວ້ໃນວັກ 1.E.8.
1.ຂ. "ໂຄງການ Gutenberg" ແມ່ນເຄື່ອງຫມາຍການຄ້າທີ່ຈົດທະບຽນ. ມັນພຽງແຕ່ອາດຈະຖືກນໍາໃຊ້ໃນຫຼືກ່ຽວຂ້ອງກັບວິທີການໃດຫນຶ່ງກັບວຽກງານເອເລັກໂຕຣນິກໂດຍປະຊາຊົນຜູ້ທີ່ຕົກລົງເຫັນດີທີ່ຈະຜູກມັດໂດຍຂໍ້ກໍານົດຂອງສັນຍານີ້. ມີບາງອັນທີ່ເຈົ້າສາມາດເຮັດໄດ້ກັບວຽກອີເລັກໂທຣນິກຂອງ Project Gutenberg™ ສ່ວນໃຫຍ່ ເຖິງແມ່ນວ່າບໍ່ໄດ້ປະຕິບັດຕາມເງື່ອນໄຂເຕັມທີ່ຂອງຂໍ້ຕົກລົງນີ້. ເບິ່ງວັກ 1.C ຂ້າງລຸ່ມນີ້. ມີຫຼາຍສິ່ງຫຼາຍຢ່າງທີ່ເຈົ້າສາມາດເຮັດໄດ້ກັບວຽກອີເລັກໂທຣນິກຂອງ Project Gutenberg™ ຖ້າທ່ານປະຕິບັດຕາມເງື່ອນໄຂຂອງຂໍ້ຕົກລົງນີ້ ແລະຊ່ວຍຮັກສາການເຂົ້າເຖິງວຽກເອເລັກໂຕຣນິກ Project Gutenberg™ ໃນອະນາຄົດໄດ້ໂດຍບໍ່ເສຍຄ່າ. ເບິ່ງວັກ 1.E ຂ້າງລຸ່ມນີ້.
1.ຄ. ໂຄງ​ການ Gutenberg Literary Archive Foundation (“ມູນ​ນິ​ທິ” ຫຼື PGLAF), ເປັນ​ເຈົ້າ​ຂອງ​ລິ​ຂະ​ສິດ​ການ​ລວບ​ລວມ​ໃນ​ການ​ເກັບ​ກໍາ​ຂອງ Project Gutenberg™ ວຽກ​ງານ​ເອ​ເລັກ​ໂຕຣ​ນິກ. ເກືອບທັງຫມົດວຽກງານສ່ວນບຸກຄົນໃນການເກັບກໍາແມ່ນຢູ່ໃນສາທາລະນະໃນສະຫະລັດ. ຖ້າວຽກງານສ່ວນບຸກຄົນບໍ່ໄດ້ຮັບການປົກປ້ອງໂດຍກົດຫມາຍລິຂະສິດໃນສະຫະລັດແລະທ່ານຕັ້ງຢູ່ໃນສະຫະລັດ, ພວກເຮົາບໍ່ອ້າງສິດທີ່ຈະປ້ອງກັນບໍ່ໃຫ້ທ່ານເຮັດສໍາເນົາ, ແຈກຢາຍ, ປະຕິບັດ, ສະແດງຫຼືສ້າງຜົນງານທີ່ອີງໃສ່ການເຮັດວຽກຕາບໃດທີ່ການອ້າງອີງທັງຫມົດຂອງໂຄງການ Gutenberg ໄດ້ຖືກໂຍກຍ້າຍ. ແນ່ນອນ, ພວກເຮົາຫວັງວ່າທ່ານຈະສະຫນັບສະຫນູນພາລະກິດຂອງ Project Gutenberg™ ຂອງການສົ່ງເສີມການເຂົ້າເຖິງວຽກງານເອເລັກໂຕຣນິກໂດຍບໍ່ເສຍຄ່າໂດຍການແບ່ງປັນໂຄງການ Gutenberg™ ເຮັດວຽກທີ່ສອດຄ່ອງກັບເງື່ອນໄຂຂອງຂໍ້ຕົກລົງນີ້ສໍາລັບການຮັກສາຊື່ໂຄງການ Gutenberg™ ທີ່ກ່ຽວຂ້ອງກັບວຽກງານ. ທ່ານສາມາດປະຕິບັດຕາມເງື່ອນໄຂຂອງຂໍ້ຕົກລົງນີ້ໄດ້ຢ່າງງ່າຍດາຍໂດຍການຮັກສາການເຮັດວຽກນີ້ໃນຮູບແບບດຽວກັນກັບໃບອະນຸຍາດໂຄງການ Gutenberg™ ເຕັມທີ່ຕິດຄັດມາໃນເວລາທີ່ທ່ານແບ່ງປັນມັນໂດຍບໍ່ເສຍຄ່າກັບຜູ້ອື່ນ.
1.D. ກົດຫມາຍລິຂະສິດຂອງສະຖານທີ່ທີ່ທ່ານຕັ້ງຢູ່ຍັງປົກຄອງສິ່ງທີ່ທ່ານສາມາດເຮັດໄດ້ກັບວຽກງານນີ້. ກົດໝາຍລິຂະສິດໃນປະເທດສ່ວນໃຫຍ່ຢູ່ໃນສະພາບປ່ຽນແປງຢ່າງຕໍ່ເນື່ອງ. ຖ້າທ່ານຢູ່ນອກສະຫະລັດ, ກວດເບິ່ງກົດຫມາຍຂອງປະເທດຂອງທ່ານນອກເຫນືອຈາກເງື່ອນໄຂຂອງສັນຍານີ້ກ່ອນທີ່ຈະດາວໂຫລດ, ຄັດລອກ, ການສະແດງ, ປະຕິບັດ, ແຈກຢາຍຫຼືສ້າງຜົນງານທີ່ອີງໃສ່ການເຮັດວຽກນີ້ຫຼືວຽກງານອື່ນໆຂອງ Project Gutenberg™. ມູນນິທິບໍ່ໄດ້ເປັນຕົວແທນກ່ຽວກັບສະຖານະພາບລິຂະສິດຂອງວຽກງານໃດໆໃນປະເທດອື່ນນອກເຫນືອຈາກສະຫະລັດ.

1.E.1. ປະໂຫຍກຕໍ່ໄປນີ້, ດ້ວຍການເຊື່ອມຕໍ່ກັບ, ຫຼືການເຂົ້າເຖິງທັນທີທັນໃດ, ໃບອະນຸຍາດໂຄງການ Gutenberg™ ສະບັບເຕັມຈະຕ້ອງປາກົດຂຶ້ນຢ່າງເດັ່ນຊັດເມື່ອໃດກໍ່ຕາມທີ່ສໍາເນົາຂອງວຽກຂອງ Project Gutenberg™ (ວຽກໃດກໍ່ຕາມທີ່ຄໍາວ່າ "Project Gutenberg" ປາກົດ, ຫຼືຄໍາວ່າ "Project Gutenberg" ທີ່ກ່ຽວຂ້ອງ) ຖືກເຂົ້າເຖິງ, ສະແດງ, ປະຕິບັດ, ເບິ່ງ, ຄັດລອກຫຼືແຈກຢາຍ:

1.F.1. ອາສາສະໝັກ ແລະພະນັກງານຂອງໂຄງການ Gutenberg ໃຊ້ຄວາມພະຍາຍາມຢ່າງຫຼວງຫຼາຍເພື່ອກໍານົດ, ເຮັດການຄົ້ນຄວ້າລິຂະສິດກ່ຽວກັບ, ຖອດຂໍ້ຄວາມ ແລະອ່ານບົດບັນທຶກທີ່ບໍ່ໄດ້ຮັບການປົກປ້ອງໂດຍກົດໝາຍລິຂະສິດຂອງສະຫະລັດ ໃນການສ້າງຄໍເລັກຊັນ Project Gutenberg™. ເຖິງວ່າຈະມີຄວາມພະຍາຍາມເຫຼົ່ານີ້, ວຽກງານອີເລັກໂທຣນິກຂອງ Project Gutenberg™, ແລະສື່ທີ່ເຂົາເຈົ້າອາດຈະຖືກເກັບໄວ້, ອາດຈະມີ “ຂໍ້ບົກພ່ອງ” ເຊັ່ນວ່າ, ແຕ່ບໍ່ຈໍາກັດ, ຂໍ້ມູນບໍ່ຄົບຖ້ວນ, ບໍ່ຖືກຕ້ອງ ຫຼືເສຍຫາຍ, ຄວາມຜິດພາດການຖອດຂໍ້ຄວາມ, ການລະເມີດລິຂະສິດ ຫຼືຊັບສິນທາງປັນຍາອື່ນໆ, ແຜ່ນທີ່ຜິດປົກກະຕິ ຫຼືເສຍຫາຍ ຫຼືສື່ອື່ນໆ, ໄວຣັສຄອມພິວເຕີ ຫຼືບໍ່ສາມາດອ່ານລະຫັດຄອມພິວເຕີຂອງທ່ານໄດ້.
1.F.2. ການຮັບປະກັນແບບຈຳກັດ, ການປະຕິເສດຄວາມເສຍຫາຍ - ຍົກເວັ້ນ "ສິດທິໃນການທົດແທນ ຫຼື ການຄືນເງິນ" ທີ່ອະທິບາຍໄວ້ໃນວັກ 1.F.3, ໂຄງການມູນນິທິເກັບວັນນະຄະດີ Gutenberg, ເຈົ້າຂອງເຄື່ອງໝາຍການຄ້າຂອງ Project Gutenberg™, ແລະພາກສ່ວນອື່ນທີ່ແຈກຢາຍວຽກອີເລັກໂທຣນິກຂອງ Project Gutenberg™ ພາຍໃຕ້ຂໍ້ຕົກລົງນີ້, ຄ່າທຳນຽມທາງກົດໝາຍ ແລະ ຄ່າໃຊ້ຈ່າຍທັງໝົດ. ທ່ານຕົກລົງເຫັນດີວ່າທ່ານບໍ່ມີການແກ້ໄຂສໍາລັບການລະເລີຍ, ຄວາມຮັບຜິດຊອບຢ່າງເຂັ້ມງວດ, ການລະເມີດການຮັບປະກັນຫຼືການລະເມີດສັນຍາຍົກເວັ້ນສິ່ງທີ່ລະບຸໄວ້ໃນ PARAGRAPH 1.F.3. ທ່ານຕົກລົງເຫັນດີວ່າຮາກຖານ, ເຈົ້າຂອງເຄື່ອງຫມາຍການຄ້າ, ແລະຜູ້ຈັດຈໍາຫນ່າຍໃດໆພາຍໃຕ້ຂໍ້ຕົກລົງນີ້ຈະບໍ່ຮັບຜິດຊອບຕໍ່ເຈົ້າສໍາລັບຕົວຈິງ, ໂດຍກົງ, ໂດຍທາງອ້ອມ, ຜົນສະທ້ອນ, ລົງໂທດຫຼືຄວາມເສຍຫາຍທີ່ເກີດຂຶ້ນໂດຍບັງເອີນ. ຂອງຄວາມເສຍຫາຍດັ່ງກ່າວ.
1.F.3. ຈໍາກັດສິດທິໃນການທົດແທນຫຼືການຄືນເງິນ - ຖ້າທ່ານຄົ້ນພົບຂໍ້ບົກພ່ອງໃນວຽກງານເອເລັກໂຕຣນິກນີ້ພາຍໃນ 90 ມື້ທີ່ໄດ້ຮັບມັນ, ທ່ານສາມາດໄດ້ຮັບເງິນຄືນ (ຖ້າມີ) ທີ່ທ່ານຈ່າຍໃຫ້ກັບມັນໂດຍການສົ່ງຄໍາອະທິບາຍເປັນລາຍລັກອັກສອນກັບບຸກຄົນທີ່ທ່ານໄດ້ຮັບການເຮັດວຽກຈາກ. ຖ້າທ່ານໄດ້ຮັບວຽກງານໃນສື່ກາງ, ທ່ານຕ້ອງສົ່ງຄືນສື່ທີ່ມີຄໍາອະທິບາຍເປັນລາຍລັກອັກສອນຂອງທ່ານ. ບຸກຄົນ ຫຼື ນິຕິບຸກຄົນທີ່ສະໜອງວຽກງານທີ່ບົກພ່ອງໃຫ້ທ່ານສາມາດເລືອກໃຫ້ສະໜອງສໍາເນົາແທນແທນການຄືນເງິນໄດ້. ຖ້າທ່ານໄດ້ຮັບວຽກທາງອີເລັກໂທຣນິກ, ບຸກຄົນ ຫຼື ໜ່ວຍງານທີ່ສະໜອງໃຫ້ເຈົ້າອາດຈະເລືອກໃຫ້ໂອກາດທີສອງແກ່ເຈົ້າໃນການຮັບວຽກທາງອີເລັກໂທຣນິກແທນການສົ່ງເງິນຄືນ. ຖ້າສໍາເນົາທີສອງມີຂໍ້ບົກພ່ອງ, ທ່ານອາດຈະຕ້ອງການເງິນຄືນເປັນລາຍລັກອັກສອນໂດຍບໍ່ມີໂອກາດທີ່ຈະແກ້ໄຂບັນຫາ.
1.F.4. ຍົກເວັ້ນສິດຈໍາກັດການທົດແທນ ຫຼືການຄືນເງິນທີ່ໄດ້ກໍານົດໄວ້ໃນວັກ 1.F.3, ວຽກງານນີ້ແມ່ນສະຫນອງໃຫ້ທ່ານ 'AS-IS', ໂດຍບໍ່ມີການຮັບປະກັນອື່ນໆຂອງທຸກປະເພດ, ສະແດງອອກຫຼືໂດຍຄວາມຫມາຍ, ລວມທັງແຕ່ບໍ່ຈໍາກັດການຮັບປະກັນການໃຫ້ບໍລິການ.
1.F.5. ບາງລັດບໍ່ອະນຸຍາດໃຫ້ປະຕິເສດການຮັບປະກັນບາງສ່ວນ ຫຼືການຍົກເວັ້ນ ຫຼືການຈຳກັດຄວາມເສຍຫາຍບາງປະເພດ. ຖ້າການປະຕິເສດຄວາມຮັບຜິດຊອບຫຼືຂໍ້ຈໍາກັດໃດໆທີ່ລະບຸໄວ້ໃນຂໍ້ຕົກລົງນີ້ລະເມີດກົດຫມາຍຂອງລັດທີ່ໃຊ້ໃນຂໍ້ຕົກລົງນີ້, ຂໍ້ຕົກລົງຈະຖືກຕີຄວາມຫມາຍເພື່ອເຮັດໃຫ້ການປະຕິເສດຄວາມຮັບຜິດຊອບສູງສຸດຫຼືຂໍ້ຈໍາກັດທີ່ອະນຸຍາດໂດຍກົດຫມາຍຂອງລັດທີ່ໃຊ້ໄດ້. ຄວາມບໍ່ເປັນຈິງ ຫຼື ຄວາມບໍ່ບັງຄັບໃຊ້ຂອງຂໍ້ສະ ເໜີ ໃດໆຂອງຂໍ້ຕົກລົງນີ້ຈະບໍ່ເປັນໂມຄະຂໍ້ ກຳ ນົດທີ່ຍັງເຫຼືອ.
1.F.6. INDEMNITY - ທ່ານຕົກລົງເຫັນດີທີ່ຈະຊົດເຊີຍແລະຖືມູນນິທິ, ເຈົ້າຂອງເຄື່ອງຫມາຍການຄ້າ, ຕົວແທນຫຼືພະນັກງານຂອງມູນນິທິ, ຜູ້ໃດທີ່ສະຫນອງສໍາເນົາຂອງວຽກງານເອເລັກໂຕຣນິກຂອງ Project Gutenberg™ ຕາມຂໍ້ຕົກລົງນີ້, ແລະອາສາສະຫມັກທີ່ກ່ຽວຂ້ອງກັບການຜະລິດ, ການສົ່ງເສີມແລະການຈໍາຫນ່າຍຂອງ Project Gutenberg™ ວຽກງານເອເລັກໂຕຣນິກ, ບໍ່ມີອັນຕະລາຍຈາກຄວາມຮັບຜິດຊອບທັງຫມົດ, ຄ່າໃຊ້ຈ່າຍແລະຄ່າໃຊ້ຈ່າຍ, ລວມທັງຄ່າທໍານຽມທາງກົດຫມາຍ, ທີ່ເກີດຂຶ້ນໂດຍກົງຫຼືໂດຍທາງອ້ອມຂອງການຈໍາຫນ່າຍນີ້ (ຫຼືໂດຍທາງອ້ອມ) ທີ່ທ່ານເຮັດຈາກນີ້ (ຫຼືໂດຍທາງອ້ອມ) ການເຮັດວຽກຂອງ Project Gutenberg™, (b) ການປ່ຽນແປງ, ການດັດແກ້, ຫຼືການເພີ່ມເຕີມຫຼືການລົບຕໍ່ການເຮັດວຽກໂຄງການ Gutenberg™, ແລະ (c) ຂໍ້ບົກພ່ອງໃດໆທີ່ທ່ານເຮັດໃຫ້ເກີດ.
ພາກທີ 2. ຂໍ້ມູນກ່ຽວກັບພາລະກິດຂອງໂຄງການ Gutenberg™
ໂຄງ​ການ Gutenberg™ ແມ່ນ​ຄ້າຍ​ຄື​ກັນ​ກັບ​ການ​ແຈກ​ຢາຍ​ຟຣີ​ຂອງ​ວຽກ​ງານ​ເອ​ເລັກ​ໂຕຣ​ນິກ​ໃນ​ຮູບ​ແບບ​ທີ່​ສາ​ມາດ​ອ່ານ​ໄດ້​ໂດຍ​ການ​ຫຼາກ​ຫຼາຍ​ຂອງ​ຄອມ​ພິວ​ເຕີ​ລວມ​ທັງ​ຄອມ​ພິວ​ເຕີ​ລ້າ​ສະ​ໄຫມ​, ເກົ່າ​, ກາງ​ແລະ​ໃຫມ່​. ມັນ​ເປັນ​ໄປ​ໄດ້​ຍ້ອນ​ຄວາມ​ພະ​ຍາ​ຍາມ​ຂອງ​ຫຼາຍ​ຮ້ອຍ​ຄົນ​ຂອງ​ອາ​ສາ​ສະ​ຫມັກ​ແລະ​ການ​ບໍ​ລິ​ຈາກ​ປະ​ຊາ​ຊົນ​ໃນ​ທຸກ​ຊັ້ນ​ຄົນ​ຂອງ​ຊີ​ວິດ.
ອາສາສະໝັກ ແລະ ການສະໜັບສະໜູນທາງດ້ານການເງິນເພື່ອສະໜອງການຊ່ວຍເຫຼືອແກ່ອາສາສະໝັກທີ່ເຂົາເຈົ້າຕ້ອງການແມ່ນມີຄວາມສຳຄັນຕໍ່ການບັນລຸເປົ້າໝາຍຂອງ Project Gutenberg™ ແລະ ຮັບປະກັນວ່າຊຸດສະສົມຂອງ Project Gutenberg™ ຈະຍັງຄົງມີຢູ່ຢ່າງເສລີສຳລັບຄົນລຸ້ນຕໍ່ໆໄປ. ໃນ​ປີ 2001, ມູນ​ນິ​ທິ Archive ວັນ​ນະ​ຄະ​ດີ​ຂອງ​ໂຄງ​ການ Gutenberg ໄດ້​ຖືກ​ສ້າງ​ຕັ້ງ​ຂຶ້ນ​ເພື່ອ​ສະ​ຫນອງ​ອະ​ນາ​ຄົດ​ທີ່​ປອດ​ໄພ​ແລະ​ຖາ​ວອນ​ສໍາ​ລັບ​ໂຄງ​ການ Gutenberg™ ແລະ​ຄົນ​ລຸ້ນ​ໃນ​ອະ​ນາ​ຄົດ. ເພື່ອຮຽນຮູ້ເພີ່ມເຕີມກ່ຽວກັບໂຄງການ Gutenberg Literary Archive Foundation ແລະວິທີທີ່ຄວາມພະຍາຍາມແລະການບໍລິຈາກຂອງທ່ານສາມາດຊ່ວຍໄດ້, ເບິ່ງພາກທີ 3 ແລະ 4 ແລະຫນ້າຂໍ້ມູນມູນນິທິຢູ່ www.gutenberg.org.

ໂຄງການມູນນິທິເກັບວັນນະຄະດີ Gutenberg ເປັນບໍລິສັດການສຶກສາທີ່ບໍ່ຫວັງຜົນກຳໄລ 501(c)(3) ທີ່ຈັດຂື້ນພາຍໃຕ້ກົດໝາຍຂອງລັດ Mississippi ແລະໄດ້ຮັບການຍົກເວັ້ນພາສີຈາກການບໍລິການລາຍຮັບພາຍໃນ. EIN ຂອງມູນນິທິ ຫຼືໝາຍເລກປະຈໍາຕົວພາສີຂອງລັດຖະບານກາງແມ່ນ 64-6221541. ການປະກອບສ່ວນເຂົ້າໃນໂຄງການ Gutenberg Literary Archive Foundation ແມ່ນສາມາດຫັກພາສີໄດ້ໃນຂອບເຂດເຕັມທີ່ອະນຸຍາດໂດຍກົດໝາຍຂອງລັດຖະບານກາງສະຫະລັດ ແລະກົດໝາຍຂອງລັດຂອງເຈົ້າ.
ຫ້ອງການທຸລະກິດຂອງມູນນິທິຕັ້ງຢູ່ທີ່ 809 North 1500 West, Salt Lake City, UT 84116, (801) 596-1887. ລິ້ງຕິດຕໍ່ທາງອີເມລ໌ ແລະຂໍ້ມູນການຕິດຕໍ່ຫຼ້າສຸດສາມາດພົບໄດ້ທີ່ເວັບໄຊທ໌ຂອງມູນນິທິ ແລະໜ້າທາງການທີ່ www.gutenberg.org/contact
ພາກທີ 4. ຂໍ້ມູນກ່ຽວກັບການບໍລິຈາກໃຫ້ແກ່ໂຄງການ Gutenberg Literary Archive Foundation
ໂຄງການ Gutenberg ™ແມ່ນຂຶ້ນກັບແລະບໍ່ສາມາດຢູ່ລອດໄດ້ໂດຍບໍ່ມີການສະຫນັບສະຫນູນສາທາລະນະຢ່າງກວ້າງຂວາງແລະການບໍລິຈາກເພື່ອປະຕິບັດພາລະກິດຂອງການເພີ່ມຈໍານວນສາທາລະນະແລະວຽກງານທີ່ມີໃບອະນຸຍາດທີ່ສາມາດແຈກຢາຍຢ່າງເສລີໃນຮູບແບບທີ່ເຄື່ອງຈັກສາມາດອ່ານໄດ້ໂດຍອຸປະກອນທີ່ກວ້າງຂວາງທີ່ສຸດລວມທັງອຸປະກອນທີ່ລ້າສະໄຫມ. ການບໍລິຈາກເລັກນ້ອຍຫຼາຍອັນ (1 ໂດລາຫາ 5,000 ໂດລາ) ແມ່ນມີຄວາມສຳຄັນເປັນພິເສດຕໍ່ການຮັກສາສະຖານະພາບການຍົກເວັ້ນພາສີກັບ IRS.
ມູນນິທິມຸ່ງໝັ້ນທີ່ຈະປະຕິບັດຕາມກົດໝາຍທີ່ຄວບຄຸມການກຸສົນ ແລະ ການບໍລິຈາກເພື່ອການກຸສົນໃນທັງໝົດ 50 ລັດຂອງສະຫະລັດ. ຂໍ້ກໍານົດການປະຕິບັດຕາມແມ່ນບໍ່ເປັນເອກະພາບແລະມັນຕ້ອງໃຊ້ຄວາມພະຍາຍາມຫຼາຍ, ເອກະສານຫຼາຍແລະຄ່າທໍານຽມຫຼາຍເພື່ອຕອບສະຫນອງແລະປະຕິບັດຕາມຂໍ້ກໍານົດເຫຼົ່ານີ້. ພວກເຮົາບໍ່ໄດ້ຂໍເງິນບໍລິຈາກໃນສະຖານທີ່ທີ່ພວກເຮົາບໍ່ໄດ້ຮັບການຢືນຢັນເປັນລາຍລັກອັກສອນກ່ຽວກັບການປະຕິບັດຕາມ. ເພື່ອສົ່ງການບໍລິຈາກຫຼືກໍານົດສະຖານະຂອງການປະຕິບັດຕາມລັດໃດນຶ່ງເຂົ້າໄປເບິ່ງທີ່ www.gutenberg.org/donate .
ໃນຂະນະທີ່ພວກເຮົາບໍ່ສາມາດ ແລະບໍ່ຂໍການປະກອບສ່ວນຈາກລັດທີ່ພວກເຮົາບໍ່ໄດ້ປະຕິບັດຕາມເງື່ອນໄຂການຮ້ອງຂໍ, ພວກເຮົາຮູ້ວ່າບໍ່ມີຂໍ້ຫ້າມໃດໆຕໍ່ການຮັບເອົາການບໍລິຈາກທີ່ບໍ່ໄດ້ຮັບການຮ້ອງຂໍຈາກຜູ້ໃຫ້ທຶນໃນລັດດັ່ງກ່າວທີ່ເຂົ້າຫາພວກເຮົາດ້ວຍການສະເຫນີໃຫ້ບໍລິຈາກ.
ການບໍລິຈາກລະຫວ່າງປະເທດໄດ້ຮັບການຍອມຮັບຢ່າງກະຕັນຍູ, ແຕ່ພວກເຮົາບໍ່ສາມາດອອກຖະແຫຼງການໃດໆກ່ຽວກັບການປິ່ນປົວພາສີຂອງການບໍລິຈາກທີ່ໄດ້ຮັບຈາກພາຍນອກສະຫະລັດ. ກົດໝາຍຂອງສະຫະລັດພຽງຜູ້ດຽວ swamp ພະນັກງານຂະຫນາດນ້ອຍຂອງພວກເຮົາ.
ກະລຸນາກວດເບິ່ງໜ້າເວັບຂອງໂຄງການ Gutenberg ສໍາລັບວິທີການບໍລິຈາກ ແລະທີ່ຢູ່ໃນປັດຈຸບັນ. ການບໍລິຈາກແມ່ນໄດ້ຮັບການຍອມຮັບໃນຫຼາຍວິທີອື່ນໆ ລວມທັງເຊັກ, ການຈ່າຍເງິນອອນໄລນ໌ ແລະການບໍລິຈາກບັດເຄຣດິດ. ເພື່ອບໍລິຈາກ, ກະລຸນາເຂົ້າໄປທີ່: www.gutenberg.org/donate
ພາກທີ 5. ຂໍ້ມູນທົ່ວໄປກ່ຽວກັບໂຄງການ Gutenberg™ ວຽກງານເອເລັກໂຕຣນິກ

Project Gutenberg™ eBooks ມັກຈະຖືກສ້າງຂື້ນມາຈາກຫຼາຍສະບັບທີ່ພິມອອກ, ທັງໝົດນີ້ຖືກຢືນຢັນວ່າບໍ່ໄດ້ຮັບການປົກປ້ອງລິຂະສິດໃນສະຫະລັດ ເວັ້ນເສຍແຕ່ວ່າມີການແຈ້ງລິຂະສິດ. ດັ່ງນັ້ນ, ພວກເຮົາບໍ່ຈໍາເປັນຕ້ອງຮັກສາ eBooks ໃຫ້ສອດຄ່ອງກັບສະບັບເຈ້ຍໂດຍສະເພາະ.
ຄົນສ່ວນໃຫຍ່ເລີ່ມຕົ້ນຢູ່ໃນເວັບໄຊທ໌ຂອງພວກເຮົາທີ່ມີສະຖານທີ່ຄົ້ນຫາ PG ຕົ້ນຕໍ: www.gutenberg.org .
