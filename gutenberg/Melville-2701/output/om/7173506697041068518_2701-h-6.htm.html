Tokkoffaa: Haalli mirkanaa’aa hin taane, tasgabbii hin qabne saayinsii Seetoloojii kanaa, bakka tokko tokkotti ammallee qurxummii qurxummii ta’uu fi dhiisuu isaa qabxii falmisiisaa ta’ee akka hafu, vestibule sana keessa jira. Liineus, Sirna Uumamaa Dh.K.D. Garuu akka ani beekutti, hanga bara 1850tti, shaarkiin fi shaad, alewives fi herring, faallaa labsii Linnæus ifa ta’e, amma iyyuu qabeenya galaana tokko Leviathan wajjin qooduun akka argaman nan beeka.

Sababni Liineus qurxummii galaanaa bishaan keessaa ari’ee akkas jedheera: “Sababa onnee isaanii isa bilocular ho’aa ta’e, somba isaanii, qoochoo ija isaanii socho’aa, gurra isaanii isa qaawwa qabu, penem intrantem feminam mammis lactantem,” fi dhumarratti, “ex lege naturæ jure meritoque.” Kana hunda hiriyyoota koo Simeon Macey fi Charley Coffin, kan Nantucket, lamaan isaanii iyyuu imala murtaa’e tokko irratti messmates kootiif dhiheesseen, sababoonni ka’an walumaa galatti gahaa miti jedhanii yaada tokkoon tokko ta’an. Charley profanely isaan humbug akka ta'an akeeke.

Falmii hundumaa dhiisee, lafa gaarii durii kan qurxummii qurxummii jedhu fudhadhee, qulqulluu Yoonaas akka na deeggaru nan waama. Wanti bu'uuraa kun qubate, qabxiin itti aanu, gama keessoo kamiin qurxummii qurxummii biroo irraa adda ta'a. Armaan olitti Linnæus meeshaalee sana siif kenneera. Garuu gabaabumatti isaan kanneen: sombaa fi dhiiga ho’aa; yeroo ta’u, qurxummiin kaan hundinuu somba kan hin qabnee fi dhiiga qabbanaawaa kan qabani dha.

Itti aansuudhaan: akkamitti qurxummii galaanaa, alaa isaa isa ifa ta’een, yeroo dhufu hundumaaf akka ifatti maqaa itti moggaasuuf ibsina? Kanaafuu, gabaabsee yoo dubbanne, qurxummii qurxummii dhangalaʼee fi funyaan isaa garagalfamee dha . Achitti isa qabda. Haa ta’u malee waliigaltee qaba, hiikni sun bu’aa xiinxala babal’ateeti. Walrus baay'ee akka qurxummii galaanaa dhangalaasa, garuu walrus qurxummii miti, sababiin isaas inni bishaan keessa jiraatudha. Garuu jechi dhumaa hiikichaa ammallee caalaatti amansiisaa dha, akkuma isa jalqabaa wajjin walqabatee. Namni kamiyyuu jechuun ni danda’ama qurxummiin namoonni lafa qaban beekan hundinuu funyaan diriiraa osoo hin taane, funyaan dhaabbataa ykn ol-fi-gadi akka qaban hubatee ta’uu qaba. Yeroo ta’u, qurxummii dhangala’u gidduutti funyaan, boca walfakkaataa qabaachuu danda’us, yeroo hunda bakka wal-qixa ta’e fudhata.

Hiika armaan olitti qurxummii galaanaa maal akka ta’e kenneen, uumama galaanaa hanga ammaatti namoota Naantucketers odeeffannoo gaarii qabaniin qurxummii galaanaa wajjin adda baafame kamiyyuu obbolummaa leviathanic keessaa gonkumaa hin hambisu; akkasumas, gama biraatiin, qurxummii hanga ammaatti aangoodhaan akka ormaatti ilaalame kamiyyuu isa waliin walqabsiisuu.* Kanaaf, qurxummiin xixiqqoo, kan dhangala’aa fi funyaan isaanii wal-qixa ta’e hundi karoora lafaa Seetoloojii kana keessatti hammatamuu qabu. Amma garuu qoqqoodinsi guddaan guutummaa keessummeessituu qurxummii galaanaa dhufa.

*Hanga yeroo ammaatti qurxummiin akkaataa Lamatins fi Dugongs (Pig-fish and Sow-fish of the Coffins of Nantucket) jedhaman ogeeyyii uumamaa hedduudhaan qurxummiin qurxummii galaanaa keessaa akka hammataman nan beeka. Garuu qurxummiin allaattii kun tuuta sagalee guddaa qabu, tuffatamaa, irra caalaa afaan lageen keessaa kan dhokatanii fi marga jiidha qabu waan nyaataniif, keessumaa immoo waan hin dhangalaasneef, akka qurxummii galaanaatti ragaa isaanii nan haale; akkasumas Mootummaa Seetoloojii akka gadhiisan paaspoortii isaanii isaaniif dhiyeessan.

Tokkoffaa: Akka guddinaatti ani qurxummii galaanaa KITAABA jalqabaa sadiitti nan qooda (BOQONNAAtti qoodamu), kunis hunda isaanii xiqqaa fi guddaa kan hammatu ta’a.

I. WHALE FOLIO; II. kan OCTAVO WHALE jedhamu; III. kan DUODECIMO WHALE jedhamu.

Akka gosa FOLIO tti ani dhiheessa Sperm Whale ; kan OCTAVO, kan Grampus ; kan DUODECIMO, kan Porpoise .

FOLIOS. Kanneen keessaa ani asitti boqonnaawwan armaan gadii dabalata:—I. Qurxummii Ispeermii ; II. qurxummii Mirgaa ; III. kan Fin-Back Whale jedhamu ; IV. kan Hump-backed Whale jedhamu ; V. qurxummii Dugdaa Raazor ; VI. kan Salfar Bottom Whale jedhamu .

KITAABA I. ( Folio ), BOQONNAA I. ( Sperm Whale ).—Whale kun, warra Ingilizii durii keessaa kan haala hin beekamneen Trumpa whale, fi Physeter whale, fi Anvil Headed whale jedhamuun kan beekamu, amma Cachalot warra Faransaayii, fi Pottsfich warra Jarmanii, fi Macrocephalus of the Long Words dha. Inni, shakkii tokko malee, jiraataa addunyaa kanaa isa guddaa dha; qurxummii galaanaa hunda caalaa sodaachisaa ta’ee fi qunnamuu danda’u; gama isaatiin kan hunda caalaa ulfina qabeessa ta’e; fi dhumarratti, daldala keessatti baay’ee gatii guddaa kan qabu; inni uumama tokkicha wanti gatii guddaa qabu sun, ispeermaaseeti, irraa argamu ta’uu isaati. Addaan isaa hundi, bakka biroo hedduutti, irratti ni guddata. Amma kanan hojjechuu qabu adda durummaan maqaa isaa wajjin. Akka filooloojiitti yoo ilaalamu, waan abshaalummaadha. Jaarraa muraasa dura, yeroo qurxummiin Ispeermii dhuunfaa isaa isa sirrii taʼeen guutummaatti jechuun ni dandaʼama hin beekamne, akkasumas yeroo zayitiin isaa akka tasaa qurxummii qabame irraa qofa argame; bara sanatti ispeermaaseetiin, uumama yeroo sanatti Ingilaanditti Giriinlaandi ykn Right Whale jedhamuun beekamu wajjin wal fakkaatu irraa kan argame taʼuu isaa kan yaadame fakkaata. Yaadni isaas ture, spermaceti wal fakkaataan kun qoosaa ariifachiisaa Greenland Whale kan qubeen jalqabaa jechichaa jecha jechaatti ibsu sana ture. Yeroo sanatti, akkasumas, ispeermaaseetiin garmalee hanqata ture, ifaaf osoo hin taane, akka dibataa fi qoricha qofaatti kan itti fayyadamu ture. Yeroo ammaa kana ruubarbii awunsii tokko waan bitattaniif ogeeyyii qoricha sammuu namaa hadoochan irraa argachuu qofa ture. Akkuma ani yaada kootti, adeemsa yeroo keessatti, maalummaa dhugaa ispeermaaseetii yeroo beekamu, maqaan isaa inni jalqabaa ammallee daldaltootaan qabamee ture; yaada hanqina isaa haala ajaa’ibaa ta’een barbaachisaa ta’een gatii isaa guddisuuf akka ta’e hin shakkisiisu. Kanaaf maqaan kun dhumarratti qurxummii galaanaa ispeermaaseetiin kun dhuguma irraa argame irratti kennamuu qaba.

KITAABA I. ( Folio ), BOQONNAA II. ( Right Whale ).—Gama tokkoon kun leviathans keessaa isa hunda caalaa kabajamaa waan ta’eef, isa jalqaba yeroo hunda namni adamsudha. Barreeffama yeroo baay’ee lafee qurxummii galaanaa ykn baleen jedhamuun beekamu argamsiisa; akkasumas zayita addatti “zayitii qurxummii galaanaa” jedhamuun kan beekamu siʼa taʼu, kunis daldala keessatti meeshaa gadi aanaadha. Qurxummii kan qaban keessaa, loogii tokko malee mata dureewwan armaan gadii hundaan moggaafama: The Whale; qurxummii Giriinlaand Weel; qurxummii Gurraacha; qurxummii Guddaa; Whale Dhugaa; kan jedhu Right Whale jedhamuun beekama. Eenyummaa gosa akkasitti baay’inaan cuuphame ilaalchisee wanti dukkanaa’aan jira. Egaa maali qurxummii galaanaa, kan ani gosa lammaffaa Folios koo keessatti hammate? Innis Mysticetus Guddaa ogeeyyii uumamaa Ingilizii ti; qurxummii Giriinlaand kan warra qurxummii galaanaa Ingilizii; Baleine Ordinaire kan warra qurxummii galaanaa Faransaay; kan Growlands Walfish kan warra Siwiidin. Innis qurxummii galaanaa jaarraa lamaa oliif galaana Arktiik keessatti warra Nezerlaandiifi Ingiliziin adamfamaa turedha; innis qurxummii kan qaban qurxummii Ameerikaa yeroo dheeraaf galaana Hindii keessatti, Baankii Biraazil irratti, qarqara galaanaa Nor' West irratti, fi kutaalee addunyaa adda addaa irratti hordofaa turan yoo ta'u, isaanis Right Whale Cruising Grounds jedhamanii moggaafaman.

Gariin isaanii garaagarummaa qurxummii Giriinlaand kan Ingilizii fi qurxummii mirgaa warra Ameerikaa gidduu jiru waan argan fakkeessani. Garuu bifa gurguddaa isaanii hunda keessatti sirriitti walii galu; akkasumas dhugaan murtaa’aan tokko kan addaan baafamuu hundee qabu irratti hundaa’u hanga ammaatti hin dhiyaanne. Garaagarummaa baay’ee murteessaa hin taane irratti hundaa’uun qoqqoodinsa xiqqaa dhuma hin qabneen, kutaaleen seenaa uumamaa tokko tokko akkasitti walxaxaa ta’an. Qurxummiin mirgaa bakka biraatti dheerina tokko tokkoon ni ilaalama, qurxummii isparmii qulqulleessuu eeruun.

KITAABA I. ( Folio ), BOQONNAA III. ( Fin-Back ).—Mataa kana jalatti ani bineensa hamaa kan maqaa adda addaa Fin-Back, Tall-Spout, fi Long-John jedhamuun galaana hunda keessatti jechuun ni danda’ama jedhamee kan argamee fi yeroo baay’ee qurxummii galaanaa jeetiin isaa fagoo yeroo baay’ee imaltoota galaana Atlaantiik qaxxaamuraniin ibsamu, daandii paakeetiiwwan Niiw Yoorki keessatti ta’e tokkon tilmaama. Dheerinni inni argatu, fi baleen isaa keessatti, Fin-back qurxummii mirgaa fakkaata, garuu naannoo portly xiqqaa, fi halluu salphaa, gara zayitiitti dhiyaata. Funyaan isaa guddaan gama keebilii fakkaatu kan dhiyeessu yoo ta’u, kunis dacha’iinsa qinxirii gurguddaa wal-xaxaa fi qaxxaamuraa ta’een kan uumame dha. Wanti adda isa guddaa taʼe, fincaan, kan maqaa isaa irraa argate, yeroo baayʼee wanta mulʼatudha. Finni kun gara meetira sadii ykn afur kan dheeratu yoo ta’u, kutaa duubaa dugda irraa gara dhaabbataatti kan guddatu, boca kofa kan qabuu fi dhuma isaa baay’ee qara kan qabuu fi fiixee kan qabuu dha. Kutaan uumama sanaa inni kaan xiqqoo illee yoo mul’achuu baatellee, fincaan adda baafame kun, yeroo tokko tokko, fuula irraa ifatti kan ba’u ni mul’ata. Galaanni giddu galeessaan tasgabbaa’ee, fi xiqqoo ripheewwan geengoo ta’aniin kan mallatteeffamu, fi qoonqoo gnomon fakkaatu kun ka’ee gaaddidduu kan darbatu, geengoon bishaanii isa marsee jiru hamma tokko daayilii fakkaata, akkaataa isaa fi sararoonni sa’aatii dalgaa qaban irratti bocamee akka jiru tilmaamuun ni danda’ama. Ahaz-dial sana irratti yeroo baayyee gaaddidduun duubatti deebi'a. Fin-Back kan walitti dhufeenya qabu miti. Dhiironni tokko tokko nama jibbu waan ta'aniif, nama qurxummii galaanaa jibbu fakkaata. Baayyee qaana'aa; yeroo hunda kophaa deemuu; bishaan fagoo fi baay’ee dukkanaa’aa ta’e keessatti osoo hin eegin gara gubbaatti ol ba’uu; jeettii isaa isa qajeelaa fi tokkicha olka’aa akka eeboo dheeraa misanthropic dirree gogaa irratti ol ka’aa jiru; humna ajaa’ibaa fi saffisa dambalii keessatti kennaa kan qaban, akka nama irraa ari’amuu amma jiru hunda mormuuf; leviathan kun Qaayin sanyii isaa kan ari’amee fi kan hin mo’amne fakkaata, mallattoo isaaf akkaataa sana dugda isaa irratti baatee. Baleen afaan isaa keessa qabaachuu irraa, Fin-Back yeroo tokko tokko qurxummii mirgaa wajjin hammatama, gosa ti'oorii Whalebone whales , jechuunis qurxummii baleen qaban keessaa. Whalebone whale jedhaman kana keessaa gosootni hedduun kan jiran fakkaatu, irra caalaan isaanii garuu baayʼee kan beekamanidha. Qurxummii hidhii bal’aa fi qurxummii afaan qaban; qurxummii galaanaa mataa paayikii qabu; qurxummii galaanaa tuuta ta’e; qurxummii jawwee jalaa fi qurxummii rostrated whales, maqaa qurxummii qaban gosa muraasaaf.

Maqaa “Whalebone whales” jedhu kanaan walqabatee, kaasuun barbaachisummaa guddaa qaba, haa ta’u malee moggaasni akkasii kun gosa qurxummii galaanaa tokko tokkoof allusions haala mijeessuu keessatti mijataa ta’uu danda’a, ta’us garuu ramaddii ifa ta’e Leviathan, kan baleen isaa, yookaan hump, yookaan fin, yookaan ilkaan isaa irratti hundaa’e yaaluun akkasumaan akka ta’e; kutaaleen ykn amala mallattoo qaban sun baay’ee ifatti sirna idilee Seetoloojii bu’uura ta’uuf kan madaqan fakkaatus, addaan baafamuu qaamaa adda baafame kamiyyuu caalaa, kan qurxummii galaanaa, gosa isaatiin, dhiyeessu. Akkamitti egaa? Baleen, hump, back-fin, fi ilkaan; kunniin wantoota adda ta’uun isaanii gosa qurxummii galaanaa hunda gidduutti loogii malee faca’aniidha, maalummaa caasaa isaanii adda addaa kanneen biroo fi caalaatti barbaachisoo ta’an keessatti maal ta’uu akka danda’u osoo hin ilaalin. Kanaaf, qurxummii isparmii fi qurxummiin humpbacked, tokkoon tokkoon isaanii hump qabu; achitti garuu walfakkeenyi sun ni dhaabata. Achiis, qurxummii humpbacked kun fi Greenland whale, tokkoon tokkoon isaanii baleen qabu; achitti garuu ammas fakkeenyi sun ni dhaabata. Kutaalee armaan olitti ibsaman kanneen biroo wajjinis akkasuma. Gosa qurxummii galaanaa adda addaa keessatti, walnyaatinsa sirrii hin taane akkasii uumu; yookiin, isaan keessaa tokko yoo addaan ba’e, adda baafamuu sirnaan hin taane akkasii; akka mala waliigalaa bu’uura akkasii irratti uumame hunda guutummaatti mormuuf. Dhagaa kana irratti warri qurxummii uumamaa (whale-naturalists) hundi gargar ba'aniiru.

Garuu, kutaalee keessoo qurxummii galaanaa keessatti, anatoomii isaa keessatti-achitti, yoo xiqqaate, ramaddii sirrii rukutuu ni dandeenya jedhamee yaadamuu danda’a. Lakki; fakkeenyaaf, anatoomii qurxummii galaanaa Giriinlaand keessatti wanti baleen isaa caalaa nama dinqisiisu maali? Ta’us baleen isaatiin qurxummii Giriinlaand sirritti ramaduun akka hin danda’amne argineerra. Akkasumas garaacha leviathans adda addaa keessa yoo gadi buute, maaliif achitti addaan baafannaa kutaa shantamaffaa akka warra alaa duraan lakkaa'aman sanaatti systematizer'f argamu hin argattu. Ergasii maaltu hafe? qurxummii galaanaa qaamaan, jildii isaanii isa bilisaa guutuu keessatti qabachuu fi ija jabinaan akkasitti sassaabuu malee homaa hin qabu. Kunis sirna Bibliyoogiraafii asitti fudhatamedha; akkasumas kan milkaa’uu danda’u isa qofaadha, sababiin isaas inni qofti hojiirra ooluu danda’a. Itti fufuuf.

KITAABA I. ( Folio ) BOQONNAA IV. ( Hump Back ).—Whale kun yeroo baayʼee qarqara galaanaa kaaba Ameerikaa irratti argama. Yeroo baayyee achitti qabamee, gara buufata doonii harkifamee jira. Akka nama daldalaatti paakkii guddaa ofirraa qaba; ykn immoo Elephant and Castle whale jettee waamuu dandeessa. Haala kamiinuu yoo ta’e, maqaan beekamaan isaaf kenname adda isa hin godhu, sababiin isaas, qurxummiin isparmii xiqqaa ta’us hump qaba. Zayitiin isaa gatii guddaa hin qabu. Baleen qaba. Inni qurxummii galaanaa hunda caalaa taphaafi garaa salphaadha, akka waliigalaatti foomii saala walfakkaataa fi bishaan adii kanneen biroo caalaa hojjeta.

KITAABA I. ( Folio ), BOQONNAA V. ( Razor Back ).—Waa’ee qurxummii galaanaa kanaa maqaa isaa malee waan xiqqootu beekama. Keep Hoorn irraa fagootti isa argeera. Uumama soorama ba'uu kan qabu yoo ta'u, adamsitootaas ta'e falaasama irraa jalaa miliqa. Abshaala ta’us, dugda isaa isa cirracha dheeraa qara qabu keessatti ol ka’u malee kutaa isaa agarsiisee hin beeku. Haa deemu. Kana caalaa waa'ee isaa xiqqoo beeka, namni biraas hin beeku.

KITAABA I. ( Folio ), BOQONNAA VI. ( Sulphur Bottom ).—Namni gaariin soorama baʼaa jiru kan biraan, garaa isaa kan dhagaa cilee qabu, bishaan keessa cuuphamuu isaa isa gadi fagoo taʼe tokko tokko keessatti, tilmaama Tartara irratti cirrachuudhaan akka argate hin shakkisiisu. Yeroo muraasaaf kan mul’atudha; yoo xiqqaate galaana kibbaa fagoo ta’e keessatti malee, achiis yeroo hunda fageenya guddaa irratti fuula isaa qorachuuf malee isa argee hin beeku. Inni gonkumaa hin ari’amu; funyoo-deemsa sararaa qabate ni baqata ture. Prodigies waa'ee isaa ni himama. Adieu, Salfarii Jala! Kana caalaa waan dhugaa ta'e isin irratti dubbachuu hin danda'u, Nantucketer inni dulloomaanis hin danda'u.

OCTAVOES.*—Isaan kun qurxummii galaanaa guddina giddu galeessaa qaban kan hammatan yoo ta’u, isaan keessaa kanneen argaman lakkoofsisuun ni danda’ama:—I., the Grampus ; II., Qurxummii Gurraacha ; III., kan Narwhale ; IV., kan Thrasher ; V., Ajjeesaa .

*Kitaabni qurxummii galaanaa kun maaliif Quarto jedhamee akka hin moggaafamne baay'ee ifaadha. Sababni isaas, qurxummiin galaanaa tartiiba kanaa, kanneen tartiiba duraanii caalaa xiqqaa ta’anis, kanas ta’e sana fakkiidhaan isaan wajjin walfakkeenya walmadaalu kan qaban yoo ta’u, ammas jildii Quarto kan kitaaba hidhu bifa safara isaatiin boca jildii Folio hin eegu, garuu jildii Octavo ni eega.

KITAABA II. ( Octavo ), BOQONNAA I. ( Grampus ).—Qurxummiin kun, kan sagalee guddaadhaan hafuura baafachuun isaa, ykn irra caalaa afuufuu isaa, namoota lafa jiraataniif mammaaksa kan kenne taʼus, jiraataa gadi fagoo keessa jiraatu taʼuun isaa baayʼee beekamaa taʼus, taʼus, qurxummii galaanaa keessaa akka beekamaatti hin ramadamu. Garuu amala adda ta’e gurguddaa leviathan hunda waan qabuuf, ogeeyyiin uumamaa baay’een isaanii tokkoof isa beekaniiru. Inni guddina oktaavoo giddu galeessaa kan qabu yoo ta’u, dheerinni isaa meetira kudha shan hanga digdamii shanii kan ta’ee fi naannoo mudhiitti safara walgitu qaba. Tuuta tuuta ta’ee dabsa; zayitiin isaa baay'inaan guddaa ta'us, ifaaf baay'ee gaarii ta'us, yeroo hunda adamfamee hin beeku. Qurxummii kan qaban tokko tokko biratti mala inni itti dhihaatu akka waan dursee tarkaanfii qurxummii qurxummii guddaa sperm whale dursee mul’isutti ilaalama.

KITAABA II. ( Octavo ), BOQONNAA II. ( Qurxummii Gurraacha ).—Qurxummii kana hundaaf maqaa qurxummii qabattoota beekamoo ta’aniif nan kenna, sababiin isaas walumaa galatti isaan hunda caalaa gaarii dha. Bakka maqaan kamiyyuu akka tasaatti ifa hin taane ykn kan hin ibsamne ta’etti akkas jedheen, kan biraa yaada nan dhiheessa. Amma akkas nan godha, Qurxummii Gurraacha, kan jedhamu, tuqee, sababiin isaas gurraachummaan seera qurxummii galaanaa hunda jechuun ni danda’ama. Kanaafuu, yoo feetan Hyena Whale jedhaa. Nyaata nyaachuun isaa akka gaariitti kan beekamu yoo ta’u, haala kofoon funyaan isaa keessaa gara oliitti qaxxaamuree jiru irraa kan ka’e, fuula isaa irratti kolfa bara baraa Mephistophelean ta’e qabata. Whale kun giddu galeessaan gara meetira kudha jaha ykn kudha saddeet dheerata. Latitude hunda keessatti jechuun ni danda'ama. Inni akkaataa addaa kan qabuu fi fincaan isaa kan dugda isaa kan hooked ta'e dambalii keessatti agarsiisu qaba, kunis waan hidhii Roomaa fakkaata. Yeroo bu'aa caalaa hin qacaramne, adamsitoonni qurxummii isparmii yeroo tokko tokko qurxummii Hyena whale qabatu, dhiyeessii zayita gatii salphaa hojii manaa itti fufuuf-akka hojjettoonni manaa qusatan tokko tokko, bakka dhaabbati hin jirretti, fi kophaa isaanii, waxii urgooftuu dhiisanii, qamadii hin mi'oofne gubu. Blubber isaanii baayyee haphii ta'us, whales kun tokko tokko zayita gaalonii soddomaa ol siif argamsiisu.

KITAABA II. ( Octavo ), BOQONNAA III. ( Narwhale ), jechuunis, Nostril whale .—Fakkeenya biraa kan maqaan isaa nama hawwatu, kan akkas jedhamee moggaafame, gaanfi isaa isa adda taʼe jalqaba irratti hidhii fiixee qabu jedhamee dogoggorfamuu isaa irraa kan kaʼe natti fakkaata. Uumamni kun gara meetira kudha jahaa kan dheeratu yoo ta’u, gaanfi isaa giddu galeessaan meetira shan yoo ta’u, tokko tokko kudhan kan caalu ta’us, gara meetira kudha shan illee ni ga’a. Cimsinee yoo ilaalle, gaanfi kun qoochoo dheerate qofa, sarara xiqqoo garagalcha irraa gad-dhiifameen jilba irraa bahee guddata. Garuu gama hamaa qofaan argama, kunis bu’aa hamaa kan qabu yoo ta’u, abbaa qabeenyaa isaatiif waan gama nama bitaa jal’aa ta’ee wajjin wal fakkaatu kenna. Gaanfi gaanfa ykn eebbi kun kaayyoo sirrii akkamii akka deebisu dubbachuun rakkisaa ta’a. Akka billaa qurxummii billaa fi qurxummii billaa kan itti fayyadamu hin fakkaatu; dooniiwwan tokko tokko dooniin Narwhale nyaataf jala galaanaa garagalchuu keessatti raakiidhaaf akka itti fayyadamu natti himan taʼus. Charley Coffin akka jedhetti, meeshaa bubbee bocuuf akka oolu; sababiin isaas, Naarwhaal, gara gubbaa Galaana Poolaariitti ol baʼee, bubbeedhaan marfamee argee, gaanfa isaa ol dhiibee, akkasitti cabsa. Garuu tilmaama kana keessaa tokkollee sirrii ta’uu isaa mirkaneessuu hin dandeessu. Yaadni mataa kootii, haa ta’u malee gaanfi gama tokkoo kun dhuguma Narwhale fayyadamuu danda’a-haata’u malee sun ta’uu danda’a-galfata barruulee dubbisuu keessatti baay’ee akka isaaf mijatu beekamaadha. Narwhale ani dhaga'e Tusked whale, Horned whale fi Unicorn whale jedhama. Inni fakkeenya hawwataa Unicornism mootummaa uumama socho’aa hunda jechuun ni danda’ama keessatti argamu ta’uun isaa hin oolu. Barreessitoota durii cufamanii jiran tokko tokko irraa gaanfi yuunikoornii galaanaa kun bara durii akka qoricha summii guddaatti akka ilaalamu, akkasumas akka kanaatti, qophiin isaa gatii guddaa akka fide walitti qabeera. Akkasumas, dubartoota of wallaalaniif soogidda jijjiiramaa taʼetti kan qophaaʼe siʼa taʼu, kunis akkuma gaanfi saree dhiiraa gara hartshorn taʼutti hojjetame. Jalqaba irratti mataan isaa wanta fedhii beekumsaa guddaa qabu jedhamee fudhatama ture. Xalayaan Gurraacha akka natti himutti, Sir Maartiin Firoobisher imala sana irraa yeroo deebi’u, yeroo mootittii Bees harka ishee faayaan hojjetame foddaa Masaraa Giriinwiich irraa gootummaan itti raasaa turte, yeroo dooniin isaa ija jabinaan laga Teems gadi bu’u; “Sir Maartiin imala sanarraa yeroo deebi’u, jilba jilbeenfatee gaanfa dheeraa Naarwhaal isa dinqisiisaa ta’ee fi yeroo dheeraaf masaraa Wiinsar keessatti erga fannifamee booda ol’aantummaa isheef dhiheesse.” Barreessaan lammii Aayilaandi tokko akka avers, Earl of Leicester, jilba jilbeenfattee, akkasuma olaantummaa isheef gaanfa biraa, bineensa lafaa uumama unicorn ilaallatu dhiheesse.

Narwhale baayyee fakkii kan qabu, bifa risaa fakkaatu kan qabu yoo ta’u, halluu lafa adii aannani kan qabu yoo ta’u, tuqaawwan geengoo fi dheeraa gurraacha kan qaban dha. Zayitiin isaa baay’ee ol’aanaa, ifaafi gaarii dha; garuu xiqqaadha, yeroo muraasaaf adamfama. Irra caalaa galaana circumpolar keessatti argama.

KITAABA II. ( Octavo ), BOQONNAA IV. ( Killer ).—Waa’ee qurxummii galaanaa kanaa nama Naantucketer sirriitti beekamu xiqqaadha, nama uumamaan beeka jedhee of waamu immoo homaa hin beeku. Waanan fagootti isa arge irraa waa’ee guddina giraampii ture jechuu qaba. Inni baay'ee gara jabeessadha-gosa qurxummii Feegee. Yeroo tokko tokko qurxummii guddaa Folio whales funyaan isaa qabatee, akka leech achitti fannifama, hanga bineensi jabaan sun hamma du'aatti yaadda'utti. Ajjeesaan gonkumaa hin adamsu. Zayitii akkamii akka qabu dhaga'ee hin beeku. Maqaa qurxummii galaanaa kanaaf kenname irraa kan hafe fudhatamuu danda’a, sababa adda baafamuu dhabuu isaatiin. Hundi keenya lafaa fi galaana irratti ajjeestoota waan taaneef; Bonapartes fi Sharks dabalatee.

KITAABA II. ( Octavo ), BOQONNAA V. ( Thrasher ).—Namni gaariin kun funyaan isaa kan diinota isaa dha’uu keessatti ferule ta’eef itti fayyadamuudhaan beekamaadha. Dugda qurxummii Folio whale irra kan yaabbatu siʼa taʼu, yeroo dambaliʼu immoo isa reebuudhaan karaa isaa hojjeta; akkuma barsiisonni mana barumsaa tokko tokko adeemsa walfakkaatuun addunyaa kana irratti walii galan. Ammas waa'ee Thrasher kan Killer caalaa wanti beekamu xiqqaadha. Lamaanuu galaana seera hin qabne keessattillee seeraan ala.

Akkasitti KITAABA II xumurama. ( Octavo ), fi KITAABA III jalqaba. ( Duodecimo ) jedhamuun beekama.

DUODECIMOES.—Kanneen keessaa qurxummii galaanaa xixiqqoo taʼan dabalata. I. Poorpoosii Huzzaa. II. Porpoose Aljeeriyaa jedhamtu. III. Porpoise afaan Daakuun qabu.

Namoota akka carraa ta’ee dhimma kana qorachuuf hin arganneef, qurxummiin yeroo baay’ee miila afur ykn shan hin caalle WHALES gidduutti walitti qabamuu qaba jechuun waan ajaa’ibaa fakkaachuu danda’a—jechi kun, hiika beekamaatiin, yeroo hundumaa yaada guddaa ta’uu kan dabarsudha. Garuu uumamtoonni armaan olitti akka Duodecimoes jedhamanii kaa’aman dogoggora malee qurxummii galaanaa dha, jechoota hiika kootiin qurxummii qurxummii maal akka ta’e— jechuunis qurxummii dhangala’u, funyaan isaa wal-qixa ta’e.

KITAABA III. ( Duodecimo ), BOQONNAA 1. ( Huzza Porpoise ).—Kun porpoise beekamaa addunyaa guutuutti jechuun ni danda’ama. Maqaan kun kan kooti; sababiin isaas, qoochoowwan gosa tokkoo ol waan taʼaniif, adda baasuuf wanti godhamuu qaba. Yeroo hundumaa shoals qoosaa, kan galaana bal’aa irratti akka kophee tuuta Foorth-of-July keessatti gara samiitti of darbachuu waan itti fufaniif, akkas jedheen isa waama. Bifti isaanii akka waliigalaatti nama doonii oofu gammachuudhaan jajama. Hafuurota gaarii taʼaniin kan guutaman siʼa taʼu, qilleensa qilleensaa irraa gara qilleensaatti dhufuun isaanii hin oolu. Isaan gurbaa yeroo hunda qilleensa dura jiraatanidha. Isaan akka carraa carraatti lakkaa'amu. Ati ofii keetii qurxummii jajjaboo kana ilaaluudhaan gammachuu sadii dandamachuu yoo dandeesse, samiin si gargaari; hafuurri tapha waaqayyoo isin keessa hin jiru. Huzza Porpoise akka gaariitti nyaatamee fi furdaa ta'e zayita gaarii gaalonii gaarii tokko siif argamsiisa. Garuu dhangala’aan xiqqaa fi micciiramaan qoochoo isaa keessaa baafamu gatii guddaa qaba. Warra faaya hojjetaniifi sa'aatii hojjetan biratti gaaffii keessa jira. Namoonni doonii oofan hones isaanii irratti kaa'u. Foon porpoise nyaata gaarii dha, beekta. Porpoise akka dhangala'u gonkumaa sitti hin dhufne ta'a. Dhugumayyuu, dhangala’aan isaa baay’ee xiqqaa waan ta’eef baay’ee salphaatti adda baafamuu hin danda’u. Garuu yeroo itti aanutti carraa yoo argatte isa ilaalaa; achiis qurxummii guddaa Sperm whale ofii isaatii xiqqaa ta'ee ni argita.

KITAABA III. ( Duodecimo ), BOQONNAA II. ( Algerine Porpoise ).—Nama saamtota galaanaa. Baayyee gara jabeessa. Inni kan argamu, akkan yaadutti, Paasifikii qofa keessatti. Inni hamma tokko Huzza Porpoise caalaa guddaadha, garuu baay'ee wal fakkaatu general make. Isa aarsi, inni immoo shaarkii tokkotti ni hidhama. Yeroo baayyee isaaf gadi bu'eera, garuu hanga ammaatti booji'amee argee hin beeku.

KITAABA III. ( Duodecimo ), BOQONNAA III. ( Mealy-mouthed Porpoise ).—Gosa Porpoise isa guddaa; akkasumas hanga beekamutti Paasifikii qofa keessatti argama. Maqaan Ingiliffaa tokkichi, kan hanga ammaatti ittiin moggaafame, kan qurxummii qabattootaati—Right-Whale Porpoise, haala inni irra caalaa naannoo Folio sanaatti argamu irraa. Boca isaatiin, Huzza Porpoise irraa hamma tokko adda, naannoo geengoo fi gammachiisaa hin taane; dhugumatti, inni nama sirrii fi nama gaarii fakkaatu qaba. Dugda isaa irratti fincaan hin qabu (poorpoises biroo baay'een isaanii qabu), funyaan jaalalaa qaba, ija Hindii miiraan guutame kan halluu hazel qabu. Garuu afaan isaa kan daakuun hunda balleessa. Dugda isaa guutuun hanga qoochoo cinaacha isaatti gadi fagoo ta’us, sarara daangaa, akka mallattoo qaama doonii keessatti argamutti adda ta’ee, “mudhii ifaa” jedhamee kan waamamu, sararri sun hundee irraa hanga duubatti isa sarara, halluu adda addaa lama, gubbaa gurraachaafi gadiitti adii qaba. Adiin kun kutaa mataa isaa, fi guutummaa afaan isaa kan of keessaa qabu yoo ta’u, kunis akka waan reefuu daawwannaa yakkaa boorsaa nyaataa jalaa miliqeetti akka mul’atu isa taasisa. Gama baay'ee gadhee fi mealy! Zayitiin isaa baayʼee kan porpoozii beekamaa wajjin wal fakkaata.

DUODECIMO bira darbee sirni kun hin adeemu, sababiin isaas Porpoise qurxummii galaanaa keessaa isa xiqqaadha. Gubbaa irratti, Lewwiyaataanota hunda kan hubataman qabda. Garuu tuuta qurxummiin qurxummii mirkanaa’aa hin taane, baqataa, walakkaa fabulous, kan akka nama qurxummii Ameerikaa qurxummii qabuutti maqaa gaariidhaan beeka, garuu akka dhuunfaatti miti. Maqaa masaraa fuulduraa isaaniitiin nan lakkaa’a; sababiin isaas tarii tarreen akkasii qorattoota gara fuula duraatiif gatii guddaa qabaachuu danda’a, isaanis waan ani asitti qabu garuu jalqabe xumuruu danda’u. Yoo qurxummii armaan gadii keessaa tokko, kana booda qabamee mallatteeffamee, akka guddina Folio, Octavo, ykn Duodecimo isaatiin salphaatti Sirna kana keessatti hammatamuu danda’a:—The Bottle-Nose Whale; kan jedhamu Junk Whale jedhamu; qurxummii Puudiing-Heeded Whale jedhamu; kan Keep Whale jedhamu; Whale Dursaa (Leading Whale) jedhamtu; kan jedhamu kan Cannon Whale jedhamu; kan jedhamtu Scragg Whale jedhamu; qurxummii Koopparii (Coppered Whale) jedhamu; kan jedhamtu Elephant Whale jedhamu; qurxummii Iceberg Whale jedhamu; Quog Whale jedhamuun beekamu; qurxummii Biluu (Blue Whale) jedhamu; fi kkf Aangawoota Aayislaandi, Daach fi Ingilizii durii irraa, tarreewwan qurxummii galaanaa mirkanaa’aa hin taane kan biroo, maqaawwan safuu hin qabne gosa hundaan kan eebbifaman caqasamuu danda’u. Garuu akka guutummaatti dulloomeetti nan dhiisa; akkasumas sagalee qofaaf, kan Leewwaataniin guutame, garuu homaa kan hin agarsiifne ta’eef isaan shakkuu gargaaruu hin danda’u.

Dhumarratti: Jalqaba irratti ibsameera, sirni kun asitti akka hin argamne, fi yeroo tokkotti, akka fiixaan bahu. Jecha koo akkan eege ifatti arguu dhiisuu hin dandeessu. Garuu amma Sirni seetooloojii koo akkasitti osoo hin xumuramne dhaabatee dhiisa, akkuma Kaatediraalli guddaan Kolon hafettillee, kireeniin ammallee gubbaa masaraa hin xumuramne sanaa irra dhaabatee jira. Sababni isaas, dhaabbii xixiqqoo arkiteektoota isaanii jalqabaatiin xumuramuu danda’a; kanneen gurguddoon, kanneen dhugaa ta’an, yoomiyyuu dhagaa kooppii sana dhalootaaf dhiisu. Waaqayyo yoomiyyuu waan tokkollee akkan hin xumurre na eegu. Kitaabni kun guutuun wixinee qofa-lakki, wixinee wixinee malee. Yaa Yeroo, Humna, Maallaqa, fi Obsa!
