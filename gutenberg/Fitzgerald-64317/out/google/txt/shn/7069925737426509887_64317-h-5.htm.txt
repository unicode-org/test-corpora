1.ၶ. “ၶူင်းၵၢၼ်ၵူဝ်ႇတႅၼ်ႇပႃႇၵျီႇ” ၼႆႉ ပဵၼ်ၶိူင်ႈမၢႆၵႃႉၶၢႆ ဢၼ်လႆႈမၢႆတွင်းဝႆႉ။ မၼ်းတေလႆႈၸႂ်ႉတိုဝ်း ၼိူဝ် ဢမ်ႇၼၼ် ၵွင်ႉၵၢႆႇၵၼ်တင်း ၵၢၼ်ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ ဢၼ်လႂ်သေဢမ်ႇဝႃႈ လူၺ်ႈၵူၼ်းၸိူဝ်း ၽွမ်ႉၸႂ် တႃႇတေလႆႈ မၵ်းမၼ်ႈ ၸွမ်းၼင်ႇ ၶေႃႈတူၵ်းလူင်း ဢၼ်ၼႆႉၼၼ်ႉၵွၺ်း။ ဢမ်ႇႁဵတ်းၸွမ်း ၶေႃႈတူၵ်းလူင်း ဢၼ်ၼႆႉ တဵမ်ထူၼ်ႈသေတႃႉၵေႃႈ ၸဝ်ႈၵဝ်ႇ ႁဵတ်းလႆႈ ၸွမ်း ၵၢၼ်ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ Project GutenbergTM ၵမ်ႈၼမ် ၵမ်ႈၽွင်ႈယူႇ။ တူၺ်းၶေႃႈ 1.C ၽၢႆႇတႂ်ႈ။ သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇ ႁဵတ်းၸွမ်း ၶေႃႈတူၵ်းလူင်း ဢၼ်ၼႆႉသေ ၸွႆႈထႅမ် တႃႇတေ ထိင်းသိမ်းလႆႈ လွင်ႈၶဝ်ႈထိုင် ၵၢၼ်ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ Project GutenbergTM လူၺ်ႈဢမ်ႇလႆႈသဵင်ႈငိုၼ်း မိူဝ်းၼႃႈၼႆၸိုင် ၸဝ်ႈၵဝ်ႇ တေႁဵတ်းလႆႈ ၵၢၼ်ငၢၼ်း ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ Project GutenbergTM ၼၼ်ႉ တင်းၼမ်ယူႇ။ တူၺ်းၶေႃႈ 1.E ၽၢႆႇတႂ်ႈ။

1.ၵ. ၸုမ်းၵမ်ႉၸွႆႈ ႁူင်းလိၵ်ႈလၢႆး ၶူင်းၵၢၼ် ၵူဝ်ႇတႅၼ်ႇပႃႇၵျီႇ (“ၸုမ်းၵမ်ႉၸွႆႈ” ဢမ်ႇၼၼ် PGLAF) ၼႆႉ ပဵၼ်ၸဝ်ႈၶွင် သုၼ်ႇလႆႈ တႃႇၵဵပ်းႁွမ် ၶူဝ်းၶွင် ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ ၶူင်းၵၢၼ် ၵူဝ်ႇတႅၼ်ႇပႃႇၵျီႇTM ၼၼ်ႉယဝ်ႉ။ ၸမ်ၸမ် ၵၢၼ်ငၢၼ်း ဢၼ်လႂ်ဢၼ်ၼၼ်ႉ ဢၼ်မီးၼႂ်း ၵၢၼ်ၵဵပ်းႁွမ်ၼၼ်ႉ ပဵၼ်ၶွင်ၵူၼ်းမိူင်း ၼႂ်းမိူင်းဢမေႇရိၵၢၼ်ႇယဝ်ႉ။ သင်ဝႃႈ ၵၢၼ်ၵေႃႉလႂ်ၵေႃႉၼၼ်ႉ ဢမ်ႇလႆႈထုၵ်ႇ ပၵ်းပိူင်သုၼ်ႇလႆႈသုၼ်ႇပဵၼ် တီႈမိူင်းဢမေႇရိၵၢၼ်ႇ ၵႅတ်ႇၶႄဝႆႉသေ ၸဝ်ႈၵဝ်ႇ မီးဝႆႉ တီႈမိူင်းဢမေႇရိၵၢၼ်ႇၼႆၸိုင် ႁဝ်းၶႃႈ ဢမ်ႇတုၵ်းယွၼ်း သုၼ်ႇႁႄႉႁၢမ်ႈ ၸဝ်ႈၵဝ်ႇ တႃႇတေ ၶိုၼ်းမိူၼ်၊ ႁၢမ်ႉတေႉ ႁဝ်းၶႃႈမုင်ႈမွင်းဝႃႈ ၸဝ်ႈၵဝ်ႇတေၵမ်ႉထႅမ်ပၼ် ၼႃႈၵၢၼ် Project GutenbergTM ဢၼ်တေပၼ်ႁႅင်း လွင်ႈၶဝ်ႈထိုင်လႆႈ ၵၢၼ်ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ လူၺ်ႈဢမ်ႇလႆႈသဵင်ႈငိုၼ်း လူၺ်ႈ လွင်ႈၽေႃႇၼမ်ႉၽေႃႇတူဝ် ၵၢၼ် Project GutenbergTM လွတ်ႈလွတ်ႈလႅဝ်းလႅဝ်း ၸွမ်းၼင်ႇ ၶေႃႈတူၵ်းလူင်း ဢၼ်ၼႆႉ တႃႇႁႂ်ႈၸိုဝ်ႈ Project GutenbergTM ၵွင်ႉၵၢႆႇၵၼ်တင်း ၵၢၼ်ၼၼ်ႉယူႇ။ ၸဝ်ႈၵဝ်ႇ ၸၢင်ႈႁဵတ်းၸွမ်း ၶေႃႈတူၵ်းလူင်း ဢၼ်ၼႆႉ ငၢႆႈငၢႆႈလူမ်လူမ် လူၺ်ႈ လွင်ႈဝႆႉ ၵၢၼ်ၼႆႉ ၼႂ်းၾွၼ်ႉ ဢၼ်မိူၼ်ၵၼ် တင်း မၢႆမီႈ ၶူင်းၵၢၼ် GutenbergTM ဢၼ်တဵမ်ထူၼ်ႈ ဢၼ်မၼ်း ၸပ်းဝႆႉၼၼ်ႉ မိူဝ်ႈၸဝ်ႈၵဝ်ႇ ၸႅၵ်ႇၽႄပၼ် ၵူၼ်းတၢင်ႇၵေႃႉ လူၺ်ႈဢမ်ႇလႆႈ သႂ်ႇၵႃႈၸႂ်ႉၸၢႆႇသင်ၼၼ်ႉယဝ်ႉ။

1.ၻီႊ။ မၢႆမီႈသုၼ်ႇလႆႈသုၼ်ႇပဵၼ် တီႈဢၼ်ၸဝ်ႈၵဝ်ႇယူႇၼၼ်ႉၵေႃႈ ဢုပ်ႉပိူင်ႇပၼ် လွင်ႈဢၼ်ၸဝ်ႈၵဝ်ႇ တေႁဵတ်းလႆႈ ၸွမ်းၵၢၼ်ဢၼ်ၼႆႉ။ မၢႆမီႈသုၼ်ႇလႆႈသုၼ်ႇပဵၼ် ၼႂ်းၸိုင်ႈမိူင်းၵမ်ႈၼမ်ၼႆႉ မၼ်းလႅၵ်ႈလၢႆႈယူႇတႃႇသေႇ။ သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇ ယူႇၼွၵ်ႈမိူင်း ဢမေႇရိၵၢၼ်ႇၼႆၸိုင် ၵွၼ်ႇပႆႇ လူင်း၊ ၶိုၼ်းမိူၼ်၊ ၼႄ၊ ၼႄ၊ ႁဵတ်းဢွၵ်ႇ၊ ပိုၼ်ၽႄ ဢမ်ႇၼၼ် ႁဵတ်းဢွၵ်ႇ ၵၢၼ်ဢၼ်လုၵ်ႉတီႈ ၵၢၼ်ၼႆႉ ဢမ်ႇၼၼ် ၵၢၼ် Project GutenbergTM တၢင်ႇဢၼ်ၼၼ်ႉ ႁႂ်ႈထတ်းတူၺ်း မၢႆမီႈ မိူင်းၸဝ်ႈၵဝ်ႇ လိူဝ်သေ ၶေႃႈတူၵ်းလူင်း ဢၼ်ၼႆႉ။ ၸုမ်းၵမ်ႉၸွႆႈ ဢမ်ႇႁဵတ်းပၼ် လွင်ႈတၢင်တူဝ် ၵဵဝ်ႇလူၺ်ႈ ၸၼ်ႉထၢၼ်ႈ သုၼ်ႇလႆႈ ၶွင် ၵၢၼ်ငၢၼ်း ဢၼ်လႂ်သေဢမ်ႇဝႃႈ ၼႂ်းမိူင်း တၢင်ႇမိူင်း လိူဝ်သေ မိူင်းဢမေႇရိၵၢၼ်ႇ။

ပပ်ႉလိၵ်ႈဢီးပပ်ႉၼႆႉ ပဵၼ်တႃႇၸႂ်ႉတိုဝ်းလႆႈ တီႈလႂ်သေဢမ်ႇဝႃႈ ၼႂ်းမိူင်းဢမေႇရိၵၢၼ်ႇ လႄႈ ၼႂ်းလုမ်ႈၾႃႉ ၵမ်ႈၼမ် ဢမ်ႇလႆႈသဵင်ႈငိုၼ်းသင် လႄႈ ဢမ်ႇမီးလွင်ႈႁႄႉႁၢမ်ႈသင် ၸမ်ၸမ်။ ၸဝ်ႈၵဝ်ႇၸၢင်ႈ ၶိုၼ်းဢဝ်မၼ်း ပၼ်မၼ်း ဢမ်ႇၼၼ် ၶိုၼ်းၸႂ်ႉတိုဝ်းမၼ်း ၸွမ်းၼင်ႇ ၶေႃႈမၵ်းမၼ်ႈ ၶူင်းၵၢၼ် ၵူဝ်ႇတႅၼ်ႇပႃႇၵျီႇ ဢၼ်ပႃးၸွမ်း ပပ်ႉလိၵ်ႈဢီးပပ်ႉၼႆႉ ဢမ်ႇၼၼ် ၼိူဝ်ဢွၼ်ႊလၢႆႊ တီႈ www.gutenberg.org ၼၼ်ႉယူႇ။ သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇ ဢမ်ႇမီးတီႈယူႇ တီႈမိူင်းဢမေႇရိၵၢၼ်ႇၼႆၸိုင် တေလႆႈထတ်းတူၺ်း မၢႆမီႈ မိူင်းဢၼ်ၸဝ်ႈၵဝ်ႇ ယူႇသဝ်းၼၼ်ႉ ၵွၼ်ႇၸင်ႇၸႂ်ႉတိုဝ်း ပပ်ႉလိၵ်ႈဢီးပပ်ႉၼႆႉ။

• ၸဝ်ႈၵဝ်ႇတေလႆႈပၼ် ၵႃႈၶွၼ်ႇ 20% ၶွင် ၼမ်ႉတွၼ်း ဢၼ်ၸဝ်ႈၵဝ်ႇလႆႈမႃး တီႈၵၢၼ်ၸႂ်ႉတိုဝ်း ၵၢၼ်ငၢၼ်း Project GutenbergTM ဢၼ်ၼပ်ႉသွၼ်ႇဝႆႉ လူၺ်ႈလွၵ်းလၢႆး ဢၼ်ၸဝ်ႈၵဝ်ႇၸႂ်ႉတိုဝ်းဝႆႉ တႃႇၼပ်ႉသွၼ်ႇ ၶွၼ်ႇဢၼ်ထုၵ်ႇမႅၼ်ႈ ၸဝ်ႈၵဝ်ႇၼၼ်ႉယဝ်ႉ။ ၵႃႈၸႂ်ႉၸၢႆႇၼႆႉ ပဵၼ်ၼီႈၸဝ်ႈၶွင် ၶိူင်ႈမၢႆၵႃႉၶၢႆ Project GutenbergTM သေတႃႉ မၼ်းလႆႈၸႂ် တႃႇတေလူႇတၢၼ်း ငိုၼ်းၶွၼ်ႇ ဢၼ်မီးဝႆႉ ၸွမ်းၼင်ႇ ၶေႃႈၼႆႉ တီႈ ၸုမ်းၵမ်ႉၸွႆႈ ႁူင်းလိၵ်ႈလၢႆး Project Gutenberg ၼၼ်ႉယဝ်ႉ။ ငိုၼ်းဢၼ်လႆႈပၼ် တႃႇၶွၼ်ႇၼၼ်ႉ တေလႆႈပၼ် ၼႂ်းၶၢဝ်းတၢင်း 60 ဝၼ်း ဢၼ်ၸွမ်းမႃး ဝၼ်းလႂ်ဝၼ်းၼၼ်ႉ ဢၼ်ၸဝ်ႈၵဝ်ႇ ႁၢင်ႈႁႅၼ်း (ဢမ်ႇၼၼ် ၸွမ်းၼင်ႇမၢႆမီႈ တေလႆႈႁၢင်ႈႁႅၼ်း) ၽိုၼ်လိၵ်ႈ ၶွၼ်ႇ ဢၼ်တေလႆႈပၼ် ၸွမ်းၶၢဝ်းယၢမ်းၼၼ်ႉယဝ်ႉ။ ငိုၼ်းဢၼ်လႆႈပၼ် တႃႇၸဝ်ႈၾႃႉၼၼ်ႉ ထုၵ်ႇလီမၢႆဝႆႉ ၸႅင်ႈၸႅင်ႈလႅင်းလႅင်းသေ သူင်ႇၸူး ၸုမ်းၵမ်ႉၸွႆႈ ႁူင်းလိၵ်ႈလၢႆး ၶူင်းၵၢၼ် ၵူဝ်ႇတႅၼ်ႇပႃႇၵျီႇ တီႈတီႈယူႇ ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၼႂ်းတွၼ်ႈ 4 ဢၼ်ဝႃႈ “ၶေႃႈမုၼ်း ၵဵဝ်ႇလူၺ်ႈ ငိုၼ်းလူႇတၢၼ်း တီႈ ႁူင်းၵၢၼ် ႁူင်းလိၵ်ႈလၢႆး ၶူင်းၵၢၼ် ၵူဝ်ႇတႅၼ်ႇပႃႇၵျီႇ” ၼၼ်ႉယဝ်ႉ။

• ၸဝ်ႈၵဝ်ႇ တေပၼ်ၶိုၼ်းငိုၼ်း ဢၼ်ၽူႈၸႂ်ႉတိုဝ်း ပၼ်ၶိုၼ်း တင်းမူတ်း ဢၼ်တႅမ်ႈလိၵ်ႈ (ဢမ်ႇၼၼ် လူၺ်ႈ ဢီးမေးလ်) ၼႂ်းၶၢဝ်းတၢင်း 30 ဝၼ်း ဝၢႆးသေ လႆႈႁပ်ႉငိုၼ်းၼၼ်ႉဝႃႈ မၼ်း/မၼ်းဢမ်ႇၽွမ်ႉၸွမ်း ၶေႃႈမၵ်းမၼ်ႈ ၶေႃႈမၵ်းမၼ်ႈ ၶူင်းၵၢၼ် GutenbergTM တဵမ်ထူၼ်ႈၼၼ်ႉ။ ၸဝ်ႈၵဝ်ႇတေလႆႈတုၵ်းယွၼ်းႁႂ်ႈၽူႈၸႂ်ႉတိုဝ်းၸိူင်ႉၼၼ် ဢဝ်ၶိုၼ်း ဢမ်ႇၼၼ် ယႃႉပႅတ်ႈ ၽိုၼ်မိူၼ် ၵၢၼ်ငၢၼ်း ဢၼ်မီးဝႆႉ ၼႂ်းၶိူင်ႈမိုဝ်း ဢၼ်ႁၼ်လႆႈၼၼ်ႉ တင်းမူတ်းသေ ၵိုတ်းယိုတ်း လွင်ႈၸႂ်ႉတိုဝ်း လႄႈ လွင်ႈၶဝ်ႈထိုင် ၽိုၼ်မိူၼ် ၵၢၼ်ငၢၼ်း Project GutenbergTM တၢင်ႇဢၼ် တင်းမူတ်း။

• ၸဝ်ႈၵဝ်ႇ ပၼ်ၶိုၼ်း ၸွမ်းၼင်ႇ ၶေႃႈ 1.F.3 ၼၼ်ႉ ငိုၼ်းဢၼ်လႆႈပၼ် တႃႇၵၢၼ် ဢမ်ႇၼၼ် ၽိုၼ်မိူၼ် ဢၼ်တႅၼ်းတၢင် သင်ဝႃႈ လႆႈႁၼ် လွင်ႈဢမ်ႇတဵမ်ထူၼ်ႈ ၼႂ်းၵၢၼ် ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ သေ လႆႈတၢင်ႇလၢတ်ႈ တီႈၸဝ်ႈၵဝ်ႇ ၼႂ်းၶၢဝ်းတၢင်း 90 ဝၼ်း ဝၢႆးသေ လႆႈႁပ်ႉၵၢၼ်ၼၼ်ႉယဝ်ႉ။

• ၸဝ်ႈၵဝ်ႇ ႁဵတ်းၸွမ်း ၶေႃႈမၵ်းမၼ်ႈ တၢင်ႇဢၼ် ဢၼ်ပႃးၼႂ်း လိၵ်ႈႁူမ်ႈမၢႆ ဢၼ်ၼႆႉ တႃႇပိုၼ်ၽႄ ၵၢၼ်ငၢၼ်း Project GutenbergTM လူၺ်ႈဢမ်ႇလႆႈသဵင်ႈငိုၼ်း။

1.င.9. သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇ ၶႂ်ႈၵဵပ်းၶွၼ်ႇ ဢမ်ႇၼၼ် ၶႂ်ႈပိုၼ်ၽႄ ၵၢၼ် ဢမ်ႇၼၼ် ၸုမ်းၵၢၼ် ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ ၶွင် Project GutenbergTM ၸွမ်းၼင်ႇ ၶေႃႈမၵ်းမၼ်ႈ ဢၼ်ပိူင်ႈၵၼ်တင်း ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၼႂ်းလိၵ်ႈႁူမ်ႈမၢႆၼႆႉၼႆၸိုင် ၸဝ်ႈၵဝ်ႇ တေလႆႈ ႁပ်ႉၶႂၢင်ႉ လူၺ်ႈလိၵ်ႈ တီႈ ၸုမ်းၵမ်ႉၸွႆႈ ႁူင်းလိၵ်ႈလၢႆး Project Gutenberg ဢၼ်ပဵၼ် ၽူႈၸတ်းၵၢၼ် ၶိူင်ႈမၢႆ ၵၢၼ်ၵႃႉၶၢႆ Project GutenbergTM ၼၼ်ႉယဝ်ႉ။ ၵပ်းသိုပ်ႇ ၸုမ်းၵမ်ႉၸွႆႈ ၸွမ်းၼင်ႇ တွၼ်ႈ 3 ၽၢႆႇတႂ်ႈၼႆႉ။

1.ၾ.2. လွင်ႈႁပ်ႉပဝ်ႊ ဢၼ်မၵ်းမၼ်ႈဝႆႉ၊ လွင်ႈလူႉသုမ်း ၵႃႈၸႂ်ႉၸၢႆႇ လႄႈ ၵႃႈၸႂ်ႉၸၢႆႇ ဢိၵ်ႇပႃး ၵႃႈမၢႆမီႈ။ ၸဝ်ႈၵဝ်ႇ လႆႈၸႂ်ဝႃႈ ၸဝ်ႈၵဝ်ႇ ဢမ်ႇမီး လွၵ်းလၢႆး တႃႇတေ ၵႄႈလိတ်ႈ လွင်ႈဢမ်ႇၾၢင်ႉမႅၼ်ႈ၊ ၸဝ်ႈၵဝ်ႇ လႆႈၸႂ်ဝႃႈ ၸုမ်းၵမ်ႉၸွႆႈ ၸဝ်ႈၶွင် ၶေႃႈမုၼ်း လႄႈ ၽူႈပိုၼ်ၽႄ ဢၼ်မီးဝႆႉ ၽၢႆႇတႂ်ႈ လိၵ်ႈႁူမ်ႈမၢႆၼႆႉ တေဢမ်ႇလႆႈ ဢဝ်ပုၼ်ႈၽွၼ်း ၼိူဝ်ၸဝ်ႈၵဝ်ႇ ၵဵဝ်ႇလူၺ်ႈ လွင်ႈဢၼ်ပဵၼ်တႄႉ၊ ပၼ်ၶေႃႈၸီႉသင်ႇ လွင်ႈဢၼ်တေၸၢင်ႈပဵၼ်လွင်ႈလူႉသုမ်းၸိူင်ႉၼႆ။

1.ၾ.3. သုၼ်ႇလႆႈ ဢၼ်မၵ်းမၼ်ႈဝႆႉ တႃႇတေတႅၼ်းတၢင် ဢမ်ႇၼၼ် တႃႇတေပၼ်ၶိုၼ်းငိုၼ်း - သင်ဝႃႈ ၼႂ်းၶၢဝ်းတၢင်း 90 ဝၼ်း ဝၢႆးသေ လႆႈႁပ်ႉၵၢၼ် ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉၼႆႉ မီးလွင်ႈဢမ်ႇတဵမ်ထူၼ်ႈၼႆၸိုင် ၸဝ်ႈၵဝ်ႇ တေလႆႈၶိုၼ်းပၼ်ငိုၼ်း (သင်ဝႃႈ မီး) ဢၼ်ၸဝ်ႈၵဝ်ႇ လႆႈပၼ်ၶိုၼ်း လူၺ်ႈ သူင်ႇလိၵ်ႈသပ်းလႅင်း ၸူး ၵူၼ်းဢၼ်ၸဝ်ႈၵဝ်ႇ လႆႈႁပ်ႉၵၢၼ် တီႈမၼ်းၼၼ်ႉယူႇ။ သင်ဝႃႈ လႆႈႁပ်ႉၵၢၼ် တီႈၼိူဝ် ၶိူင်ႈမိုဝ်းထိုဝ် ဢၼ်ႁၼ်လႆႈၼၼ်ႉၸိုင် တေလႆႈဢဝ် ၶေႃႈသပ်းလႅင်း ဢၼ်တႅမ်ႈဝႆႉၼၼ်ႉ သူင်ႇၶိုၼ်း ၶိူင်ႈမိုဝ်းထိုဝ်ၼၼ်ႉယဝ်ႉ။ ၵူၼ်း ဢမ်ႇၼၼ် ၸုမ်းဢၼ်ပၼ် ၵၢၼ်ဢၼ်ဢမ်ႇတဵမ်ထူၼ်ႈၼၼ်ႉ တေလိူၵ်ႈပၼ် ၽိုၼ်မိူၼ် ဢၼ်တႅၼ်းတၢင် ငိုၼ်းၶိုၼ်းၼၼ်ႉယူႇ။ သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇလႆႈႁပ်ႉၵၢၼ် လူၺ်ႈလၢႆးဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉၼႆၸိုင် ၵူၼ်း ဢမ်ႇၼၼ် ၸုမ်းဢၼ်ပၼ်ၵၢၼ်ၼၼ်ႉ တေလိူၵ်ႈပၼ် သုၼ်ႇလႆႈ ပွၵ်ႈၵမ်းသွင် တႃႇႁပ်ႉၵၢၼ် လူၺ်ႈလၢႆးဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ တႅၼ်းတၢင် ငိုၼ်းၶိုၼ်း။ သင်ဝႃႈ ၽိုၼ်မိူၼ်မၢႆသွင်ၼၼ်ႉၵေႃႈ ဢမ်ႇတဵမ်ထူၼ်ႈၼႆၸိုင် ဢမ်ႇမီးတိုဝ်ႉတၢင်း တႃႇတေမႄးပၼ်ပၼ်ႁႃၼၼ်ႉထႅင်ႈသေ တုၵ်းယွၼ်းၶိုၼ်းငိုၼ်း လူၺ်ႈလိၵ်ႈ။

1.ၽ.4။ လိူဝ်သေ သုၼ်ႇလႆႈ ဢၼ်မၵ်းမၼ်ႈဝႆႉ တႃႇတေတႅၼ်းတၢင် ဢမ်ႇၼၼ် တႃႇတေပၼ်ၶိုၼ်းငိုၼ်း ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၼႂ်းၶေႃႈ 1.F.3 ၼၼ်ႉ ၵၢၼ်ၼႆႉ လႆႈပၼ်ဝႆႉ တီႈၸဝ်ႈၵဝ်ႇ 'AS-IS' သေ ဢမ်ႇမီး လွင်ႈႁပ်ႉပဝ်ႊ တၢင်ႇဢၼ် ဢၼ်ပဵၼ် လွင်ႈႁပ်ႉပဝ်ႊ ဢၼ်ၸႅင်ႈလႅင်း ဢမ်ႇၼၼ် ဢၼ်ၼႄဝႆႉ ၸႅင်ႈၸႅင်ႈလႅင်းလႅင်း ဢိၵ်ႇပႃး လွင်ႈႁပ်ႉပဝ်ႊ ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၵူၺ်းၵႃႈ ဢမ်ႇလႆႈမၵ်းမၼ်ႈဝႆႉ လွင်ႈႁပ်ႉပဝ်ႊ ယိူင်းဢၢၼ်းသင်ၵေႃႈယဝ်ႉ၊

1.ၽ.5။ ၸေႊမိူင်းမၢင်ႊၸိူဝ်း ဢမ်ႇပၼ်ၶႂၢင်ႉ တႃႇတေ ထဵင်ၶိုၼ်း လွင်ႈႁပ်ႉပဝ်ႊ ဢၼ်ၼႄဝႆႉ ဢမ်ႇၼၼ် လွင်ႈဢမ်ႇပႃး ဢမ်ႇၼၼ် လွင်ႈမၵ်းမၼ်ႈ လွင်ႈလူႉသုမ်း မၢင်ၸိူဝ်း။ သင်ဝႃႈ လွင်ႈထဵင်ၶိုၼ်း ဢမ်ႇၼၼ် လွင်ႈမၵ်းမၼ်ႈ ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၼႂ်းလိၵ်ႈႁူမ်ႈမၢႆၼႆႉ ပူၼ်ႉပႅၼ် မၢႆမီႈ ၸေႊမိူင်း ဢၼ်ၸႂ်ႉတိုဝ်း လိၵ်ႈႁူမ်ႈမၢႆၼႆႉၼႆၸိုင် လိၵ်ႈႁူမ်ႈမၢႆၼၼ်ႉ တေလႆႈ ပိုတ်ႇတီႈပွင်ႇ ႁႂ်ႈပဵၼ် လွင်ႈထဵင်ၶိုၼ်း ဢမ်ႇၼၼ် လွင်ႈမၵ်းမၼ်ႈ ဢၼ်မၢႆမီႈ ၸေႊမိူင်း ဢၼ်ၸႂ်ႉတိုဝ်းၼၼ်ႉ ပၼ်ၶႂၢင်ႉဝႆႉ သုင်သုတ်းၼၼ်ႉယဝ်ႉ။ လွင်ႈဢၼ် ၶေႃႈတူၵ်းလူင်း ဢၼ်လႂ်သေဢမ်ႇဝႃႈ ဢမ်ႇၸႂ်ႈ ဢမ်ႇၼၼ် ဢမ်ႇၸၢင်ႈႁဵတ်းၸွမ်းလႆႈၼၼ်ႉ တေဢမ်ႇႁဵတ်းႁႂ်ႈ ၶေႃႈတူၵ်းလူင်း ဢၼ်ၵိုတ်းဝႆႉၼၼ်ႉ ဢမ်ႇမၢၼ်ႇမႅၼ်ႈ။

Hart ၼႆႉ ပဵၼ်ၽူႈတႄႇတင်ႈ ၶူင်းၵၢၼ် GutenbergTM ဢၼ်ပဵၼ် ႁူင်းတူၺ်းလိၵ်ႈ ၵၢၼ်ဢီႇလႅတ်ႇထရေႃးၼိၵ်ႉ ဢၼ်တေၸၢင်ႊ ၸႅၵ်ႇၽႄပၼ် ၽႂ်မၼ်း လွတ်ႈလွတ်ႈလႅဝ်းလႅဝ်းၼၼ်ႉယဝ်ႉ။ ၶၢဝ်းတၢင်းသီႇသိပ်းပီႊ မၼ်းႁဵတ်းဢွၵ်ႇ လႄႈ ပိုၼ်ၽႄ ပပ်ႉလိၵ်ႈဢီးပပ်ႉ Project GutenbergTM ဢၼ်မီး ၼႅၵ်ႉဝွၵ်ႉ ဢၼ်လွတ်ႈလႅဝ်း ဢၼ်ၸွႆႈထႅမ် ၽူႈၸွႆႈထႅမ်ၶဝ်ၼၼ်ႉၵွႆး။

ပပ်ႉလိၵ်ႈဢီးပပ်ႉ Project GutenbergTM ၼႆႉ ၵႆႉႁဵတ်းဢွၵ်ႇမႃး တီႈပပ်ႉဢၼ်ဢိတ်ႇဢွၵ်ႇမႃးလၢႆလၢႆဢၼ်သေ တင်းမူတ်းၼႆႉ လႆႈမၵ်းမၼ်ႈဝႃႈ ဢမ်ႇလႆႈထုၵ်ႇ သုၼ်ႇလႆႈသုၼ်ႇပဵၼ် တီႈမိူင်းဢမေႇရိၵၢၼ်ႇ ၵႅတ်ႇၶႄဝႆႉ သင်ဝႃႈ ဢမ်ႇပႃး ၶေႃႈပိုၼ်ၽၢဝ်ႇ သုၼ်ႇလႆႈသုၼ်ႇပဵၼ် ၼႆယဝ်ႉ။ ယွၼ်ႉၼၼ် ႁဝ်းၶႃႈ ဢမ်ႇလူဝ်ႇဝႆႉ ပပ်ႉလိၵ်ႈဢီးပပ်ႉ ႁႂ်ႈမႅၼ်ႈၸွမ်း ပပ်ႉလိၵ်ႈ ဢၼ်ဢွၵ်ႇမႃး တီႈၽိုၼ်လိၵ်ႈ ဢၼ်လႂ်သေဢမ်ႇဝႃႈ။

ၵူၼ်းၵမ်ႈပႃႈၼမ် တႄႇတီႈ ဝႅပ်ႉသၢႆႉႁဝ်း ဢၼ်မီး ၶိူင်ႈသွၵ်ႈႁႃ PG ပိူင်လူင် ဢၼ်ပဵၼ် www.gutenberg.org ၼၼ်ႉ ယဝ်ႉ။
