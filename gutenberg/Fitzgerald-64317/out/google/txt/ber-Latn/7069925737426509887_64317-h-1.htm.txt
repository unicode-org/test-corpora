Ttmeslayen-d fell-as ccwal n tkerkas – amek ay tt-tufa yemma-s tettḥewwis taɛkemt-nnes deg yiwen n yiḍ n unebdu akken ad teddu ɣer New York ad d-tini ar tufat n yiwen n lɛeskeṛ ay la iteddun ɣer tmura n beṛṛa. Tettwaḥbes s tɣawla, maca ur telli ara deg tmuɣli n tmesliwt d twacult-is acḥal n ddurt. Deffir waya ur tetturar ara s yiserdasen, maca ala s kra n yilemẓiyen s yiḍarren iḍarren, s tmuɣli n tɣerɣert deg temdint, ur zmiren ara ad kecmen ɣer lɛesker akk.

Armi d lexrif i d-iteddun tuɣal d tameṭṭut, d tameṭṭut am zik. Tesɛa tikkelt tamezwarut deffir ma tfuk tmenɣiwt, yerna deg Yebrir, ttwalin-tt d yiwen n wergaz seg New Orleans. Deg Yunyu tezweǧ d Tom Buchanan n Chicago, s tmeqbert d tegnit ugar n wayen ay tessen Louisville zik. Yewweḍ-d s mya n yemdanen deg 4 n tkeryas tusligin, yerna yekrez yiwen n lqaɛa s lekmal-is n lḥutel n Muhlbach, yerna ass uqbel tmeɣra yefka-as-d yiwet n tmeẓẓuɣt n lmerjan s wazal n kraḍ n tmerwin d xemsin n yimelyunen n yidulaṛen.

Nekk lliɣ d tameṭṭut n teslit. Usiɣ-d ɣer texxamt-is azgen n ssaɛa uqbel imensi n teslit, ufiɣ-tt tettḍeggiɛ ɣef usu-ines s lḥir am yiḍ n Yunyu s tmeqbert-is n yijeǧǧigen – yerna teswa am lbaz. Tesɛa deg yiwen ufus yiwet n tbuciḍant n Sauterne, deg wayeḍ yiwet n tebrat.

“ ‘Snemmer- iyi,’ i d- tenna s tmuɣli n tmeṭṭut. “Ur lliɣ ara sswiɣ uqbel, maca oh amek i t-ḥemmleɣ.”

“Dagi, ay iɛzizen.” Tezzi-d deg yiwen n lbaṭaṭa n lqut i tesɛa yid-s ɣef usu, tessufeɣ-d lḥirfa n lmerjan. "Awi-ten ɣer ddaw, rret-ten i win yellan d ayla-nsen. Ini-asen akk Daisy tbeddel' tin-is. Ini-d: 'Daisy tbeddel' tin-is!" ”

Tebda tettrun — tettrun, tettrun. Fɣeɣ-d s tɣawla, ufiɣ-d taqeddact n yemma-s, nḥebbes tawwurt, nessekcem-itt deg yiwen n usgunfu asemmaḍ. Ur tezmir ara ad teǧǧ tabratt-nni. Tewwi-tt yid-s deg tbuciḍant, tessemɣer-itt deg yiwet n tbuciḍant n tmeẓẓuɣt, yerna teǧǧa-yi kan ad tt-ǧǧeɣ deg tmeẓẓuɣt n ssabun mi ara twali belli tettkemmil tettfeggiḍ am udfel.

Maca ur d-tenni ara awal nniḍen. Nefka-as ṛṛuḥ n l’ammoniac yerna nessers-as glaz ɣef tɣerɣert-is yerna nḥebbes-itt ɣer deffir ɣer tmeqbert-is, yerna azgen n ssaɛa sakkin, mi d-neffeɣ seg texxamt, llant lmerǧan-nni ɣef tɣerɣert-is yerna tefra tedyant-nni. Azekka-nni ɣef xemsa n tmeddit tezweǧ d Tom Buchanan war ma teḥbes ula d yiwen n lḥir, yerna tebda tikli n kraḍ n wayyuren ɣer yilel n Wenẓul.

Ẓriɣ-ten di Santa Barbara mi d-uɣalen, ɣileɣ ur ẓriɣ ara taqcict tettḥulfu s wergaz-is am wa. Ma yeffeɣ-d seg texxamt kra n tedqiqt ad tmuqel s lḥir, ad as-tini: “Anida i yedda Tom?” yerna telsa-d awal abstrait akk alamma t-twala iteddu-d deg tewwurt. Tella tettɣimi ɣef lqaɛa s uqerruy-is deg tfekka-s s ssaɛa, tettḥezzib iḍarren-is ɣef wallen-is yerna tettmuqul-it s lferḥ ur nesɛi lmeɛna. Yella yettḥaz-ik mi ara ten-twaliḍ akk d yiwen —yeḍsa-k s lḥir, s lḥir. D ayen yellan deg Ɣuct. Ddurt deffir ma ffɣeɣ seg Santa Barbara, Tom yuzzel ɣer yiwet n tkeṛṛust deg webrid n Ventura deg yiwen n yiḍ, yerna yeqceɛ-d yiwet n tɣawsa n zdat seg tkeṛṛust-nnes. Taqcict-nni yellan yid-s tekcem ula d nettat deg yiḍrisen, imi afus-is yeqreɛ – d yiwet seg tmeṭṭutin n tḥanut deg lḥutel n Santa Barbara.

Deg yebrir i d-iteddun, Daisy tesɛa taqcict-is tamecṭuḥt, ddan ɣer Fransa aseggas. Ẓriɣ-ten yiwet n tefsut deg Cannes, sakkin deg Deauville, sakkin uɣalen ɣer Chicago akken ad sbedden iman-nsen. Daisy tella tettwassen deg Chicago, akken teẓriḍ. Ttmuqulen s lɣaci s tɣawla, akk d ilemẓiyen d yimerkantiyen d yiberkanen, maca nettat teffeɣ-d s ccbaḥa ikemlen s lekmal-is. Ahat imi ur teswi ara. D lfayda tameqqrant ma ur teswiḍ ara gar yimdanen yesswen s lǧehd. Tzemreḍ ad teṭṭfeḍ iles-ik yerna, nnig waya, tzemreḍ ad tessemlilḍ kra n lbaṭel ameẓyan n yiman-ik akken yal yiwen nniḍen ad yili d aderɣal armi ur ttwalin ara neɣ ur ttḥulfun ara. Ahat Daisy ur tekcim ara akk deg tmeṭṭut-is – yerna ɣas akken yella kra deg ṣṣut-is-nni...

Ihi, azal n 6 n ddurt aya, tesla i yisem n Gatsby i tikkelt tamezwarut deg yiseggasen. D asmi i k-steqsaɣ – tecfiḍ ? – ma yella tessneḍ Gatsby deg West Egg. Mi tṛuḥeḍ ɣer uxxam tekcem-d ɣer texxamt-iw, tessekcem-iyi-d, tenna-yi-d: “D acu n Gatsby?” yerna mi t-id-sfehmeɣ — lliɣ azgen n yiḍes — tenna-d s ṣṣut-nni igerrzen akk belli ilaq d argaz-nni i tessen zik. Armi d asmi i qqneɣ Gatsby-agi ɣer ufesyan-nni yellan deg tkerrust-is tamellalt.
