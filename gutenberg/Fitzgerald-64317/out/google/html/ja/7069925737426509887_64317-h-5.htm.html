Project Gutenberg™の電子書籍は、多くの場合、複数の印刷版から作成されています。これらの印刷版はすべて、著作権表示が含まれていない限り、米国では著作権保護の対象ではないことが確認されています。したがって、電子書籍は必ずしも特定の紙の版に準拠しているわけではありません。
ほとんどの人は、メインの PG 検索機能を備えた当社の Web サイト ( www.gutenberg.org ) から始めます。
この Web サイトには、Project Gutenberg Literary Archive Foundation への寄付の方法、新しい電子書籍の制作に協力する方法、新しい電子書籍に関する情報を受け取るための電子メール ニュースレターの購読方法など、Project Gutenberg™ に関する情報が掲載されています。
