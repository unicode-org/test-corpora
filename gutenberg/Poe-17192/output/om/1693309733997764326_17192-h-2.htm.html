Yoo haala waliigaltee kanaa hunda eeguuf walii hin galle, kooppiiwwan hojiiwwan elektirooniksii Project GutenbergTM harka keessan jiran hunda fayyadamuu dhiisuu fi deebisuu ykn balleessuu qabdu. Hojii elektirooniksii Pirojektii GutenbergTM waraabbii argachuuf ykn argachuuf kaffaltii yoo kaffaltee fi haala waliigaltee kanaatiin akka hidhamtu yoo walii hin galle, nama ykn qaama kaffaltii kaffalte irraa akka keewwata 1.E.8 jalatti ibsametti deebi’ee argachuu dandeessa.
1.B. “Project Gutenberg” mallattoo daldalaa galmaa’eedha. Hojii elektirooniksii irratti ykn karaa kamiinuu itti fayyadamuu kan danda’u namoota haala waliigaltee kanaatiin akka hidhaman waliigalan qofa. Hojiiwwan elektirooniksii Project GutenbergTM irra caalaan isaanii haala waliigaltee kanaa guutuu osoo hin eegin illee wantoota muraasni hojjechuu dandeessan jiru. Keewwata 1.C armaan gadii ilaali. Hojiiwwan elektirooniksii Project GutenbergTM wajjin wantoota gochuu dandeessan baay’eedha yoo haala waliigaltee kanaa hordoftanii fi gara fuulduraatti hojiiwwan elektirooniksii Project GutenbergTM bilisaan argachuuf gargaartan. Keewwata 1.E armaan gadii ilaali.
1.C. Faawundeeshiniin Kuusaa Ogbarruu Pirojektii Gutenberg (“the Foundation” ykn PGLAF), walitti qabama hojiiwwan elektirooniksii Pirojektii GutenbergTM keessatti mirga waraabbii qindeessaa qaba. Hojiiwwan dhuunfaa walitti qabama kana keessa jiran hundi jechuun ni danda’ama Ameerikaa keessatti kan ummataaf ifa ta’u. Hojiin dhuunfaa tokko seera mirga waraabbii Ameerikaa keessatti eegumsa kan hin qabne yoo ta’ee fi Ameerikaatti kan argamtan yoo ta’e, hanga wabiiwwan Pirojektii Gutenberg hunduu haqamanitti hojiiwwan hojii sana irratti hundaa’an waraabuu, raabsuu, raawwachuu, agarsiisuu ykn uumuu irraa mirga isin dhorkuu hin gaafannu. Hojii kana yeroo kaffaltii malee namoota biroof qooddan Hayyama Pirojektii GutenbergTM guutuu isaa waliin walqabatee bifa walfakkaatuun haala waliigaltee kanaa salphaatti eeguu dandeessu.
1.D. Seerri mirga waraabbii iddoo ati argamtus hojii kanaan maal gochuu akka dandeessu ni bulcha. Seerri mirga waraabbii biyyoota baay’ee keessatti jijjiirama dhaabbataa keessa jira. Yoo Ameerikaan ala jirta ta’e, hojii kana ykn hojii Project GutenbergTM biroo kamiyyuu irratti hundaa’uun hojiiwwan bu’aa irraa argamanii buufachuu, waraabuu, agarsiisuu, raawwachuu, raabsuu ykn uumuu dura haala waliigaltee kanaa dabalatee seera biyya keetii ilaali. Faawundeeshinichi haala mirga waraabbii hojii kamiyyuu Ameerikaa malee biyya kamiyyuu keessatti bakka bu’ummaa hin kennu.
