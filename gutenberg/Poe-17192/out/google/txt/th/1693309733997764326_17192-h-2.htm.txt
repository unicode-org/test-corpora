*** สิ้นสุดโครงการ GUTENBERG EBOOK THE RAVEN ***

ฉบับที่อัปเดตจะแทนที่ฉบับก่อนหน้า โดยฉบับเก่าจะถูกเปลี่ยนชื่อ

การสร้างผลงานจากฉบับพิมพ์ที่ไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา หมายความว่าไม่มีใครเป็นเจ้าของลิขสิทธิ์ของสหรัฐอเมริกาในงานเหล่านี้ ดังนั้น มูลนิธิ (และคุณ!) จึงสามารถคัดลอกและแจกจ่ายในสหรัฐอเมริกาโดยไม่ได้รับอนุญาตและไม่ต้องจ่ายค่าลิขสิทธิ์ลิขสิทธิ์ กฎพิเศษที่กำหนดไว้ในส่วนข้อกำหนดการใช้งานทั่วไปของใบอนุญาตนี้ ใช้กับการคัดลอกและแจกจ่ายงานอิเล็กทรอนิกส์ของ Project Gutenberg™ เพื่อปกป้องแนวคิดและเครื่องหมายการค้าของ PROJECT GUTENBERG™ Project Gutenberg เป็นเครื่องหมายการค้าจดทะเบียน และไม่สามารถใช้ได้หากคุณเรียกเก็บเงินสำหรับ eBook เว้นแต่จะปฏิบัติตามเงื่อนไขของใบอนุญาตเครื่องหมายการค้า รวมถึงการชำระค่าลิขสิทธิ์สำหรับการใช้เครื่องหมายการค้า Project Gutenberg หากคุณไม่เรียกเก็บเงินใดๆ สำหรับสำเนาของ eBook นี้ การปฏิบัติตามใบอนุญาตเครื่องหมายการค้านั้นง่ายมาก คุณสามารถใช้ eBook นี้เพื่อวัตถุประสงค์ใดๆ ก็ได้ เช่น การสร้างผลงานลอกเลียนแบบ รายงาน การแสดง และการวิจัย eBooks ของ Project Gutenberg อาจได้รับการแก้ไข พิมพ์ และแจกจ่าย—คุณสามารถดำเนินการใดๆ ก็ได้ในสหรัฐอเมริกาโดยที่ eBooks ไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา การแจกจ่ายซ้ำต้องอยู่ภายใต้ใบอนุญาตเครื่องหมายการค้า โดยเฉพาะอย่างยิ่งการแจกจ่ายซ้ำในเชิงพาณิชย์

เริ่มต้น: ใบอนุญาตเต็มรูปแบบ

ใบอนุญาตโครงการ GUTENBERG เต็มรูปแบบ

โปรดอ่านสิ่งนี้ก่อนที่คุณจะแจกจ่ายหรือใช้งานนี้

เพื่อปกป้องพันธกิจของ Project Gutenberg™ ในการส่งเสริมการแจกจ่ายงานอิเล็กทรอนิกส์อย่างเสรี โดยการใช้หรือแจกจ่ายงานนี้ (หรืองานอื่นใดที่เกี่ยวข้องกับวลี “Project Gutenberg”) คุณตกลงที่จะปฏิบัติตามเงื่อนไขทั้งหมดของสิทธิ์ใช้งาน Full Project Gutenberg™ ที่มีอยู่ในไฟล์นี้หรือทางออนไลน์ที่ www.gutenberg.org/license

ส่วนที่ 1 ข้อกำหนดทั่วไปในการใช้งานและการแจกจ่ายงานอิเล็กทรอนิกส์ของ Project Gutenberg™

1.ก. การอ่านหรือใช้งานส่วนใดๆ ของงานอิเล็กทรอนิกส์ของ Project Gutenberg™ แสดงว่าคุณได้อ่าน ทำความเข้าใจ ตกลงและยอมรับเงื่อนไขทั้งหมดของข้อตกลงใบอนุญาตและทรัพย์สินทางปัญญา (เครื่องหมายการค้า/ลิขสิทธิ์) หากคุณไม่ตกลงที่จะปฏิบัติตามข้อกำหนดทั้งหมดของข้อตกลงนี้ คุณต้องหยุดใช้และส่งคืนหรือทำลายสำเนางานอิเล็กทรอนิกส์ของ Project Gutenberg™ ทั้งหมดที่คุณครอบครอง หากคุณชำระค่าธรรมเนียมในการรับสำเนาหรือการเข้าถึงงานอิเล็กทรอนิกส์ของ Project Gutenberg™ และคุณไม่ตกลงที่จะผูกพันตามเงื่อนไขของข้อตกลงนี้ คุณอาจได้รับเงินคืนจากบุคคลหรือนิติบุคคลที่คุณชำระค่าธรรมเนียมให้ตามที่กำหนดไว้ในย่อหน้าที่ 1.จ.8

1.บี. “โครงการกูเทนเบิร์ก” เป็นเครื่องหมายการค้าจดทะเบียน อาจใช้หรือเกี่ยวข้องกับงานอิเล็กทรอนิกส์ในทางใดทางหนึ่งโดยบุคคลที่ตกลงที่จะผูกพันตามเงื่อนไขของข้อตกลงนี้ มีบางสิ่งที่คุณสามารถทำได้กับงานอิเล็กทรอนิกส์ส่วนใหญ่ของ Project Gutenberg™ แม้ว่าจะไม่ปฏิบัติตามข้อกำหนดทั้งหมดของข้อตกลงนี้ก็ตาม ดูย่อหน้าที่ 1.ค ด้านล่าง มีหลายสิ่งที่คุณสามารถทำได้กับงานอิเล็กทรอนิกส์ของ Project Gutenberg™ หากคุณปฏิบัติตามข้อกำหนดของข้อตกลงนี้ และช่วยรักษาการเข้าถึงงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ได้ฟรีในอนาคต ดูย่อหน้าที่ 1.E ด้านล่าง

1.ค. Project Gutenberg Literary Archive Foundation (“มูลนิธิ” หรือ PGLAF) เป็นเจ้าของลิขสิทธิ์การรวบรวมในการรวบรวมผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ผลงานเกือบทั้งหมดในคอลเลกชันนี้เป็นสาธารณสมบัติในสหรัฐอเมริกา หากงานแต่ละชิ้นไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ในสหรัฐอเมริกา และคุณอยู่ในสหรัฐอเมริกา เราไม่เรียกร้องสิทธิที่จะป้องกันไม่ให้คุณคัดลอก แจกจ่าย ดำเนินการ แสดง หรือสร้างผลงานลอกเลียนแบบจากงานดังกล่าว ตราบใดที่การอ้างอิงถึง Project Gutenberg ทั้งหมดถูกลบออก แน่นอน เราหวังว่าคุณจะสนับสนุนภารกิจของ Project Gutenberg™ ในการส่งเสริมการเข้าถึงงานอิเล็กทรอนิกส์โดยไม่เสียค่าใช้จ่ายด้วยการแบ่งปันงานของ Project Gutenberg™ อย่างอิสระตามเงื่อนไขของข้อตกลงนี้เพื่อรักษาชื่อ Project Gutenberg™ ที่เชื่อมโยงกับงานนี้ คุณสามารถปฏิบัติตามข้อกำหนดของข้อตกลงนี้ได้อย่างง่ายดายโดยทำให้งานนี้อยู่ในรูปแบบเดียวกันพร้อมกับสิทธิ์ใช้งาน Project Gutenberg™ แบบเต็มที่แนบมาด้วย เมื่อคุณแบ่งปันกับผู้อื่นโดยไม่คิดค่าใช้จ่าย

1.D. กฎหมายลิขสิทธิ์ของสถานที่ที่คุณอยู่ยังควบคุมสิ่งที่คุณสามารถทำได้กับงานนี้ด้วย กฎหมายลิขสิทธิ์ในประเทศส่วนใหญ่มีการเปลี่ยนแปลงอยู่ตลอดเวลา หากคุณอยู่นอกสหรัฐอเมริกา โปรดตรวจสอบกฎหมายในประเทศของคุณนอกเหนือจากข้อกำหนดของข้อตกลงนี้ ก่อนที่จะดาวน์โหลด คัดลอก แสดง ดำเนินการ แจกจ่าย หรือสร้างผลงานลอกเลียนแบบจากงานนี้หรืองาน Project Gutenberg™ อื่น ๆ มูลนิธิไม่ได้รับรองเกี่ยวกับสถานะลิขสิทธิ์ของงานใดๆ ในประเทศใดๆ นอกเหนือจากสหรัฐอเมริกา

1.จ. เว้นแต่คุณได้ลบการอ้างอิงถึง Project Gutenberg ทั้งหมดแล้ว:

1.จ.1. ประโยคต่อไปนี้ ซึ่งมีลิงก์ที่ใช้งานอยู่หรือการเข้าถึงโดยทันที สิทธิ์การใช้งาน Project Gutenberg™ แบบเต็มจะต้องปรากฏอย่างเด่นชัดเมื่อใดก็ตามที่สำเนาของงาน Project Gutenberg™ (งานใดๆ ที่มีวลี “Project Gutenberg” ปรากฏ หรือที่เกี่ยวข้องกับวลี “Project Gutenberg” ที่เกี่ยวข้อง) มีการเข้าถึง แสดง ดำเนินการ ดู คัดลอก หรือแจกจ่าย:

eBook นี้มีไว้สำหรับทุกคนในสหรัฐอเมริกาและส่วนอื่นๆ ของโลกโดยไม่มีค่าใช้จ่ายและแทบไม่มีข้อจำกัดใดๆ เลย คุณสามารถคัดลอก แจกให้ หรือใช้ซ้ำได้ภายใต้เงื่อนไขของ Project Gutenberg License ที่มาพร้อมกับ eBook นี้ หรือทางออนไลน์ที่ www.gutenberg.org หากคุณไม่ได้อยู่ในสหรัฐอเมริกา คุณจะต้องตรวจสอบกฎหมายของประเทศที่คุณอยู่ก่อนใช้ eBook เล่มนี้

1.จ.2. หากงานอิเล็กทรอนิกส์ของ Project Gutenberg™ แต่ละรายการได้มาจากข้อความที่ไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา (ไม่มีประกาศที่ระบุว่ามีการโพสต์โดยได้รับอนุญาตจากผู้ถือลิขสิทธิ์) งานดังกล่าวสามารถคัดลอกและแจกจ่ายให้กับใครก็ได้ในสหรัฐอเมริกาโดยไม่ต้องจ่ายค่าธรรมเนียมหรือค่าใช้จ่ายใดๆ หากคุณกำลังแจกจ่ายต่อหรือให้การเข้าถึงงานที่มีวลี "Project Gutenberg" เกี่ยวข้องหรือปรากฏในงาน คุณต้องปฏิบัติตามข้อกำหนดของย่อหน้า 1.E.1 ถึง 1.E.7 หรือได้รับอนุญาตสำหรับการใช้งานและเครื่องหมายการค้า Project Gutenberg™ ตามที่กำหนดไว้ในย่อหน้า 1.E.8 หรือ 1.E.9

1.จ.3. หากมีการโพสต์งานอิเล็กทรอนิกส์ Project Gutenberg™ แต่ละชิ้นโดยได้รับอนุญาตจากผู้ถือลิขสิทธิ์ การใช้และการแจกจ่ายของคุณจะต้องเป็นไปตามทั้งย่อหน้า 1.E.1 ถึง 1.E.7 และข้อกำหนดเพิ่มเติมใด ๆ ที่กำหนดโดยผู้ถือลิขสิทธิ์ ข้อกำหนดเพิ่มเติมจะเชื่อมโยงกับสิทธิ์ใช้งาน Project Gutenberg™ สำหรับงานทั้งหมดที่โพสต์โดยได้รับอนุญาตจากผู้ถือลิขสิทธิ์ที่พบในตอนเริ่มต้นของงานนี้

1.จ.4. อย่ายกเลิกการเชื่อมโยงหรือแยกหรือลบข้อกำหนดสิทธิ์การใช้งาน Project Gutenberg™ ฉบับเต็มออกจากงานนี้ หรือไฟล์ใดๆ ที่มีส่วนหนึ่งของงานนี้หรืองานอื่นใดที่เกี่ยวข้องกับ Project Gutenberg™

1.จ.5. ห้ามคัดลอก แสดง ดำเนินการ แจกจ่ายหรือแจกจ่ายงานอิเล็กทรอนิกส์นี้หรือส่วนใดส่วนหนึ่งของงานอิเล็กทรอนิกส์นี้ โดยไม่แสดงประโยคที่กำหนดไว้ในย่อหน้า 1.E.1 อย่างเด่นชัดพร้อมลิงก์ที่ใช้งานได้หรือเข้าถึงข้อกำหนดทั้งหมดของสิทธิ์การใช้งาน Project Gutenberg™ ได้ในทันที

1.จ.6. คุณสามารถแปลงและแจกจ่ายงานนี้ในรูปแบบไบนารี บีบอัด มาร์กอัป ไม่มีกรรมสิทธิ์หรือเป็นกรรมสิทธิ์ รวมถึงรูปแบบการประมวลผลคำหรือไฮเปอร์เท็กซ์ อย่างไรก็ตาม หากคุณให้สิทธิ์ในการเข้าถึงหรือแจกจ่ายสำเนาของงาน Project Gutenberg™ ในรูปแบบอื่นที่ไม่ใช่ "Plain Vanilla ASCII" หรือรูปแบบอื่นที่ใช้ในเวอร์ชันอย่างเป็นทางการที่โพสต์บนเว็บไซต์ทางการของ Project Gutenberg™ (www.gutenberg.org) คุณจะต้องจัดเตรียมสำเนา วิธีการส่งออกสำเนา หรือวิธีการขอรับสำเนาของงานใน "Plain Vanilla ASCII" ต้นฉบับหรือรูปแบบอื่น ๆ ให้กับผู้ใช้โดยไม่มีค่าใช้จ่ายเพิ่มเติม รูปแบบทางเลือกใดๆ จะต้องมี Project Gutenberg™ License ฉบับเต็มตามที่ระบุไว้ในย่อหน้าที่ 1.E.1

1.จ.7. อย่าเรียกเก็บค่าธรรมเนียมในการเข้าถึง ดู แสดง ดำเนินการ คัดลอก หรือแจกจ่ายผลงานของ Project Gutenberg™ เว้นแต่คุณจะปฏิบัติตามย่อหน้าที่ 1.จ.8 หรือ 1.จ.9

1.จ.8. คุณอาจเรียกเก็บค่าธรรมเนียมที่เหมาะสมสำหรับสำเนาหรือให้การเข้าถึงหรือแจกจ่ายงานอิเล็กทรอนิกส์ของ Project Gutenberg™ โดยมีเงื่อนไขว่า:

• คุณชำระค่าลิขสิทธิ์ 20% ของกำไรขั้นต้นที่คุณได้รับจากการใช้งาน Project Gutenberg™ ซึ่งคำนวณโดยใช้วิธีที่คุณใช้ในการคำนวณภาษีที่เกี่ยวข้องของคุณอยู่แล้ว ค่าธรรมเนียมดังกล่าวเป็นของเจ้าของเครื่องหมายการค้า Project Gutenberg™ แต่เขาตกลงที่จะบริจาคค่าลิขสิทธิ์ภายใต้ย่อหน้านี้ให้กับ Project Gutenberg Literary Archive Foundation การชำระค่าลิขสิทธิ์จะต้องชำระภายใน 60 วันนับจากวันที่คุณจัดเตรียม (หรือจำเป็นต้องจัดเตรียมตามกฎหมาย) การขอคืนภาษีเป็นระยะ การชำระค่าลิขสิทธิ์ควรทำเครื่องหมายไว้อย่างชัดเจน และส่งไปยัง Project Gutenberg Literary Archive Foundation ตามที่อยู่ที่ระบุไว้ในมาตรา 4 “ข้อมูลเกี่ยวกับการบริจาคให้กับ Project Gutenberg Literary Archive Foundation”

• คุณคืนเงินเต็มจำนวนที่จ่ายโดยผู้ใช้ที่แจ้งให้คุณทราบเป็นลายลักษณ์อักษร (หรือทางอีเมล) ภายใน 30 วันนับจากวันที่ได้รับ โดยที่ผู้ใช้ไม่ยอมรับเงื่อนไขของ Project Gutenberg™ License ฉบับเต็ม คุณต้องกำหนดให้ผู้ใช้ดังกล่าวส่งคืนหรือทำลายสำเนาของงานทั้งหมดที่มีอยู่ในสื่อบันทึกจริง และหยุดการใช้งานและการเข้าถึงสำเนาอื่น ๆ ของงาน Project Gutenberg™ ทั้งหมด

• คุณคืนเงินเต็มจำนวนตามวรรค 1.F.3 ที่จ่ายให้กับงานหรือสำเนาทดแทน หากพบข้อบกพร่องในงานอิเล็กทรอนิกส์และรายงานให้คุณทราบภายใน 90 วันนับจากวันที่ได้รับงาน

• คุณปฏิบัติตามข้อกำหนดอื่นๆ ทั้งหมดของข้อตกลงนี้สำหรับการเผยแพร่งาน Project Gutenberg™ ฟรี

1.จ.9. หากคุณต้องการเรียกเก็บค่าธรรมเนียมหรือแจกจ่ายงานอิเล็กทรอนิกส์หรือกลุ่มผลงานของ Project Gutenberg™ ในเงื่อนไขที่แตกต่างจากที่กำหนดไว้ในข้อตกลงนี้ คุณต้องได้รับอนุญาตเป็นลายลักษณ์อักษรจาก Project Gutenberg Literary Archive Foundation ซึ่งเป็นผู้จัดการของเครื่องหมายการค้า Project Gutenberg™ ติดต่อมูลนิธิตามที่กำหนดไว้ในมาตรา 3 ด้านล่าง

1.ฟ.

1.ฟ.1. อาสาสมัครและพนักงานของ Project Gutenberg ใช้ความพยายามอย่างมากในการระบุ วิจัยลิขสิทธิ์ ถอดเสียง และตรวจทานผลงานที่ไม่ได้รับการคุ้มครองโดยกฎหมายลิขสิทธิ์ของสหรัฐอเมริกาในการสร้างคอลเลกชัน Project Gutenberg™ แม้จะมีความพยายามเหล่านี้ งานอิเล็กทรอนิกส์ของ Project Gutenberg™ และสื่อที่อาจจัดเก็บไว้นั้นอาจมี "ข้อบกพร่อง" เช่น แต่ไม่จำกัดเพียงข้อมูลที่ไม่สมบูรณ์ ไม่ถูกต้องหรือเสียหาย ข้อผิดพลาดในการถอดความ ลิขสิทธิ์หรือการละเมิดทรัพย์สินทางปัญญาอื่นๆ ดิสก์หรือสื่ออื่นๆ ที่มีข้อบกพร่องหรือเสียหาย ไวรัสคอมพิวเตอร์ หรือรหัสคอมพิวเตอร์ที่ทำให้อุปกรณ์ของคุณเสียหายหรือไม่สามารถอ่านได้

1.ฉ.2. การรับประกันแบบจำกัด การปฏิเสธความรับผิดต่อความเสียหาย - ยกเว้น "สิทธิ์ในการเปลี่ยนหรือการคืนเงิน" ที่อธิบายไว้ในย่อหน้าที่ 1.F.3, Project Gutenberg Literary Archive Foundation เจ้าของเครื่องหมายการค้า Project Gutenberg™ และฝ่ายอื่นใดที่เผยแพร่งานอิเล็กทรอนิกส์ของ Project Gutenberg™ ภายใต้ข้อตกลงนี้ จะไม่รับผิดชอบต่อความเสียหาย ต้นทุน และค่าใช้จ่าย รวมถึงค่าธรรมเนียมทางกฎหมายทั้งหมดต่อคุณ คุณยอมรับว่าคุณไม่มีวิธีแก้ไขใด ๆ สำหรับความประมาทเลินเล่อ ความรับผิดที่เข้มงวด การละเมิดการรับประกัน หรือการละเมิดสัญญา ยกเว้นที่ระบุไว้ในย่อหน้า 1.F.3 คุณยอมรับว่ามูลนิธิ เจ้าของเครื่องหมายการค้า และผู้จัดจำหน่ายใดๆ ภายใต้ข้อตกลงนี้จะไม่รับผิดต่อคุณสำหรับความเสียหายที่เกิดขึ้นจริง โดยตรง โดยอ้อม ที่เป็นผลสืบเนื่อง การลงโทษ หรือโดยบังเอิญ แม้ว่าคุณจะแจ้งให้ทราบถึงความเป็นไปได้ของความเสียหายดังกล่าวก็ตาม

1.ฉ.3. สิทธิ์ในการเปลี่ยนหรือคืนเงินอย่างจำกัด - หากคุณพบข้อบกพร่องในงานอิเล็กทรอนิกส์นี้ภายใน 90 วันหลังจากได้รับงาน คุณสามารถรับเงินคืน (ถ้ามี) ที่คุณจ่ายไปโดยการส่งคำอธิบายเป็นลายลักษณ์อักษรไปยังบุคคลที่คุณได้รับงานจาก หากคุณได้รับงานในรูปแบบสื่อทางกายภาพ คุณจะต้องส่งคืนสื่อดังกล่าวพร้อมกับคำอธิบายที่เป็นลายลักษณ์อักษรของคุณ บุคคลหรือนิติบุคคลที่มอบผลงานที่มีข้อบกพร่องให้กับคุณอาจเลือกที่จะจัดเตรียมสำเนาทดแทนแทนการคืนเงิน หากคุณได้รับงานทางอิเล็กทรอนิกส์ บุคคลหรือนิติบุคคลที่จัดหางานให้กับคุณอาจเลือกที่จะให้โอกาสครั้งที่สองแก่คุณในการรับงานทางอิเล็กทรอนิกส์แทนการคืนเงิน หากสำเนาที่สองมีข้อบกพร่อง คุณสามารถขอเงินคืนเป็นลายลักษณ์อักษรได้โดยไม่ต้องมีโอกาสแก้ไขปัญหาอีก

1.ฉ.4. ยกเว้นสิทธิ์ที่จำกัดในการเปลี่ยนหรือการคืนเงินที่กำหนดไว้ในย่อหน้า 1.F.3 งานนี้จัดเตรียมไว้ให้คุณ 'ตามที่เป็น' โดยไม่มีการรับประกันอื่นใดทั้งโดยชัดแจ้งหรือโดยนัย รวมถึงแต่ไม่จำกัดเพียงการรับประกันความสามารถในการค้าขายหรือความเหมาะสมสำหรับวัตถุประสงค์ใดๆ

1.F.5. บางรัฐไม่อนุญาตให้ปฏิเสธความรับผิดชอบต่อการรับประกันโดยนัยบางประเภท หรือการยกเว้นหรือจำกัดความเสียหายบางประเภท หากข้อจำกัดความรับผิดชอบหรือข้อจำกัดใดๆ ที่กำหนดไว้ในข้อตกลงนี้ฝ่าฝืนกฎหมายของรัฐที่ใช้บังคับกับข้อตกลงนี้ ข้อตกลงนี้จะถูกตีความเพื่อให้เป็นข้อจำกัดความรับผิดชอบหรือข้อจำกัดสูงสุดที่ได้รับอนุญาตโดยกฎหมายของรัฐที่ใช้บังคับ ความเป็นโมฆะหรือไม่สามารถบังคับใช้ได้ของข้อกำหนดใด ๆ ของข้อตกลงนี้จะไม่ทำให้ข้อกำหนดที่เหลือเป็นโมฆะ

1.ฉ.6. การชดใช้ค่าเสียหาย - คุณตกลงที่จะชดใช้ค่าเสียหายและถือครองมูลนิธิ เจ้าของเครื่องหมายการค้า ตัวแทนหรือพนักงานของมูลนิธิ ใครก็ตามที่จัดหาสำเนาของงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ตามข้อตกลงนี้ และอาสาสมัครใด ๆ ที่เกี่ยวข้องกับการผลิต การส่งเสริมการขาย และการจัดจำหน่ายงานอิเล็กทรอนิกส์ของ Project Gutenberg™ จะไม่เป็นอันตรายต่อความรับผิด ต้นทุนและค่าใช้จ่ายทั้งหมด รวมถึงค่าธรรมเนียมทางกฎหมายที่เกิดขึ้นโดยตรงหรือโดยอ้อมจากสิ่งต่อไปนี้ที่คุณทำหรือก่อให้เกิด: (a) การแจกจ่ายสิ่งนี้หรืองานของ Project Gutenberg™ ใด ๆ (b) การเปลี่ยนแปลง ดัดแปลง หรือเพิ่มเติมหรือลบงาน Project Gutenberg™ และ (c) ข้อบกพร่องใดๆ ที่คุณก่อขึ้น

ส่วนที่ 2 ข้อมูลเกี่ยวกับภารกิจของ Project Gutenberg™

Project Gutenberg™ มีความหมายเหมือนกันกับการเผยแพร่ผลงานอิเล็กทรอนิกส์ในรูปแบบที่คอมพิวเตอร์หลากหลายประเภทสามารถอ่านได้ฟรี รวมถึงคอมพิวเตอร์ที่ล้าสมัย เก่า วัยกลางคน และใหม่ เกิดขึ้นได้เพราะความพยายามของอาสาสมัครหลายร้อยคนและการบริจาคจากผู้คนทุกสาขาอาชีพ

อาสาสมัครและการสนับสนุนทางการเงินเพื่อมอบความช่วยเหลือที่จำเป็นแก่อาสาสมัครมีความสำคัญอย่างยิ่งต่อการบรรลุเป้าหมายของ Project Gutenberg™ และสร้างความมั่นใจว่าคอลเลกชัน Project Gutenberg™ จะยังคงใช้ได้อย่างอิสระสำหรับคนรุ่นต่อ ๆ ไป ในปี 2544 มูลนิธิ Project Gutenberg Literary Archive Foundation ถูกสร้างขึ้นเพื่อมอบอนาคตที่มั่นคงและถาวรสำหรับ Project Gutenberg™ และคนรุ่นต่อๆ ไป หากต้องการเรียนรู้เพิ่มเติมเกี่ยวกับ Project Gutenberg Literary Archive Foundation และวิธีที่ความพยายามและการบริจาคของคุณสามารถช่วยเหลือได้ โปรดดูส่วนที่ 3 และ 4 และหน้าข้อมูลของมูลนิธิที่ www.gutenberg.org

หมวดที่ 3 ข้อมูลเกี่ยวกับมูลนิธิหอจดหมายเหตุวรรณกรรมโครงการ Gutenberg

Project Gutenberg Literary Archive Foundation เป็นองค์กรการศึกษาที่ไม่แสวงหาผลกำไรตามมาตรา 501(c)(3) ซึ่งจัดตั้งขึ้นภายใต้กฎหมายของรัฐมิสซิสซิปปี้ และได้รับสถานะได้รับการยกเว้นภาษีจาก Internal Revenue Service EIN ของมูลนิธิหรือหมายเลขประจำตัวผู้เสียภาษีของรัฐบาลกลางคือ 64-6221541 การบริจาคให้กับ Project Gutenberg Literary Archive Foundation สามารถนำไปหักลดหย่อนภาษีได้เต็มขอบเขตที่ได้รับอนุญาตโดยกฎหมายรัฐบาลกลางของสหรัฐอเมริกาและกฎหมายของรัฐของคุณ

สำนักงานธุรกิจของมูลนิธิตั้งอยู่ที่ 809 North 1500 West, Salt Lake City, UT 84116, (801) 596-1887 ลิงค์ติดต่อทางอีเมลและข้อมูลติดต่อล่าสุดสามารถดูได้ที่เว็บไซต์ของมูลนิธิและหน้าอย่างเป็นทางการที่ www.gutenberg.org/contact

หมวดที่ 4 ข้อมูลเกี่ยวกับการบริจาคให้กับมูลนิธิหอจดหมายเหตุวรรณกรรมโครงการ Gutenberg

Project Gutenberg™ พึ่งพาและไม่สามารถดำรงอยู่ได้หากไม่ได้รับการสนับสนุนจากสาธารณะและการบริจาคอย่างกว้างขวาง เพื่อดำเนินภารกิจในการเพิ่มจำนวนสาธารณสมบัติและผลงานที่ได้รับลิขสิทธิ์ ซึ่งสามารถเผยแพร่ได้อย่างอิสระในรูปแบบที่เครื่องอ่านได้ ซึ่งสามารถเข้าถึงได้โดยอุปกรณ์ที่หลากหลายที่สุด รวมถึงอุปกรณ์ที่ล้าสมัย การบริจาคจำนวนเล็กน้อยจำนวนมาก ($1 ถึง $5,000) มีความสำคัญอย่างยิ่งต่อการรักษาสถานะการยกเว้นภาษีกับ IRS

มูลนิธิมุ่งมั่นที่จะปฏิบัติตามกฎหมายที่ควบคุมองค์กรการกุศลและการบริจาคเพื่อการกุศลใน 50 รัฐของสหรัฐอเมริกา ข้อกำหนดในการปฏิบัติตามข้อกำหนดไม่เหมือนกัน และต้องใช้ความพยายามอย่างมาก ต้องใช้เอกสารจำนวนมาก และค่าธรรมเนียมมากมายในการปฏิบัติตามและปฏิบัติตามข้อกำหนดเหล่านี้ เราไม่ขอรับบริจาคในสถานที่ที่เราไม่ได้รับการยืนยันเป็นลายลักษณ์อักษรเกี่ยวกับการปฏิบัติตามข้อกำหนด หากต้องการส่งเงินบริจาคหรือกำหนดสถานะของการปฏิบัติตามข้อกำหนดสำหรับรัฐใด ๆ โปรดไปที่ www.gutenberg.org/donate

แม้ว่าเราจะไม่สามารถและไม่ร้องขอการบริจาคจากรัฐที่เราไม่ปฏิบัติตามข้อกำหนดในการชักชวน แต่เราทราบดีว่าไม่มีข้อห้ามใด ๆ ที่จะยอมรับการบริจาคที่ไม่พึงประสงค์จากผู้บริจาคในรัฐดังกล่าวที่เข้ามาหาเราเพื่อเสนอที่จะบริจาค

เรายอมรับการบริจาคระหว่างประเทศอย่างสุดซึ้ง แต่เราไม่สามารถแถลงใดๆ เกี่ยวกับการรักษาภาษีของการบริจาคที่ได้รับจากนอกสหรัฐอเมริกาได้ กฎหมายของสหรัฐอเมริกาเพียงอย่างเดียวทำให้พนักงานเล็กๆ ของเราล้นหลาม
