*** ТҮСЭЛЭЙ ДҮҮРГЭЛ ГУТЕНБЕРГЭЙ ЭЛЕКТРОННО НОМ ХЭРЭГЛЭЛ ***
Шэнэлэгдэһэн хэблэлнүүд урданайхиие һэлгэхэ — хуушан хэблэлнүүдэй нэрэ һэлгэгдэхэ.
АНУ-ай зохёогшын эрхын хуулиар хамгаалагдаагүй хэблэлэй хэблэлнүүдһээ бүтээлнүүдые бүтээхэ гээшэ эдэ бүтээлнүүдтэ АНУ-ай зохёогшын эрхэ хэншье эзэмшэхэгүй гэһэн удхатай, тиимэһээ Сан (ба та!) зүбшөөлгүйгөөр, зохёогшын эрхын гонорар түлэхэгүйгөөр АНУ-да хуулбарилжа, тараажа болохо. Энэ лицензиин Хэрэглэхэ юрэнхы дүримүүдэй хэһэгтэ зааһан тусхай дүримүүд ProJECT GUTENBERGTM ойлгосо болон наймаанай тэмдэгые хамгаалхын тулада Project GutenbergTM электрон бүтээлнүүдые хуулбарилха ба тарааха хэрэгтэ хабаатай. Project Gutenberg хадаа бүридхэлдэ абтаһан наймаанай тэмдэг болоод, хэрбээ та электронно номой түлөө түлбэри абаа һаа, наймаанай тэмдэгэй лицензиин дүримүүдые дагажа, тэрэ тоодо Project Gutenberg наймаанай тэмдэгые хэрэглэһэнэй түлөө гонорар түлэхэһөө бэшэ хэрэглэжэ болохогүй. Хэрбээ энэ электронно номой хуулбаринуудай түлөө юушье абахагүй һаа, наймаанай тэмдэгэй лицензи дүүргэхэнь тон бэлэн. Та энэ электронно номые ямаршье зорилгоор хэрэглэжэ болохот, жэшээнь, гаралтай ажалнуудые, мэдээсэлнүүдые, тоглолтонуудые болон шэнжэлэлгэнүүдые зохёохо. Project Gutenberg электрон номуудые хубилгажа, хэблэжэ, үгэжэ болохо — та АНУ-да АНУ-ай зохёогшын эрхын хуулиар хамгаалагдаагүй электрон номуудаар юушье хэжэ болохот. Дахин хубаарилалта наймаанай тэмдэгэй лицензидэ хамаарна, илангаяа олзын хэрэг эрхилэлгын дахин хубаарилалта.
ЭХИЛЭЛ: ДҮҮРЭН ЛИЦЕНЗИ

ГУТЕНБЕРГЭЙ БҮТЭН ТҮСЭЛЭЙ ЛИЦЕНЗИ

ЭНЭ АЖАЛЫЕ ТУРААХА ГҮ, АЛИ ХЭРЭГЛЭХЭҺЭЭ УРДА ЭНЭЭ УНШАГТЫ
Электрон бүтээлнүүдые түлбэригүй тарааха зорилготой Project GutenbergTM-эй зорилгые хамгаалхын тулада энэ бүтээлые (гү, али “Project Gutenberg” гэһэн үгэтэй ямар нэгэн аргаар холбоотой бусад бүтээлнүүдые) хэрэглэжэ гү, али тараажа, та энэ файлтай гү, али www.gutenberg.org/licenberg гэһэн сайт дээрэ үгтэһэн Full Project GutenbergTM License-эй бүхы нэрэ томьёонуудые дүүргэхэеэ зүбшөөнэ.
1-дэхи хуби. Хэрэглэхэ ба дахин хубааха ниитэ дүримүүд Project GutenbergTM электронно бүтээлнүүдые
1.А. Энэ Project GutenbergTM электронно ажал уншажа гү, али хэрэглэжэ байхадаа, та энэ лицензиин болон оюун ухаанай зөөриин (наймаанай тэмдэг/зохиогшын эрхэ) хэлсээнэй бүхы нэрэ томьёонуудые уншаһанаа, ойлгоһоноо, зүбшөөһэнөө, зүбшөөһэнээ харуулна. Хэрбээ та энэ хэлсээнэй бүхы дүримүүдые дүүргэхэеэ зүбшөөнэгүй һаа, та өөрынгөө мэдэлдэ байһан Project GutenbergTM электронно бүтээлнүүдэй бүхы хуулбаринуудые хэрэглэхэеэ болижо, бусааха гү, али үгы хэхэ ёһотойт. Хэрбээ та Project GutenbergTM электронно бүтээлэй хуулбари абаха гү, али нэбтэрхэ түлбэри түлэһэн байгаад, энэ хэлсээнэй дүримүүдые дүүргэхэеэ зүбшөөнэгүй һаа, 1.E.8-дахи абзацта зааһан ёһоор түлбэри түлэһэн хүнһөө гү, али байгуулгаһаа бусаалга абажа болохот.
1.Б. “Проект Гутенберг” хадаа бүридхэлдэ абтаһан наймаанай тэмдэг юм. Энэ хэлсээнэй дүримүүдые дүүргэхэеэ зүбшөөһэн хүнүүд гансал электронно бүтээл дээрэ хэрэглэжэ гү, али ямаршье аргаар холбожо болохо. Энэ хэлсээнэй дүүрэн дүримүүдые дүүргээгүйшье һаа, Project GutenbergTM электронно бүтээлнүүдэй ехэнхидээ хэжэ болохо хэдэн зүйлнүүд бии. Доорохи 1.С абзац харагты. Хэрбээ та энэ хэлсээнэй дүримүүдые дагажа, Project GutenbergTM электронно бүтээлнүүдтэ ерээдүйдэ түлбэригүй нэбтэрхые хадгалхада туһалбал, Project GutenbergTM электронно бүтээлнүүдтэй олон юумэ хэжэ болохот. Доорохи 1.Е абзац харагты.
1.C. Гутенбергын түсэлэй уран зохёолой архивай жаса ("Сан" гү, али PGLAF), Гутенбергын түсэлэй электронно бүтээлнүүдэй суглуулбариин зохёолой эрхэтэй. Суглуулбари соохи бүхы шахуу бүтээлнүүд АНУ-да олониитын мэдэлдэ байна. Хэрбээ тусхай бүтээл АНУ-да зохёогшын эрхын хуулиин ёһоор хамгаалагдаагүй байгаад, та АНУ-да байрладаг һаа, Гутенбергын түсэл тухай бүхы тэмдэглэлнүүдые усадхаһан байгаа һаа, бидэ таанадые энэ бүтээл дээрэ үндэһэлһэн бүтээлнүүдые хуулбарилха, тарааха, гүйсэдхэхэ, харуулха гү, али бүтээхэһээ һаалта хэхэ эрхэтэй гэжэ мэдүүлнэгүйбди. Мэдээжэ, энэ хэлсээнэй ёһоор Project GutenbergTM нэрэтэй холбоотой байхын тулада Project GutenbergTM бүтээлнүүдые сүлөөтэй хубаалдажа, электронно бүтээлнүүдтэ түлбэригүй нэбтэрхые дэмжэхэ Project GutenbergTM зорилго дэмжэхэ гэжэ найданабди. Та энэ ажалые бусадтай түлбэригүйгөөр хубаалдахадаа хабсарһан Project GutenbergTM License-тэй адли форматтай байлгажа, энэ хэлсээнэй дүримүүдые амархан дүүргэжэ болохот.
1.Д. Танай байрладаг газарай зохёогшын эрхын хуулинууд энэ ажалаар юу хэжэ болохоб гэжэ баһа заана. Ехэнхи гүрэнүүдтэ зохёогшын эрхын хуулинууд хододоо хубилжа байдаг. Хэрбээ та АНУ-һаа гадуур байгаа һаа, энэ ажал гү, али Project GutenbergTM-эй бусад ажал дээрэ үндэһэлһэн бүтээлнүүдые татажа абаха, хуулбарилха, харуулха, гүйсэдхэхэ, тарааха гү, али бүтээхэһээ урид энэ хэлсээнэй нэрэ томьёонуудһаа гадна өөрынгөө уласай хуулинуудые шалгаха хэрэгтэй. Сан АНУ-һаа бэшэ ямаршье ороной ямаршье зохёолой зохёогшын эрхын байдал тухай ямаршье мэдүүлэл үгэнэгүй.
1.Э. Гутенбергын түсэлэй бүхы тэмдэглэлнүүдые усадхаагүй һаа:
1.Е.1. Project GutenbergTM лицензиин эдэбхитэй холбооһонуудтай гү, али бусад шууд нэбтэрхэй толитой иимэ мэдүүлэл Project GutenbergTM ажалай хуулбари (“Project Gutenberg” гэһэн үгэ бии болоһон алибаа ажал, али “Project Gutenberg” гэһэн үгэтэй холбоотой ямаршье ажал) орохо, харуулха, хуулбарилха гү, али тарааха үедэ эли тодоор харагдаха ёһотой:
Энэ электронно ном АНУ-ай болон дэлхэйн бусад нютагуудта хэншье үнэгүйгөөр, ямаршье хизаарлалтагүйгөөр хэрэглэхэ аргатай. Та энэ электронно номтой хамта гү, али www.gutenberg.org сайт дээрэ оруулагдаһан Гутенбергын түсэлэй лицензиин ёһоор хуулбарилжа, үгэжэ гү, али дахин хэрэглэжэ болохот. Хэрбээ та АНУ-да бэшэ һаа, энэ электронно ном хэрэглэхэһээ урид байһан оронойнгоо хуули шалгаха хэрэгтэй.
1.Е.2. Хэрбээ Project GutenbergTM электронно бүтээл АНУ-ай зохёогшын эрхын хуулиар хамгаалагдаагүй текстнүүдһээ абтаһан байгаа һаань (зохёогшын эрхын эзэнэй зүбшөөлтэйгээр табигдаһан тухай мэдээсэл үгы), энэ бүтээлые хуулбарилжа, ямаршье түлбэри түлэхэгүйгөөр АНУ-да хэншье тараажа болохо. Хэрбээ та “Project Gutenberg” гэһэн үгэтэйгээр ажалаа дахин тараажа гү, али нэбтэрүүлжэ байгаа һаа, 1.E.1-һээ 1.E.7 хүрэтэрхи абзацуудай эрилтэнүүдые дүүргэхэ гү, али 1.E.1 гү, али 1.E.8-дахи абзацуудта зааһан ёһоор Project GutenbergTM наймаанай тэмдэгые хэрэглэхэ зүбшөөл абаха ёһотойт.
1.Е.3. Хэрбээ Project GutenbergTM электронно бүтээл зохёогшын эрхын эзэнэй зүбшөөлтэйгээр табигдаһан байгаа һаань, танай хэрэглэлгэ ба тараалга 1.E.1-һээ 1.E.7 хүрэтэрхи абзацуудта болон зохёогшын эрхын эзэнэй табиһан нэмэлтэ нэрэ томьёонуудта тааруу байха ёһотой. Энэ ажалай эхиндэ зохёогшын эрхын эзэнэй зүбшөөлтэйгээр табигдаһан бүхы бүтээлнүүдтэ нэмэлтэ нэрэ томьёонууд Project GutenbergTM License-тэй холбогдохо.
1.Е.4. Энэ ажалһаа, энэ ажалһаа гү, али Project GutenbergTM-тэй холбоотой бусад ажалһаа Project GutenbergTM Лицензиин дүүрэн нэрэ томьёонуудые бү һалгаагты, бү һалгаагты, бү усадхагты.
1.Е.5. Энэ электронно бүтээлые гү, али энэ электронно бүтээлэй али нэгэ хэһэгые 1.E.1-дэхи абзацта зааһан мэдүүлэлые эдэбхитэй холбооһонуудтай гү, али Project GutenbergTM License-эй дүүрэн нэрэ томьёонуудта дары түргэн нэбтэрхэгүйгөөр хуулбарилха, харуулха, гүйсэдхэхэ, тарааха гү, али дахин тараахагүй.
1.Е.6. Та энэ ажалые ямаршье хоёртын, шахагдаһан, тэмдэглэгдэһэн, өөрын бэшэ гү, али өөрын хэлбэридэ, тэрэ тоодо ямаршье үгэ болбосоруулха гү, али гипертекст хэлбэридэ хубилгажа, тараажа болохот. Гэбэшье, хэрбээ та Project GutenbergTM-эй ажалай хуулбаринуудые “Plain Vanilla ASCII” гү, али Project GutenbergTM-эй албан ёһоной сайт дээрэ (www.gutenberg.org) табигдаһан албан ёһоной хубилалтада хэрэглэгдэдэг ондоо форматтай нэбтэрхэ гү, али тарааха эрхэ олгобол, хэрэглэгшэдэй нэмэлтэ түлбэри, түлбэригүйгөөр хуулбари, хуулбариие экспортировалха арга боломжо гү, али хуулбари абаха арга боломжо олгохо ёһотойт хүсэлтээр хуулбари, анханай “Plain Vanilla ASCII” гү, али ондоо түхэлөөр. Ямаршье ондоо формат 1.E.1-дэхи абзацта зааһан ёһоор Project GutenbergTM License-ые дүүрэн оруулха ёһотой.
1.Е.7. Хэрбээ та 1.E.8 гү, али 1.E.9-дэхи абзацуудые дүүргээгүй һаа, ямаршье Project GutenbergTM бүтээлнүүдтэ орохо, хараха, харуулха, гүйсэдхэхэ, хуулбарилха, тарааха түлбэри бү абагты.
1.Е.8. Та Project GutenbergTM электронно бүтээлнүүдэй хуулбаринуудай түлөө гү, али нэбтэрхэ эрхэ олгоһон гү, али тарааһанайнгаа түлөө таарамжатай түлбэри абажа болохот, хэрбээ:
  • • Та Project GutenbergTM-эй ажалнуудые хэрэглэжэ абаһан ниитэ олзо оршынгоо 20%-иин гонорар түлэхэт, энэнь танай хэрэгтэй татабаринуудые тоолоходоо хэрэглэжэ байһан аргаар тоосогдоһон байна. Түлбэри Project GutenbergTM наймаанай тэмдэгэй эзэндэ үритэй, зүгөөр тэрэ энэ абзацын ёһоор Project Gutenberg Literary Archive Foundation-да гонорар хандиблахые зүбшөөһэн байна. Танай үе үеын татабариин мэдүүлгэ бэлдэһэн (гү, али хуулиин ёһоор бэлдэхэ ёһотой) үдэр бүхэнһөө хойшо 60 хоногой дотор гонорар түлэгдэхэ ёһотой. Гонорар түлбэри тиимэ гэжэ тодорхойгоор тэмдэглэгдэжэ, 4-дэхи хубида зааһан хаягаар Гутенбергын түсэлэй уран зохёолой архивай жасада эльгээгдэхэ ёһотой, “Гутенбергын түсэлэй уран зохёолой архивай жасада хандиб тухай мэдээсэл.”
  • • Та 30 хоногой дотор Project GutenbergTM License-эй дүримүүдтэй зүбшөөнэгүй гэжэ бэшэгээр (гү, али электронно бэшэгээр) мэдээсэһэн хэрэглэгшын түлэһэн мүнгэеэ дүүрэн бусааха аргатайт. Та иимэ хэрэглэгшэһээ физическэ оршондо байһан бүтээлнүүдэй бүхы хуулбаринуудые бусааха гү, али үгы хэхэ, Project GutenbergTM бүтээлнүүдэй бусад хуулбаринуудые хэрэглэхэ болон нэбтэрхэ эрхэеэ болюулхыень эрилтэ табиха ёһотойт.
  • • Хэрбээ электронно ажалда алдуу элирүүлэгдэжэ, ажал абаһанай һүүлдэ 90 хоногой дотор танда дуулгабал, та 1.F.3-дахи абзацын ёһоор, ажал гү, али һэлгэхэ хуулбариин түлөө түлэһэн мүнгэеэ дүүрэн бусааха аргатайт.
  • • Та Project GutenbergTM бүтээлнүүдые түлбэригүй тарааха тухай энэ хэлсээнэй бусад бүхы дүримүүдые дүүргэнэ.
1.Е.9. Хэрбээ та энэ хэлсээндэ зааһанһаа ондоохон нэрэ томьёогоор Project GutenbergTM электронно бүтээл гү, али бүтээлнүүдэй бүлэгые түлбэри абаха гү, али тарааха гэбэл, Project GutenbergTM наймаанай тэмдэгэй менеджер болохо Project Gutenberg Literary Archive Foundation-һаа бэшэмэл зүбшөөл абаха хэрэгтэй. Доорохи 3-дахи хубида хэлэгдэһэнэй ёһоор Сантай холбоо баригты.
1.Ф.

Хэрбээ хоёрдохи хуулбаринь баһал гажуудалтай байгаа һаань, асуудалые заһаха арга боломжогүйгөөр бэшэгээр мүнгэ бусааха тухай эрилтэ табижа болохот.

4-дэхи хуби. Гутенбергын уран зохёолой архивай жасада хандибууд тухай мэдээсэл
ГутенбергTM түсэл олониитын дэмжэлгэ болон хандибгүйгөөр амидаржа шадахагүй, олониитын мэдэлэй болон лицензитэй бүтээлнүүдэй тоо олошоруулха, энэнь хуушарһан түхеэрэлгэнүүдые багтааһан эгээл үргэн түхеэрэлгэнүүдээр нэбтэрхэ машинаар уншаха хэлбэритэйгээр сүлөөтэй тарааха зорилготой юм. Олон багахан хандибууд ($1-һээ $5,000 хүрэтэр) IRS-тэй татабариһаа сүлөөлэгдэһэн байдалые хадгалхада илангаяа шухала.
Энэ жаса АНУ-ай бүхы 50 можо уласуудта буян үйлэдэлгын болон буян үйлэдэлгын хандибуудые зохисуулха хуулинуудые сахиха үүргэтэй. Дабталгын эрилтэнүүд нэгэ янзын бэшэ, тиимэһээ эдэ эрилтэнүүдые хангахын тулада ехэ оролдолго, олон баримта бэшэгүүд, олон түлбэри хэрэгтэй. Бидэ дүримүүдые сахиха тухай бэшэмэл баталамжа абаагүй газарнуудта хандиб эридэггүйбди. ХАНДИЛГА ЭЛГЭХЭ гү, али ямар нэгэ уласай дүримүүдые сахиха байдалые тодорхойлхын тула www.gutenberg.org/donate руу орожо үзэгты.
Хэдыгээр бидэ эрилтэнүүдые хангажа шадаагүй можо уласуудһаа хандиб эрижэ шадахагүйбди, эридэггүйшье һаа, хандиб үргэхэ тухай дурадхалнуудтай хандиб үргэһэн иимэ можо уласуудта эриһэнгүй хандиб абаха хорюулгүй гэжэ мэдэнэбди.
Уласхоорондын хандибуудые баяртайгаар хүлеэн абадаг, зүгөөр АНУ-һаа гадуур ороһон хандибуудай татабариин талаар ямаршье мэдүүлэл хэжэ шадахагүйбди. Ганса АНУ-гай хуулинууд манай багахан ажалшадые һандаргадаг.
Мүнөөнэй хандиб үргэлгын арганууд болон хаягуудые Project Gutenberg вэб хуудаһануудһаа шалгажа үзыт. Хандиб үргэлнүүдые чек, онлайн түлбэри, кредит картаар хандиб оруулха зэргэ олон аргаар абадаг. Хандиб үргэхэ гэбэл, www.gutenberg.org/donate руу орожо үзэгты.
