ASEGZI ƔEF USEfru.

Lbaḍna n usefru, ur d-yeqqim ara d rrbeḥ n tmedyazt, tella deg umeẓẓuɣ n win i s-yeslan. Yiel i tmeẓẓuɣt-is, qebel lḥal n umedyaz : aya, s tɣawla, d ayen i d-ttaran yimusnawen mi ara ten-testeqsiḍ ɣef wazal-is. Ula ma amedyaz s timmad-is, deg lḥal-is nniḍen, yenna-yak-d belli taẓuri-s d lḥirfa kan n ufus, d tuččit n umseḍru-ines n tuččit, yerna yettak-d ad ak-d-yessken lḥirfa-ines, — ur t-ttamneḍ ara. Raǧu ssaɛa-ines n nnbi; syin efk iman-ik i lḥir-is, i lferḥ-is neɣ i lḥif-is. « Nekni deg ufus n Tayri ass-a !» yecna Gautier, s tmuɣli n Swinburne i d-yesmektayen s tmeẓẓuɣt-is, — yerna seg ṣṣbeḥ armi d aɣelluy n yiṭij nettwaḥbes ɣef yilel n tkerkas : tella kan yiwet n tayri, yiwet n Mayu, yiwet n tmeẓẓuɣt n yijeǧǧigen. Tayri d tin n lebda, ayen akk nniḍen ur yelli ara d tidet yerna ssersen-t ɣer yidis. Tamuɣli tesɛa taggara, tadyant tettbeddil; maca nekkni nekkes-d kra, d lmektub n yiwen n usḥissef. Akken aṭas n yimedyazen, aṭas n yiserdasen. Yella lḥir n Evanescence, win i d-yettaken i ccbaḥa d lxir tameqqrant aureole n Pathos. Bḍu d Landor yiwen n "iḍ-is n tmuɣliwin d tmeɣriwin" i Rose Aylmer, yerna tesɛiḍ aya s lekmal-is.

Yerna tura ṭṭef afus n yiwen n umseḍru n umaḍal amaynut, iɛeddan seg kra n tmeddurt iwatan ɣer Marikan-nni ur nesɛi lḥeqq yerna ur nesɛi lḥeqq ay yellan, akken ay d-iwala Baudelaire, « yella i Poe ala d lḥebs ameqran aydeg ay d-yezger, dagi d din, s tmeẓẓuɣt n yiwen n wemdan ay d-yettwaxelqen akken ad yessufeɣ nnefs deg umaḍal n tudert zeddigen ugar », yerna anda ay d-yella ṛṛuḥ-nnes am « ṛṛuḥ-nnes am drunkard, yella kan d yiwen n lmejhud n lebda akken ad yerwel seg tmeẓẓuɣt n tmeẓẓuɣt-agi n tmeẓẓuɣt." Ḥebset afus n tḥila n yiwen n ucennay yettwaḥeqren yettḥulfu akka s tɣawsiwin-is tiberkanin, u bḍu yid-s lḥala ideg yufa, — werǧin deg wass, dima deg yiḍ, — ma yella mačči d Atlantis seg d-yeffeɣ, ma ulac d amkan n tɣellist seg tlisa ideg yella deg wass yettwaḥbes.

Ar yiwet n tmurt kan i yesɛa tazmert ad k-id-yawi, yerna i yiwen n yiḍ kan i tzemreḍ ad tbeḍneḍ targit-is. Tazrart ur nelli n Tmurt neɣ n Yigenni : « No-man’s-land », seg Ugensa, seg Wakud. Hatan wid yettwasqerdec, s wallen-nsen, am tid n Cenci, yettaf-d ṛṛuḥ tiwwura ɣas akken lbal yettwaḥbes ; dagi lǧennet, ttnadin ɣef ubrid i d-yettawin ɣer Lebda, ttwaḥebsen yerna ttwaɛeṭṭlen. D limbo n "yiṛuḥaniyen n yiplanet", anda llant akk lbaṭel n tafat n wayyur, akk tayri d tmuɣliwin n tkerkas i yewḍen. Dagi kra ttwaḥerzen deg tmeẓẓuɣt, d lḥir kan i d-yettwaweḍen; wiyaḍ

"Murked s wudem ifazen
 Ɣer yiwen n uẓawan ur nemsefham ara :"

Ma deg yal amkan llan

"Imektiyen n wayen yezrin—
 Talɣiwin yettwaḍelqen i yebdan u ttnefxaren
 Akken ttɛeddayen ɣef uɛewwiq."

Akka i tella tmurt, yerna i yiwen n yiḍ ad tt-nekcem, — d iḍ n tfaskiwin n yiṭij d tmeẓẓuɣin yettwaɛerḍen. Monodies-is d mraw d sin n yisefra, aẓawan-nsen yettnadi ad ibeddel maca yal tikkelt d yiwen. Yiwen yiwen ttmeslayen, am tmeẓẓuɣt n tesraft n lḥenni d lḥenni, deg « The Masque of the Red Death », i d-yeǧǧan imedyazen n waltz ad ḥebsen s « lxuf d tmeẓẓuɣt d tmuɣli », akken kan ara d-teḍru teswiɛt.

Seg wakk tikadimin-agi timsirin, aɛeyyen n The Raven , yettḥerriqen s tewwurt, s umata yessewham amaḍal n berra. D acu n tɣawsiwin i d-yettawin ɣer usnulfu n usefru, — yerna acḥal d tidet deg waya, am deg tuget n tɣawsiwin nniḍen, lǧens-nni i d-yefkan isem i yibardan-is "imseḍruyen" ? Axeddim yettwasemma-d seg lbaṭel. Anda ur yelli kra, yeqqim, — d axlaq amaynut, d aḥric seg ugerruj n wemdan. Yerna kra n wawalen ur nelli am wiyaḍ, ugar n wiyaḍ i d-yesnulfan s tdukli, ḥettmen-aɣ ad nxemmem seg tamaynut amek s tebɣest i d-yettbeddilen uqlam n umedyaz tiɣawsiwin ur nettwassen ara .

"I talɣiwin, yerna ur yettak ara i lhawa kra
 Tazedɣa tadigant, d yisem."

Yal yiwen iban-d war ma yella-d umdan (prototype), d acu kan akk ttḥemmilen-aɣ s yiferdisen yettwasfeḍen seg tlemmast n Uɛebbuḍ. Tura lxalat n tlemmast ttḥulfunt s ṛṛuḥ n tɣawsiwin; yezmer neɣ ur yezmir ara ad d-yesmekti Fantasy i yiɣerfan s tugniwin n tmeẓẓuɣt gar wayyur. Maca, yiwen seg wawalen-a, s ubrid-is ameẓyan, yettḥaz-aɣ s tmuɣli n tmeẓẓuɣt, s tɣawla am yixeddimen n tmeẓẓuɣt n yiwen n ukabar n tmeẓẓuɣt, - "Faustus" n Marlowe, "Faust" n Goethe, "Manfred", neɣ ula d tizlatin-nni n tmeẓdit, "The D'Tempedsure". Ugar n yiwen, ɣas akken d unṣib, yesɛa kra n burden neɣ refrain i yettḥazen lmeɛqul, — yiwet n tikkelt tesla, ur tettwattu ara, am lḥess n yiwen n usḥulfu n tfekka ur yettwasqedcen ara mliḥ maca yettwassen. Yettwassen gar-asent "Lenore" n Bürger, tazlit-nni n tmedyazt n tmeṭṭut d tmeṭṭut, d lḥirfa d lḥirfa n yemsuqqlen. D cwiṭ kan ara inekren d akken "Rime n Mariner Aqbur" n Coleridge ay d-yessewhamen, ibedd deg uqerru-nsen s timmad-nnes. « Le Juif-Errant » ad yesɛu lḥeqq, lemmer d Beranger ay d amedyaz ameqran ugar ; yerna, maca ɣef ljal n tbeddi-nsen seg tḥemla n tmeɣra n medden, "Tameṭṭut n Shalott" ed "Damozel yettwabarek" zemren ad rnun ɣer llista-nni. Tettwanefk i Edgar Allan Poe akken ad d-yessuffeɣ sin n yimeslayen, "The Bells" ed The Raven , yal yiwen seg-sen, ɣas akken ahat d ayen ur nesɛi ara aṭas n ccbaḥa am win n Tennyson ed Rossetti, maca d ayen ay d-yellan s tɣawla. « Ulalume », ɣas akken d ayen igerrzen yerna d ayen ay d-yettwaxemmen s tdukli, ur yesɛi ara taɣawsa tamaḍlant ay yellan d aḥric seg uɛeyyen-nneɣ.

Raven deg yiferdisen n tmedyazt s timmad-is yeɣli ddaw n yiḍrisen am "Aqṣer n tmeṭṭut", "Tamdint deg yilel", "Iḍes" d "Israfel". Akk-is ad t-beddlen, ttcukkuɣ, yimeɣriyen n yiwen n ukabar n tmeṭṭut, s yiḍrisen am wigi :

Yexṣer-as lḥirfa n yigenni n umedyaz-nni s lḥirfa n wul-is d l’lute :

« Yerna qqaren-d (tadukli n yitran
 U tiɣawsiwin nniḍen n tmesliwt)
 D times n Israfeli .

Maca The Raven , am "The Bells" d "Annabel Lee", yettḥemmil iman-is i waṭas d wid ur nesɛi ara aṭas. Nniɣ-d anda niḍen d akken isura n Poe ay d-yellan s lqella ttbanen-iyi-d "d wid aydeg yella, ma ulac, tmuɣli n tɣawsiwin s timmad-nnes, aydeg yettḥulfu s tḥulfa-nnes, deg wakud aydeg yettmettat yettɣellit yerna yettḍurru amur ameqran seg uẓawan, amur ameqran seg lḥir, ttasen-d seg-s war ma yeḥṣa." Mazal-it d l’iman-iw; yerna ɣas akken, ɣef yiwet n tezrawt tamaynut n usefru-a, yessewham-iyi ugar n yal tallit seg wasmi ay lliɣ d aqcic. Tamusni qrib tettmeslay-d s lfayda n yal axeddim n tẓuri n tidet. Ssexdem argaz-nni, ur nessin ara taẓuri, ur tt-iḥemmel ara, ad yessefqed kra n usefru neɣ n tfelwit, d wamek ara t-teṭṭef tezmert-is s tɣawla ! Deg tidet, ad yefk azal n wazal n uxeddim yelhan amezwaru ay s-d-yeṭṭef lbal-nnes s lḥeqq. The Raven , daɣen, yesɛa tizemmar yettwasdukklen ay yessefk ad tent-iḥemmel ula d amasay. Ulac deg yisefra nniḍen n umeskar-is ur d-yettwasbeggen ara s tmuɣli n tmuɣli s tɣawla ugar. « The Haunted Palace » d ayen i d-yesbanen akken ilaq i umeɣri yettwafernen, maca Poe qrib ad d-yeṭṭef allegorie-nni n tmeẓẓuɣt i lmend n uskasi n tmeẓẓuɣt. Raven yettwaḥbes akk s usentel n tmedyazt n umeskar-a – d axessar ur nesɛi lmeɛna n tmeṭṭut yettwaɛezlen yerna tecbeḥ; maca ɣef ljal nniḍen, daɣen, lḥirfa taɣelnawt telha deg uxemmem-is d asefru-ines agensas.

Argaz n tmussni s umata yettaf-d aḍar s rrbeḥ n kra n yiwen n lmejhud, yerna aya mačči dima d ayen-is ameqqran. Taɛessast d tin i d-yettawin ugar n teswiɛt imi ttwaɛeṭṭlen. Ur t-yettaf ara, am lbaḍna n yiwen n umeẓyan, ssuma deffir ssuma, maca "ulac akk neɣ akk deg wakk." Yerna akka s tmuɣliwin nniḍen : aɛessas n lḥebs, aserdas, asegbar, — akken yebɣu yili lḥal, — yal yiwen yettḥessis rrbeḥ-is seg kra n tmeẓẓuɣt n lxir. Raven n Poe, ɣas akken d ayen igerrzen, yella ɣur-s "d agḍiḍ ay d-yesbedden aḍu ad d-yessufeɣ." Amedyaz-a yezdeɣ deg New-York, deg unebdu n 1844-’45, yufa-d ccɣel ɣef wedlis n Willis, "The Evening Mirror", yerna yessuffeɣ-d idrimen-nnes s ttawilat ay d-yettunefken anda niḍen. Sḍis n yiseggasen aya seg wasmi ay yella d amaru (écrivain) axeddam, yerna yesɛa ccbaḥa taɣelnawt (professionnelle); yettwaḍḍef ama deg leqder, ama deg ccek, yerna ur yelli ara deg lxir ɣef ljal n weḥric-nnes deg tɣamsa ur yettwaxelṣen ara s lxir n wass-nni. Yella daɣen yexdem aṭas seg leqdic-is igerrzen akk, am "Ligeia" d "Aɣellug n Uxxam n Usher", (taneggarut-a tesɛa amcic-nni n tmussni, deg yifyar, n "A Lost Mind" n Elihu Vedder) am "The Gold Bug" d "The Mystery of Marie Roget". Yella iga-d imesbaṭliyen (prosélytes) deg berra, yerna yuwey-d amkan ara idumen deg wallaɣ afransis. Yella yelmed tazmert-is d lqella-s, yerna yella deg tmezwarut-is, yerna mačči mebla kra n tmussni. Maca ur yuri ara kra yellan deg yiles n yal yiwen. I uxeddim d aẓiḍan yerna d aḥlawan yessefk ad d-nernu kra n tmuɣli taɣerfant akken ad teṭṭef aɣref amatu n tallit-is.

S temguri n yemseḍruyen n tmeddurt n Poe ay d-yeddan deffir-s, aḍebsi ay d-yessuffeɣ The Raven yuɣal d tamacahut ay ttmeslayen s waṭas. Tameṭṭut tameẓyant n umedyaz, Virginia, tella tettfeggiḍ zdat wallen-is, maca teqqim aseggas nniḍen daxel n tlemmast n lmut. Taxxamt ɣezzifen, d tameẓyant deg wexxam zdat webrid n Bloomingdale tella am yettwassen s texxamt anda i d-yesnulfa Rouget de l’Isle tazlit n Marseilla. Slan akk-n d akken asefru-a, ay d-yesmekta "Quarles", yeffeɣ-d deg "American Review", s yiwen n usmekti n tẓuri n tkerkas ɣef talɣa-nnes; belli Poe yewwi-d mraw n yidularen fell-as; belli Willis, win iḥemmlen aṭas yerna d win ur nesɛi ara lḥeqq ɣef yiɛessasen n lmudda, yeɛreḍ-itt-id s yiwen n usmekti (eulogie) ay t-yerran deg yiwen n wakud d awal n temdint. Akk ccek deg umeskar-nnes yeffeɣ-d asmi ay t-yeɣra Poe s timmad-nnes deg yiwen n wemjadel aseklan, yerna kra n lweqt, d netta ay d win ay d-yettwasmektin akk gar yimedyazen imarikaniyen. Hit-nni yessebɣes-it yerna yessebɣas-it. Am umseḍru niḍen yerna d win ay d-yesfaydin seg-s, ula d netta yufa-d « kra n unebdu » ula « deg tmeẓẓuɣt n yiɣersiwen ». S lḥif, maca ɛeddan kraḍ n yiseggasen, - tallit n tɣawsa, n lfeṛḥ, n lḥif, maca dima n uxeddim n tmuɣli neɣ n tmuɣli, uqbel ma teɣli-d tmeɣra taneggarut, uqbel ma teɣli tkurt ɣef tudert ay yellan ɣur-s s tidett d taɣawsa n tmeɣra, yerna yefka iman-nnes i « Uɣersiw n Uɣella ».

"The American Review: A Whig Journal" d tasɣunt ay yesɛan azal deg tallit-nni, tesɛa snat n tɣawsiwin, tettwaẓreg ɣef lkaɣeḍ yelhan s ṣṣenf ibanen, yerna ttwaseknent-d s tfelwiyin n mezzotint. Gar waṭas n tɣawsiwin yellan ddaw n lqidar n wass-a, tella deg-s kra ara yefṛeḥ yal amazrag ad tt-id-yawi. Iḍris amezwaru, i useggas n 1845, yesɛa imagraden n Huras Grili, Donald Mitcel, Walter Whitman, Marsh, Tukerman, d Whipple. Asefru n Ralph Hoyt, "Old", yeffeɣ-d deg udlis-a. Yerna atan kraḍ n wawalen n Poe: « Tamdint deg yilel », « Tagzirt n ccwal », ed The Raven . Sin seg-sen bnan-ten, — akka i yella ubrid-is, — seg tezrawin n zik, maca isem-is aneggaru yeffeɣ-d amzun d amaynut i d-yesnulfa, yerna qrib akken i t-nesɛa tura. Awal-nni ay d-yennan ur yettwasbedd ara deffir-s d ayen ixuṣṣen. Mraw d yiwen n yibeddilen imeẓyanen seg uḍris-nni n tesɣunt ttbanen-d deg The Raven and Other Poems , 1845, d adlis ay ɣef ur yeɛjib ara umedyaz-a ad t-yesnemmer akken ad t-id-yefk i wegdud. Wigi d amur ameqran seg yibeddilen n tmeẓri, neɣ n wawalen yiwen, ṣṣenf-a aneggaru yettwaxdem akken ad yessiweḍ ad yessiweḍ lḥeqq n tmeẓri. Deg teẓrigt n Mass Lang n tmedyazt n Poe, ay d-yessuffeɣ deg « Tmeḍrit n Uḍris », yessken-d lḥirfa n umusnaw, yerna yexdem s leɛqel, deg tuɣalin-nnes ɣer uḍris yellan deg wedlis ay d-nebder yakan, am wakken ay d-yettunefk deg wuṭṭun n London n 1846. Wid-a ɛawden-ten-id yemsuddsen ay d-yusan sdeffir-s, yerna ula d nitni xedmen tuccḍiwin s timmad-nsen. Maca aḍris n The Raven , ɣef ljal n yisuter ay d-yettwaqeddmen i umeskar ɣef lqem n yiḍrisen n ufus, mazal-it yettwasseqdec sɣur-s ugar; deg tidet, yessuffeɣ-itt-id deg Richmond, qbel kan ad yemmet, s ubeddel amedyaz n « serafim ay d-yeɣlin yiḍarren-nsen » s « lmalayekkat ay d-yeɣlin yiḍarren-nsen s tɣawla », deg tmenḍawt tis mraw d mraw. Aḍris-nneɣ n wass-a, ihi, ɣas akken d win n 1845 s lsas, yettwabeddel kra s tɣuri n umedyaz ay d-yellan sakkin, yerna d netta, ɣef leḥsab-inu, ay d aḍris ay d-yellan s tɣawla yerna d win ay d-yellan s tɣawla deg usefru-a yiwen kan. Abeddel-nni ay d-yellan s lǧehd akk seg tesɣunt tamezwarut iban-d, d acu kan, deg udlis-nni ay d-yellan deg 1845; tmezgunt tis mraw d yiwet deg tazwara tesɛa iḍrisen-a, lxilaf deg tmedyazt yerna s wayen nniḍen d lxilaf deg usefru-nni :

"Yettwaṭṭef seg kra n bab ur nesɛi lferḥ, win ur nesɛi ara rreḥma Tawaɣit
 Yeḍfer s tɣawla yerna yeḍfer s tɣawla ugar- akka, mi ara d-yesmekti Asirem,
 Stern Despair yuɣal-d, deg umkan n Usirem ẓiden i yezmer ad t-id-yesmekti—
 Tiririt-nni n lḥif, 'Ur tt-id-ttafeɣ ara!'"

Ad yili d ayen yelhan ma yella imedyazen niḍen, yerna yettwassnen, zemren ad ttwaqnen akken ad rren ibeddalen-nsen dima d iselhuyen. Poe yella yettkemmil yettkemmil yettḥadar asaru-nnes n yifyar ay d-yellan s tɣawla, yerna s umata, yettkemmil yettnerni-t. Raven d amezwaru seg kra n yisefra ay qrib ad ten-yessiweḍ uqbel ad ten-id-yessuffeɣ. Yezmer lḥal wid iḥemmlen tamedyazt drus i sxeṣren s tmettant-is. Yessen s tmedyazt, ur yuri ara aḍris i lmend n usnulfu n usefru. Mi t-iḥuza yiwen n refrain n tugna, l’lyrique i d-yeffɣen seg-s d lḥirfa, akken i d-yenna netta s timmad-is, n lḥir, mačči n yiswi. Aya yella-d deg tmiḍrant (intervalles) ay d-yellan s lqella armi ay qrib ad d-yesbedd tiẓri n Fairfield ay d-yennan dakken yal yiwet deg-sent d afarisen n yiwet n tezɣent n tfekka.

D acu ihi i d-yefkan i umedyaz lḥeqq-is ɣef The Raven ? Seg wacu n llsas n tmeẓẓuɣt i d-yekkan s tɣawla ɣer uẓawan i d-yessuffuɣen s tɣawla? Am yal tikkelt, ugar n yiwet n tɣawsa tedda ɣer lebni n usefru yettwassnen akka. Ma nwala tiɣriwin tiɣezfanin ay d-yettwaberrḥen s waṭas ɣef yiḍrisen igejdanen yerna ur tt-id-ttawin ara mliḥ, ssarameɣ ur ttwaɛaqben ara ɣef uskasi-a n yiwen n uḍris n tmedyazt, — ladɣa win ay d-yerra unaẓur am Doré d asentel n tmuɣli n tmeẓẓuɣt (illustration) n tmeẓẓuɣt. Ar ass-a lliɣ ttwaliɣ d akken aḍris-a, d kra kan ay d-yessuffeɣ umeskar-nnes deffir ma yeffeɣ-d, d ayen ur nelli d lxilaf imi ur d-yennerna ara seg yigerdan deg uḍris-nnes n tmeṭṭut. Maca Mass Fearing Gill yessken-iyi-d kra n tmedyazin ur d-yeffɣen ara n Poe, yura-tent deg useggas-is wis tmanya d mraw, yerna s yisem-nsent : « Aciṭan n tmes ». Tira-nni iban-d d akken tella deg tira n zik n umedyaz, yerna tidet-nnes tettwaḍmen-d sɣur twacult aydeg teqqim azgen n lqern aya. Menɣir igerdan igerrzen akk n "The Bells" d "The Haunted Palace" yella deg-s kra n yiḍrisen ay d-yesmektayen kra n uldi d uqelleq n The Raven . Ma d aẓawan n usefru-nneɣ, ma nessemgerrad gar wussan, ad nwali d akken aya yettwaḍḍef s uẓawan n "Lady Geraldine's Courtship". Poe yella seg wid imezwura ay d-yesmektin s tmussni n Miss Barrett; yura-as-d isefra-nnes ay d-yesnejmaɛ d « win ay d-yellan d aɛlayan akk deg lǧens-nnes », yerna yella deg 1990. tḥemmel s tmeẓri-s n tmedyazt. Iḍrisen seg usefru-ines n tayri,

"S yiwen n usḥissef ur nesɛi lḥeqq, deg yigenni, lkurṭu aberkan
 Teffeɣ-d yerna teffeɣ-d ɣef wallen-is tizegzawin ur nettmuqul ara."

yufa-d yiwen n lḥess deg wigi:

"U lḥess n leḥrir ur nettwakkes ara n yal lḥiḍ aberkan
 Yesnemmer-iyi-yeččur-iyi s tugdi igerrzen ur ḥsiɣ ara zik."

Dagi, Poe yeṭṭef yiwen n lḥeqq ay ɣef ay d-yesḥassef s lqerb Longfellow, yerna ula d yiwen ur t-id-yettnadi deg wemkan-nnes war ma yettwaḍḍef s tɣawla yerna yettwaɛaqeb. Deg talɣa n tmedyazt d tmedyazt, ad nwali belli sin yisefra-agi ur mxallafen ara, maca deg tmuɣli n tmedyazt mxallafen akk. Tamedyazt-nni n tmeṭṭut ur teḥsi ara s kra nniḍen anagar taẓuri n tidet n umedyaz. "Tira-agi yeddren!" » i d-tenna s lferḥ : « Tazmert-agi i d-yettwaḥulfan !... Amedyaz-nneɣ ameqqran, Mass Browning, ameskar n 'Paracelsus', &c., d aḥemmel n tmeẓẓuɣt-is." Mass Ingram, deffir ma yessawel-d ɣef « Lady Geraldine », s tmussni, yesmekta-d s yiwen n uɣbalu niḍen aydeg Poe yezmer ad d-yeṭṭef lḥir. Deg 1843, Albert Pike, azgen n Yigrikkiyen, azgen n tlisa, amedyaz n Arkansas, yessuffeɣ-d deg "The New Mirror", ay ɣef yella yettaru Poe imir-nni, kra n yifyar s uzwel n "Isadore", maca seg wasmi ay d-yella waya, ameskar-nnes yesseqdec-iten-id yerna isemma-as "Ul n Tmeɣra". Nekk ttferriɣ seg tmuɣli n Mass Pike tazlit-agi i d-iteddun, ideg ttqabalen-d tifyar tigejdanin d tmuɣli taẓurant :

"Ur ttḥulfuɣ ara s lḥir, ttedduɣ deg texxamin-nneɣ tiweḥciyin, ur ttceggiɛeɣ ara tizlatin-nneɣ,
 Iṭij n tmeẓẓuɣt yettfeǧǧiǧ s lferḥ ɣef lqaɛa ur yettwasfeḍen ara;
 Mazal yeqqim ugḍiḍ-nni n tmeḥqranit, icennu, a lḥirfa n lḥif !
 Minzi ul inu am ijj n useynu n lexrif i d-iffyen s udfel;
 Kečč teɣliḍ-iyi i lebda, a Isadore !»

Dagi nesɛa lqidar iɣezfanen, lemcawrat n tmeẓẓuɣt, d usmekti n yiwen n yiɣersiwen i tesnernay tezlit-is lḥif. Llan yiberdan nniḍen i izemren ad ten-ḍefren wid iḥemmlen lḥir; ladɣa, d aḥric ay d-yefren Mass Ingram seg tmuɣli taneggarut n Poe ɣef "Barnaby Rudge" :

"Ula d aɣersiw n tmeẓẓuɣt, * * * yezmer ad yili yettwaxdem, ugar n wayen ay t-nettwali tura, d aḥric seg tmuɣli n Barnaby n tmeẓẓuɣt. * * * Awadem-nnes yezmer ad yili ixeddem, deg wayen yerzan win n uɛewwiq, aḥric am wakken ay ixeddem, deg uẓawan, aḍebsi deg wayen yerzan aḍu."

Dacu kan, deffir ma nbeggen-d lgerman-a d tmiḍrant-a, azal n usefru-a mazal-it yettwaf deg tgemmi-ines. Aẓawan yettkemmilen, aḍris n tmuɣli d tafat d tlemmast yemgaraden, — nnig waya, lḥir n ṛṛuḥ n yiḍ, rran-t d axeddim n yiwen n ugensas (génie) i d-yettaken isalan. Ma d agḍiḍ-nni n tmeẓẓuɣt, ur yelli am wakk iɣersiwen niḍen n lɛerc-nnes, seg « twa corbies » ed « kraḍ n yiɣersiwen » n yiballadisten arma d « Grip » n Barnaby ay d-yesnulfan lḥir. Dagi ulac lbenna n tmeẓẓuɣt n tmeẓẓuɣt (cawing rook) ay yettɛeddayen ɣef tɣerɣert n lejdud yerna tettḍeggiɛ annar n tmeẓẓuɣt (heraldique); Ulac aḥeqqani n Tickell i d-yecna belli, asmi,

"Tessuɣu deg ṭṭaq-is tlata n tikkal.
 Yewwet-d Uqjan afriwen-is.
 Teẓra mliḥ teqcict-nni ixuṣṣen deg tayri
 Ṣṣut n tmeɣra."

Raven n Poe d asentel yemgaraden; lǧeţţa n lḥif d lqella n usirem n win iḥemmlen; a sable embodied Memory, d asekkil n lḥebs i d-yeqqimen, d ṣṣenf n Irreparable. Yerwel ɣef tama n Styx, seg « tlemmast n Pluton n Yiḍ », iban-d d ṛṛuḥ yettwasnulfan n win i d-yessutren s timmad-is, — n win ur nezmir ara, ur nebɣi ara, ad iqabel nepenthe n ṣṣenf, acku lmeɛqul n Lenore d ayen kan i s-d-yeqqimen, yerna s tmeɣra n lḥif-is ula d win ara yettwassersen ɣer yidis.

Yezmer daɣen ad yettwaṭṭef Raven d asefru i d-yemmalen ameskar-is, imi d amedya-ines n wakk tiktiwin-is ɣef wayen yessefk ad yili d asefru. Wid-a ttwafkan-d deg yiḍrisen-is ɣef « Amenzaḍ aseklan », « Asentel n yifyar », d « Tafelsuft n usnulfu ». Poe yenna-d d akken "deg Uẓawan, ahat, qrib akk ad yaweḍ ṛṛuḥ ɣer taggara tameqrant ay ɣef, mi ara t-id-yesmekti Uḥulfu Aseklan, yettnaɣ-d – d asnulfu n Cbaḥa n yigenni.... Aḍris ur yezmir ad yettwasemma s lxir ugar n uẓawan n ddaw neɣ n tzemmar ddaw n tmurt"; maca daɣen, aḍris yellan s tidett d "Tasekla n wawalen" d "Asnulfu n ccbaḥa s tmeẓẓuɣt,"—aya mačči d ayen yugaren aya. Lḥess n Cbaḥa n tlemmast d win n Lḥif. Ayen i t-yesḥissifen akk deg yisental d Lmut. Aya ilaq ad yili d amdakkel n Beauty. "Lmut, ihi, n tmeṭṭut icebḥen, war ccek, d asentel amedyaz akk deg umaḍal, - yerna s tdukli, ur yelli ccek dakken iɣes ay iwatan akk i usentel am wa d win n umeɛḍur ay t-yesɛan." Imeslayen-a ineggura ttwaqeddmen-d seg uskasi n tmeẓẓuɣt n Poe ɣef usefru-a s timmad-nnes, maca ttbegginen-d s tɣawla aḥric amatu n uḍris-nnes. Taggara n "The Bells" d aẓawan n tmeẓẓuɣt (monotone) n yiɣersiwen (ghouls) ay d-yettḥemmilen ad sneqsen ul n wemdan. « Lenore », The Raven , « Win iḍeṣen », « I yiwen deg lǧennet », d « Ulalume » ttgen-d yiwet n tmedyazt n tenebrose, — yerna « Annabel Lee », ay d-yura d aneggaru akk, yesskanay-d belli yiwen usentel ay t-yeṭṭef armi d taggara. I tikkelt niḍen, wigi akk ur llin d kra ma yella ur llin ara d aẓawan, yerna kra ttwaḥettmen s tɣawsa-nni n Fantastic ay d-yesnekren lḥir, yerna yerna-d tugdi tamaynut i tmeɣra s timmad-nnes. S kullec yettwasbedd s tmuɣli n tmeẓẓuɣt, ddaw n tmeẓẓuɣt n tmeẓẓuɣt, tayri n umedyaz n tmeẓẓuɣt, — s waya, s waṭas ay d-tettwassken azgen anda niḍen, — tameṭṭut n Usher ay yettwaḍḍlen, Lady Ligeia, deg 1990. tidett d amcic n tgemmi-s. Ɣileɣ belli tayri n unaẓur ɣer yiwen "deg talɣa" ur tezmir ara ad tqabel s lekmal-is lḥir-is ɣef kra n tmuɣli (idéal). Tameṭṭut yellan qrib-is yessefk ad tesseqdec tiḥerci-ines, ad tili akk s tmuɣli-s yerna ulac ayen ɣezzifen, ad t-tessewham s tɣawsiwin ur nesɛi tagara, neɣ ad tqebleḍ ad teǧǧ amur seg lḥeqq-is. Ilaq ad t-id-yesqerdec Unattainable, yerna aya yella yal tikkelt nnig-s kan deg tḥemla-nnes n tẓuri n usnulfu.

Poe, am Hawthorne, yekcem-d s tmeqbert n uɣerbaz n Ruman, yerna ulac win i t-yesferḥen ugar-is akken ad yeḍseḍ ɣef tmeɣra-ines. Maca ul-is yella d irumyen d yifecka-nsen n Ugmuḍ neɣ n Tguṭ. Asnulfu-ines, i d-yesfaydin s tmucuha n tmedyazt, amzun d ayen i t-yeǧǧan mi ara yaru aḍris; yerna lḥeqq-is yenna-as belli isefra n tmeṭṭut ɣezzifen ttkemmilen ugar ɣef tedyant wala ɣef usmekti, — yerna, akken ad d-tessufɣeḍ tamedyazt n tmeṭṭut, ad kfen wawalen. S wakka tiẓri-ines, iban-d belli tkemmel s tlisa-ines, belli "'asefru ɣezzifen' d ayen ixuṣṣen deg wawalen." Iferdisen n The Raven drus yerna d isehlen : argaz, agḍiḍ, d tmeckukal n tmeṭṭut. Maca aḍris-nni yefka-d yiwen n uskasi yelhan n tɣawsa n tmeṭṭut. D acu nesɛa? Tnaṣfa n yiḍ; texxamt n tlemmast s yiḍrisen-is n tmedyazt yettwattun; anelmad,—d Hieronymus n wass-a; lbaṭaṭa n lqaɛa ɣef tqacuct; iḍ n unebdu d tmes yettmettaten; lqecc n leḥrir i d-yesfeḍ waḍu; tirga d lqella n laman ur neẓri ara n tlemmast n tlemmast i d-yettqerriben; agḍiḍ-nni aberkan, ur nesɛi lmeɛna ɣef tbuciḍant-nni n tmeẓẓuɣt; aksesuar n lqaḍi n lqaɛa d lfeṭṭa n tmeẓẓuɣt. Akk aḍris-a n tmezgunt n liḥala, tafat, ccbaḥa, ṣṣut, d ayen n tmeṭṭut kan, yerna ula d amedyaz, maca s tɣawsa tamedyazt ay ur d-tesskanay ara melodrama s waṭas, yerna tettban-d s lqerb deg « lḥir n lebda, aḍḍan n lebda » n umedyaz.

Tuṣkiwt n ritm n The Raven tella s tidet ad d-tessuffeɣ lḥir. Tazlit, aliter, aɛeqqa, talɣa n tmedyazt, ttwaxedmen s tmussni n yiwen. Ur yelli ccek, amedyaz-a yettwaḥbes s tɣawsa n ttrochaïque n uẓawan n Miss Barrett, deg « tmanya », yerna ladɣa s usnefli ay d-yettwaqeblen qrib ad tfak « Lady Geraldine » :

"'Allen,' i d-yenna, 'tura ttbeddilent deg-i! D keččini i d allen i yi-d-yesfeḍḍen?
 Allen ttceɛciɛent, am tẓegwa tiqburin yettwasbedden deg uẓru n tmeṣkiwt n Parian!
 Ddaw n uɛebbuḍ-nni amellal yessusem, tettqerribeḍ dima s tmes
 O'er ṣṣeḥṛa n lḥemmam n wul-iw d tudert-iw ur ttwaḥebsen ara?'"

Tasekkirt-nnes tanaẓurant n tmedyazt tis kraḍt ama deg tfelwit tis snat, ama deg tfelwit tis ṛebɛa, d tmerna n tfelwit tis xemsa d tfelwit taneggarut, d aya ay d-yesbedden tazlit n The Raven . Alliteration persistante amzun d tin i d-iteddun war lǧehd, yerna s waṭas, tifyar yellan daxel n yiḍrisen ttḥezzibent; ma d refrain neɣ lḥebs yettḥakaṛ akk axeddim. Dagi daɣen yella yesfaydi-d seg tezrawt n Miss Barrett ɣef tmedyazin n tmeṭṭut d tmedyazt n tmeṭṭut s yiles-is d yiles nniḍen. "Refrain" d lḥirfa s wayes yettḍeggiɛ umedyaz neɣ aẓawan ameẓẓuɣ yettɛeddayen, — lḥirfa n Ugama i d-yettkemmilen ɣef tɣawsa tamezwarut. Ssexdameɣ dima tizlatin icebḥen n tcennayt taglizit, — « Iɣersiwen n yiḍ, iɣersiwen n yiḍ », « Margret, Margret », « Ul-iw d nekk », « Toll s tɣawla », « Asif yettkemmil », « Pan, Pan yemmut », atg. Yeḥya-t-id:

"A yimdanen iseɛdiyen! ulac iles
 Cbaḥa-nsen tezmer ad d-tesmekti:
 Teffeɣ-d tefsut n tayri seg wul-iw,
 U necc barkeɣ ten, war ẓriɣ.
 S tidett, amqeddes-iw yelhan iḥemmel-iyi.
 U sbarkeɣ-ten war ma ẓriɣ."

Poe ixleq-d tazlit tis xemsa n tmedyazt-is i tmeẓẓuɣt n repetent. Yettkel fell-as armi d taggara deg kra n yisefra i d-yeddan sdeffir-s, - "Lenore", "Annabel Lee", "Ulalume" d "I Annie". Tewwi-d aẓawan n tmeẓẓuɣt yerna d aḥeqqani, ɣef leḥsab-inu, seg "yiḍrisen ẓiḍen", n yiɣeblan d tmeẓẓuɣin tifriqiyin ay as-d-cnan deg temẓi, ttṣeggimen s ccna-nsen anaṣli taɣect n umedyaz-nneɣ n Wenẓul.

"Tafelsuft n usnulfu", askasi-ines ɣef The Raven , d asenqed atikniki n tmuɣli-ines d tgemmi-ines. La d ayen ay d-yenna s tmeẓdit (artifice) s yidammen isemmaḍen, la d ayen ay d-yenna sdeffir-s i yemdukal-nnes dakken askasi n tmeẓdit-a d yiwen kan seg yiḥezziben-nnes niḍen n tmussni, ur yessefk ad yettwaḥseb s lekmal-nnes. Lemmer ad d-yesnulfu uqbel axeddim-nni akk, qrib ad d-yesnulfu aḍris n tmedyazin n tmeẓẓuɣt (rattle-pan rhymes) s lḥirfa n tmeẓẓuɣt n tmeẓẓuɣt tis snat, — ula ɣef tiẓri-ines n tmeẓẓuɣt n tmeẓẓuɣt. D tidet anaẓur, mi ara ikemmel axeddim, yettwali, am Anaẓur amezwaru, belli yelha, yerna yettwali acimi yelha. Askasi ara d-yeḍrun sdeffir-s, rnu-as asenqes n yal timest taɣerfant, yezmer ad d-yili s tɣawla. Tamenna-inu d akken tikti tamezwarut d usnulfu n usefru-agi usan-d akken i d-yettas dima usmekti; belli ameskar-is iwala imir-nni amek ara yettwakemmel, yefka-as tiɣawsiwin tineggura i d-yesmekta deg yixef-is ɣef Usnulfu, yerna aneggaru-a, ɣef waya, ur yelli ara d lekdeb s lekmal-is, ur yelli ara d tidet s lekmal-is. Iban-d belli yella kra i d-yesdukklen gar tmuɣliwin n uxeddim n Poe d Doré. Aya ad d-yeffeɣ iban ugar lemmer aneggaru-a ad yeɛreḍ afus-is ɣef "Timucuha n Grotesque d Arabesque". Sin-nni ttnadin s waṭas ɣer tmurt n yilefḍan n tmedyazt d tmedyazt n tmeṭṭut. Deg tmeɣriwin n tmedyazt (feats melodramatiques) sin-nni, s tmehla-nsen n wayen nnig n tfelsafit, ttḥeqriren-d lḥirfa n tḥila gar wayen yellan d ayen ur nelli d ayen yelhan. Poe yella d aɛebban n tidet n Ucebḥan; tayri-s fell-as tella d lḥirfa i t-yesqeddcen, yerna dagi i yettbeddilen d umseḍru-ines. Tameṭṭut tamarikanit tga-d tugna n Genius s wallen n yitran s tmeẓẓuɣt taneggarut, d ṣṣifa yelhan n win ay yettwali, maca iɣes-nnes ur yezmir ara ad d-yessuffeɣ, lbaḍna n tmurt nnig "tewwurt n uẓekka n tmedyazt".
