Эдгар Аллан По суруйбут «Ворона» диэн «Проект Гутенберг» электроннай кинигэтэ.

*** БЫРААХ ТҮМҮГЭ ГУТЕНБЕРГ ЭЛЕКТРОННАЙ КИНИГЭ ВОРОН ***

Саҥардыллыбыт таһаарыылар уруккуну солбуйуохтара — урукку таһаарыылар ааттара уларыйыаҕа.

АХШ авторскай бырааптарын сокуонунан көмүскэллээх буолбатах бэчээт таһаарыыларыттан айымньылары оҥоруу диэн ким да бу айымньыларга АХШ авторскай быраабыгар бас билбэтин биллэрэр, онон Фонд (итиэннэ эн!) АХШ-ка көҥүлэ суох уонна авторскай быраап гонорар төлөммөккө көһөрөн тарҕатыаххын сөп. Бу лицензия Уопсай Туһаныы усулуобуйатыгар ыйыллыбыт анал быраабылалар PROJECT GUTENBERGTM өйдөбүлүн уонна атыы бэлиэтин харыстыыр ​​туһугар Project GutenbergTM электроннай үлэлэрин көһөрүүгэ уонна тарҕатыыга сыһыаннаахтар. Project Gutenberg диэн регистрацияламмыт атыы бэлиэтэ, уонна электроннай кинигэ иһин төлөбүр ыллаххына, атыы бэлиэтин лицензиятын усулуобуйатын тутуһууттан ураты, ол иһигэр Project Gutenberg атыы бэлиэтин туһаныы иһин гонорар төлүүртэн ураты, туһаныллыа суохтаах. Бу электроннай кинигэ копиятын иһин тугу да ылбат буоллаххына, атыы бэлиэтин лицензиятын тутуһуу олус судургу. Бу электроннай кинигэни ханнык баҕарар сыалга туһаныаххын сөп, холобур, атын үлэлэри, отчуоттары, испэктээктэри уонна чинчийиилэри оҥорууга. Project Gutenberg электроннай кинигэлэрэ уларыйан, бэчээттэнэн, бэриллиэн сөп — АХШ-ка АХШ авторскай быраабыгар сокуонунан көмүскэллээх буолбатах электроннай кинигэлэри кытта тугу баҕарар оҥоруоххун сөп. Саҥа тарҕатыы атыы бэлиэтин лицензиятыгар олоҕурар, ордук коммерческай хаттаан тарҕатыыга.

СТАРТ: ТОЛОРУ ЛИЦЕНЗИЯ

ТОЛОРУ ПРОЕКТ ГУТЕНБЕРГ ЛИЦЕНЗИЯТА

БУ ҮЛЭҔИ ТАРГАТЫАҔЫТ ЭБЭЭ ТУҺАНЫАҔЫТ иннинэ МАНИ ААҔЫҤ

Электроннай үлэлэри босхо тарҕатыыны көҕүлүүр Project GutenbergTM сорудаҕын көмүскүүр туһугар, бу үлэни (эбэтэр “Project Gutenberg” диэн этиини кытта сибээстээх атын үлэни) туһанан эбэтэр тарҕатан, эн бу файлга эбэтэр www.gutenberg.org/licenseberg аадырыска баар Full Project GutenbergTM License бары усулуобуйаларын тутуһарга сөбүлэһэҕин.

Раздел 1. Уопсай усулуобуйалар туһаныы уонна тарҕатыы Project GutenbergTM электроннай үлэлэрин

1.А. Бу Project GutenbergTM электроннай үлэтин ханнык баҕарар чааһын ааҕан эбэтэр туһанан, бу лицензия уонна интеллектуальнай бас билии (товарнай бэлиэ/авторскай быраап) дуогабарын бары усулуобуйаларын аахпыккын, өйдөөбүккүн, сөбүлэһэргин уонна ылынаргын биллэрэҕин. Бу дуогабар бары усулуобуйаларын тутуһарга сөбүлэспэт буоллаххына, Project GutenbergTM электроннай үлэлэргин бары копияларын туһаныыны тохтотуохтааххын уонна төннөрүөхтээххин эбэтэр суох оҥоруохтааххын. Өскөтүн эн Project GutenbergTM электроннай үлэтин куопуйатын ыларгар эбэтэр онно киирэргэр төлөбүрү төлөөбүт буоллаххына уонна бу дуогабар усулуобуйатыгар сөп түбэспэт буоллаххына, 1.E.8 пууҥҥа этиллибитин курдук, төлөбүрү төлөөбүт киһиттэн эбэтэр тэрилтэттэн харчыгын төнүннэриэххин сөп.

1.Б. “Проект Гутенберг” диэн регистрацияламмыт атыы бэлиэтэ. Бу дуогабар усулуобуйатыгар сөбүлэһэр дьон эрэ электроннай үлэҕэ туттуохтарын эбэтэр ханнык баҕарар ньыманан сибээстэһиэхтэрин сөп. Бу дуогабар толору усулуобуйатын тутуспакка даҕаны, үгүс Project GutenbergTM электроннай үлэлэригэр оҥоруохха сөптөөх аҕыйах дьыала баар. Аллараа 1.С пууну көр. Бу дуогабар усулуобуйатын тутуһар буоллаххына уонна Project GutenbergTM электроннай үлэлэригэр босхо киириини харыстыырга көмөлөһөр буоллаххына, Project GutenbergTM электроннай үлэлэригэр элбэҕи оҥоруоххун сөп. Аллараа 1.Е пууну көр.

1.С. Проект Гутенберг Литературнай Архыыбын Фондата (“Фонда” эбэтэр PGLAF), Проект ГутенбергTM электроннай айымньыларын хомуурунньугар авторскай бырааптаах. Хомуурунньукка баар биирдиилээн үлэлэр бары кэриэтэ АХШ-ка уопсастыбаннай бас билиигэ бааллар. Биирдиилээн айымньы АХШ-ка авторскай быраап сокуонунан көмүскэллээх буолбатах буоллаҕына уонна эн АХШ-ка баар буоллаххына, биһиги эйиэхэ айымньыга олоҕурбут айымньылары көһөрөргө, тарҕатаргар, толороргор, көрдөрөргө эбэтэр атын айымньылары оҥорорго бобор бырааптаахпытын ирдээбэппит, өскөтүн Project Gutenberg туһунан бары ыйыылар суох буоллахтарына. Биллэн турар, эһиги Project GutenbergTM аатын үлэҕэ сыһыары тутар туһунан сөбүлэһии усулуобуйатыгар сөп түбэһиннэрэн, Project GutenbergTM үлэлэрин көҥүллүк үллэстэн, электроннай үлэлэргэ босхо киириини көҕүлүүр Project GutenbergTM соруктарын өйүөххүт диэн эрэнэбит. Бу үлэни атын дьону кытта төлөбүрэ суох үллэстэр кэмҥитигэр, толору Project GutenbergTM Лицензиятын кытта биир формакка хаалларан, бу дуогабар усулуобуйаларын чэпчэкитик тутуһуоххутун сөп.

1.Д. Эн баар сиргин авторскай быраап сокуоннара эмиэ бу үлэни тугунан дьарыктаныаххын сөбүн быһаараллар. Үгүс дойдуларга авторскай быраап сокуоннара өрүү уларыйа тураллар. АХШ таһыгар баар буоллаххына, бу үлэҕэ эбэтэр атын Project GutenbergTM үлэтигэр олоҕурбут үлэлэри хачайдыыр, көһөрөр, көрдөрөр, толорор, тарҕатар эбэтэр оҥорор иннинэ, бу дуогабар усулуобуйатыгар эбии дойдуҥ сокуоннарын бэрэбиэркэлээ. Фонд АХШ-тан атын дойдуга ханнык баҕарар айымньы авторскай быраабын статуһун туһунан туох да этиини оҥорбот.

1.Е. Гутенберг бырайыакка бары ыйыылары суох оҥорботоххуна:

1.Е.1. Маннык этии, толору Project GutenbergTM Лицензиятыгар актыыбынай сигэлэрдээх эбэтэр атын быһаччы киириилээх, Project GutenbergTM үлэтин ханнык баҕарар куопуйатын (ханнык баҕарар үлэ, онно “Project Gutenberg” диэн этии баар, эбэтэр “Project Gutenberg” диэн этии баар) киирдэххэ, көрдөрүллэр, тарҕатыллар түгэнигэр көстүөхтээх:

Бу электроннай кинигэни АХШ уонна аан дойду үгүс муннуктарыгар ким баҕарар босхо уонна туох да хааччахтааһына суох туһаныан сөп. Бу электроннай кинигэҕэ киирбит Гутенберг бырайыагын лицензиятын усулуобуйатынан эбэтэр www.gutenberg.org сайтка көһөрөн, биэриэххитин эбэтэр хаттаан туһаныаххытын сөп. АХШ-ка суох буоллаххына, бу электроннай кинигэни туһаныаххын иннинэ, баар дойдуҥ сокуоннарын бэрэбиэркэлиэххин наада.

1.Е.2. Биирдиилээн Project GutenbergTM электроннай үлэтэ АХШ авторскай быраабыгар сокуонунан көмүскэллээх буолбатах тиэкистэртэн ылыллыбыт буоллаҕына (авторскай быраап хаһаайынын көҥүлүнэн бэчээттэммитин туһунан иһитиннэрии суох буоллаҕына), үлэни АХШ-ка ханнык баҕарар төлөбүрү уонна төлөбүрү төлөөбөккө көһөрөн тарҕатыахха сөп. Өскөтүн эн үлэҕэр “Проект Гутенберг” диэн этиилээх эбэтэр онно көстөр үлэҕэ тарҕатар эбэтэр киириини биэрэр буоллаххына, эн 1.Е.1-тэн 1.Е.7 пууннар ирдэбиллэрин тутуһуохтааххын эбэтэр үлэни уонна Проект ГутенбергTM атыы бэлиэтин 1.Е.1 эбэтэр 1.Е.9 пууннарга этиллибитин курдук туһанарга көҥүл ылыахтааххын.

1.Е.3. Биирдиилээн Project GutenbergTM электроннай үлэтэ авторскай быраап хаһаайынын көҥүлүнэн тахсыбыт буоллаҕына, эн туһаныыҥ уонна тарҕатыыҥ 1.E.1-тан 1.E.7 пууннарга уонна авторскай быраап хаһаайына туруорбут эбии усулуобуйаларыгар сөп түбэһиэхтээх. Эбии усулуобуйалар бу үлэ саҕаланыытыгар баар авторскай быраап хаһаайынын көҥүлүнэн тахсыбыт бары үлэлэргэ Project GutenbergTM лицензиятыгар ситимнээх буолуохтара.

1.Е.4. Бу үлэттэн, эбэтэр бу үлэ чааһын эбэтэр Project GutenbergTM-ы кытта сибээстээх атын үлэттэн баар ханнык баҕарар файллары Project GutenbergTM Лицензиятын толору усулуобуйаларын араарымаҥ, араарымаҥ, эбэтэр суох оҥорумаҥ.

1.Е.5. Бу электроннай үлэни, эбэтэр бу электроннай үлэ ханнык баҕарар чааһын, 1.E.1 пууҥҥа этиллибит этиини активнай сигэлэрдээх эбэтэр Project GutenbergTM Лицензиятын толору усулуобуйатыгар түргэнник киириитэ суох көрдөрбөккө, көһөрөн, көрдөрбөккө, толорбокко, тарҕатыма эбэтэр хаттаан тарҕатыма.

1.Е.6. Бу үлэни ханнык баҕарар бинарнай, баттаабыт, бэлиэтэммит, бас билиитэ суох эбэтэр бас билиитэ суох көрүҥҥэ, ол иһигэр ханнык баҕарар тиэкис обработкатын эбэтэр гипертекст көрүҥүнэн, уларытыаххын уонна тарҕатыаххын сөп. Ол гынан баран, эһиги Project GutenbergTM үлэтин копияларын “Plain Vanilla ASCII” эбэтэр Project GutenbergTM официальнай саайтыгар (www.gutenberg.org) ыйыллыбыт официальнай версияҕа туттуллар атын формакка киириини эбэтэр тарҕатыыны биэрэр буоллаххытына, туһанааччыга эбии ороскуота, төлөбүрэ эбэтэр ороскуота суох, копияны, экспортка таһаарыы ньыматын, эбэтэр ылыы ньыматын биэриэхтээххит көрдөһүү быһыытынан, үлэ оригинальнай “Plain Vanilla ASCII” эбэтэр атын көрүҥүнэн. Ханнык баҕарар атын формат 1.E.1 пууҥҥа ыйыллыбытынан, Project GutenbergTM лицензиятын толору киллэриэхтээх.

1.Е.7. 1.E.8 эбэтэр 1.E.9 пууннары тутуспатаххына, ханнык баҕарар Project GutenbergTM үлэлэригэр киирии, көрүү, көрдөрүү, толоруу, көһөрүү эбэтэр тарҕатыы иһин төлөбүрү ылыма.

1.Е.8. Project GutenbergTM электроннай үлэлэрин копияларыгар эбэтэр кинилэргэ киириини биэрэргэ эбэтэр тарҕатыыга сөптөөх төлөбүрү ылыаххын сөп, ол эрээри:

• Эһиги сөптөөх нолуоктары ааҕаргытыгар туттар ньымаҕытынан ааҕыллыбыт Project GutenbergTM үлэлэрин туһаныыттан ылыллыбыт валовай барыһыҥ 20% гонорар төлүүгүт. Төлөбүр Project GutenbergTM атыы бэлиэтин хаһаайыныгар иэстээх, ол гынан баран кини бу пуун быһыытынан гонорардары Project Gutenberg Literary Archive Foundation-ка биэрэргэ сөбүлэстэ. Гонорар төлөбүрдэрэ кэмиттэн кэмигэр нолуок декларациятын бэлэмнээбит (эбэтэр сокуонунан бэлэмнииргэ ирдэнэр) хас биирдии күнүттэн 60 күн иһигэр төлөнүөхтээхтэр. Гонорар төлөбүрдэрэ оннук диэн чуолкайдык бэлиэтэниэхтээхтэр уонна Гутенберг бырайыагын литературнай архыыбын пуондатыгар 4-с чааска ыйыллыбыт аадырыска ыытыллыахтаахтар, “Гутенберг бырайыагын литературнай архыыбын пуондатыгар көмө харчы туһунан иһитиннэрии”.

• Эһиги толору Project GutenbergTM Лицензиятын усулуобуйатыгар сөбүлэспэтин туһунан суругунан (эбэтэр электроннай почтанан) эһиэхэ биллэрбит туһанааччы төлөөбүт харчытын толору төнүннэриэххит. Маннык туһанааччыттан физическэй эйгэҕэ баар айымньылар бары копияларын төннөрөргө эбэтэр суох оҥорорго уонна Project GutenbergTM үлэлэрин атын копияларын туһаныыны уонна бары киириини тохтоторго ирдиэхтээххин.

• 1.F.3 пуун быһыытынан, электроннай үлэҕэ кэһии булуллан, үлэ ылыллыбытын кэннэ 90 күн иһигэр эһиэхэ биллэриллибит буоллаҕына, үлэҕэ эбэтэр солбуйар копияҕа төлөммүт харчыны толору төнүннэриэххит.

• Project GutenbergTM үлэлэрин босхо тарҕатыыга бу дуогабар атын бары усулуобуйаларын тутуһаҕын.

1.Е.9. Бу дуогабарга этиллибиттэн атын усулуобуйаҕа Project GutenbergTM электроннай айымньытын эбэтэр үлэлэр бөлөхтөрүн төлөбүрү ылыаххын эбэтэр тарҕатыаххын баҕарар буоллаххына, Project GutenbergTM атыы бэлиэтин салайааччытыттан, Project Gutenberg Literary Archive Foundation-тан суругунан көҥүл ылыахтааххын. Аллараа 3-с чааска этиллибитин курдук, Фонданы кытта сибээстэһиҥ.

1.Ф.

1.Ф.1. Gutenberg бырайыагын волонтердара уонна үлэһиттэрэ АХШ авторскай бырааптарын сокуонунан көмүскэллээх буолбатах айымньылары буларга, авторскай бырааптары чинчийэргэ, транскрипциялыырга уонна корректуралыырга Project GutenbergTM хомуурунньугун оҥорорго улахан кыһамньыны уураллар. Бу дьаһаллар баалларын үрдүнэн, Project GutenbergTM электроннай үлэлэрэ уонна кинилэри харайар ньымалар “Кэһиилэр” баар буолуохтарын сөп, ол курдук, ол эрээри манныктарынан эрэ муҥурдаммакка, ситэтэ суох, сөбө суох эбэтэр алдьаммыт даннайдар, транскрипция алҕастара, авторскай быраап эбэтэр атын интеллектуальнай бас билии кэһиитэ, алдьаммыт эбэтэр алдьаммыт дискэ эбэтэр атын ньыма, вирус эбэтэр көмпүүтэр, эбэтэр көмпүүтэр алдьатыан сөбө суох тэрилгитинэн ааҕыллыҥ.

1.Ф.3. УЛАРЫЫ ЭБЭЭ ТӨНӨРҮҮ КЫРААХ БЫРААПТААХ - Бу электроннай үлэҕэ ылбыккыттан 90 күн иһигэр кэһиини буллаххытына, үлэни ылбыт киһиэхэ суругунан быһаарыы ыытан, төлөөбүт харчыгытын (баар буоллаҕына) төннөрүөххүтүн сөп. Үлэни физическэй эйгэҕэ ылбыт буоллаххына, суругунан быһаарыыгын кытта эйгэни төннөрүөхтээххин. Эһиэхэ алдьаммыт үлэни биэрбит киһи эбэтэр тэрилтэ харчы төннөрүн оннугар солбуйар копияны биэриэн сөп. Үлэни электроннай ньыманан ылбыт буоллаххына, эйиэхэ биэрбит киһи эбэтэр тэрилтэ эйиэхэ харчыгын төннөрүү оннугар үлэни электроннай ньыманан ылар иккис кыаҕы биэриэн сөп. Иккис экземпляр эмиэ алдьаммыт буоллаҕына, кыһалҕаны быһаарарга атын кыах суох суругунан харчы төннөрүүтүн ирдиэххин сөп.

1.Ф.4. 1.F.3 пууҥҥа этиллибит солбуйуу эбэтэр төннөрүү кыра бырааптан ураты, бу үлэ эһиэхэ 'БАРЫТЫНАН' бэриллэр, АТЫН ГАРАНТИЯЛАРА СУОХ, ЭБЭРГЭ ЭБЭТЭР СУОХ, ОЛ КИИРГЭР, ОЛ ЭРЭН КИИРГЭРЭБЭТ ГАРАНТИЯЛАРЫНАН МАРЧАНТИЯ ГАРАНТИЯЛАРА АНАЛ.

1.Ф.5. Сорох штаттар сорох өйдөммөт гарантиялартан аккаастаныыны эбэтэр сорох көрүҥнээх хоромньуну таһаарыыны эбэтэр хааччахтааһыны көҥүллээбэттэр. Бу сөбүлэһиигэ этиллибит ханнык баҕарар эппиэтинэһиттэн аккаастаныы эбэтэр хааччахтааһын бу сөбүлэһиигэ сыһыаннаах судаарыстыба сокуонун кэһэр буоллаҕына, сөбүлэһии судаарыстыба сокуонунан көҥүллэммит улахан эппиэтинэһиттэн аккаастаныыны эбэтэр хааччахтааһыны оҥорор курдук быһаарыллыахтаах. Бу сөбүлэһии ханнык баҕарар пуунун сөбө суох буолуута эбэтэр толоруллубат буолуута хаалбыт пууннары суох оҥорбот.

ҮӨҺЭ - Эн Фонданы, табаар бэлиэтин хаһаайынын, Фонда ханнык баҕарар агенын эбэтэр үлэһитин, бу дуогабарга сөп түбэһиннэрэн Project GutenbergTM электроннай үлэлэрин куопуйаларын биэрэр ким баҕарар, уонна Project GutenbergTM электроннай оҥоһуктарын оҥорууга, тарҕатыыга уонна тарҕатыыга сыһыаннаах ханнык баҕарар волонтердары, бары ороскуота суох үлэттэн босхолуурга уонна тутарга сөбүлэһэҕин уонна ороскуоттар, ол иһигэр юридическай төлөбүрдэр, маннык дьыалалартан быһаччы эбэтэр көскө үөскүүллэр: (а) бу эбэтэр ханнык баҕарар Project GutenbergTM үлэтин тарҕатыы, (б) Project GutenbergTM үлэтин уларытыы, уларытыы, эбэтэр эбии эбэтэр суох оҥоруу, уонна (в) эн оҥорбут ханнык баҕарар кэһииҥ.

2-с чааһа. GutenbergTM бырайыак сорудаҕын туһунан информация

Project GutenbergTM диэн эргэрбит, эргэ, орто саастаах уонна саҥа көмпүүтэрдэри киллэрэн туран, араас көмпүүтэрдэр ааҕар форматтарыгар электроннай үлэлэри босхо тарҕатыыга синоним буолар. Сүүсчэкэ волонтердар кыһамньыларынан уонна олох араас эйгэтин дьоно көмөлөрүнэн баар.

Волонтердар уонна волонтердарга наадалаах көмөнү оҥорорго үп-харчы өйөбүлэ Project GutenbergTM сыалын ситиһэргэ уонна Project GutenbergTM хомуурунньуга кэлэр көлүөнэлэргэ көҥүл баар буоларын хааччыйарга олус наадалаах. 2001 сыллаахха, Гутенберг бырайыагын литературнай архыыбын пуондата Гутенберг бырайыакка уонна кэлэр көлүөнэлэргэ куттала суох уонна үйэлээх инникини хааччыйар туһугар тэриллибитэ. Гутенберг бырайыагын литературнай архыыбын пуондатын туһунан уонна эн кыһамньыҥ уонна көмөлөрүҥ хайдах көмөлөһүөн сөбүн туһунан сиһилии билэргэ, 3 уонна 4 чаастары уонна пуонда информационнай страницатын www.gutenberg.org көрүҥ.

3-с чааһа. Гутенберг аатынан литературнай архыып пуондатын бырайыагын туһунан информация

Project Gutenberg Literary Archive Foundation диэн Миссисипи штат сокуоннарынан тэриллибит уонна Ис дохуот сулууспатынан нолуоктан босхолонуу статуһун ылбыт коммерческайа суох 501(c)(3) үөрэхтээһин корпорацията. Фонд EIN эбэтэр федеральнай нолуок идентификационнай нүөмэрэ 64-6221541. Гутенберг литературнай архыыбын бырайыагар үп-харчы АХШ федеральнай сокуоннарынан уонна эһиги штаткыт сокуоннарынан көҥүллэммит кээмэйинэн нолуоктан босхолонор.

Фонд дьыалабыай офиса 809 Хотугу 1500 Арҕаа, Солт-Лейк-Сити, Юта 84116, (801) 596-1887 аадырыска баар. Электроннай почтанан сибээс сигэлэрин уонна билиҥҥи сибээс информациятын Фонд сайтыгар уонна официальнай страницатыттан www.gutenberg.org/contact булуохха сөп

4-с чааһа. Гутенберг аатынан литературнай архыып пуондатыгар көмө харчы туһунан информация

GutenbergTM бырайыак эргэрбит тэриллэри киллэрэн туран, массыынанан ааҕар көрүҥүнэн көҥүл тарҕаныан сөптөөх уопсастыбаннай уонна лицензиялаах үлэлэр ахсааннарын элбэтии соруктарын толорорго киэҥник тарҕаммыт уопсастыбаннай өйөбүл уонна көмө харчыта суох тутулуктаах уонна тыыннаах хаалар кыаҕа суох. Элбэх кыра көмө ($1-тан $5,000 диэри) IRS-ка нолуоктан босхолонуу статуһун тутуһарга ордук суолталаах.

Фонда АХШ бары 50 штатыгар үтүө дьыалалары уонна үтүө дьыалаҕа биэриилэри дьаһайар сокуоннары тутуһарга дьулуһар. Тутуу ирдэбиллэрэ биир кэлим буолбатахтар уонна бу ирдэбиллэри толорорго уонна тутуһарга улахан үлэ, элбэх докумуон уонна элбэх төлөбүр наада. Биһиги ирдэбиллэри тутуһуу туһунан суругунан бигэргэтиини ылбатах сирдэргэ көмө харчыны көрдөөбөппүт. КӨМӨ ЫЫТЫЫ эбэтэр ханнык баҕарар судаарыстыбаҕа сөп түбэһии туругун быһаарарга www.gutenberg.org/donate киирэн көрүҥ.

Биһиги көрдөһүү ирдэбиллэригэр эппиэттээбэт судаарыстыбалартан үп-харчы ирдиир кыахпыт суох уонна ирдээбэппит эрээри, биһиэхэ бэлэх биэрэргэ этии киллэрэр судаарыстыбаларга ирдэммэтэх бэлэхтэри ылыыга бобуу суоҕун билэбит.

Аан дойдутааҕы көмө харчыны махталлаахтык ылынабыт, ол гынан баран АХШ таһыттан киирбит көмө харчыга нолуокка сыһыан туһунан туох да этиини оҥорор кыахпыт суох. АХШ сокуоннара эрэ кыра үлэһиттэрбитин ыга анньаллар.

Билиҥҥи көмө ньымаларын уонна аадырыһын туһунан Project Gutenberg веб-сирэйдэрин көрүҥ. Көмө харчыны атын да ньыманан ылаллар, ол иһигэр чек, онлайн төлөбүр уонна кредитнай каартанан бэлэх. Көмө биэрэргэ, баһаалыста, киирэн көрүҥ: www.gutenberg.org/donate.

5-с чааһа. GutenbergTM бырайыак электроннай үлэлэрин туһунан уопсай информация

Профессор Майкл С. Харт ким баҕарар кытта көҥүллүк үллэстиллиэн сөптөөх электроннай үлэлэр библиотекаларын туһунан Project GutenbergTM өйдөбүлүн төрүттээбитэ. Түөрт уон сыл устата кини Project GutenbergTM электроннай кинигэлэрин волонтердар өйөбүллэрин сымнаҕас ситиминэн эрэ оҥорон тарҕаппыта.

Project GutenbergTM электроннай кинигэлэрэ үгүстүк хас да бэчээттэммит таһаарыылартан оҥоһуллаллар, кинилэр бары АХШ-ка авторскай быраапынан көмүскэллээх буолбатахтарынан бигэргэтиллэр, авторскай быраап туһунан иһитиннэрии киирбэтэҕинэ. Онон, биһиги электроннай кинигэлэри ханнык эрэ туспа кумааҕы таһаарыыга сөп түбэһиннэрэн тутуспаппыт.

Үгүс дьон биһиги сайтпытыттан саҕалыыллар, онно сүрүн PG көрдүүр сир баар: www.gutenberg.org .

Бу саайтка Project GutenbergTM туһунан информация баар, ол иһигэр Project Gutenberg Literary Archive Foundation-ка хайдах үп-харчы киллэриэххэ сөбүй, саҥа электроннай кинигэлэрбитин оҥорорго хайдах көмөлөһүөххэ сөбүй уонна саҥа электроннай кинигэлэр тустарынан истээри электроннай почтабытыгар хайдах суруттарыахха сөбүй.
