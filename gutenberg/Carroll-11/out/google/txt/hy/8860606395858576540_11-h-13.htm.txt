*** ՆԱԽԱԳԾԻ ՎԵՐՋ ԳՈՒՏԵՆԲԵՐԳԻ ԷԼԳԻՐՔ ԱԼԻՍԻ ԱՐԿԱԾՆԵՐԸ ՀՐԱՇՔՆԵՐԻ ԵՐԵՎԱՆՈՒՄ ***

Թարմացված հրատարակությունները կփոխարինեն նախորդին. հին հրատարակությունները կվերանվանվեն:

Ստեղծագործությունները տպագիր հրատարակություններից ստեղծելը, որոնք պաշտպանված չեն ԱՄՆ-ի հեղինակային իրավունքի մասին օրենքով, նշանակում է, որ ոչ ոք չունի Միացյալ Նահանգների հեղինակային իրավունք այս ստեղծագործությունների վրա, ուստի Հիմնադրամը (և դուք!) կարող է պատճենել և տարածել այն Միացյալ Նահանգներում առանց թույլտվության և առանց հեղինակային իրավունքի հոնորար վճարելու: Հատուկ կանոններ, որոնք սահմանված են սույն լիցենզիայի Օգտագործման ընդհանուր պայմանների մասում, կիրառվում են Project Gutenberg™ էլեկտրոնային աշխատանքների պատճենման և տարածման համար՝ PROJECT GUTENBERG™ հայեցակարգի և ապրանքանիշի պաշտպանության համար: Project Gutenberg-ը գրանցված ապրանքային նշան է և չի կարող օգտագործվել, եթե դուք վճարում եք էլեկտրոնային գրքի համար, բացառությամբ ապրանքանիշի լիցենզիայի պայմաններին հետևելու, ներառյալ Project Gutenberg ապրանքանիշի օգտագործման համար հոնորարների վճարումը: Եթե դուք ոչինչ չեք գանձում այս էլեկտրոնային գրքի պատճենների համար, ապրանքանիշի լիցենզիայի համապատասխանությունը շատ հեշտ է: Դուք կարող եք օգտագործել այս էլեկտրոնային գիրքը գրեթե ցանկացած նպատակով, ինչպիսիք են ածանցյալ աշխատանքների, զեկույցների, ներկայացումների և հետազոտությունների ստեղծումը: Project Gutenberg էլեկտրոնային գրքերը կարող են փոփոխվել, տպագրվել և տրվել. Միացյալ Նահանգներում դուք կարող եք գործնականում ամեն ինչ անել էլեկտրոնային գրքերով, որոնք պաշտպանված չեն ԱՄՆ հեղինակային իրավունքի մասին օրենքով: Վերաբաշխումը ենթակա է ապրանքային նշանի լիցենզիայի, հատկապես առևտրային վերաբաշխման:

ՍԿԶԲԸ՝ ԼԻՎԱԾ ԼԻՑԵՆԶԻԱ

ԱՄԲՈՂՋԱԿԱՆ ԾՐԱԳԻՐ ԳՈՒՏԵՆԲԵՐԳԻ ԼԻՑԵՆԶԻԱ

ԽՆԴՐՈՒՄ ԵՆՔ ԿԱՐԴԱՑԵՔ ԱՅՍ ԱՇԽԱՏԱՆՔԸ ՏԱՐԱԾԵԼՈՒ ԿԱՄ ՕԳՏԱԳՈՐԾԵԼՈՒ առաջ

Էլեկտրոնային աշխատանքների անվճար տարածումը խթանելու Project Gutenberg™ առաքելությունը պաշտպանելու համար, օգտագործելով կամ տարածելով այս աշխատանքը (կամ որևէ այլ աշխատանք, որը որևէ կերպ կապված է «Project Gutenberg» արտահայտության հետ), դուք համաձայնում եք պահպանել Full Project Gutenberg™ լիցենզիայի բոլոր պայմանները, որոնք հասանելի են այս ֆայլով կամ առցանց www.gutenberg.org/license կայքում:

Բաժին 1. Օգտագործման ընդհանուր պայմաններ և վերաբաշխում Project Gutenberg™ էլեկտրոնային աշխատանքները

1.Ա. Ընթերցելով կամ օգտագործելով այս Project Gutenberg™ էլեկտրոնային աշխատանքի որևէ մաս՝ դուք նշում եք, որ կարդացել եք, հասկանում, համաձայնում և ընդունում եք այս լիցենզիայի և մտավոր սեփականության (ապրանքանիշի/հեղինակային իրավունքի) համաձայնագրի բոլոր պայմանները: Եթե դուք համաձայն չեք պահպանել սույն համաձայնագրի բոլոր պայմանները, դուք պետք է դադարեցնեք օգտագործել և վերադարձնեք կամ ոչնչացնեք Project Gutenberg™ էլեկտրոնային աշխատանքների բոլոր պատճենները, որոնք գտնվում են ձեր մոտ: Եթե դուք վճարել եք Project Gutenberg™ էլեկտրոնային աշխատանքի պատճենը ստանալու կամ մուտք գործելու համար, և համաձայն չեք, որ պարտավորված եք լինելու սույն պայմանագրի պայմաններով, կարող եք վերադարձ ստանալ այն անձից կամ կազմակերպությունից, ում վճարել եք վճարը, ինչպես նշված է պարագրաֆ 1.E.8-ում:

1.Բ. «Project Gutenberg»-ը գրանցված ապրանքային նշան է: Այն կարող է օգտագործվել միայն էլեկտրոնային աշխատանքի վրա կամ որևէ կերպ կապված լինել այն մարդկանց կողմից, ովքեր համաձայնում են պարտավորված լինել սույն պայմանագրի պայմաններով: Կան մի քանի բաներ, որոնք դուք կարող եք անել Project Gutenberg™ էլեկտրոնային աշխատանքների մեծ մասի հետ, նույնիսկ առանց սույն պայմանագրի ամբողջական պայմաններին համապատասխանելու: Տես ստորև 1.C պարբերությունը: Կան շատ բաներ, որոնք դուք կարող եք անել Project Gutenberg™ էլեկտրոնային աշխատանքների հետ, եթե հետևեք այս պայմանագրի պայմաններին և օգնեք պահպանել ապագայում Project Gutenberg™ էլեկտրոնային աշխատանքների անվճար մուտքը: Տես ստորև 1.E պարբերությունը:

1.C. Project Gutenberg Literary Archive Foundation-ը («Հիմնադրամ» կամ PGLAF) պատկանում է Project Gutenberg™ էլեկտրոնային աշխատանքների հավաքածուի հեղինակային իրավունքին: Հավաքածուի գրեթե բոլոր անհատական աշխատանքները Միացյալ Նահանգներում հանրային սեփականություն են: Եթե առանձին ստեղծագործությունը պաշտպանված չէ Միացյալ Նահանգներում հեղինակային իրավունքի մասին օրենքով, և դուք գտնվում եք Միացյալ Նահանգներում, մենք իրավունք չենք պահանջում ձեզ արգելելու պատճենել, տարածել, կատարել, ցուցադրել կամ ստեղծել ստեղծագործության հիման վրա ածանցյալ ստեղծագործություններ, քանի դեռ Գուտենբերգ նախագծի բոլոր հղումները հանված են: Իհարկե, մենք հուսով ենք, որ դուք կաջակցեք Project Gutenberg™ առաքելությանը` խթանելու էլեկտրոնային ստեղծագործությունների ազատ մուտքը` ազատորեն տարածելով Project Gutenberg™ աշխատանքները՝ համաձայն սույն պայմանագրի պայմանների, որպեսզի պահպանվի Project Gutenberg™ անունը աշխատանքի հետ: Դուք կարող եք հեշտությամբ հետևել այս պայմանագրի պայմաններին` պահպանելով այս աշխատանքը նույն ձևաչափով, որն ունի իր կցված ամբողջական Project Gutenberg™ լիցենզիան, երբ այն առանց վճարի համօգտագործեք ուրիշների հետ:

1.Դ. Ձեր գտնվելու վայրի հեղինակային իրավունքի մասին օրենքները նաև կարգավորում են այն, ինչ դուք կարող եք անել այս աշխատանքի հետ: Հեղինակային իրավունքի մասին օրենքները շատ երկրներում անընդհատ փոփոխվում են: Եթե դուք գտնվում եք Միացյալ Նահանգներից դուրս, ստուգեք ձեր երկրի օրենքները՝ ի լրումն սույն պայմանագրի պայմանների, նախքան ներբեռնելը, պատճենելը, ցուցադրելը, կատարելը, տարածելը կամ ստեղծելով ածանցյալ ստեղծագործություններ՝ հիմնված այս աշխատանքի կամ Project Gutenberg™ որևէ այլ աշխատանքի վրա: Հիմնադրամը որևէ հայտարարություն չի ներկայացնում որևէ ստեղծագործության հեղինակային իրավունքի կարգավիճակի վերաբերյալ որևէ այլ երկրում, բացի Միացյալ Նահանգներից:

1.Է. Եթե չեք հանել Գուտենբերգ նախագծի բոլոր հղումները.

1.E.1. Հետևյալ նախադասությունը՝ Project Gutenberg™ լրիվ լիցենզիայի ակտիվ հղումներով կամ այլ անմիջական հասանելիությամբ, պետք է ընդգծված երևա, երբ Project Gutenberg™ աշխատանքի որևէ պատճեն (ցանկացած ստեղծագործություն, որի վրա հայտնվում է «Project Gutenberg» արտահայտությունը կամ որի հետ կապված է «Project Gutenberg» արտահայտությունը) հասանելի է, ցուցադրվում, կատարվում, դիտվում, պատճենվում կամ տարածվում է.

Այս էլեկտրոնային գիրքը ցանկացած անձի համար է ցանկացած վայրում Միացյալ Նահանգներում և աշխարհի շատ այլ մասերում առանց որևէ գնի և գրեթե առանց որևէ սահմանափակումների: Դուք կարող եք պատճենել այն, հանձնել կամ նորից օգտագործել այն Project Gutenberg լիցենզիայի պայմաններով, որոնք ներառված են այս էլեկտրոնային գրքում կամ առցանց www.gutenberg.org կայքում: Եթե ​​դուք գտնվում եք Միացյալ Նահանգներում, դուք պետք է ստուգեք այն երկրի օրենքները, որտեղ գտնվում եք, նախքան այս էլեկտրոնային գիրքն օգտագործելը:

1.E.2. Եթե Project Gutenberg™-ի անհատական էլեկտրոնային աշխատանքը բխում է ԱՄՆ-ի հեղինակային իրավունքի մասին օրենքով չպաշտպանված տեքստերից (չի պարունակում ծանուցում, որը ցույց է տալիս, որ այն տեղադրվել է հեղինակային իրավունքի սեփականատիրոջ թույլտվությամբ), աշխատանքը կարող է պատճենվել և տարածվել Միացյալ Նահանգներում ցանկացած անձի՝ առանց որևէ վճար կամ վճար վճարելու: Եթե դուք վերաբաշխում եք կամ մուտք եք գործում աշխատանքի՝ «Project Gutenberg» արտահայտությամբ, որը կապված է կամ հայտնվում է աշխատանքի վրա, դուք պետք է համապատասխանեք կամ 1.E.1-ից մինչև 1.E.7 պարբերությունների պահանջներին կամ ստանաք աշխատանքի և Project Gutenberg™ ապրանքանիշի օգտագործման թույլտվությունը, ինչպես սահմանված է 1.E.8 կամ 1.E. պարագրաֆներում:

1.E.3. Եթե անհատական Project Gutenberg™ էլեկտրոնային աշխատանքը տեղադրվում է հեղինակային իրավունքի սեփականատիրոջ թույլտվությամբ, ձեր օգտագործումը և տարածումը պետք է համապատասխանի 1.E.1-ից մինչև 1.E.7 պարբերություններին և հեղինակային իրավունքի սեփականատիրոջ կողմից սահմանված ցանկացած լրացուցիչ պայմաններին: Լրացուցիչ պայմանները կկապվեն Project Gutenberg™ լիցենզիայի հետ բոլոր աշխատանքների համար, որոնք տեղադրված են հեղինակային իրավունքի սեփականատիրոջ թույլտվությամբ, որը գտնվել է այս աշխատանքի սկզբում:

1.E.4. Մի անջատեք կամ անջատեք կամ հեռացնեք Project Gutenberg™ լիցենզիայի ամբողջական պայմաններն այս աշխատանքից կամ որևէ ֆայլ, որը պարունակում է այս աշխատանքի մի մասը կամ Project Gutenberg™-ի հետ կապված որևէ այլ աշխատանք:

1.E.5. Մի պատճենեք, ցուցադրեք, կատարեք, տարածեք կամ վերաբաշխեք այս էլեկտրոնային աշխատանքը կամ այս էլեկտրոնային աշխատանքի որևէ մասը՝ առանց 1.E.1 պարագրաֆում նշված նախադասության ակնհայտ ցուցադրման՝ ակտիվ հղումներով կամ Project Gutenberg™ լիցենզիայի ամբողջական պայմաններին անմիջապես հասանելիության հետ:

1.E.6. Դուք կարող եք վերափոխել և տարածել այս աշխատանքը ցանկացած երկուական, սեղմված, նշագրված, ոչ սեփականության կամ սեփականության ձևով, ներառյալ ցանկացած բառի մշակման կամ հիպերտեքստի ձև: Այնուամենայնիվ, եթե դուք տրամադրում եք Project Gutenberg™ ստեղծագործության պատճենները կամ տարածում եք այլ ձևաչափով, քան «Plain Vanilla ASCII» կամ այլ ձևաչափով, որն օգտագործվում է պաշտոնական տարբերակում, որը տեղադրված է պաշտոնական Project Gutenberg™ կայքում (www.gutenberg.org), դուք պետք է առանց լրացուցիչ ծախսերի, վճարների կամ ծախսերի օգտագործողին տրամադրեք պատճենը, միջոցը, որն արտահանում է գործի պատճենը, պատճենը կամ պատճենը արտահանելու միջոց: Vanilla ASCII» կամ այլ ձև: Ցանկացած այլընտրանքային ձևաչափ պետք է ներառի Project Gutenberg™ լիցենզիան, ինչպես նշված է պարագրաֆ 1.E.1-ում:

1.E.7. Project Gutenberg™ աշխատանքների մուտքի, դիտման, ցուցադրման, կատարման, պատճենման կամ տարածման համար վճար մի գանձեք, քանի դեռ չեք համապատասխանում 1.E.8 կամ 1.E.9 պարբերությանը:

1.E.8. Դուք կարող եք ողջամիտ վճար գանձել Project Gutenberg™ էլեկտրոնային աշխատանքների պատճենների կամ մուտքի տրամադրման կամ տարածման համար, պայմանով, որ.

• Դուք վճարում եք Project Gutenberg™ աշխատանքների օգտագործումից ստացված համախառն շահույթի 20%-ի չափով հոնորար, որը հաշվարկվում է այն մեթոդով, որն արդեն օգտագործում եք ձեր կիրառելի հարկերը հաշվարկելու համար: Վճարը պարտք է Project Gutenberg™ ապրանքանիշի սեփականատիրոջը, սակայն նա համաձայնել է այս պարբերության համաձայն հոնորարները նվիրաբերել Project Gutenberg Literary Archive Foundation-ին: Ռոյալթիի վճարումները պետք է վճարվեն 60 օրվա ընթացքում՝ սկսած յուրաքանչյուր օրվանից, երբ դուք պատրաստում եք (կամ օրինականորեն պարտավոր եք պատրաստել) ձեր պարբերական հարկային հայտարարագրերը: Ռոնալթի վճարումները պետք է հստակորեն նշվեն որպես այդպիսին և ուղարկվեն Project Gutenberg Literary Archive Foundation՝ Բաժին 4-ում նշված հասցեով՝ «Տեղեկություններ Project Gutenberg Literary Archive Foundation-ին նվիրատվությունների մասին»:

• Դուք տրամադրում եք ամբողջական փոխհատուցում այն ​​օգտատիրոջ կողմից վճարված ցանկացած գումարի համար, ով ձեզ գրավոր (կամ էլ. փոստով) տեղեկացնում է ստանալուց հետո 30 օրվա ընթացքում, որ նա համաձայն չէ Project Gutenberg™ լրիվ լիցենզիայի պայմաններին: Դուք պետք է պահանջեք այդպիսի օգտատիրոջից վերադարձնել կամ ոչնչացնել ֆիզիկական միջավայրում տիրող ստեղծագործությունների բոլոր օրինակները և դադարեցնել Project Gutenberg™ աշխատանքների այլ օրինակների օգտագործումն ու մուտքը:

• Դուք տրամադրում եք, համաձայն պարբերության 1.F.3-ի, աշխատանքի կամ փոխարինող պատճենի համար վճարված ցանկացած գումարի ամբողջական փոխհատուցում, եթե էլեկտրոնային աշխատանքում թերություն հայտնաբերվի և ձեզ զեկուցվի աշխատանքը ստանալուց հետո 90 օրվա ընթացքում:

• Դուք կատարում եք սույն պայմանագրի բոլոր այլ պայմանները՝ Project Gutenberg™ աշխատանքների անվճար բաշխման համար:

1.E.9. Եթե ցանկանում եք վճար գանձել կամ տարածել Project Gutenberg™ էլեկտրոնային աշխատանք կամ ստեղծագործությունների խումբ տարբեր պայմաններով, քան սահմանված են սույն պայմանագրում, դուք պետք է գրավոր թույլտվություն ստանաք Project Gutenberg Literary Archive Foundation-ից՝ Project Gutenberg™ ապրանքանիշի կառավարիչից: Կապվեք հիմնադրամի հետ, ինչպես նշված է ստորև Բաժին 3-ում:

1.Ֆ.

1.F.1. Project Gutenberg-ի կամավորներն ու աշխատակիցները զգալի ջանքեր են ծախսում Project Gutenberg™ հավաքածուն ստեղծելու համար հեղինակային իրավունքի վերաբերյալ հետազոտություններ կատարելու, արտագրելու և սրբագրելու համար ԱՄՆ-ի հեղինակային իրավունքի մասին օրենքով չպաշտպանված աշխատանքները: Չնայած այս ջանքերին, Project Gutenberg™ էլեկտրոնային աշխատանքները և կրիչը, որի վրա դրանք կարող են պահվել, կարող են պարունակել «թերություններ», ինչպիսիք են, բայց չսահմանափակվելով, թերի, ոչ ճշգրիտ կամ կոռումպացված տվյալներ, տառադարձման սխալներ, հեղինակային իրավունքի կամ այլ մտավոր սեփականության խախտում, թերի կամ վնասված սկավառակը կամ այլ սարքավորումը կամ համակարգիչը չի կարող կարդալ ձեր համակարգչի կողմից վնասված կամ վնասված կոդ:

1.F.2. ՍԱՀՄԱՆԱՓԱԿ ԵՐԱՇԽԻՔ, ՎՆԱՍՆԵՐԻ ՀՐԱԺԱՐՈՒՄ - Բացառությամբ պարագրաֆ 1.F.3-ում նկարագրված «Փոխարինման կամ վերադարձման իրավունքի», Project Gutenberg Literary Archive Foundation-ը, Project Gutenberg™ ապրանքանիշի սեփականատերը և ցանկացած այլ կողմ, որը բաշխում է Project Gutenberg™-ի կողմից Project Gutenberg™-ի բոլոր պարտավորությունները, ներառյալ ձեր էլեկտրոնային աշխատանքի համար նախատեսված բոլոր վնասները, օրինական ծախսերը և սույն պայմանագրով ձեր էլեկտրոնային աշխատանքի համար նախատեսված ծախսերը: ԴՈՒ ՀԱՄԱՁԱՅՆ ԵՔ, ՈՐ ԴՈՒՔ ՉՈՒՆԵՔ ԱՆՓՈԽՈՒԹՅԱՆ, ԽԻՍՏ ՊԱՏԱՍԽԱՆԱՏՎՈՒԹՅԱՆ, ԵՐԱՇԽԻՔԻ ԿԱՄ ՊԱՅՄԱՆԱԳԻՐԸ ԽԱԽՏԵԼՈՒ ՀԱՄԱՐ, Բացառությամբ 1.F.3-ում նախատեսվածներից: ԴՈՒ ՀԱՄԱՁԱՅՆՈՒՄ ԵՔ, ՈՐ ՀԻՄՆԱԴՐԱՄԸ, ԱՊՐԱՆՔԱՅԻՆ ՆՇԱՆԻ ՍԵՓԱԿԱՆԱՏԵՐԸ ԵՎ ԱՅՍ ՊԱՅՄԱՆԱԳՐՈՎ ՈՐԵՎԱ ԲԱՂՈՐԴ ՉԵՆ ՊԱՏԱՍԽԱՆԱՏՎԵԼ ՁԵԶ ԱՌԱՋ ՓԱՍՏԱԿԱՆ, ՈՒՂԻՂ, ԱՆՈՒՂԻՂ, ՀԵՏԵՎԱՆԱԿԱՆ, ՊԱՏԺԻ ԿԱՄ ԴԵՊՔԻ ՀԱՄԱՐ: ՆՄԱՆ ՎՆԱՍԻ ՀՆԱՐԱՎՈՐՈՒԹՅՈՒՆԸ.

1.F.3. ՓՈԽԱՐԻՆԵԼՈՒ ԿԱՄ ՎԵՐԱԴԱՐՁՄԱՆ ՍԱՀՄԱՆԱՓԱԿ ԻՐԱՎՈՒՆՔԸ. Եթե այս էլեկտրոնային աշխատանքում թերություն հայտնաբերեք այն ստանալուց հետո 90 օրվա ընթացքում, կարող եք վերադարձնել դրա համար վճարած գումարը (եթե այդպիսիք կան)՝ ուղարկելով գրավոր բացատրություն այն անձին, ումից ստացել եք աշխատանքը: Եթե աշխատանքն ստացել եք ֆիզիկական կրիչով, ապա պետք է վերադարձնեք կրիչը ձեր գրավոր բացատրությամբ: Անձը կամ կազմակերպությունը, որը տրամադրել է ձեզ թերի աշխատանքը, կարող է ընտրել փոխարինող պատճենը վերադարձնելու փոխարեն: Եթե աշխատանքն ստացել եք էլեկտրոնային եղանակով, ապա այն ձեզ տրամադրող անձը կամ կազմակերպությունը կարող է ընտրել ձեզ երկրորդ հնարավորությունը՝ աշխատանքն էլեկտրոնային եղանակով ստանալու փոխարեն, գումարի վերադարձի փոխարեն: Եթե երկրորդ օրինակը նույնպես թերի է, դուք կարող եք գրավոր պահանջել փոխհատուցում` առանց խնդիրը շտկելու հետագա հնարավորությունների:

1.F.4. Բացառությամբ 1.F.3 պարագրաֆում սահմանված փոխարինման կամ վերադարձման սահմանափակ իրավունքի, այս աշխատանքը տրամադրվում է ձեզ «Ինչպես կա», ՉՈՒՆԵՑՈՎ ՈՐԵՎԷ ՏԵՍԱԿԻ ԱՅԼ ԵՐԱՇԽԻՔՆԵՐՈՎ, ՀԱՅՏԱՐԱՐ ԿԱՄ ՆԱԽԱՏԵՍՎԱԾ, ՆԵՐԱՌՅԱԼ ԲԱՅՑ ՉՍԱՀՄԱՆԱՓԱԿՎՈՂ ԱՊՐԱՆՔԱՅԻՆ ՓՈԽԱՐԿՈՒԹՅԱՆ ԵՐԱՇԽԻՔՆԵՐՈՎ:

1.F.5. Որոշ նահանգներ թույլ չեն տալիս հրաժարվել որոշակի ենթադրյալ երաշխիքներից կամ վնասների որոշակի տեսակների բացառումը կամ սահմանափակումը: Եթե ​​սույն պայմանագրում ամրագրված որևէ հրաժարում կամ սահմանափակում խախտում է սույն համաձայնագրի նկատմամբ կիրառելի պետության օրենքը, համաձայնագիրը պետք է մեկնաբանվի որպես առավելագույն հրաժարում կամ սահմանափակում, որը թույլատրվում է կիրառելի նահանգի օրենսդրությամբ: Սույն պայմանագրի որևէ դրույթի անվավեր լինելը կամ անգործունակությունը չի վերացնում մնացած դրույթները:

Բաժին 2. Տեղեկություն Գուտենբերգ ծրագրի առաքելության մասին

Project Gutenberg™-ը հոմանիշ է էլեկտրոնային աշխատանքների անվճար բաշխման ձևաչափերով, որոնք ընթեռնելի են համակարգիչների ամենալայն տեսականիով, ներառյալ հնացած, հին, միջին տարիքի և նոր համակարգիչները: Այն գոյություն ունի հարյուրավոր կամավորների ջանքերի և կյանքի բոլոր ոլորտների մարդկանց նվիրատվությունների շնորհիվ:

Կամավորները և ֆինանսական աջակցությունը՝ կամավորներին անհրաժեշտ օգնություն տրամադրելու համար, կարևոր նշանակություն ունեն Project Gutenberg™-ի նպատակներին հասնելու և Project Gutenberg™ հավաքածուի ազատ հասանելիության համար գալիք սերունդների համար ապահովելու համար: 2001 թվականին Project Gutenberg Literary Archive Foundation-ը ստեղծվել է Գուտենբերգ ծրագրի և ապագա սերունդների համար ապահով և մշտական ապագա ապահովելու համար: Ծրագրի Գուտենբերգի գրական արխիվի հիմնադրամի մասին ավելին իմանալու համար, թե ինչպես կարող են օգնել ձեր ջանքերն ու նվիրատվությունները, տես 3-րդ և 4-րդ բաժինները և հիմնադրամի տեղեկատվական էջը www.gutenberg.org կայքում:

Բաժին 4. Տեղեկություններ Project Gutenberg գրական արխիվի հիմնադրամին նվիրատվությունների մասին

Project Gutenberg™-ը կախված է և չի կարող գոյատևել առանց լայնածավալ հանրային աջակցության և նվիրատվությունների՝ իրականացնելու իր առաքելությունը՝ ավելացնելու հանրային տիրույթի և լիցենզավորված աշխատանքների քանակը, որոնք կարող են ազատորեն տարածվել մեքենայաընթեռնելի ձևով, որը հասանելի է սարքավորումների ամենալայն տեսականիով, ներառյալ հնացած սարքավորումները: Շատ փոքր նվիրատվություններ ($1-ից $5000) հատկապես կարևոր են հարկերից ազատված կարգավիճակը IRS-ում պահպանելու համար:

Հիմնադրամը պարտավորվում է պահպանել բարեգործությունները և բարեգործական նվիրատվությունները կարգավորող օրենքները Միացյալ Նահանգների բոլոր 50 նահանգներում: Համապատասխանության պահանջները միատեսակ չեն, և այդ պահանջները բավարարելու և համընթաց պահելու համար պահանջվում է զգալի ջանք, շատ փաստաթղթեր և շատ վճարներ: Մենք նվիրատվություններ չենք պահանջում այն վայրերում, որտեղ մենք չենք ստացել համապատասխանության գրավոր հաստատում: ՆՎԻՐԱՏՎՈՒԹՅՈՒՆՆԵՐ ՈՒՂԱՐԿԵԼՈՒ կամ որևէ պետության համապատասխանության կարգավիճակը որոշելու համար այցելեք www.gutenberg.org/donate :

Թեև մենք չենք կարող և չենք հայցում ներդրումներ այն նահանգներից, որտեղ մենք չենք բավարարել խնդրանքների պահանջները, մենք չգիտենք այնպիսի նահանգներում, որոնք մեզ դիմում են նվիրատվության առաջարկներով դոնորներից չպահանջված նվիրատվություններ ընդունելու արգելքի մասին:

Միջազգային նվիրատվություններն ընդունվում են երախտագիտությամբ, սակայն մենք չենք կարող որևէ հայտարարություն անել Միացյալ Նահանգներից դուրս ստացված նվիրատվությունների հարկային վերաբերմունքի վերաբերյալ: Միայն ԱՄՆ օրենքները ճահճում են մեր փոքրաթիվ անձնակազմին:

Խնդրում ենք ստուգել Ծրագրի Գուտենբերգի վեբ էջերը նվիրատվության ընթացիկ եղանակների և հասցեների համար: Նվիրատվություններն ընդունվում են մի շարք այլ եղանակներով՝ ներառյալ չեկերը, առցանց վճարումները և կրեդիտ քարտի նվիրատվությունները: Նվիրատվության համար այցելեք՝ www.gutenberg.org/donate:

Բաժին 5. Ընդհանուր տեղեկություններ Project Gutenberg™ էլեկտրոնային աշխատանքների մասին

Պրոֆեսոր Մայքլ Ս. Հարթը եղել է Project Gutenberg™ հայեցակարգի հեղինակը էլեկտրոնային ստեղծագործությունների գրադարանի մասին, որը կարող է ազատորեն կիսվել ցանկացածի հետ: Քառասուն տարի նա արտադրում և տարածում էր Project Gutenberg™ էլեկտրոնային գրքերը միայն կամավորների աջակցության թույլ ցանցով:

Project Gutenberg™ էլեկտրոնային գրքերը հաճախ ստեղծվում են մի քանի տպագիր հրատարակություններից, որոնք բոլորն էլ հաստատված են որպես ԱՄՆ-ում հեղինակային իրավունքով չպաշտպանված, եթե ներառված չէ հեղինակային իրավունքի մասին ծանուցում: Այսպիսով, մենք պարտադիր չէ, որ էլեկտրոնային գրքերը պահպանենք որևէ կոնկրետ թղթային հրատարակության համաձայն:

Մարդկանց մեծ մասը սկսում է մեր կայքէջից, որն ունի PG որոնման հիմնական հնարավորությունը՝ www.gutenberg.org :

Այս կայքը պարունակում է տեղեկություններ Project Gutenberg™-ի մասին, ներառյալ՝ ինչպես նվիրատվություններ կատարել Project Gutenberg գրական արխիվի հիմնադրամին, ինչպես օգնել մեր նոր էլեկտրոնային գրքերի արտադրությանը և ինչպես բաժանորդագրվել մեր էլ. տեղեկագրին՝ նոր էլեկտրոնային գրքերի մասին լսելու համար:
