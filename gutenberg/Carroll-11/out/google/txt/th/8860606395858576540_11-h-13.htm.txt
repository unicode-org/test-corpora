การผจญภัยของอลิซในแดนมหัศจรรย์ | โปรเจกต์กูเทนเบิร์ก

*** จบโครงการกูเทนเบิร์ก อีบุ๊ก การผจญภัยของอลิซในแดนมหัศจรรย์ ***

รุ่นที่อัปเดตจะเข้ามาแทนที่รุ่นก่อนหน้า โดยรุ่นเก่าจะเปลี่ยนชื่อ

การสร้างผลงานจากฉบับพิมพ์ที่ไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา หมายความว่าไม่มีใครเป็นเจ้าของลิขสิทธิ์ของสหรัฐอเมริกาในผลงานเหล่านี้ ดังนั้นมูลนิธิ (และคุณ!) จึงสามารถคัดลอกและเผยแพร่ผลงานเหล่านี้ในสหรัฐอเมริกาได้โดยไม่ได้รับอนุญาตและโดยไม่ต้องจ่ายค่าลิขสิทธิ์ กฎพิเศษที่กำหนดไว้ในส่วนข้อกำหนดการใช้งานทั่วไปของใบอนุญาตนี้ มีผลบังคับใช้กับการคัดลอกและเผยแพร่ผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ เพื่อปกป้องแนวคิดและเครื่องหมายการค้าของ PROJECT GUTENBERG™ Project Gutenberg เป็นเครื่องหมายการค้าจดทะเบียน และไม่อนุญาตให้นำไปใช้หากคุณเรียกเก็บเงินสำหรับ eBook ยกเว้นโดยปฏิบัติตามข้อกำหนดของใบอนุญาตเครื่องหมายการค้า ซึ่งรวมถึงการจ่ายค่าลิขสิทธิ์สำหรับการใช้เครื่องหมายการค้า Project Gutenberg หากคุณไม่เรียกเก็บเงินใดๆ สำหรับสำเนา eBook นี้ การปฏิบัติตามใบอนุญาตเครื่องหมายการค้าก็ง่ายมาก คุณสามารถใช้ eBook นี้ได้เกือบทุกวัตถุประสงค์ เช่น การสร้างผลงานดัดแปลง รายงาน การแสดง และการวิจัย eBook ของ Project Gutenberg สามารถดัดแปลง พิมพ์ และแจกจ่ายได้ คุณสามารถทำอะไรก็ได้ในสหรัฐอเมริกากับ eBook ที่ไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา การแจกจ่ายซ้ำต้องอยู่ภายใต้ใบอนุญาตเครื่องหมายการค้า โดยเฉพาะการแจกจ่ายซ้ำเชิงพาณิชย์

เริ่มต้น: ใบอนุญาตเต็มรูปแบบ

ใบอนุญาตโครงการ GUTENBERG ฉบับเต็ม

โปรดอ่านสิ่งนี้ก่อนที่คุณจะเผยแพร่หรือใช้ผลงานนี้

เพื่อปกป้องภารกิจของ Project Gutenberg™ ในการส่งเสริมการแจกจ่ายผลงานอิเล็กทรอนิกส์ฟรี โดยการใช้หรือแจกจ่ายผลงานนี้ (หรือผลงานอื่นใดที่เกี่ยวข้องในทางใดทางหนึ่งกับวลี “Project Gutenberg”) คุณตกลงที่จะปฏิบัติตามข้อกำหนดทั้งหมดของใบอนุญาต Project Gutenberg™ ฉบับเต็มที่มีอยู่พร้อมกับไฟล์นี้หรือทางออนไลน์ที่ www.gutenberg.org/license

ส่วนที่ 1 เงื่อนไขการใช้งานทั่วไปและการแจกจ่ายผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™

1.A. การอ่านหรือใช้งานส่วนใดส่วนหนึ่งของผลงานอิเล็กทรอนิกส์ Project Gutenberg™ นี้ หมายความว่าคุณได้อ่าน เข้าใจ ตกลง และยอมรับข้อกำหนดทั้งหมดของสัญญาอนุญาตและข้อตกลงทรัพย์สินทางปัญญา (เครื่องหมายการค้า/ลิขสิทธิ์) ฉบับนี้แล้ว หากท่านไม่ตกลงที่จะปฏิบัติตามข้อกำหนดทั้งหมดของข้อตกลงนี้ ท่านต้องหยุดใช้งานและส่งคืนหรือทำลายสำเนาผลงานอิเล็กทรอนิกส์ Project Gutenberg™ ทั้งหมดที่ท่านครอบครองอยู่ หากท่านได้ชำระค่าธรรมเนียมในการขอรับสำเนาหรือการเข้าถึงผลงานอิเล็กทรอนิกส์ Project Gutenberg™ และท่านไม่ตกลงที่จะผูกพันตามข้อกำหนดของข้อตกลงนี้ ท่านสามารถขอรับเงินคืนจากบุคคลหรือนิติบุคคลที่ท่านได้ชำระค่าธรรมเนียมตามที่ระบุไว้ในวรรค 1.E.8

1.B. “Project Gutenberg” เป็นเครื่องหมายการค้าจดทะเบียน เครื่องหมายการค้านี้สามารถใช้ได้เฉพาะกับหรือเชื่อมโยงกับงานอิเล็กทรอนิกส์ไม่ว่าในรูปแบบใดก็ตาม โดยผู้ที่ตกลงผูกพันตามข้อกำหนดของข้อตกลงนี้เท่านั้น มีบางสิ่งที่คุณสามารถทำได้กับงานอิเล็กทรอนิกส์ส่วนใหญ่ของ Project Gutenberg™ แม้จะไม่ได้ปฏิบัติตามข้อกำหนดฉบับเต็มของข้อตกลงนี้ก็ตาม โปรดดูวรรค 1.C ด้านล่าง มีหลายสิ่งที่คุณสามารถทำได้กับงานอิเล็กทรอนิกส์ของ Project Gutenberg™ หากคุณปฏิบัติตามข้อกำหนดของข้อตกลงนี้และช่วยรักษาสิทธิ์การเข้าถึงงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ในอนาคตโดยเสรี โปรดดูวรรค 1.E ด้านล่าง

1.C. มูลนิธิคลังวรรณกรรม Project Gutenberg (“มูลนิธิ” หรือ PGLAF) เป็นเจ้าของลิขสิทธิ์ในการรวบรวมผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ผลงานเกือบทั้งหมดในคอลเลกชันนี้เป็นสาธารณสมบัติในสหรัฐอเมริกา หากผลงานใดไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ในสหรัฐอเมริกา และคุณอาศัยอยู่ในสหรัฐอเมริกา เราไม่มีสิทธิ์ในการห้ามไม่ให้คุณคัดลอก เผยแพร่ แสดง หรือสร้างผลงานดัดแปลงจากผลงานดังกล่าว ตราบใดที่ได้ลบการอ้างอิงถึง Project Gutenberg ออกไปทั้งหมดแล้ว เราหวังว่าคุณจะสนับสนุนภารกิจของ Project Gutenberg™ ในการส่งเสริมการเข้าถึงผลงานอิเล็กทรอนิกส์อย่างเสรี โดยการแบ่งปันผลงาน Project Gutenberg™ อย่างเสรีตามข้อกำหนดของข้อตกลงนี้ เพื่อรักษาชื่อ Project Gutenberg™ ที่เกี่ยวข้องกับผลงาน คุณสามารถปฏิบัติตามข้อกำหนดของข้อตกลงนี้ได้อย่างง่ายดายโดยการเก็บรักษาผลงานนี้ในรูปแบบเดิมพร้อมกับใบอนุญาต Project Gutenberg™ ฉบับเต็มที่แนบมา เมื่อคุณแบ่งปันผลงานนี้กับผู้อื่นโดยไม่เสียค่าใช้จ่าย

1.D. กฎหมายลิขสิทธิ์ของสถานที่ที่คุณอาศัยอยู่ยังควบคุมสิ่งที่คุณสามารถทำได้กับผลงานนี้ด้วย กฎหมายลิขสิทธิ์ในประเทศส่วนใหญ่มีการเปลี่ยนแปลงอยู่ตลอดเวลา หากคุณอยู่นอกสหรัฐอเมริกา โปรดตรวจสอบกฎหมายของประเทศของคุณนอกเหนือจากข้อกำหนดของข้อตกลงนี้ก่อนดาวน์โหลด คัดลอก แสดง ดำเนินการ จัดจำหน่าย หรือสร้างผลงานดัดแปลงจากผลงานนี้หรือผลงานอื่นๆ ของ Project Gutenberg™ มูลนิธิไม่รับรองสถานะลิขสิทธิ์ของผลงานใดๆ ในประเทศอื่นใดนอกจากสหรัฐอเมริกา

1.E. เว้นแต่คุณจะลบการอ้างอิงทั้งหมดถึง Project Gutenberg:

1.E.1. ประโยคต่อไปนี้ พร้อมลิงก์ที่ใช้งานได้ หรือการเข้าถึงโดยตรงอื่นๆ ไปยังใบอนุญาต Project Gutenberg™ ฉบับเต็ม ต้องปรากฏอย่างชัดเจนทุกครั้งที่มีการเข้าถึง แสดง ดำเนินการ ดู คัดลอก หรือเผยแพร่สำเนาผลงาน Project Gutenberg™ (ผลงานใดๆ ที่มีวลี “Project Gutenberg” ปรากฏอยู่ หรือเชื่อมโยงกับวลี “Project Gutenberg”):

อีบุ๊กเล่มนี้เหมาะสำหรับทุกคนในสหรัฐอเมริกาและส่วนอื่นๆ ของโลก โดยไม่มีค่าใช้จ่ายและแทบไม่มีข้อจำกัดใดๆ ทั้งสิ้น คุณสามารถคัดลอก แจกจ่าย หรือนำกลับมาใช้ใหม่ได้ภายใต้เงื่อนไขของใบอนุญาต Project Gutenberg ที่มาพร้อมกับอีบุ๊กเล่มนี้ หรือทางออนไลน์ที่ www.gutenberg.org หากคุณไม่ได้อยู่ในสหรัฐอเมริกา คุณจะต้องตรวจสอบกฎหมายของประเทศที่คุณอาศัยอยู่ก่อนใช้งานอีบุ๊กเล่มนี้

1.E.2 หากผลงานอิเล็กทรอนิกส์ Project Gutenberg™ แต่ละรายการมีที่มาจากข้อความที่ไม่ได้รับความคุ้มครองตามกฎหมายลิขสิทธิ์ของสหรัฐอเมริกา (ไม่มีประกาศระบุว่าเผยแพร่โดยได้รับอนุญาตจากเจ้าของลิขสิทธิ์) ผลงานดังกล่าวสามารถถูกคัดลอกและเผยแพร่ให้แก่บุคคลใดๆ ในสหรัฐอเมริกาได้โดยไม่ต้องเสียค่าธรรมเนียมหรือค่าใช้จ่ายใดๆ หากท่านเผยแพร่ซ้ำหรือให้สิทธิ์การเข้าถึงผลงานที่มีวลี “Project Gutenberg” เกี่ยวข้องหรือปรากฏอยู่ในผลงานดังกล่าว ท่านต้องปฏิบัติตามข้อกำหนดในวรรค 1.E.1 ถึง 1.E.7 หรือได้รับอนุญาตให้ใช้งานผลงานดังกล่าวและเครื่องหมายการค้า Project Gutenberg™ ตามที่ระบุไว้ในวรรค 1.E.8 หรือ 1.E.9

1.E.3 หากมีการเผยแพร่ผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ โดยได้รับอนุญาตจากเจ้าของลิขสิทธิ์ การใช้งานและการเผยแพร่ผลงานของคุณจะต้องเป็นไปตามทั้งข้อ 1.E.1 ถึง 1.E.7 และข้อกำหนดเพิ่มเติมใดๆ ที่เจ้าของลิขสิทธิ์กำหนด ข้อกำหนดเพิ่มเติมจะเชื่อมโยงกับใบอนุญาต Project Gutenberg™ สำหรับผลงานทั้งหมดที่เผยแพร่โดยได้รับอนุญาตจากเจ้าของลิขสิทธิ์ ซึ่งระบุไว้ที่ตอนต้นของผลงานนี้

1.E.4 ห้ามยกเลิกการเชื่อมโยง แยก หรือลบข้อกำหนดสิทธิ์การใช้งาน Project Gutenberg™ ฉบับเต็มออกจากงานนี้ หรือไฟล์ใดๆ ที่มีส่วนหนึ่งของงานนี้ หรืองานอื่นใดที่เกี่ยวข้องกับ Project Gutenberg™

1.E.5 ห้ามคัดลอก แสดง ดำเนินการ แจกจ่าย หรือแจกจ่ายซ้ำผลงานอิเล็กทรอนิกส์นี้ หรือส่วนใดส่วนหนึ่งของผลงานอิเล็กทรอนิกส์นี้ โดยไม่แสดงประโยคที่ระบุไว้ในวรรค 1.E.1 อย่างชัดเจน พร้อมลิงก์ที่ใช้งานได้ หรือเข้าถึงข้อกำหนดฉบับเต็มของใบอนุญาต Project Gutenberg™ ได้ทันที

1.E.6 คุณสามารถแปลงและเผยแพร่ผลงานนี้ในรูปแบบไบนารี บีบอัด มาร์กอัป ไม่เป็นกรรมสิทธิ์ หรือเป็นกรรมสิทธิ์ รวมถึงรูปแบบการประมวลผลคำหรือไฮเปอร์เท็กซ์ใดๆ ก็ได้ อย่างไรก็ตาม หากคุณให้สิทธิ์เข้าถึงหรือเผยแพร่สำเนาผลงานของ Project Gutenberg™ ในรูปแบบอื่นที่ไม่ใช่ “Plain Vanilla ASCII” หรือรูปแบบอื่นที่ใช้ในเวอร์ชันอย่างเป็นทางการที่โพสต์บนเว็บไซต์อย่างเป็นทางการของ Project Gutenberg™ (www.gutenberg.org) คุณต้องจัดหาสำเนา วิธีการส่งออกสำเนา หรือวิธีการขอรับสำเนาของผลงานในรูปแบบ “Plain Vanilla ASCII” ต้นฉบับหรือรูปแบบอื่นใดให้แก่ผู้ใช้ โดยไม่มีค่าใช้จ่าย ค่าธรรมเนียม หรือค่าใช้จ่ายเพิ่มเติมใดๆ แก่ผู้ใช้ รูปแบบอื่นใดๆ จะต้องรวมใบอนุญาต Project Gutenberg™ ฉบับเต็มตามที่ระบุไว้ในวรรค 1.E.1

1.E.7. ห้ามเรียกเก็บค่าธรรมเนียมสำหรับการเข้าถึง การดู การแสดง การดำเนินการ การคัดลอก หรือการแจกจ่ายผลงานใดๆ ของ Project Gutenberg™ เว้นแต่คุณจะปฏิบัติตามข้อ 1.E.8 หรือ 1.E.9

1.E.8. คุณอาจเรียกเก็บค่าธรรมเนียมที่สมเหตุสมผลสำหรับการคัดลอกหรือการให้สิทธิ์เข้าถึงหรือการแจกจ่ายผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ โดยมีเงื่อนไขว่า:

• คุณจ่ายค่าลิขสิทธิ์ 20% ของกำไรขั้นต้นที่คุณได้รับจากการใช้ผลงาน Project Gutenberg™ ซึ่งคำนวณโดยใช้วิธีที่คุณใช้คำนวณภาษีที่เกี่ยวข้องอยู่แล้ว ค่าธรรมเนียมนี้เป็นหนี้เจ้าของเครื่องหมายการค้า Project Gutenberg™ แต่เจ้าของเครื่องหมายการค้าได้ตกลงที่จะบริจาคค่าลิขสิทธิ์ภายใต้วรรคนี้ให้แก่มูลนิธิ Project Gutenberg Literary Archive Foundation การชำระค่าลิขสิทธิ์จะต้องชำระภายใน 60 วันหลังจากวันที่คุณจัดทำ (หรือวันที่กฎหมายกำหนดให้จัดทำ) แบบแสดงรายการภาษีประจำงวด การชำระค่าลิขสิทธิ์ควรระบุไว้อย่างชัดเจนและส่งไปยังมูลนิธิ Project Gutenberg Literary Archive Foundation ตามที่อยู่ที่ระบุไว้ในหัวข้อที่ 4 “ข้อมูลเกี่ยวกับการบริจาคให้แก่มูลนิธิ Project Gutenberg Literary Archive Foundation”

• คุณจะคืนเงินเต็มจำนวนที่ผู้ใช้ได้ชำระไปแล้ว หากผู้ใช้แจ้งเป็นลายลักษณ์อักษร (หรือทางอีเมล) ให้คุณทราบภายใน 30 วันนับจากวันที่ได้รับแจ้งว่าผู้ใช้ไม่ยอมรับข้อกำหนดของใบอนุญาต Project Gutenberg™ ฉบับเต็ม คุณต้องขอให้ผู้ใช้ดังกล่าวส่งคืนหรือทำลายสำเนาผลงานทั้งหมดที่มีอยู่ในสื่อทางกายภาพ และยุติการใช้งานและการเข้าถึงสำเนาผลงาน Project Gutenberg™ อื่นๆ ทั้งหมด

• คุณจัดเตรียมการคืนเงินเต็มจำนวนที่ชำระสำหรับงานหรือสำเนาทดแทนตามวรรค 1.F.3 หากพบข้อบกพร่องในงานอิเล็กทรอนิกส์และรายงานให้คุณทราบภายใน 90 วันนับจากวันที่ได้รับงาน

• คุณปฏิบัติตามข้อกำหนดอื่นๆ ทั้งหมดของข้อตกลงนี้สำหรับการแจกจ่ายผลงาน Project Gutenberg™ ฟรี

1.E.9 หากท่านต้องการเรียกเก็บค่าธรรมเนียมหรือเผยแพร่ผลงานอิเล็กทรอนิกส์หรือกลุ่มผลงานของ Project Gutenberg™ ในเงื่อนไขที่แตกต่างจากที่กำหนดไว้ในข้อตกลงนี้ ท่านต้องได้รับอนุญาตเป็นลายลักษณ์อักษรจากมูลนิธิคลังวรรณกรรม Project Gutenberg™ ซึ่งเป็นผู้จัดการเครื่องหมายการค้า Project Gutenberg™ กรุณาติดต่อมูลนิธิตามที่ระบุไว้ในข้อ 3 ด้านล่าง

1.ฉ.

1.F.1 อาสาสมัครและพนักงานของ Project Gutenberg ได้ทุ่มเทความพยายามอย่างมากในการระบุ ค้นคว้าลิขสิทธิ์ ถอดความ และพิสูจน์อักษรผลงานที่ไม่ได้รับการคุ้มครองตามกฎหมายลิขสิทธิ์ของสหรัฐอเมริกาในการสร้างคอลเลกชัน Project Gutenberg™ แม้จะมีความพยายามเหล่านี้ ผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ และสื่อบันทึกข้อมูลเหล่านั้นอาจประกอบด้วย “ข้อบกพร่อง” เช่น แต่ไม่จำกัดเพียง ข้อมูลที่ไม่สมบูรณ์ ไม่ถูกต้อง หรือเสียหาย ข้อผิดพลาดในการถอดความ การละเมิดลิขสิทธิ์หรือทรัพย์สินทางปัญญาอื่นๆ ดิสก์หรือสื่อบันทึกข้อมูลอื่นที่ชำรุดหรือเสียหาย ไวรัสคอมพิวเตอร์ หรือรหัสคอมพิวเตอร์ที่ทำให้อุปกรณ์ของคุณเสียหายหรือไม่สามารถอ่านได้

1.F.2. การรับประกันแบบจำกัด การปฏิเสธความรับผิดต่อความเสียหาย - ยกเว้น “สิทธิ์ในการเปลี่ยนหรือคืนเงิน” ตามที่อธิบายไว้ในวรรค 1.F.3 มูลนิธิคลังวรรณกรรม Project Gutenberg เจ้าของเครื่องหมายการค้า Project Gutenberg™ และบุคคลอื่นใดที่เผยแพร่ผลงานอิเล็กทรอนิกส์ Project Gutenberg™ ภายใต้ข้อตกลงนี้ ขอปฏิเสธความรับผิดทั้งหมดต่อท่านสำหรับความเสียหาย ค่าใช้จ่าย และค่าใช้จ่ายต่างๆ รวมถึงค่าธรรมเนียมทางกฎหมาย ท่านตกลงว่าท่านไม่มีสิทธิเยียวยาใดๆ สำหรับความประมาทเลินเล่อ ความรับผิดโดยเด็ดขาด การละเมิดการรับประกัน หรือการผิดสัญญา ยกเว้นที่ระบุไว้ในวรรค 1.F.3 คุณตกลงว่ามูลนิธิ เจ้าของเครื่องหมายการค้า และผู้จัดจำหน่ายใดๆ ภายใต้ข้อตกลงนี้จะไม่รับผิดต่อคุณสำหรับความเสียหายที่เกิดขึ้นจริง โดยตรง โดยอ้อม เป็นผลสืบเนื่อง เป็นการลงโทษ หรือโดยบังเอิญ แม้ว่าคุณจะแจ้งให้ทราบถึงความเป็นไปได้ของความเสียหายดังกล่าวก็ตาม

1.F.3 สิทธิ์จำกัดในการเปลี่ยนหรือคืนเงิน - หากคุณพบข้อบกพร่องในงานอิเล็กทรอนิกส์นี้ภายใน 90 วันหลังจากได้รับ คุณสามารถขอรับเงินคืน (ถ้ามี) ที่คุณจ่ายไป โดยส่งคำอธิบายเป็นลายลักษณ์อักษรไปยังบุคคลที่คุณรับงานมา หากคุณได้รับงานในสื่อทางกายภาพ คุณต้องส่งคืนสื่อพร้อมคำอธิบายเป็นลายลักษณ์อักษร บุคคลหรือหน่วยงานที่มอบงานที่มีข้อบกพร่องให้คุณอาจเลือกที่จะมอบสำเนางานทดแทนแทนการคืนเงิน หากคุณได้รับงานทางอิเล็กทรอนิกส์ บุคคลหรือหน่วยงานที่มอบงานให้คุณสามารถเลือกที่จะให้โอกาสครั้งที่สองแก่คุณในการรับงานทางอิเล็กทรอนิกส์แทนการคืนเงิน หากสำเนางานที่สองมีข้อบกพร่องเช่นกัน คุณสามารถเรียกร้องเงินคืนเป็นลายลักษณ์อักษรได้โดยไม่ต้องรอโอกาสในการแก้ไขปัญหาเพิ่มเติม

1.F.4 ยกเว้นสิทธิ์ในการเปลี่ยนหรือคืนเงินที่จำกัดตามที่ระบุไว้ในวรรค 1.F.3 ผลงานนี้มอบให้คุณ "ตามสภาพ" โดยไม่มีการรับประกันอื่นใด ไม่ว่าโดยชัดแจ้งหรือโดยนัย รวมถึงแต่ไม่จำกัดเพียงการรับประกันความสามารถในการขายหรือความเหมาะสมสำหรับวัตถุประสงค์ใดๆ

1.F.5. บางรัฐไม่อนุญาตให้มีการปฏิเสธการรับประกันโดยนัยบางประการ หรือการยกเว้นหรือการจำกัดความเสียหายบางประเภท หากการปฏิเสธหรือข้อจำกัดใดๆ ที่ระบุไว้ในข้อตกลงนี้ละเมิดกฎหมายของรัฐที่ใช้บังคับกับข้อตกลงนี้ ให้ตีความข้อตกลงนี้เพื่อให้การปฏิเสธหรือข้อจำกัดสูงสุดเป็นไปตามที่กฎหมายของรัฐที่ใช้บังคับอนุญาต ความไม่สมบูรณ์หรือการไม่สามารถบังคับใช้ได้ของข้อกำหนดใดๆ ในข้อตกลงนี้จะไม่ทำให้ข้อกำหนดที่เหลือเป็นโมฆะ

1.F.6 การชดใช้ค่าเสียหาย - ท่านตกลงที่จะชดใช้ค่าเสียหายและถือว่ามูลนิธิ เจ้าของเครื่องหมายการค้า ตัวแทนหรือพนักงานของมูลนิธิ บุคคลใดก็ตามที่จัดหาสำเนาผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ ตามข้อตกลงนี้ และอาสาสมัครใดๆ ที่เกี่ยวข้องกับการผลิต การส่งเสริม และการจัดจำหน่ายผลงานอิเล็กทรอนิกส์ของ Project Gutenberg™ พ้นจากความรับผิด ค่าใช้จ่าย และค่าใช้จ่ายใดๆ รวมถึงค่าธรรมเนียมทางกฎหมาย ที่เกิดขึ้นโดยตรงหรือโดยอ้อมจากการกระทำใดๆ ต่อไปนี้ที่ท่านได้กระทำหรือทำให้เกิดขึ้น: (ก) การจัดจำหน่ายผลงานนี้หรือผลงานใดๆ ของ Project Gutenberg™ (ข) การแก้ไข ดัดแปลง หรือการเพิ่มหรือการลบผลงานใดๆ ของ Project Gutenberg™ และ (ค) ข้อบกพร่องใดๆ ที่ท่านก่อขึ้น

ส่วนที่ 2 ข้อมูลเกี่ยวกับภารกิจของ Project Gutenberg™

โครงการกูเทนเบิร์ก™ มีความหมายเหมือนกับการเผยแพร่ผลงานอิเล็กทรอนิกส์ในรูปแบบที่คอมพิวเตอร์หลากหลายประเภทสามารถอ่านได้ฟรี ไม่ว่าจะเป็นคอมพิวเตอร์ที่ล้าสมัย เก่า กลางคน หรือใหม่ โครงการนี้ดำรงอยู่ได้ด้วยความพยายามของอาสาสมัครหลายร้อยคนและการบริจาคจากผู้คนทุกระดับชั้น

อาสาสมัครและการสนับสนุนทางการเงินเพื่อช่วยเหลืออาสาสมัครตามที่ต้องการนั้นมีความสำคัญอย่างยิ่งต่อการบรรลุเป้าหมายของ Project Gutenberg™ และเพื่อให้มั่นใจว่าคอลเล็กชัน Project Gutenberg™ จะยังคงเผยแพร่ต่อไปอย่างเสรีสำหรับคนรุ่นต่อๆ ไป ในปี พ.ศ. 2544 มูลนิธิคลังวรรณกรรม Project Gutenberg ก่อตั้งขึ้นเพื่อมอบอนาคตที่มั่นคงและยั่งยืนให้กับ Project Gutenberg™ และคนรุ่นต่อๆ ไป หากต้องการเรียนรู้เพิ่มเติมเกี่ยวกับมูลนิธิคลังวรรณกรรม Project Gutenberg และวิธีที่ความพยายามและการบริจาคของคุณจะสามารถช่วยเหลือได้ โปรดดูส่วนที่ 3 และ 4 และหน้าข้อมูลมูลนิธิที่ www.gutenberg.org

ส่วนที่ 3 ข้อมูลเกี่ยวกับมูลนิธิโครงการ Gutenberg Literary Archive Foundation

มูลนิธิคลังวรรณกรรมโครงการกูเทนเบิร์ก (Project Gutenberg Literary Archive Foundation) เป็นองค์กรการศึกษาที่ไม่แสวงหาผลกำไร 501(c)(3) จัดตั้งขึ้นภายใต้กฎหมายของรัฐมิสซิสซิปปี และได้รับสถานะยกเว้นภาษีจากกรมสรรพากร หมายเลข EIN หรือหมายเลขประจำตัวผู้เสียภาษีของรัฐบาลกลางของมูลนิธิคือ 64-6221541 เงินบริจาคที่มอบให้มูลนิธิคลังวรรณกรรมโครงการกูเทนเบิร์กสามารถนำไปหักลดหย่อนภาษีได้ตามขอบเขตสูงสุดที่กฎหมายของรัฐบาลกลางสหรัฐอเมริกาและกฎหมายของรัฐของคุณอนุญาต

สำนักงานธุรกิจของมูลนิธิตั้งอยู่ที่ 809 North 1500 West, Salt Lake City, UT 84116 โทร. (801) 596-1887 สามารถดูลิงก์ติดต่อทางอีเมลและข้อมูลติดต่อล่าสุดได้ที่เว็บไซต์ของมูลนิธิและหน้าเพจอย่างเป็นทางการที่ www.gutenberg.org/contact

ส่วนที่ 4 ข้อมูลเกี่ยวกับการบริจาคให้กับมูลนิธิ Project Gutenberg Literary Archive Foundation

โครงการ Gutenberg™ พึ่งพาและไม่สามารถอยู่รอดได้หากปราศจากการสนับสนุนและการบริจาคจากสาธารณชนอย่างกว้างขวาง เพื่อดำเนินภารกิจในการเพิ่มจำนวนผลงานสาธารณสมบัติและผลงานที่ได้รับอนุญาต ซึ่งสามารถเผยแพร่ได้อย่างอิสระในรูปแบบที่เครื่องอ่านได้และเข้าถึงได้โดยอุปกรณ์ที่หลากหลายที่สุด รวมถึงอุปกรณ์ที่ล้าสมัย การบริจาคเล็กๆ น้อยๆ จำนวนมาก (ตั้งแต่ 1 ถึง 5,000 ดอลลาร์สหรัฐ) มีความสำคัญอย่างยิ่งต่อการรักษาสถานะการยกเว้นภาษีกับกรมสรรพากร

มูลนิธิมุ่งมั่นที่จะปฏิบัติตามกฎหมายที่ควบคุมองค์กรการกุศลและการบริจาคเพื่อการกุศลในทั้ง 50 รัฐของสหรัฐอเมริกา ข้อกำหนดในการปฏิบัติตามข้อกำหนดไม่ได้กำหนดไว้เป็นมาตรฐานเดียวกัน จึงต้องใช้ความพยายามอย่างมาก เอกสารจำนวนมาก และค่าธรรมเนียมจำนวนมากเพื่อให้เป็นไปตามข้อกำหนดเหล่านี้ เราไม่ขอรับบริจาคในสถานที่ที่ยังไม่ได้รับการยืนยันเป็นลายลักษณ์อักษรถึงการปฏิบัติตามข้อกำหนด หากต้องการส่งเงินบริจาคหรือตรวจสอบสถานะการปฏิบัติตามข้อกำหนดของรัฐใดรัฐหนึ่ง โปรดไปที่ www.gutenberg.org/donate

แม้ว่าเราจะไม่สามารถและไม่ขอรับบริจาคจากรัฐที่เราไม่ได้ปฏิบัติตามข้อกำหนดในการขอรับบริจาค แต่เราไม่ทราบว่ามีข้อห้ามใดๆ เกี่ยวกับการยอมรับการบริจาคที่ไม่ได้ร้องขอจากผู้บริจาคในรัฐดังกล่าวที่ติดต่อมาเสนอบริจาคกับเรา

เรายินดีรับบริจาคจากต่างประเทศ แต่เราไม่สามารถออกแถลงการณ์ใดๆ เกี่ยวกับการจัดเก็บภาษีจากเงินบริจาคที่ได้รับจากนอกสหรัฐอเมริกาได้ กฎหมายของสหรัฐอเมริกาเพียงอย่างเดียวก็ทำให้พนักงานจำนวนน้อยของเราล้นหลามแล้ว

กรุณาตรวจสอบวิธีการและที่อยู่ปัจจุบันของเว็บไซต์โครงการกูเทนเบิร์ก ท่านสามารถบริจาคได้หลายวิธี เช่น เช็ค การชำระเงินออนไลน์ และการบริจาคผ่านบัตรเครดิต หากต้องการบริจาค โปรดไปที่: www.gutenberg.org/donate

ส่วนที่ 5 ข้อมูลทั่วไปเกี่ยวกับงานอิเล็กทรอนิกส์ Project Gutenberg™

ศาสตราจารย์ไมเคิล เอส. ฮาร์ต เป็นผู้ริเริ่มแนวคิด Project Gutenberg™ ซึ่งเป็นห้องสมุดผลงานอิเล็กทรอนิกส์ที่สามารถแบ่งปันให้ทุกคนได้อย่างอิสระ ตลอดระยะเวลาสี่สิบปี เขาผลิตและจัดจำหน่ายอีบุ๊ก Project Gutenberg™ โดยอาศัยเครือข่ายอาสาสมัครที่กระจายตัวอย่างหลวมๆ

อีบุ๊ก Project Gutenberg™ มักสร้างขึ้นจากฉบับพิมพ์หลายฉบับ ซึ่งทั้งหมดได้รับการยืนยันว่าไม่ได้รับการคุ้มครองลิขสิทธิ์ในสหรัฐอเมริกา เว้นแต่จะมีประกาศลิขสิทธิ์แนบมาด้วย ดังนั้น เราจึงไม่จำเป็นต้องจัดทำอีบุ๊กให้สอดคล้องกับฉบับพิมพ์ฉบับใดฉบับหนึ่ง

คนส่วนใหญ่เริ่มต้นที่เว็บไซต์ของเราซึ่งมีสิ่งอำนวยความสะดวกการค้นหา PG หลัก: www.gutenberg.org

เว็บไซต์นี้มีข้อมูลเกี่ยวกับ Project Gutenberg™ รวมถึงวิธีการบริจาคให้กับ Project Gutenberg Literary Archive Foundation วิธีช่วยผลิต eBook ใหม่ของเรา และวิธีการสมัครรับจดหมายข่าวทางอีเมลของเราเพื่อรับข่าวสารเกี่ยวกับ eBook ใหม่
