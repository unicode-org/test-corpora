미국 저작권법으로 보호되지 않는 인쇄판에서 작품을 제작한다는 것은 누구도 이러한 작품에 대한 미국 저작권을 소유하지 않는다는 것을 의미하므로 재단(그리고 귀하!)은 허가 없이 미국에서 저작권 로열티를 지불하지 않고 이를 복사하고 배포할 수 있습니다. 이 라이선스의 일반 이용 약관 부분에 명시된 특별 규칙은 Project Gutenberg™ 전자 작품의 복사 및 배포에 적용되어 PROJECT GUTENBERG™의 개념과 상표를 보호합니다. Project Gutenberg은 등록 상표이므로 Project Gutenberg 상표 사용에 대한 로열티 지불을 포함하여 상표 라이선스 조건을 준수하는 경우를 제외하고는 eBook에 대한 비용을 청구하는 경우 사용할 수 없습니다. 이 eBook의 사본에 대해 아무런 비용을 청구하지 않는 경우 상표 라이선스를 준수하는 것은 매우 쉽습니다. 이 eBook은 파생 작품, 보고서, 공연 및 연구 작성 등 거의 모든 목적으로 사용할 수 있습니다. Project Gutenberg eBook은 수정, 인쇄 및 배포가 가능합니다. 미국 저작권법으로 보호되지 않는 eBook으로 미국에서 사실상 모든 것을 할 수 있습니다. 재배포는 상표권 라이선스를 받아야 하며, 특히 상업적 재배포의 경우 더욱 그렇습니다.

전자 작품의 자유로운 배포를 촉진하는 Project Gutenberg™의 사명을 보호하기 위해, 이 작품(또는 "Project Gutenberg"라는 문구와 어떤 식으로든 관련된 다른 작품)을 사용하거나 배포하는 경우, 이 파일이나 www.gutenberg.org/license에서 온라인으로 제공되는 전체 Project Gutenberg™ 라이선스의 모든 조건을 준수하는 데 동의하는 것입니다.

제1절 Project Gutenberg™ 전자 저작물의 일반 이용 약관 및 재배포

1.A. 본 Project Gutenberg™ 전자 저작물의 일부를 읽거나 사용함으로써 귀하는 본 라이선스 및 지적 재산권(상표/저작권) 계약의 모든 조건을 읽고, 이해하고, 동의하며, 수락함을 나타냅니다. 본 계약의 모든 조건을 준수하는 데 동의하지 않는 경우, 귀하는 소지하고 있는 Project Gutenberg™ 전자 저작물의 사용을 중단하고 모든 사본을 반환하거나 파기해야 합니다. Project Gutenberg™ 전자 저작물의 사본 또는 접근 권한을 얻기 위해 요금을 지불했으며 본 계약 조건에 동의하지 않는 경우, 1.E.8항에 명시된 대로 요금을 지불한 개인 또는 단체로부터 환불을 받을 수 있습니다.

1.B. "Project Gutenberg"는 등록 상표입니다. 본 계약의 조건에 동의하는 사람만 전자 저작물에 사용하거나 어떤 방식으로든 연결할 수 있습니다. 본 계약의 전체 조건을 준수하지 않더라도 대부분의 Project Gutenberg™ 전자 저작물에 대해 몇 가지 작업을 수행할 수 있습니다. 아래 1.C항을 참조하십시오. 본 계약의 조건을 준수하고 Project Gutenberg™ 전자 저작물에 대한 향후 무료 접근을 유지하는 데 도움을 주면 Project Gutenberg™ 전자 저작물에 대해 많은 작업을 수행할 수 있습니다. 아래 1.E항을 참조하십시오.

1.C. Project Gutenberg Literary Archive Foundation(이하 "재단" 또는 PGLAF)은 Project Gutenberg™ 전자 저작물 컬렉션에 대한 편집물 저작권을 소유합니다. 컬렉션에 포함된 거의 모든 개별 저작물은 미국에서 퍼블릭 도메인입니다. 개별 저작물이 미국의 저작권법에 의해 보호되지 않고 귀하가 미국에 거주하는 경우, Project Gutenberg에 대한 모든 참조를 제거하는 한, 저희는 귀하가 해당 저작물을 기반으로 하는 복제, 배포, 공연, 전시 또는 파생 저작물을 제작하는 것을 금지할 권리를 주장하지 않습니다. 물론, Project Gutenberg™ 명칭을 저작물과 연결하는 본 계약 조건을 준수하여 Project Gutenberg™ 저작물을 자유롭게 공유함으로써 전자 저작물에 대한 무료 접근을 장려하는 Project Gutenberg™의 사명을 지지해 주시기를 바랍니다. 귀하는 다른 사람과 무료로 공유할 때 첨부된 전체 Project Gutenberg™ 라이선스와 함께 동일한 형식으로 이 저작물을 유지함으로써 본 계약 조건을 쉽게 준수할 수 있습니다.

1.D. 귀하가 거주하는 지역의 저작권법은 귀하가 이 저작물을 어떻게 사용할 수 있는지에도 적용됩니다. 대부분의 국가에서 저작권법은 끊임없이 변화하고 있습니다. 미국 외 지역에 거주하는 경우, 이 저작물이나 기타 Project Gutenberg™ 저작물을 다운로드, 복사, 전시, 공연, 배포하거나 이를 기반으로 하는 파생 저작물을 제작하기 전에 본 계약 조건 외에도 거주 국가의 법률을 확인하십시오. 재단은 미국 외 다른 국가에서 저작물의 저작권 상태에 대해 어떠한 진술도 하지 않습니다.

1.E. Project Gutenberg에 대한 모든 참조를 삭제하지 않은 경우:

1.E.1. Project Gutenberg™ 저작물("Project Gutenberg"라는 문구가 표시되거나 "Project Gutenberg"라는 문구가 연관된 모든 저작물)의 사본을 접근, 표시, 공연, 열람, 복사 또는 배포할 때마다 다음 문장을 Project Gutenberg™ 라이선스 전체에 대한 활성 링크 또는 기타 직접 접근 권한과 함께 눈에 잘 띄게 표시해야 합니다.

이 전자책은 미국 및 전 세계 대부분 지역의 누구나 무료로, 거의 제한 없이 사용할 수 있습니다. 이 전자책에 포함된 Project Gutenberg 라이선스 또는 www.gutenberg.org 에서 온라인으로 이용 약관에 따라 복사, 배포 또는 재사용할 수 있습니다. 미국에 거주하지 않으시는 경우, 이 전자책을 사용하기 전에 거주 국가의 법률을 확인하시기 바랍니다.

1.E.2. 개별 Project Gutenberg™ 전자 저작물이 미국 저작권법의 보호를 받지 않는 텍스트에서 파생된 경우(저작권자의 허가를 받아 게시되었음을 나타내는 고지가 없는 경우), 해당 저작물은 수수료나 요금을 지불하지 않고 미국 내 누구에게나 복사 및 배포될 수 있습니다. "Project Gutenberg"라는 문구가 저작물과 연관되거나 저작물에 표시된 저작물을 재배포하거나 이에 대한 액세스를 제공하는 경우, 1.E.1항부터 1.E.7항까지의 요건을 준수하거나 1.E.8항 또는 1.E.9항에 명시된 대로 저작물 및 Project Gutenberg™ 상표 사용에 대한 허가를 받아야 합니다.

1.E.3. 개별 Project Gutenberg™ 전자 저작물이 저작권자의 허가를 받아 게시되는 경우, 귀하의 사용 및 배포는 1.E.1항부터 1.E.7항까지, 그리고 저작권자가 부과하는 추가 조건을 모두 준수해야 합니다. 추가 조건은 저작권자의 허가를 받아 게시된 모든 저작물에 대한 Project Gutenberg™ 라이선스(본 저작물 시작 부분에 있음)에 링크되어 있습니다.

1.E.4. 본 저작물 또는 본 저작물의 일부 또는 Project Gutenberg™와 관련된 다른 저작물을 포함하는 파일에서 Project Gutenberg™ 라이선스 조건의 전체 링크를 해제하거나 분리하거나 제거하지 마십시오.

1.E.5. 1.E.1항에 명시된 문장을 명확하게 표시하고 Project Gutenberg™ 라이선스의 전체 조건에 대한 활성 링크 또는 즉각적인 접근 권한을 제공하지 않고는 본 전자 저작물 또는 그 일부를 복사, 전시, 실행, 배포 또는 재배포할 수 없습니다.

1.E.6. 귀하는 이 저작물을 바이너리, 압축, 마크업, 비독점적 또는 독점적 형식(워드 프로세싱 또는 하이퍼텍스트 형식 포함)으로 변환하고 배포할 수 있습니다. 단, Project Gutenberg™ 저작물에 대한 접근을 제공하거나 사본을 배포하는 경우, "Plain Vanilla ASCII" 또는 Project Gutenberg™ 공식 웹사이트(www.gutenberg.org)에 게시된 공식 버전에서 사용된 다른 형식 이외의 형식으로 배포하는 경우, 사용자에게 추가 비용, 수수료 또는 경비를 부과하지 않고 원본 "Plain Vanilla ASCII" 또는 다른 형식의 저작물 사본, 사본을 내보내는 방법 또는 요청 시 사본을 얻을 수 있는 방법을 제공해야 합니다. 모든 대체 형식에는 1.E.1항에 명시된 Project Gutenberg™ 라이선스 전체가 포함되어야 합니다.

1.E.7. 1.E.8항 또는 1.E.9항을 준수하지 않는 한, Project Gutenberg™ 저작물에 대한 접근, 열람, 전시, 공연, 복사 또는 배포에 대해 요금을 청구하지 마십시오.

1.E.8. 귀하는 Project Gutenberg™ 전자 저작물의 복사, 접근 제공 또는 배포에 대해 합리적인 수수료를 청구할 수 있습니다. 단, 다음의 경우에 한합니다.

• 귀하는 Project Gutenberg™ 저작물 사용으로 발생하는 총수익의 20%를 로열티로 지불합니다. 로열티는 귀하가 이미 적용 가능한 세금 계산에 사용하는 방식을 사용하여 계산됩니다. 이 수수료는 Project Gutenberg™ 상표 소유자에게 지급되어야 하지만, 소유자는 본 조항에 따라 로열티를 Project Gutenberg Literary Archive Foundation에 기부하기로 동의했습니다. 로열티는 정기 세금 보고서를 작성하거나 법적으로 작성해야 하는 각 날짜로부터 60일 이내에 지불해야 합니다. 로열티는 해당 사실을 명확하게 표시하고 4항 "Project Gutenberg Literary Archive Foundation 기부 정보"에 명시된 주소로 Project Gutenberg Literary Archive Foundation으로 보내야 합니다.

• Project Gutenberg™ 작품의 무료 배포에 대한 본 계약의 모든 기타 조건을 준수해야 합니다.

1.E.9. 본 계약에 명시된 조건과 다른 조건으로 Project Gutenberg™ 전자 저작물 또는 저작물 집합을 유료로 배포하거나 배포하려는 경우, Project Gutenberg™ 상표 관리자인 Project Gutenberg Literary Archive Foundation의 서면 허가를 받아야 합니다. 아래 3항에 명시된 대로 재단에 문의하세요.

1.F.

1.F.1. Project Gutenberg 자원봉사자와 직원들은 Project Gutenberg™ 컬렉션을 제작하는 과정에서 미국 저작권법으로 보호되지 않는 저작물을 식별하고, 저작권 조사를 수행하고, 필사하고, 교정하는 데 상당한 노력을 기울이고 있습니다. 이러한 노력에도 불구하고, Project Gutenberg™ 전자 저작물과 그 저작물이 저장될 수 있는 매체에는 불완전하거나 부정확하거나 손상된 데이터, 필사 오류, 저작권 또는 기타 지적 재산권 침해, 결함이 있거나 손상된 디스크 또는 기타 매체, 컴퓨터 바이러스, 또는 사용자 장비에서 손상되거나 읽을 수 없는 컴퓨터 코드 등을 포함하되 이에 국한되지 않는 "결함"이 포함될 수 있습니다.

1.F.2. 제한적 보증, 손해 배상 책임의 부인 - 1.F.3항에 명시된 "교환 또는 환불 권리"를 제외하고, Project Gutenberg 문학 아카이브 재단, Project Gutenberg™ 상표권자 및 본 계약에 따라 Project Gutenberg™ 전자 저작물을 배포하는 기타 모든 당사자는 귀하에게 발생하는 모든 손해, 비용 및 경비(변호사 수임료 포함)에 대한 책임을 부인합니다. 귀하는 1.F.3항에 명시된 경우를 제외하고 과실, 무과실 책임, 보증 위반 또는 계약 위반에 대한 구제책이 없다는 데 동의합니다. 귀하는 본 계약에 따른 재단, 상표 소유자 및 모든 유통업체가 실제적, 직접적, 간접적, 결과적, 징벌적 또는 우발적 손해에 대해 귀하에게 책임을 지지 않는다는 데 동의하며, 귀하가 그러한 손해 가능성에 대한 통지를 제공한 경우에도 마찬가지입니다.
